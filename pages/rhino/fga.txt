-->When I installed Rhino, I checked the option to use the Zoo, but I don't use the Zoo#
On Windows Vista or Windows 7, delete all the files in:// C:\ProgramData\McNeel\Rhinoceros\5.0\License Manager\Licenses.//\\  
On Windows XP delete all the files in: //C:\Documents and Settings\All Users\Application Data\McNeel\Rhinoceros\5.0\License Manager\Licenses//
  - **Restart** Rhino.
  - **Enter your key** when asked.

>**Note**: On some systems, instead of the //"\All Users\"// folder, the path may be your //"\Domain\"// log in name.
