=====Download der Mac-Rhino-Tutorial-Dateien unter OS X 10.7 oder neuer =====

**Hinweis:** Die Tutorials wurden für die Windows-Version von Rhino geschrieben. Da die Mac-Version noch nicht vollständig ist, wurden die Tutorials auch noch nicht an die Besonderheiten des Mac angepasst.

**Download der Tutorial-Dateien**

  * Klicken Sie im Menü **Rhino** auf **Help > Learn Rhino > Open Tutorial Models...**. 

Wenn Sie noch keine Tutorial-Dateien auf Ihrem Computer installiert haben, werden Sie aufgefordert, welche herunterzuladen. Die Datei **rh40mac_wip_tutorials_20090724.dmg** wird in Ihren Download-Ordner kopiert.

Öffnen Sie die Datei **rh40mac_wip_tutorials_20090724.dmg**, um ein neues Finder-Fenster mit der Datei **RhinocerosTutorials.mpkg** anzuzeigen.

Doppelklicken Sie auf die Datei **RhinocerosTutorials.mpkg**, um die Tutorial-Dateien zu installieren.

OS X warnt Sie eventuell davor, dass die Datei von einem unbekannten Entwickler stammt und weigert sich, die Datei auszuführen. In diesem Fall **klicken Sie mit der rechten Maustaste** (oder **Ctrl+Klick**) auf **RhinocerosTutorials.mpkg** un wählen Sie im Kontextmenü **Öffnen**. Klicken Sie im Dialogfenster auf die Schaltfläche **Öffnen**. 

Der Tutorial-Installer wird nun gestartet.  Folgen Sie den Anweisungen des Installationsprogramms.

Klicken Sie nach Abschluss der Installation erneut auf **Help > Learn Rhino > Open Tutorial Model...**, um ein Tutorial-Modell zu öffnen.
