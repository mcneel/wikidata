** [[accurender:accurender|AccuRender]] 4 con AutoCAD 2007/2008 en Vista **

======AccuRender 4.0 y Windows Vista======

\\

Este parche de la Service Release 2 (SR2) para [[accurender:accurender|AccuRender]] 4.0 es necesario para usar el programa en Windows Vista. Los nuevos CD con el texto "SR2" impreso en el borde del CD ya son compatibles con Vista.

La SR2 está disponible desde http://www.es.[[accurender:accurender|AccuRender]].com/download.htm y la pueden descargar aquellos usuarios de [[accurender:accurender|AccuRender]] que ya tienen [[accurender:accurender|AccuRender]] 4.0 instalado.

Las pruebas preliminares muestran que [[accurender:accurender|AccuRender]] 4 funciona con 2008 instalado en Windows Vista. Se necesitará hacer más pruebas para determinar la estabilidad y esta página se actualizará para informarles tan pronto sepamos más.

=====CDs de la SR1 de AccuRender y Windows Vista=====

Debido a un error de Windows Vista, la instalación del CD de [[accurender:accurender|AccuRender]] 4 SR1 se queda bloqueada. Los síntomas de este problema son que la instalación continúa sólo hasta el diálogo de selección de idioma.

Mientras este error continúe en Windows Vista, hemos publicado [[http://wiki.mcneel.com/_media/legacy/en/ar4_CD_patch.zip|este pequeño parche]] que debe ejecutarse antes de insertar el CD de AR4 y de ejecutar la instalación.

[[http://wiki.mcneel.com/_media/legacy/en/ar4_CD_patch.zip|Parche de AccuRender 4 CD para Windows Vista.]]

Descomprima el archivo y coloque el instalador en el escritorio. Reinicie el equipo.  Cuando el equipo se haya reiniciado, ejecute el instalador. El parche le solicitará insertar el CD cuando haya terminado. Inserte el CD y ejecute el instalador inmediatamente cuando se haya completado la instalación.

Tenga en cuenta que este parche no es necesario si no se encuentra con este problema. Los CD de la SR0 y SR2 no tienen este problema.

=====CD de actualización de AccuRender y Windows Vista=====

Las nuevas funciones de seguridad de Windows Vista causan algunos errores cuando se ejecuta la instalación si se solicita insertar el CD de AR3 para validar la actualización.  El error aparece en un cuadro de diálogo con el título "[[accurender:accurender|AccuRender]] 4.0: Windows - Wrong Volume".

{{:legacy:en:VistaAR4InstallerUpgradeError.png}}

Si le aparece este mensaje de error, simplemente reemplace el CD de AR4 en la unidad de CD y pulse "Reintentar". De este modo la instalación continuará normalmente.

Actualmente estamos trabajando en solucionar este problema. Sin embargo, todos los CDs antiguos tendrán este con Windows Vista a menos que el comportamiento sea modificado por Microsoft en alguna futura Service Release.

Otra solución al problema es copiar toda la carpeta de "AccuRender" del CD en el disco duro e instalar el programa desde ahí.

=====AutoCAD 2008=====
AutoCAD 2008 y los productos basados en AutoCAD 2008 (como ADT 2008) son los únicos productos de Autodesk que actualmente están certificados para Vista.

[[accurender:accurender|AccuRender]] 4 funciona en AutoCAD 2008 si utiliza el mismo procedimiento requerido para instalarlo para 2008 en Windows XP. Requiere un parche de registro antes de instalar el CD del producto y luego un parche con la SR2 actual. Para obtener más información, haga clic [[http://es.wiki.mcneel.com/default.aspx/McNeel/AutoCAD2007.html|aquí]]

Tenemos recomendaciones adicionales que cambian la configuración del icono de AutoCAD 2008 en Windows.

1) Haga clic con el botón derecho en el icono de AutoCAD 2007 o 2008 del escritorio y seleccione "Propiedades"

2) Haga clic en la ficha "Compatibilidad".

3) Marque la opción "Ejecutar este programa en modo de compatibilidad para:" y seleccione "Windows XP (Service Pack 2)" de la lista.

4) Más abajo en el diálogo, marque las siguientes 2 opciones:

   [X] Desactivar temas visuales
   [X] Desactivar composición de escritorio

Nota 1: Desactivar los temas visuales ocultan el botón rojo con la X de las barras de herramientas flotantes. Esto también hará que todos los "efectos" de Vista Ultimate se desactiven temporalmente (p. ej. ver a través de las ventanas, etc.) mientras AutoCAD está abierto.

Nota 2: Se le solicitará cargar el modelo antes de Paseo y aparecerán las fichas de renderizado. Separe "Cargar el modelo" del menú desplegable de [[accurender:accurender|AccuRender]] o separe el botón "Cargar el modelo" del administrador de Paseo. Esto es un error.

Nota 3: Las ventanas de Paseo aparecerán flotantes fuera de la ficha de presentación en Vista. Puede renderizar en ella, pero se dividirá en la parte superior que se encuentra en las barras de herramientas de AutoCAD. Si lo encuentra molesto, puede renderizar en la vista de AutoCAD como opción alternativa hasta que se solucione el problema. Haga clic en el menú [[accurender:accurender|AccuRender]] > Tareas de renderizado > Configuración de renderizado. Para el destino, seleccione Renderizar vista de AutoCAD en la ficha Principal.

=====AutoCAD 2007=====
Autodesk ha lanzado una actualización de los productos de AutoCAD 2007 para que sean compatibles con Vista. Descárguela del sitio web de AutoCAD http://www.Autodesk.com o haga clic aquí http://usa.autodesk.com/adsk/servlet/ps/dl/item?siteID=123112&id=9444984&linkID=9240618. Después de instalar el parche para la 2007, utilice el procedimiento para instalar [[accurender:accurender|AccuRender]] 4.

=====AutoCAD 2006 y versiones anteriores=====
Puesto que [[accurender:accurender|AccuRender]] es un plug-in de AutoCAD, depende de la visualización de AutoCAD, su gestión de memoria, etc. Las pruebas para determinar si [[accurender:accurender|AccuRender]] funciona bien en Vista sólo se pueden hacer en un AutoCAD que funcione bien en Vista. Puesto que "AutoCAD no va a crear un parche de AutoCAD 2006 o versiones anteriores para Vista, no podremos dar soporte de [[accurender:accurender|AccuRender]] en esas combinaciones. Probaremos [[accurender:accurender|AccuRender]] en AutoCAD 2007 y Vista cuando haya un parche disponible.

Creemos que, por nuestra experiencia, AutoCAD ni tan sólo se instalará en Vista. Un cliente de [[accurender:accurender|AccuRender]] nos dijo que actualizó de XP a Vista una máquina que ya tenía AutoCAD 2006 instalado. Sin embargo, AutoCAD no funcionaba bien y las nuevas funciones de seguridad de Vista impedían el funcionamiento de las librerías de [[accurender:accurender|AccuRender]] 4.

Actualizaremos esta página a medida que la situación progrese.
