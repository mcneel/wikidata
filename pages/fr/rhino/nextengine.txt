====== NextEngine File Import ======

> **Sommaire:** //Plugin to import NextEngine SCN files//

=====Description=====

This plugin for Rhino4 will allow you to import NextEngine SCN files.

----

\\

\\

[[http://download.mcneel.com/download.asp?id=RhinoLabsNextEngine&language=|Click here to download plugin]]

=====Installation instructions (Only has to be done once)=====

  - Copy import_SCN.rhp and NEModel.dll to your "Rhinoceros 4.0\Plug-ins" directory.
  - Run "regsvr32 pathtoplugins\NEModel.dll". (pathtoplugins will look something like "C:\Program Files\Rhinoceros 4.0\Plug-ins")
  - Start Rhino.
  - Run LoadPlugin and press the install button. Browse to your "Rhinoceros 4.0\Plug-ins" directory and select import_SCN.rhp.

===== Use Instructions=====

  - Go to the File pulldown and use the Open or Import command
  - In the Files of Type dropdown, select NextEngine .SCN
  - Select your SCN file.
  - Once Imported, you may need to run Zoom Extents to see the model.
\\
