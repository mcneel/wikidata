====== Configuración del Zoo para todos los usuarios ======

> **Producto:** //[[zoo:home|Zoo]]//
> **Resumen:** //Instrucciones para configurar el Zoo para todos los usuarios de un equipo cliente con Rhino para Windows//

===== Problema =====
Tengo varios usuarios que usan un equipo cliente con Rhino para Windows. Tengo que configurar el nombre/IP del servidor del Zoo para cada usuario. ¿Puedo configurar el servidor del Zoo para todos los usuarios?

===== Causa =====
Cuando utiliza doble clic para ejecutar Rhino y configurar el servidor del Zoo, la información del servidor del Zoo se agrega al Registro de Windows en HKEY_CURRENT_USER. **Solo para el usuario actual.**

===== Solución =====
Haga clic con el botón derecho en el icono de acceso directo de Rhino, seleccione "Ejecutar como administrador" y configure el servidor del Zoo. La información del servidor del Zoo se agrega al Registro de Windows en HKEY_LOCAL_MACHINE. **Para todos los usuarios.**
