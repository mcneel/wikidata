====Changer une licence de Rhino (6 ou 7) pour utiliser le Cloud Zoo====

Pour transférer vers votre compte Rhino une licence installée de manière autonome sur votre ordinateur, suivez ces instructions :

**1. Restituez votre validation de licence autonome**
  - Lancez Rhino.
  - Dans Rhino pour Windows utilisez le menu déroulant Outils > Options > Licences, cliquez sur **Changer de licence**. Dans Rhino pour Mac utilisez le menu déroulant Rhinoceros > Préférences > Licences. 
  - Cliquez sur **Supprimer une licence** en bas de la boîte de dialogue.
    (Si votre licence est installée sur le Zoo, sélectionnez-la dans la console d'administration du Zoo et cliquez sur l'icône de suppression.)

**2. Ajoutez votre licence sur le Cloud Zoo**
  - Cliquez sur **Connexion en tant que [Votre nom]**.
  - Vous serez averti que vous ne possédez pas de licence personnelle.
  - Cliquez sur **Ajouter une licence**.
  - Votre navigateur Internet s'ouvrira et une page Web vous indiquera les instructions.
  - Ouvrez la page **Ajouter une licence**.
  - Entrez votre clé de licence de Rhino 7 et cliquez sur **Ajouter une licence**.

**3. Lancez Rhino en utilisant le Cloud Zoo**
  - Dans la boîte de dialogue de Licence, cliquez sur **Réessayer**.
  - Rhino démarrera en utilisant votre licence issue du Cloud Zoo.

**Utiliser Rhino depuis un autre ordinateur**
  - Lancez Rhino sur un autre ordinateur.
  - Connectez-vous à votre compte Rhino.
