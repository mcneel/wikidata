====Rhino OS X está en desarrollo:====

  * **Todavía faltan** algunas funciones principales, como **Grasshopper**, plug-ins, impresión y documentación.
  * **Cada semana o dos** se publica una nueva WIP con las nuevas funciones que se han añadido.

-->Más información#

Rhino OS X todavía se encuentra en la fase de desarrollo 'Wenatchee'. La versión no está **lista para producción**. 

**Falta** terminar la interfaz de usuario, los recursos de la ayuda, los tutoriales y las herramientas de impresión/diseño de página necesiarias para las necesidades básicas de dibujo. 

Puesto que todavía no hay un kit de desarrollo de software (SDK) disponible, tampoco ha empezado el desarrollo de plug-ins externos. Como, por ejemplo, VRay. 

Dicho esto, las herramientas de creación de superficies están completas y son muy útiles. Si solo usa Rhino para el modelado de superficies, entonces le irá bien. Sin embargo, para completar cualquier tipo de trabajo real, **necesitará Rhino 5 para Windows**.

El soporte también es un algo diferente. Rhino 5 para Windows es una herramienta completa, publicada y lista para producción, de modo que nuestros recursos de soporte técnico están preparados para ayudarle.

Le invitamos a **participar en el desarrollo**, pero puesto que Rhino para Mac está en desarrollo, nos interesan principalmente los comentarios destinados a terminar el proyecto y poder lanzarlo.

Durante el desarrollo se publican versiones varias veces al mes. Cada nueva versión funcionará durante unos **60 días** y será reemplazada antes de que caduque.

====Requisitos del sistema====
  * procesadores Intel Core Duo 2 de 64 bits
  * OS X 10.7 o posterior.
  * 4 GB de RAM (se recomienda más)
  * Un ratón de varios botones (se recomiendan una rueda de desplazamiento)
  * El [[http://www.3dconnexion.com/products/spacenavigator.html|SpaceNavigator]] es compatible.

--> Hardware no compatible#
    * OS X 10.6 o versiones anteriores
    * **Spaceball** y **SpaceExplorer**
