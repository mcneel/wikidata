====== PanelingToolsは使用できますか? (Rhino 5 for Mac) ======
> **概要:** //プラグインのPanelingToolsはRhino 5 for Macで使えるようになっていますか?//

使用できます。PanelingToolsは製品としてリリースされたRhino 5 for Macには搭載されて**いません**が、[[http://www.rhino3d.com/go/download/rhino-for-mac/wip/latest|Rhino for Mac WIP]]に搭載されています。 If you own a Rhino for Mac license, then you need to download and install the Rhino for Mac WIP to access PanelingTools. Please note that the Rhino for Mac WIP is a Work-In-Progress, which means it includes the latest features and development, but it also means that there may be instability and bugs. New WIPs are released frequently as we add features and fix bugs. 

Note: In the WIP, there is no PanelingTools menu and all of the commands are available through the command line only. Type “pt” to see all the PanelingTools commands in the AutoComplete list.
