====== Zoo demande Microsoft .NET Framework 4.5 ======
> **Produit :** //[[fr:zoo:home|Zoo]]//
> **Résumé :** //Zoo et Microsoft .NET Framework 4.5//


Le Zoo a besoin de **[[https://www.microsoft.com/fr-fr/download/details.aspx?id=30653|Microsoft .NET Framework 4.5]]**. 

Vous pouvez vérifier que Microsoft .NET Framework 4.5 est installé sur votre système en utilisant **[[http://windows.microsoft.com/fr-fr/windows/windows-update|Windows Update]]**.

Pour détecter les version de Microsoft .NET Framework installée sur votre système, utilisez :

**[[http://mcneel.s3.amazonaws.com/zoo/Utils/DetectFX.exe|L'utilitaire de détection de .NET Framework (Win32)]]**

**[[http://mcneel.s3.amazonaws.com/zoo/Utils/DetectFXx64.exe|L'utilitaire de détection de .NET Framework (Win64)]]**
