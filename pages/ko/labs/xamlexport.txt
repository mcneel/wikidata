====== XAML File Export ======
> **Summary:** //XAML 파일을 내보내는 (export) 플러그인//

=====설명=====

이 Rhino 4.0 용 플러그인을 사용하여 XAML 파일을 내보낼 수 있습니다.

----

\\

\\

[[http://download.rhino3d.com/download.asp?id=XAML_EXPORT&language=|플러그인 다운로드 클릭]]

=====설치 안내=====

  - "Rhinoceros 4.0\Plug-ins" 디렉터리로 export_xaml.rhp 파일을 복사합니다.
  - Rhino 를 시작합니다.
  - 도구 - 옵션 - 플러그인 (Tools - Options - Plug-ins) 페이지로 갑니다.
  - 설치 단추를 클릭합니다.
  - 첫 번째 단계에서 복사해둔 파일이 있는 폴더를 찾습니다.
  - export_xaml.rhp 파일을 선택하고 열기 단추를 클릭합니다.
  - 확인 단추를 클릭하여 Rhino 옵션 페이지를 닫습니다.

===== 사용 안내=====

  - 파일 메뉴에서 다른 이름으로 저장 (SaveAs) 또는 내보내기 (Export) 메뉴를 사용합니다.
  - 파일 형식 (Files of Type) 에서 .xaml 를 선택합니다.

XAML 파일에 대해 잘 모르시나요? 이 프로젝트를 시작하기 전까지 저도 마찬가지였습니다. 제 생각엔, 새로운 어플리케이션의 UI 를 개발하는 데 Rhino 모델을 사용하는 게 아니라면 대부분의 Rhino 사용자들에게는 그다지 큰 가치는 없다고 봅니다. http://msdn.microsoft.com/en-us/library/cc295302.aspx 페이지에 XAML 에 대한 개요가 잘 설명되어 있습니다. Vista 를 사용하신다면 XAML 파일들을 Internet Explorer 에서 그래픽으로 보실 수 있습니다. (이 파일은 ascii 라서 메모장과 같은 텍스트 편집기에서 구조를 보거나 수정할 수 있습니다).  XP 사용자들의 경우, IE 에서 보려면 MS 의 SilverLight 2 가 필요합니다. 다음 링크를 참조하세요. http://www.microsoft.com/SILVERLIGHT/resources/install.aspx?v=2.0
