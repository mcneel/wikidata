=====Downloads=====

**SectionTools Plugin for Rhino 7 for Windows:** Last updated April 30, 2021
  * Use the new PackageManager command in Rhino 7 to search for and install SectionsTools.
  * Toolbars should be accessible  through the Rhino Toolbar command.
  * [[http://www.food4rhino.com/app/sectiontools-rhino|Documentation:]] Download the Reference Manual(pdf) and the hut example model (.3dm) from Food4Rhino:

**Need older versions of SectionTools** for Rhino 6 5?
  * Make sure you have the latest service release of Rhino 6 or 5. 
  * For Rhino 6, Download from[[https://www.food4rhino.com/app/sectiontools-rhino| SectionTools at Food4Rhino site...]]
  * For Rhino 5, download and install SectionTools installer from [[https://files.mcneel.com/rhino/5.0/labs/SectionToolsInstaller_April2_2012_Win64.zip|64-bit here...]] and [[https://files.mcneel.com/rhino/5.0/labs/SectionToolsInstaller_April2_2012_Win32.zip|32-bit here...]]
