====== Installation eines Rhino-Upgrades auf einem neuen Computer ======
> **Zusammenfassung:** //Muss ich eine Vorgängerversion von Rhino installieren, um das Upgrade auf Rhino 5 durchführen zu können?//
> ** Weitere Infos zu Installationsproblemen:** //[[installproblems|Problembehandlung bei der Installation von Rhino 5]]//

=====Frage=====
Ich habe einen neuen Computer und möchte Rhino 5 installieren.  Ich habe immer Aktualisierungen von Rhino gekauft.  Muss ich ALLE vorherigen Versionen und folgende Service Packs installieren??

=====Antwort=====
**Nein!** Sie benötigen jedoch sowohl den **Lizenzschlüssel von Rhino 5**  als auch einen **Lizenzschlüssel einer Vorgängerversion** von Rhino.\\
  - Laden Sie das neueste Installationsprogramm von Rhino 5 über [[http://www.rhino3d.com/download/rhino/5/latest|unsere Website]] herunter.
  - Führen Sie das Installationsprogramm aus und geben Sie Ihren Rhino-5-Lizenzschlüssel
  - Während der Installation werden Sie aufgefordert, den Lizenzschlüssel einer Vorgängerversion von Rhino einzugeben
  - Schließen Sie die Bestätigung Ihrer neuen Lizenz ab

Das neueste Service Release finden Sie [[http://www.rhino3d.com/download/rhino/5.0/sr|hier]]
