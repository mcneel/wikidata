====== Lo scripting con Python in Rhino per Mac ======
Rhino per Mac supporta un plug-in di prova che aggiunge lo scripting con Python a Rhino. Per aggiungere questa funzionalità:

  - [[http://www.rhino3d.com/download/wenatchee/5.0/wip|Installare l'ultima versione di Rhino OSX]]
  - Eseguire questa nuova versione di Rhino. \\ In questo modo, l'estensione .macrhi viene associata a Rhino. Basta farlo una volta sola. Ora, facendo doppio clic su qualsiasi file .macrhi, si avvierà Rhino.
  - [[http://download.rhino3d.com/IronPython/5.0/wip/download/|Scaricare il programma di installazione del plug-in IronPython .macrhi]]. \\ Si tratta del programma di installazione del plug-in per Python per Rhino per Mac. 
  - Fare doppio clic sul file per installare il plug-in in Rhino. \\ Rhino per Mac si avvierà ed installerà il plug-in.
  - Chiudere e riavviare Rhino. \\ Ora dovrebbe apparire un nuovo comando: **RunPythonScript**.

====Comando RunPythonScript====
Esistono due versioni di questo comando:

**RunPythonScript** - Mostra una finestra di dialogo di apertura file che consente di selezionare uno script di Python (file .py.) Il plug-in include un gruppo di script di esempio eseguibili. La prima volta che si esegue il comando, la finestra di dialogo dovrebbe puntare verso questa cartella.

**-RunPythonScript** - La versione del comando "preceduta da trattino" consente di digitare il nome file di uno script. Se lo script esiste nella directory Scripts del computer, esso verrà eseguito. Per eseguire uno script contenuto in una sottodirectory della directory Scripts, si può usare il punto. Per esempio, se si vuole eseguire il file CurveLength.py contenuto nella sottodirectory degli esempi (samples), si può digitare samples.curvelength al prompt.

**Nota**: Tutti gli esempi inclusi in questo plug-in dovrebbero funzionare sia sulla [[http://mac.rhino3d.com|WIP di Rhino per Mac]] che su [[http://www.rhino3d.com/|Rhino per Windows]].
