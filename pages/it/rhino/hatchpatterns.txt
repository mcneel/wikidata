====== Modelli di riempimento personalizzati per Rhino =====

Esistono diversi modi di creare un modello di riempimento in Rhino.

=====Opzione 1=====
Scaricare e utilizzare i modelli di riempimento forniti in Rhino [[https://wiki.mcneel.com/it/rhino/hatch|qui]].

=====Opzione 2=====

La prima alternativa più semplice, se si possiede AutoCAD, è importare ACAD.PAT in Rhino. In secondo luogo, è possibile scaricare i riempimenti da siti come [[http://www.dotsoft.com/acadhatch.htm|DotSoft.]]

È anche possibile creare un file *.pat in Notepad, utilizzando una delle esercitazioni riportate di seguito.
Poiché Rhino utilizza lo stesso formato di AutoCAD, qualsiasi esercitazione sulla creazione di un modello di AutoCAD funzionerà.

Consultare:
  * [[http://www.dotsoft.com/acadhatch.htm|DotSoft]]
  * [[http://www.afralisp.net/autocad/tutorials/creating-hatch-patterns.php|AfraLISP]]
  * [[http://www.cadhatch.com/free-downloads/4588167679|CADhatch free downloads]]
  * [[http://www.ellenfinkelstein.com/acadblog/creating-custom-hatch-patterns/|AutoCAD Bible, Ellen Finkelstein]]

===== Opzione 3: Modelli di esempio di DotSoft =====
[[http://www.dotsoft.com/acadhatch.htm|Sito web di DotSoft]]
Scaricare gratuitamente i [[https://s3.amazonaws.com/mcneel/rhino/4.0/misc/hatch patterns for rhino.pat|file di riempimento di Rhino]].
Importare questo file in Rhino da:
**Opzioni-> Proprietà del documento -> Riempimento -> Importa**.
