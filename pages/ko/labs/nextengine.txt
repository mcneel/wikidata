====== NextEngine File Import ======

> **Summary:** //NextEngine SCN 파일을 가져오는 (import) 플러그인//

=====설명=====

이 Rhino 4.0 용 플러그인을 사용하여 NextEngine SCN 파일을 가져올 수 있습니다.

----

\\

\\

[[http://download.mcneel.com/download.asp?id=RhinoLabsNextEngine&language=|플러그인 다운로드 클릭]]

=====설치 안내 (한 번만 실행하시면 됩니다)=====

  - import_SCN.rhp 파일과 NEModel.dll 파일을 "Rhinoceros 4.0\Plug-ins" 디렉터리에 복사합니다.
  - "regsvr32 pathtoplugins\NEModel.dll". (pathtoplugins 은 "C:\Program Files\Rhinoceros 4.0\Plug-ins" 을 말합니다) 파일을 실행합니다.
  - Rhino 를 시작합니다.
  - 도구 - 옵션 - 플러그인 (Tools - Options - Plug-ins) 페이지로 갑니다.
  - 설치 단추를 클릭합니다.
  - 두 번째 단계에서 복사해둔 세 개의 파일이 있는 폴더를 찾습니다.
  - import_SCN.rhp 파일을 선택하고 열기 단추를 클릭합니다.
  - 확인 단추를 클릭하여 Rhino 옵션 페이지를 닫습니다.

===== 사용 안내=====

  - 파일 메뉴에서 열기 (Open) 또는 가져오기 (Import) 메뉴를 사용합니다.
  - 파일 형식 (Files of Type) 에서 NextEngine .SCN 을 선택합니다.
  - 사용자의 SCN 파일을 선택합니다.
  - 일단 파일을 가져오면, 모델을 보기 위해 Zoom Extents 명령을 실행해야 할 수도 있습니다.
