======여러 IP 주소======
> **제품:** //[[ko:zoo:home|Zoo]]//
> **요약:** //여러 IP 주소인 시스템에서 LAN Zoo를 설정하는 방법을 안내합니다.//

=====질문=====
Zoo에서 TCP 80 포트가 아닌 포트를 사용할 수 있습니까?

=====답변=====
아니요. LAN Zoo는 다른 TCP 또는 UDP 포트를 통해 연결을 수신할 수 없습니다.

그러나, LAN Zoo는 Zoo 서비스를 특정한 IP 주소로 바인딩할 수 있습니다.  

{{:zoo:zoo_ip.png|Zoo 옵션}}

**Bind Server to IP Address** 필드에서 특정한 네트워크 인터페이스에서 연결을 수신하도록 Zoo 서버를 설정할 수 있습니다.    

예를 들어, 컴퓨터에 두 개의 네트워크 카드가 있다면 두 인터페이스에서 서버에 액세스할 수 있습니다. 이 옵션이라면 드롭다운 메뉴에서 해당 네트워크 카드의 IP 주소를 선택하여 Zoo에서 해당 주소를 통해서만 액세스할 수 있습니다. 

따라서, 사용하는 서버가 여러 IP 주소로 구성된 서버라면 Zoo 서비스를 하나의 IP 주소에서 실행하고, 다른 서비스(예: IIS)는 다른 IP 주소에서 실행할 수 있습니다.
