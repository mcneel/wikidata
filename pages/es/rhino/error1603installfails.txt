====== Error de instalación 1612: No se puede eliminar la versión anterior del paquete de idioma de Rhinoceros 5 (en-US)." ======
O bien se ha producido un error de instalación; código de salida 1603

> **Resumen:** //Cómo resolver el error del sistema 1612 o 1603.//
> **Ayuda con la instalación:** //[[installproblems|Solucionar problemas con la instalación de Rhino 5]]//

=====Problema=====
Rhino, el paquete de idioma o los archivos de Ayuda no se han podido instalar. El paquete del instalador puede estar dañado o incompleto debido a:
  * un detector de virus en ejecución
  * un bloqueo del equipo
  * algun programa malicioso que esté impidiendo la instalación
Reinstalar o desinstalar desde **Programas y funciones** de Windows tampoco ayuda.

=====Solución posible=====
  - Ejecute esta Utilidad de Microsoft: //[[https://support.microsoft.com/es-es/mats/program_install_and_uninstall|Solucionar problemas de programas, no se puede instalar o desinstalar]]//
  - Utilice esta utilidad para resolver el problema con el instalador de Rhino. Por ejemplo con el //paquete de idioma de Rhinoceros 5 (en-US)//. Puede ejecutar la utilidad manualmente o automáticamente.
  - Cuando haya resuelto el problema del instalador con la utilidad de MS, continúe con la instalación de Rhino, de la Ayuda o del paquete de idioma. 
  - Abra Rhino y ejecute **RestablecerBarrasDeHerramientas**. Cierre Rhino.
  - Vuelva a abrir Rhino y pruébelo.
