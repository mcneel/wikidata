====== Zoo et Microsoft .NET Framework 4 ======
> **Produit:** //[[fr:zoo:zoo5|Zoo 5]]//
> **Résumé:** //Utilisation de Zoo 5 sur Windows XP.//

===== Problème =====
J'essaie d'utiliser le Zoo sur Windows XP. Dès que je lance le Zoo sur cette machine, il se bloque et rien ne se passe. Zoo5 est-il compatible avec Windows XP ?

===== Résolution=====
Zoo 5 est pris en charge par **Microsoft Windows XP Service Pack 3**,  

mais il a besoin de **Microsoft .NET Framework 4** pour fonctionner. 

L'installation de Zoo ne contient pas cette application qui ajouterait 50 Mo au téléchargement. 

**Téléchargements**
  * [[http://support.microsoft.com/kb/322389|Microsoft Windows XP Service Pack 3]]
  * [[http://www.microsoft.com/fr-fr/download/details.aspx?id=17851|Microsoft .NET Framework 4 (Installation depuis Internet)]]
  * [[http://www.microsoft.com/fr-fr/download/details.aspx?id=17718|Microsoft .NET Framework 4 (Installation autonome)]]
