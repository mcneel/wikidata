====== Rhino en Mac ======

Oficialmente Rhino no es compatible con Mac OSX, pero aquí hay información de otros usuarios que han experimentado con Intel Macs de Apple.

//David Twelves escribió://

Hola, quiero informaros que Rhino3D funciona en OS X con **Parallels Desktop for Mac**.  Enlace al programa: http://www.parallels.com/en/products/desktop

Mi máquina es un Mac Mini con un procesador 2GHz **Intel Core Duo**, 2 GB de RAM y 100 GB de disco duro a 7200rpm.  Configuré mi sistema de Windows virtual con un procesador de un núcleo de 2GHz, 1 GB de RAM y 40 GH de disco duro.  Rhino funciona muy bien.  Hay un poco de retraso en la interfaz porque la tarjeta de vídeo es emulada, pero en general funciona bien.

"El extracto siguiente no considera Rhino, pero tiene observaciones interesantes sobre el rendimiento.  Es del artículo //"Weighing a Switch to a Mac"// de Thomas J. Fitzgerald en el //New York Times// del 10 de agosto de 2006://

"Hay dos métodos para que Windows funcione en los nuevos Macs y ambos funcionan mucho más rápido que Virtual PC, la opción principal con la antigua arquitectura de Mac.

En primer lugar, un nuevo programa llamado **Parallels Desktop** para Mac ($80; www.parallels.com) permite ejecutar Windows y Mac OS X Tiger simultáneamente.

Por ejemplo, puede ejecutar software para Windows como Internet Explorer y Microsoft en una ventana que se puede minimizar al igual que con otros programas de Mac. Se pueden copiar datos entre las plataformas, se pueden compartir archivos y carpetas y se puede escoger ejecutar Windows en modo de pantalla completa.

Parallels puede ejecutar versiones de Windows desde la versión 3.1 hasta las ediciones más actuales.  Un inconveniente de Parallels es que no es compatible con tarjetas gráficas 3D aceleradas, por lo que algunos juegos 3D y otros programas funcionarán lentos o no muy bien. Otros fatores a tener en cuenta son la reducción de la velocidad del 5 al 15 por ciento en comparación a ejecutar Windows nativamente en ordenadores con Intel y el hecho de que no todos los dispositivos periféricos son compatibles.

La otra opción para ejecutar Windows en los nuevos Macs es posible gracias a **Boot Camp** (www.apple.com/bootcamp), una utilidad gratuita de Apple.  A diferencia de Parallels, que ejecuta Windows en Mac OS X, Boot Camp crea una partición en el disco duro del ordenador e instala Windows ahí. Cuando el ordenador se inicia, se puede escoger entre ejecutar Windows o Mac OS X.

Las ventajas de Boot Camp son poder ejecutar Windows a máxima velocidad; funciona nativamente en Mac igual que en cualquier PC convencional con Windows, utilizando toda la capacidad del procesador y la tarjeta gráfica y ofreciendo compatibilidad con periféricos de hardware y dispositivos diseñados para PC.

Un inconveniente de Boot Camp es que es necesario apagar un sistema operativo antes de usar el otro. Esto significa que no se pueden ejectutar aplicaciones de Windows y de Mac simultáneamente. Otro inconveniente es que sólo funciona en dos versiones de Windows: Windows XP Home Edition con el Service Pack 2, que cuesta $200 o Windows XP Professional con el Service Pack 2, que cuesta $300.

La **seguridad** es otro aspecto de los Mac que interesa a los usuarios de Windows. En Windows, los programas antivirus y antispyware son esenciales para defenderse de un montón de amenazas. Por ahora, el sistema operativo Mac OS X no ha filtrado virus y el spyware no es una amenaza para este sistema.

Pero cuando Windows se ejecuta en Macs con Intel, por ejemplo a través de Boot Camp o Parallels, es vulnerable a las mismas amenazas de virus y spyware que pueden afectar a los PC convencionales basados en Windows.
