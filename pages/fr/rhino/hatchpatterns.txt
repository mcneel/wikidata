====== Motifs de hachures personnalisés pour Rhino= =====

Il existe plusieurs méthodes pour créer un motif de hachures dans Rhino. La première et la plus simple, si vous possédez AutoCAD, consiste à importer ACAD.PAT dans Rhino. Ensuite, vous pouvez en télécharger sur Internet à partir de sites comme [[http://www.dotsoft.com/acadhatch.htm|DotSoft.]]

Vous pouvez aussi créer un fichier PAT dans le Bloc-notes en utilisant l’un des tutoriels suivants.
Comme Rhino utilise le même format qu’AutoCAD, tous les tutoriels expliquant comment créer un motif AutoCAD vous serviront.

Consultez :
  * [[http://www.dotsoft.com/acadhatch.htm|DotSoft]]
  * [[http://www.afralisp.net/autocad/tutorials/creating-hatch-patterns.php|AfraLISP]]
  * [[http://www.cadhatch.com/free-downloads/4588167679|Téléchargements gratuits CADhatch]]
  * [[http://www.ellenfinkelstein.com/acadblog/creating-custom-hatch-patterns/|Bible d’AutoCAD, Ellen Finkelstein]]

Téléchargez un exemple de [[http://s3.amazonaws.com/mcneel/rhino/4.0/misc/hatch patterns for rhino.pat|fichier de motif de hachures de Rhino]] à utiliser.
Dans Rhino, pour l’importer dans votre fichier, allez dans :
**Options-> Propriétés du document -> hachures-> Importer.**
