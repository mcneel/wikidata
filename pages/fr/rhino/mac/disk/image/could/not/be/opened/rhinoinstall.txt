====== macOS indique "Impossible d’ouvrir les images disques suivantes." Que dois-je faire? ======
> ** Résumé:** //Après avoir téléchargé Rhino pour Mac, un message indique que l'image disque n'a pas pu être ouverte.//

Le téléchargement de Rhino pour Mac ne s'est pas déroulé correctement. 

  - Ouvrez la page de [[http://www.rhino3d.com/download/rhino-for-mac/5/evaluation|téléchargement]] et demandez un autre lien de téléchargement.

Si le deuxième fichier .dmg ne fonctionne toujours pas, suivez ce processus de validation :

  - Lancez l'utilitaire de disque **/Applications/Utilitaires/Utilitaire de disque**
  - Dans le menu, sélectionnez **Vérifier...** Une boîte de dialogue apparaît.
  - Sélectionnez le fichier .dmg que vous avez téléchargé. Il se trouve normalement dans le dossier de téléchargement et porte un nom comme : rhino_6.20.19317.02202.dmg
  - Si le fichier que vous avez téléchargé n'est pas valide, retournez sur la page de [[http://www.rhino3d.com/download/rhino-for-mac/5/evaluation|téléchargement]] et demandez un autre lien de téléchargement.
  Si ce problème se reproduit, il se peut que votre connexion Internet soit instable, essayez de télécharger le fichier depuis une autre connexion Internet.  Le fichier de téléchargement fait plus de 350 Mo.
