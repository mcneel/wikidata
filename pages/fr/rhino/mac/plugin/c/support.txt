====== La version de Rhino pour Mac sera-t-elle compatible avec mon module C++ ? ======
> **Résumé:** //La version de Rhino pour Mac sera-t-elle compatible avec mon module C/C++ ?//


Probablement dans quelques temps. Nous ne sommes pas encore prêt pour fournir une assistance sur les modules C++ mais nous espérons le faire dans le futur. Vous devrez recompiler votre module en utilisant le système de développement Xcode. Suivant la complexité de votre module, cela peut être assez facile de créer un module qui se compile et fonctionne aussi bien sur Windows que Mac.
