====== Recomendaciones de Sistema para los Exámenes de Rhino - Región de América Latina======
El sistema de exámenes es compatible con la mayoría de los navegadores de Internet estándares creados desde 1990. Por lo tanto, esta aplicación se puede utilizar en la mayoría de las computadoras de escritorio, computadoras portátiles, tabletas y teléfonos inteligentes. A continuación se detallan nuestras recomendaciones.
 
===== Recomendaciones del Navegador =====
 
Los navegadores antiguos y nuevos pueden funcionar bien, los siguientes navegadores que conocemos funcionan mejor:
  * Internet Explorer 6+ (las versiones 9+ mostrarán una interfaz de usuario mejorada)
  * Firefox 3.6+ (las versiones 12+ mostrarán una interfaz de usuario mejorada)
  * Chrome 5+ (las versiones 30+ mostrarán una interfaz de usuario mejorada)
  * Safari 4+ (las versiones 5 y superiores mostrarán una interfaz de usuario mejorada)
  * Opera 11+ (todos mostrarán una interfaz de usuario mejorada)

===== Recomendaciones de Configuración Básica =====
Los siguientes son los ajustes básicos que recomendamos. El 99,99% de las veces la configuración predeterminada de su navegador funcionará correctamente. Cada red, dispositivo y navegador es diferente, la mayoría de estos ajustes pueden modificarse en el menú Herramientas de su navegador y luego elegir Opciones de Internet.
 
  * Deshabilitar todos los bloqueadores de ventanas emergentes
  * Permitir cookies
  * Habilitar Javascript
  * Instale la versión más reciente de Adobe PDF Acrobat Reader GRATUITO
  * Agregue como SITIOS DE CONFIANZA en su navegador y dispositivos de red los sitios: test.com, testcentral.com, mytestcom.com, mytestcom.net, amazonaws.com, and adobe.com
  * Para redes corporativas con software / dispositivos de filtrado / seguridad web o cortafuegos (que utilizan productos como Websence, iPrism ..) deshabilitar, eludir o BLANQUEAR LA LISTA de los SITIOS DE CONFIANZA enumerados anteriormente
  * Para redes corporativas que requieren rangos de IP específicos para uso en listas blancas: 66.241.194.*; 76.12.115.*; 76.12.20.*; 173.239.96.*; and 204.12.0.*

===== Información para Solucionar Problemas =====
La siguiente información se proporciona para ayudar a solucionar problemas si tiene dificultades para usar esta aplicación.
 
==== ¿Han sido 'Kicked Out' o 'Logged Out' mientras toman un examen (contenido) o ANTES de que comience? ====
 Esta aplicación cierra automáticamente la sesión de los usuarios si no ha habido actividad dentro de un período de tiempo razonable.
  *  Asegúrese de seguir las configuraciones y recomendaciones básicas que se detallan arriba.
  *  Deshabilite todas las barras de herramientas del navegador y complementos mientras esté conectado a este sitio.
  *  Analice su computadora en busca de virus, malware y otras amenazas de Internet.
 
==== ¿Problemas para contestar la misma pregunta varias veces, errores de navegación o ser "expulsado" mientras toma una prueba (contenido) DESPUÉS de que comience? ====
  * Primero, vea si puede reiniciar / reanudar la prueba y vuelva a intentarlo. Si el problema persiste, tenga en cuenta lo siguiente.
  * Asegúrese de seguir las configuraciones y recomendaciones básicas que se detallan arriba.
  * Si está en una red corporativa o en una red en la que necesita iniciar sesión antes de acceder a Internet, vuelva a verificar la recomendación de LISTA BLANCA que figura más arriba en la configuración básica y el título de recomendaciones.
  * Deshabilite todas las barras de herramientas del navegador y complementos mientras esté conectado a este sitio.
  * Analice su computadora en busca de virus, malware y otras amenazas de Internet.

==== Problemas con los Bloqueadores de Ventanas Emergentes ====
Esta aplicación se puede configurar para mostrar contenido y otra información en ventanas adicionales del navegador.
  * Asegúrese de que la ventana que está buscando no esté detrás de otra ventana abierta en su escritorio.
  * La mayoría de los navegadores tienen configuraciones para evitar publicidades emergentes y pueden confundir incorrectamente nuestras ventanas abiertas como anuncios publicitarios. Deshabilite todos los bloqueadores de ventanas emergentes.
  * Muchas barras de herramientas de los motores de búsqueda también tienen características para evitar publicidades emergentes, asegúrese de que estén apagadas. Si tiene instalados varios programas en la barra de herramientas del navegador con esta función, asegúrese de desactivar la función de bloqueo de ventanas emergentes para CADA barra de herramientas instalada.
 
==== ¿Problemas para abrir un documento PDF? ====
Esta aplicación genera numerosos documentos (como certificados e informes) utilizando el formato Adobe Acrobat PDF.
  * Asegúrese de que tiene el software más reciente de lector de PDF cargado en su computadora, de manera predeterminada el software PDF se carga en la mayoría de las computadoras. Visite el sitio web de adobe para actualizar.
  * Si tiene varios navegadores instalados en su computadora, o si carga un nuevo navegador encima de un navegador anterior, es posible que uno o más de estos navegadores necesiten un software de lector de PDF.
 
==== ¿Problemas para reproducir videos? ====
Esta aplicación utiliza un reproductor de video basado en HTML5 o Adobe Flash para reproducir videos. Esto es típico para aplicaciones como esta. El jugador elegirá entre HTML5 o Adobe Flash en función de su navegador. Los navegadores antiguos que no son compatibles con HTML5 necesitarán Adobe Flash. En este caso, el reproductor de video se asegurará automáticamente de que esté disponible la versión correcta de Adobe Flash. Si la aplicación no puede acceder automáticamente a Flash debido a la configuración de seguridad, es posible que deba visitar manualmente el sitio web de Adobe para actualizar.
 
==== Using a Thin Client, Dumb Terminal or Terminal Emulator? ====
This application is compatible with most Thin Clients, if the browsers on those devices are truly compatible with those listed as recommended above. Make sure when configuring Terminal Server software all such settings mimic the above listed recommendations. Make sure ample memory is allocated to each user and user session management capabilities are enabled.
 
==== ¿Cómo usar un Servidor Proxy? ====
Si utiliza un servidor proxy, asegúrese de que los datos de la sesión del navegador no estén en caché, que es una configuración estándar. Esto permitirá que múltiples usuarios en múltiples dispositivos accedan a esta aplicación simultáneamente sin que su equipo de red mezcle a los usuarios.
 
==== Bloqueo de Inicio de Sesión ====
Al iniciar sesión, si la información de acceso incorrecta se ha ingresado repetidamente demasiadas veces, esta aplicación evitará más intentos de inicio de sesión dentro de un marco de tiempo limitado. Si el período de tiempo expiró y todavía no puede iniciar sesión, existe una configuración del navegador o un complemento del navegador que impide que la aplicación test.com elimine la configuración en su computadora. Deberá borrar su historial en caché de Internet. Para la mayoría de los navegadores, abra el menú Herramientas de su navegador, luego elija Opciones de Internet, borre el historial y las cookies.
==== ¿Problemas al usar un Nuevo Navegador? ====
A menudo, cuando las compañías de software lanzan versiones más nuevas de sus navegadores, estos navegadores tienen problemas menores de compatibilidad. La solución es que la mayoría de estos navegadores ofrecen una función de 'Vista de compatibilidad' o 'Vista de compatibilidad'. Enciéndalo para ver si eso resuelve el problema.
 
==== ¿Aún tienes problemas? ====
Nos disculpamos por cualquier inconveniente que pueda haber experimentado. Pruebe lo siguiente si todavía tiene dificultades para usar esta aplicación.
  * Cerrar sesión (y si está disponible: presione el enlace "¿Olvidó su contraseña?" En la página de inicio de sesión, luego presione el enlace "Restablecer cuenta ingrese"). Intenta acceder a este sitio nuevamente.
  * Cierre sesión e intente acceder de nuevo a este sitio usando una computadora diferente o use una computadora diferente en una red diferente.
  * Póngase en contacto con el fabricante de su computadora / dispositivo o el administrador de red de su organización para obtener asistencia adicional.
