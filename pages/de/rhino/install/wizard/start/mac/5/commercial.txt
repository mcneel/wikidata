====== Rhino 5 Kommerzielle Einzelnutzer-Lizenz ======


===Bitte auswählen:===
    * **[[https://wiki.mcneel.com/de/rhino/5/macinstall|Ein oder mehrere Computer, die einem einzigen Nutzer gehören]]**
      *Wählen Sie diese Option, wenn Sie Ihre Lizenz auf Ihren eigenen Computern teilen wollen
        * Kann nur auf jeweils einem Computer ausgeführt werden
      * Benötigt keinen Internetzugang
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Endbenutzer-Lizenzvertrag]]

    * **[[https://wiki.mcneel.com/de/zoo/rhinomacdeployment|Mehrere Nutzer über eine Büro-LAN]]**
      * Wählen Sie diese Option wenn Sie Ihre Lizenz(en) mit mehreren Nutzern über eine LAN teilen
      * Der Windows-**[[https://wiki.mcneel.com/de/zoo/home|Zoo]]**-Server wird benötigt
        * Die gleichzeitige Benutzung ist auf die Anzahl individueller Lizenzen im [[https://wiki.mcneel.com/de/zoo/home|Zoo]] beschränkt
      * Anleitung zur automatischen [[https://wiki.mcneel.com/de/zoo/rhinomacdeployment|Verteilung ]]
      * Benötigt keinen Internetzugang
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Endbenutzer-Lizenzvertrag]]
