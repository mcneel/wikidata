====== Rhino 5: Error _mcneel._zoo5 ======
**Problem:**\\ Rhino 5 cannot run and shows an error message about "_mcneel._zoo5".\\ This problem happens when Rhino 5 runs for the first time after it's been insatalled.

{{http://wiki.mcneel.com/_media/rhino/rhino5/mcneel_zoo5.png}}

**Cause:**\\ An unknown installation problem.

**Solution:**\\
  - Download **[[http://files.na.mcneel.com/misc/Rhino5LicenseReset.exe.|Rhino5LicenseReset.exe]]**.
  - Right-click on the Rhino5LicenseReset.exe and select **Run as administrator**. 
  - Run Rhino 5.
  - Use one of the options below:\\ **a)** If you have a Rhino 5 license and would like to install the license on this computer, click **Enter a License** and enter your license.\\ **b)** If you have a Rhino 5 license and the license is installed on the Zoo license server, select **Use a Zoo** and set the Zoo server name/IP.\\ **c)** If you don't have a Rhino 5 license and you are evaluating Rhino 5, click **Evaluate**.
