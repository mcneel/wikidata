====== BoxEdit - Edit Objects by Bounding Box ======
> **Summary:** //Rhino 4.0 Plug-in to size, scale, position, and rotate objects based on their bounding box.//
> **Updated:** //August 5, 2009//

=====Description=====

This plug-in for Rhino 4.0 will allow you to numerically size, scale, position, and rotate objects based on their bounding box.

This plug-in requries Rhino 4.0 SR6 or greater.

{{http://en.wiki.mcneel.com/content/upload/images/boxedit.png}}

=====Installation=====

  - Copy **BoxEdit.rhp** file your **<drive>:\Programs Files\[[rhino:rhinoceros|RhinoCeros]] 4.0\Plug-ins** folder.
  - Start Rhino.
  - Run **PluginManager** and press the install button. Browse to your **<drive>:\Programs Files\[[rhino:rhinoceros|RhinoCeros]] 4.0\Plug-ins** folder and select **BoxEdit.rhp**.

=====Usage=====

  - Run **BoxEdit**.
  - Select some geometry.
  - Manipulate the controls in the BoxEdit dialog, the press **Apply**.
