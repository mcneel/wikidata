====== Open source ======

This page provides information about the open source code used by Rhino.

== RhinoCycles ==

The full repository for RhinoCycles code is here:

https://github.com/jesterKing/CCSycles

The original Cycles project is here:

https://developer.blender.org/diffusion/C/

== QuadMesher ==

QuadMesh uses the following open source libraries:

  * Eigen
  * Gmm
  * OpenBlas

The source code, licenses, and further details for these libraries can be downloaded from:

[[http://files.na.mcneel.com/misc/CoMISo.zip]]
