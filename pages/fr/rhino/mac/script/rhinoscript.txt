====== Puis-je lancer un script de commande dans Rhino pour Mac ? ======
> **Résumé:** //Rhinoscript existe-t-il dans Rhino pour Mac ?//

Non. RhinoScript est basé sur VisualBasic et VisualBasic ne fonctionne pas sur OS X. 

Nous vous recommandons d'utiliser Python pour l'écriture de scripts sur les versions Mac et Windows. Voir le [[http://discourse.mcneel.com/category/scripting | forum d'assistance pour l'écriture de scripts dans Rhino]] pour plus d'informations sur Python pour Windows et Mac.
