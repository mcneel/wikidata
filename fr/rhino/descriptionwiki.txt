====== Description du Wiki ======
> **Sommaire:** //Une description rapide du Wiki - ce que c'est et comment ça marche.//
  * Un [[http://www.flexwiki.com/default.aspx/FlexWiki/$$$|WikiWiki]] est un site web de pages liées entre elles mais dont chacune a une idée (comme DescriptionWiki).
  * Les rubriques peuvent aussi bien être informelles et peu recherchées que très complètes et bien étudiées.
  * La modification est accessible à tous [[http://www.flexwiki.com/default.aspx/FlexWiki/$$$|(AnybodyCanChangeAnything]]) ; vous (oui, vous!) êtes invités à commencer à écrire immédiatement  !
  * La modification d'une page est très, très facile. Cliquez sur Modifier dans le menu d'une page et modifiez le contenu en utilisant les règles de format [[http://www.flexwiki.com/default.aspx/FlexWiki/$$$|(FormattingRules]]).
  * Si vous voulez **annuler la modification**, utilisez le bouton "Précédente" de votre navigateur. De cette façon vous éviter de créer des copies identiques.
  * Si vous n'osez pas modifier une page, essayez d'abord notre PageTest.
  * Le ton général de ces pages est libre et respectueux.
  * Chaque modification d'une page est enregistrée et vous pouvez donc voir l'historique des pages. (Et votre adresse IP est mémorisée, il ne vaut donc mieux pas essayer de tout effacer)
  * Vous pouvez voir les modification réalisées pour chaque version d'une page.
  * C'est tout ce que vous devez savoir pour commencer à écrire.
  * De plus amples informations sont disponibles dans le Guide de l'utilisateur [[http://www.flexwiki.com/default.aspx/FlexWiki/$$$|(UserGuide]]), le Guide de l'administrateur [[http://www.flexwiki.com/default.aspx/FlexWiki/$$$|(AdministratorGuide]]) et le Guide du développeur [[http://www.flexwiki.com/default.aspx/FlexWiki/$$$|(DeveloperGuide]])
  * Une liste exhaustive (on espère) de tous les asects de la modifications des pages //FlexWiki// est donnée ici: [[http://www.flexwiki.com/default.aspx/FlexWiki/$$$.|ListOfEditingTopics]]
