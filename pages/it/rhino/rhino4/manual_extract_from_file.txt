===== Decomprimere un Rhino 4EXE dopo averlo scaricato ed installarlo manualmente =====

Alcuni sistemi, per determinate ragioni, presentano dei problemi con il formato file .png che usiamo nella schermata di installazione di V4, per cui non riescono a mostrare correttamente il programma di installazione. Se vi succede, provate questa soluzione alternativa:

  - Scaricate l'applicazione [[http://www.7-zip.org/|7Zip]] ed installatela nel vostro sistema.
  - Fate clic destro sul file di installazione di Rhino 4.
  - Dal menu, selezionate "**7-Zip**" e quindi "**Estrai i file...**".
  - Specificate un percorso in cui estrarre i contenuti e prendetene nota.
  - Andate su questa cartella e cercate la sottocartella **Rhinoceros**.
  - Cercate **Rhino4_Setup.msi**. Fate doppio clic per eseguire questo file ed installare la V4.

Troverete la descrizione dello stesso processo per la **V5** [[http://wiki.mcneel.com/it/rhino/rhino5/manual_extract_from_file|qui]].
