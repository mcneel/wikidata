====== Débloquer des modules de Rhino ======

Si vous rencontrez des problèmes lors de l'installation d'un module, il se peut que Windows ait marqué le fichier du module pour dire "Windows ne connaît pas la provenance de ce fichier". Ceci se produit souvent avec les fichiers téléchargés sur Internet ou reçus par e-mail.

Pour des raisons de sécurité, Windows ne chargera pas le module dans Rhino avant que vous ne débloquiez ce fichier.
  * Fermez Rhino
  * Cliquez avec le bouton de droite sur le fichier et choisissez "Propriétés" dans le menu contextuel
  * Une boîte de dialogue semblable à celle de l'image ci-dessous devrait apparaître
  * Cliquez sur le bouton "Débloquer"
  * Cliquez avec le bouton de droite sur l'icône du raccourci de Rhino, sélectionnez "Exécuter en tant qu'administrateur"
  * Chargez votre module
