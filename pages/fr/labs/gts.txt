======Export GTS======

=====Installation=====

  - **[[http://download.mcneel.com/download.asp?id=export_GTS|Téléchargez]]** le module Export GTS.
  - **[[http://download.mcneel.com/download.asp?id=import_GTS|Téléchargez]]** le module Import GTS.
  - Copiez les fichiers rhp dans votre dossier **\Program Files\Rhinoceros 4.0\Plug-ins**.
  - Chargez le module avec le GestionnaireModules.
=====Utilisation=====

  - GTS devrait apparaître dans la liste des types de fichier à enregistrer ou ouvrir.
