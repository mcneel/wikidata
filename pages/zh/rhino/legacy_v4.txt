
===== 在 Windows 10 下执行 Rhino V4 =====

Rhino V4 设计用来在 Windows XP 下执行，但也可以在新版 Windows 中最初设计的那样运作，以下是在较新版本Windows下运作的 Rhino V4限制与要求。

==== 限制与要求 ====
  * **内存** - 新的64位计算机系统可能配备16GB或更高的RAM。Rhino V4 是一款32位的应用程序。这意味着无论新系统有多少RAM，都只会用到最多4GB的RAM。 
  * **旧有应用系统** -  Microsoft 在Windows加入**用户帐户控制**之前发布Rhino V4。要解决这个问题，您需要在**Windows XP SP3兼容模式**中执行Rhino V4，否则Windows将阻止Rhino关闭时保留对工具栏及其他设定的更改。 Windows兼容模式设定在"桌面快捷方式"属性的内容里。
  * **低分辨率** - 您的新计算机能够执行的屏幕分辨率，可能比旧windows XP系统高出很多。您需要限制Windows并以较低的分辨率(如：1024X768)来执行，这点在Rhino V4发布时是受到支持的，要不然像**图层面板**这样的工具就会呈现空白，而且因为尺寸太小无法阅读。

==== 考虑尝试Rhino V8 ====
除了Rhino V8[[https://www.rhino3d.com/features|的新功能与特性之外]] ，要充分利用新的64位元Windows系统的功能。 \\
