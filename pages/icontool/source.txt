====== IconTOOL Source Files ======
> **Summary:** Feel free to download any or all the of IconTOOL source files and use them as you like. IconTOOL is **no** longer supported. //**DO NOT call us!**//

=====Downloads=====


**IconTOOL Source Code**

This download contains the C++ source code required to build the ARX tools used by IconTOOL 5.9. The projects included with the download will build the ARX files required to allow IconTOOL 5.9 to run on [[rhino:autocad|AutoCAD]] 2004, 2005 and 2006. To build the projects included in this download, you will need the following:

  * Microsoft's Visual Studio .NET 2002 (C++ 7.0)
  * Autodesk's ObjectARX SDK for [[rhino:autocad|AutoCAD]] 2004.
**The source code is available as-is, and we will not answer any questions on it.**

[[http://en.wiki.mcneel.com/content/upload/files/SourceIconTool59.zip|Download (466KB)]]
