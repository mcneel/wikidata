===== Zoo 또는 독립실행형 장치에서 사용 중인 라이선스 키를 Rhino Account 계정에 추가하려면 =====

사용 중인 라이선스를 Rhino Account 계정에 추가하려면, 우선 사용 중인 Zoo 또는 장치에서 라이선스를 제거해야 합니다.

==== 기존 위치에서 라이선스를 제거하기 ====

== 라이선스가 Zoo에 있는 경우: ==
  - Zoo에서 라이선스를 제거합니다.

== 라이선스가 독립실행형 장치에 있는 경우: ==
  - **도구** -> **옵션** -> **라이선스** -> **라이선스 키 변경** 으로 갑니다.
  - 대화상자의 아래쪽에 있는 **라이선스 제거**를 클릭합니다.

==== Rhino Account 계정에 라이선스를 추가하려면 ====
  - [[https://www.rhino3d.com/licenses/?controller=add_licenses|라이선스 추가 페이지에 갑니다]].
  - 귀하의 계정 또는 귀하가 속한 팀에 라이선스를 추가합니다.
