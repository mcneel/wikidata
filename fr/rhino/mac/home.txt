====Rhino OS X est en cours de développement:====

  * De nombreuses fonctions importantes **manquent toujours**, y compris **Grasshopper**, les modules, l'impression et la documentation.
  * Une nouvelle WIP est publiée **toutes les semaines ou tous les 15 jours** lors de la sortie de nouvelles fonctions..

-->Plus d'informations#

Rhino OS X est toujours en phase de développement 'Wenatchee'. Il n'est **pas prêt pour la production**. 

Il **manque** une interface utilisateur terminée, des ressources d'aide, des tutoriels et des outils de mise en page et d'impression, même pour remplir les besoins de base d'un dessin simple. 

Puisqu'il n'existe pas de SDK (Kit de réalisation de logiciel) pour le moment, le développement externe de modules n'a pas commencé. Et ceci inclut VRay. 

En revanche, les outils de création de surfaces sont très complets et utiles. Si vous utilisez le logiciel uniquement pour la modélisation de surfaces, il peut alors vous convenir. Cependant, si vous voulez pouvoir livrer un travail réel, **vous aurez besoin de Rhino pour Windows**.

L'assistance est un peu différente également. Rhino pour Windows est un outil complet, en vente et prêt pour la production, nos ressources d'assistance visent donc à vous aider à réaliser votre travail. 

Vous pouvez **participer au développement** mais puisque Rhino pour Mac est en cours de développement, nous nous intéressons principalement aux commentaires permettant de terminer le projet afin de le publier. Pour le moment, nous ne cherchons pas à vous aider à réaliser votre travail avec cette version

De nouvelles versions seront publiées plusieurs fois par mois pendant le développement. Chaque nouvelle version fonctionnera environ **60 jours** mais sera remplacée bien avant d'arriver à expiration.

====Configuration requise====
  * Processeurs 64 bits Core 2 Duo Intel ou ultérieurs
  * OS X 10.7.5, OS X 10.8.5 ou toute version de OS X 10.9.  Les bogues des versions précédentes de 10.7 et 10.8 pouvaient donner des problèmes dans Rhino. 
  * 4 Go de RAM (plus est recommandé)
  * Une souris à plusieurs boutons (une molette est recommandée)
  * Le [[http://www.3dconnexion.com/products/spacenavigator.html|SpaceNavigator]] peut être utilisé.

--> Matériel non pris en charge#
    * OS X 10.6 ou toute version précédente
    * **Spaceball** et **SpaceExplorer**
