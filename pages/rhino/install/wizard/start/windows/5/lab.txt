====== 3-user EDU Lab License  ======


Deployment type:
    * **Stand-alone -  Not recommended** - The license can be used to manually install and validate the license on 30 individual computers.

    * **Zoo - Recommended** 
        * **[[https://wiki.mcneel.com/zoo/home|Zoo Floating license manager]]** - Add the license to a Windows Server running our free floating license manager. 
        * **[[https://wiki.mcneel.com/rhino/installingrhino/5.0|Deploy Rhino]]** - Deploy Rhino silently onto the lab workstations pre-configured to use the Zoo server.
