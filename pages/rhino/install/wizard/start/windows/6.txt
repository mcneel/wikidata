====== Rhino 7, Rhino 6, or plug-in license type ======


===Choose one:===
    * **[[rhino:install:wizard:start:windows:6:commercial|Single Commercial and 90-day Evaluation License]]** //<- Pick for more details.//
      * Allows one user per license
      * May be used stand-alone, or shared over a LAN (Zoo) or Internet based floating license manager (Cloud Zoo).
      * [[https://www.rhino3d.com/7/eula|Rhino End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:educational|Single Educational License]]** //<- Pick for more details.//
      * Allows one user per license
      * May be used stand-alone on the student's computer, or with Cloud Zoo Personal license manager
      * [[https://www.rhino3d.com/7/eula|Rhino End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:lab|30-user EDU Lab License]]** //<- Pick for more details.//
      * Allows 30 concurrent users
      * May be used over a school LAN using the [[https://wiki.mcneel.com/zoo/home|Zoo]] floating license manager (recommended) or installed stand-alone.
       * Rhino installation easily **deployed**:
          * [[https://wiki.mcneel.com/rhino/installingrhino/7|Windows]]
          * [[https://wiki.mcneel.com/zoo/rhinomacdeployment|Mac]]
      * [[https://www.rhino3d.com/7/eula|Rhino End User License Agreement]]
