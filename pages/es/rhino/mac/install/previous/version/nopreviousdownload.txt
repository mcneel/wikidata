====== ¿Puedo descargar una versión anterior de Rhino para Mac? ======
> **Resumen:** //La versión actual de Rhino para Mac tiene un problema. ¿Puedo descargar la versión anterior?//

No, las versiones anteriores de Rhino para Mac no están disponibles.  

Si hay un error importante en la versión estable actual, infórmenos del error en  [[http://discourse.mcneel.com/c/rhino-for-mac|Discourse, en la categoría Rhino for Mac]].
