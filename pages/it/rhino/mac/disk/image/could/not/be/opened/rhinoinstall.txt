====== Messaggio di OS X: "Impossibile aprire le seguenti immagini disco." Cosa faccio? ======
> **Riepilogo:** //Dopo aver scaricato Rhino per Mac, il download segnala che non è stato possibile aprire l'immagine disco.//

Il file di download di Rhino per Mac è stato danneggiato durante il download. 

  - Andate su [[http://www.rhino3d.com/download/rhino-for-mac/5/evaluation]] e richiedete un altro link di download.

Se il secondo download del file .dmg non funziona, seguite il seguente processo di convalida:

  - Avviate il programma Utility Disco **/Applicazioni/Utility/Utility Disco**
  - Dal menu, selezionare **Immagini > Verifica...** Apparirà la finestra di dialogo "Apri file".
  - Selezionate il file .dmg scaricato. Di solito si trova nella cartella Download ed ha un nome simile a: Rhinoceros_5A123.dmg
  - Se il file scaricato non è valido, andate su [[http://www.rhino3d.com/download/rhino-for-mac/5/evaluation]] e richiedete un altro link di download.
  - Se succede di nuovo, la vostra connessione Internet non è affidabile e dovreste cercare di scaricare il file usando una connessione Internet diversa.  Il file di download supera i 100 Mb.
