====== Upgrading a LAN Zoo Server ======
> **Product:** //[[zoo:home|Zoo]]//
> **Summary:** //Upgrading a LAN Zoo server.//

To support the latest Rhino, you may need to either upgrade your **LAN Zoo** or move to the **[[https://wiki.mcneel.com/rhino_accounts/home|Cloud Zoo]]** (often a better option). 

===== Prerequisite  =====

[[https://wiki.mcneel.com/zoo/zoodotnet|The LAN Zoo Requires Microsoft .NET Framework 4.8]]

===== Download and Install =====
  * [[http://www.rhino3d.com/download/zoo/7/latest|Download the LAN Zoo here.]]

  * Install on the same server as your current LAN Zoo.

===== FYI =====

  * LAN Zoo upgrades will replace your older LAN Zoo.

  * Your licenses will be automatically be moved to the new LAN Zoo. 

  * Users running Rhino at the time of installation will not be affected.

  * The installation takes about 15 seconds.
