====== Cloud Zoo - 팀 ======

>안내: 이 옵션을 사용하여 팀 구성원들 간에 라이선스를 함께 사용합니다. 귀하의 라이선스가 저희 서버에 위치하게 되므로 귀하의 팀은 어떤 컴퓨터에서도 라이선스에 액세스할 수 있습니다. 처음 사용할 때 인터넷 연결이 필요합니다. 그 이후에는 Rhino에서 일주일에 한 번 정도의 인터넷 연결을 통해 라이선스 사용 기간이 연장됩니다. 약 2주 동안 인터넷 연결 없이 Rhino를 사용할 수 있으나, 계속해서 사용하려면 인터넷 연결이 필요합니다.


===초기 웹 기반 설정===
  - **사용자의 Rhino 계정에 로그인**
    - https://accounts.rhino3d.com/?controller=home 을 방문하세요.
      - 아직 Rhino 계정이 없다면 Rhino 계정을 만듭니다.
      - 귀하의 자격 증명을 저장합니다. 나중에 필요하게 됩니다.
  - **팀 만들기**
    - "나의 계정" 페이지에서 **팀**을 클릭합니다.
      - 새로운 팀을 클릭합니다.
      - 이름과 설명을 추가합니다.
  - **귀하의 라이선스를 팀에 추가**
    - "나의 계정" 페이지에서 **라이선스**를 클릭합니다.
    - 팀 라이선스 아래에서 앞서 만든 팀을 클릭합니다.
    - 팀 페이지에서 라이선스 추가를 클릭합니다.
    - 해당하는 제품을 선택합니다.
    - 귀하의 라이선스 키를 입력합니다.
    - 라이선스 추가를 클릭합니다.
  - **팀에 가입할 사용자 초대**
    - "나의 계정" 페이지에서 **팀**을 클릭합니다.
      - 액션을 클릭합니다.
      - 구성원 초대를 클릭합니다.
        - 안내: 구성원은 자신의 Rhino 로그인 계정이 있어야 합니다.
      - 귀하가 팀 라이선스에 액세스를 허용할 사용자의 이메일을 입력합니다.
      - 보내기를 클릭합니다.
      - 사용자가 초대를 받아 수락하면, 이에 대한 확인 이메일이 팀의 소유자인 귀하에게 전송됩니다.
  - **Rhino를 시작합니다**.
    - 로그인을 클릭합니다.
    - 귀하의 계정 자격 증명을 제공합니다.
    - Rhino가 시작합니다.

===다른 컴퓨터===
  - **Rhino를 설치합니다**.
  - **Rhino를 시작합니다**.
    - 로그인을 클릭합니다.
    - 귀하의 계정 자격 증명을 제공합니다.
    - Rhino가 시작합니다.
