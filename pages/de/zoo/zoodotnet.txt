====== LAN Zoo benötigt Microsoft .NET Framework 4.8 ======
> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //LAN Zoo und Microsoft .NET Framework 4.8//


Der LAN Zoo benötigt [[https://dotnet.microsoft.com/download/dotnet-framework/net48|Microsoft .NET Framework 4.8]]. 

Um sicherzustellen, dass Microsoft .NET Framework 4.8 auf Ihrem System installiert ist, verwenden Sie das **[[http://windows.microsoft.com/windows/windows-update|Windows-Update]]**.
