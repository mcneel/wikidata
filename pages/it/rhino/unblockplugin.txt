====== Sblocco dei plug-in di Rhino ======

Se state riscontrando dei problemi con l'installazione di un plug-in, uno dei motivi può essere il fatto che Windows abbia segnalato il file come potenzialmente pericoloso perché non sa da dove proviene. Di solito ciò succede con i file scaricati da Internet o ricevuti per e-mail.

Per motivi di sicurezza, Windows non caricherà il plug-in in Rhino fino a quando l'utente non "sbloccherà" il file.
  * Chiudete Rhino
  * Cliccate con il tasto destro sul file e selezionate "Proprietà" dal menu di scelta rapida
  * Vi apparirà una finestra di dialogo come quella dell'immagine nella parte inferiore di questa pagina
  * Cliccate sul pulsante "Sblocca"
  * Cliccate con il tasto destro sull'icona di collegamento di Rhino e selezionate "Esegui come amministratore"
  * Caricate il plug-in
