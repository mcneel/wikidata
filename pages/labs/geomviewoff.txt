====== Geomview OFF File Import ======
> **Summary:** //Labs plug-in to import Geomview .OFF files.//
> **Updated:** //August 15, 2012//

=====Description=====

This Labs plug-in will allow you to open or import Geomview .OFF files. Geomview is an interactive 3D viewing program for Unix. For more information on Geomview, including details on the OFF file format, visit the [[http://www.geomview.org|Geomview web site.]]

=====Installation=====
  - Download the plug-in and unzip the archive.
  - Start Rhino.
  - Run **PluginManager** and press the **Install** button. 
  - Browse to the location where you unzipped the archive.
  - Select the **import_OFF.rhp** file and press OK.

=====Usage=====

  - From Rhino's **File** menu, select either the **Open** or **Import** command.
  - In the **Files of Type** drop down box, select the **Geomview .OFF** file type.
  - Select your .OFF file.
  - Once opened or imported, you may need to **Zoom** to see the geometry.
