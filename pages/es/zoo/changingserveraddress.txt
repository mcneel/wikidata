====== Cambiar la dirección IP de un servidor del Zoo ======

> **Productos:** //[[http://wiki.mcneel.com/es/zoo/zoo/home|Zoo]]//
> **Resumen:** //Repercusiones de cambiar la dirección IP de un servidor de Zoo LAN//

=====Pregunta=====
Queremos cambiar la dirección IP en algunos de nuestros servidores de licencias. Uno de ellos ahora aloja el servidor de licencias del Zoo para Rhino. Como se trata de un 'soft lock', ¿se desactivará alguna parte de la licencia al cambiar la dirección IP en el servidor? El nombre de host y la dirección MAC permanecen iguales.

=====Respuesta=====
Ni la instalación del Zoo ni las licencias que mantiene están vinculadas a la dirección IP o MAC del servidor. Por lo tanto, en cuanto al Zoo, no hay ningún problema con el cambio de dirección IP del servidor que aloja el Zoo.

Si las instalaciones de Rhino encuentran que el servidor del Zoo utiliza el nombre de dominio del servidor (DNS), la migración debería realizarse sin problemas. Si las instalaciones de Rhino encuentran que el Zoo utiliza la dirección IP del servidor, cada estación de trabajo tendrá que ser redirigida.
