===== Checking Your Fonts for Problems===== 

Start Font Book in Applications

In the Collection column, click All Fonts (at the top)

{{:rhino:mac:checkfonts1.png?nolink|}}


Click some font in the Fonts column.

Type Command-A to select all fonts in the Font column.

{{:rhino:mac:checkfonts2.png|}}

Select File > Validate Fonts in the application menu.  A new dialog appears and the program checks your fonts.

{{:rhino:mac:checkfonts3.png|}}

Fix any font problems you find.
