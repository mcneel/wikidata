===== Rhino 6과 Rhino 7의 라이선스 사용 방식에 대한 질문과 대답 (FAQ) =====

==== Rhino 라이선스에는 어떤 사용 방식이 있나요? ====
상세한 안내는 https://www.rhino3d.com/6/new/licensing-and-administration 페이지를 참조하세요.

====라이선스 공용====

==1개의 Rhino 7 라이선스를 여러 대의 컴퓨터에서 사용할 수 있나요?==
예, 플로팅 라이선스 관리자를 사용하시면 가능합니다. LAN Zoo로 사무실 LAN에서 사용하시거나, 이따금 인터넷 연결이 필요한 Cloud Zoo를 통해 단일 사용자 라이선스를 여러 대의 컴퓨터에서 번갈아 사용하실 수 있습니다. 

==여러 대의 컴퓨터에 단일 컴퓨터 라이선스 방식으로 설치하려고 할 때는 어떻게 하나요?==
각 컴퓨터마다 별개의 라이선스 키가 1개씩 필요합니다. Rhino 5의 경우, 단일 사용자 라이선스 키의 1회 이상 유효성 확인이 가능하였습니다. Rhino 6 및 Rhino 7 버전에서는 그렇지 않습니다. Rhino 7에서는 Cloud Zoo 개인용 라이선스를 관리할 수 있어, 1개의 단일 사용자 라이선스를 여러 대의 컴퓨터에서 번갈아 사용할 수 있습니다(가끔씩 인터넷 연결 필요합니다). 

==1개 이상의 라이선스는 다른 여러 명의 사용자와 어떻게 공용하나요?==
귀하의 라이선스를 Cloud Zoo의 팀에 추가하는 방식을 권장합니다. 또는 LAN Zoo에 라이선스를 추가하여 사용하세요.
https://wiki.mcneel.com/ko/rhino/6/license

====Rhino 7로 업그레이드 ====

==Rhino 6에서 Rhino 7로 업그레이드하면 계속해서 Rhino 6 버전을 사용할 수 있나요?==
예, 업그레이드는 Rhino 7 기능을 사용할 수 있도록 귀하의 라이선스를 확장하는 것이며, 새롭게 라이선스를 추가하는 것이 아닙니다. 즉, Rhino 7 버전과 이전 버전의 Rhino를 같은 컴퓨터에서 사용할 수 있음을 뜻합니다. 다른 컴퓨터에서 다른 사용자가 Rhino 7 업그레이드를 사용하는 동안, 귀하가 귀하의 컴퓨터에서 Rhino 6 버전을 사용하실 수 없습니다.

==나의 Rhino 7 업그레이드 라이선스를 Cloud Zoo에 추가해도 Rhino 6 버전을 여전히 설치할 수 있나요?==
예, Rhino 6 버전이 Rhino 7 버전으로 업그레이드된 이후에도 Rhino 6 버전을 설치하실 수 있습니다. 단, 두 버전 라이선스 모두 Zoo 또는 Cloud Zoo에 두고 사용하셔야 합니다.  

==나의 Rhino 5 라이선스를 Cloud Zoo에 두고 사용할 수 있나요?==
아니요, Rhino 5에서는 Cloud Zoo가 지원되지 않습니다.

==Zoo에 나의 Rhino 6와 Rhino 7 업그레이드 라이선스를 추가하면 어떻게 되나요?==
Rhino 7 라이선스 키가 지원되는 Zoo 7 SR 7 (또는 그 이후) 버전이 필요합니다. Zoo가 귀하의 Rhino 6 및 Rhino 7 업그레이드 라이선스 키를 하나의 라이선스 "묶음"으로 만들어, 언제나 하나의 라이선스가 사용되도록 합니다. 동일한 컴퓨터에서 Rhino 6 버전과 Rhino 7 버전을 실행하는 사용자는 오직 하나의 라이선스만 사용하게 됩니다. 

====Rhino 7 을 파일 뷰어로 사용하려면 ====

==Rhino를 파일 뷰어로만 사용하려면 어떻게 해야 하나요?==

Rhino를 파일 뷰어로 사용하려면 평가판 라이선스 키가 필요합니다. 평가 기한이 만료된 라이선스 키도 괜찮습니다. 평가판 라이선스 키가 없으시다면 https://www.rhino3d.com/download/rhino/evaluation 에서 무료로 발급받으시기 바랍니다. 

평가판 라이선스 키의 유효성 검사를 실행할 때, 여러 컴퓨터에서 라이선스를 공용하는 옵션을 선택하지 마시고, 하나의 컴퓨터를 지정하여 라이선스를 사용하는 옵션을 선택하세요. 

이미 라이선스가 CloudZoo에 있는 상태라면 먼저 Rhino 계정에서 해당 라이선스를 제거하셔야 합니다:

  * https://accounts.rhino3d.com/ 에 접속하셔서 본인의 이메일 주소를 사용하여 Rhino 계정에 로그인합니다.
  * 페이지 아래쪽으로 스크롤하여 라이선스 메뉴를 클릭한 후, 개인용 라이선스 항목 아래에 있는 본인 이름을 클릭합니다. 
  * 평가판 라이선스를 제거하고 Rhino 7을 시작합니다.  
  * 새 라이선스를 입력하라는 화면에서 "라이선스 키 입력" 옵션을 선택하고 평가판 라이선스 키를 입력한 후, "계속"을 클릭합니다. "귀하의 Rhino 계정에 추가" 옵션을 **선택하지 않도록** 주의하시고, 왼쪽 아래에 있는 "이 컴퓨터에서만 사용" 옵션을 클릭한 후, "예"를 클릭하세요.   

본인의 Rhino 평가판 라이선스가 유효하다면 작업 내용을 저장할 수 있습니다. 평가 기한이 만료된 이후에는 저장 기능을 실행할 수 없으나, Rhino 모델 파일을 열고, 뷰를 변경하거나, 치수를 사용하는 등의 작업은 할 수 있습니다.
