======Différences entre Rhino pour Windows et Rhino pour Mac ======
=== Ajouts dans l'interface utilisateur de Mac ===

Un certain nombre de commande et de fonctions sont uniquement disponibles sur la version Mac.  Consultez [[http://wiki.mcneel.com/fr/rhino/mac/userinterfacefeatures|cette page wiki]] pour plus d'informations. 

-->Souris#

**Recommandations**

  * Pour une navigation plus facile, une souris à deux boutons avec molette est recommandée, mais toutes les opérations sont possibles en utilisant une souris à un seul bouton.
  * Pour réaliser les actions du bouton de droite avec une souris à un seul bouton, maintenez la touche **Contrôle** et cliquez.\\ \\

**Navigation**

  * Le **déplacement avec le bouton de droite** ou avec le **bouton de droite+Maj** fonctionne comme dans Windows. Voir [[http://docs.mcneel.com/rhino/5/help/fr-fr/user_interface/shortcuts.htm|Aide de Rhino - Raccourcis]]. \\ Quel que soit le style de projection de la vue, appuyez sur la touche **Maj** pour toujours effectuer un déplacement, sur la touche **Commande** pour zoomer et sur **Commande + Maj** pour faire tourner la vue.
  *  Utilisez la molette pour zoomer dans une vue.
  * Si votre souris ne possède qu'un seul bouton, vous devez appuyer sur **Contrôle**, **Commande** et **Maj** ensemble pour faire toujours tourner la vue.

<--

-->Menu contextuel du bouton de droite#

  * Cliquez avec le bouton de droite dans une fenêtre pour afficher le  menu contextuel.
  * Par défaut, le clic droit n'agit **pas** comme la touche **Entrée** pour répéter la dernière commande. \\ Pour changer ce comportement, dans **Rhinoceros** > **Preferences** > **Mouse** > **Mouse**, désactivez l'option **Enable context menus**.

<--

-->Ligne de commandes#

Si vous êtes habitué à la ligne de commandes dans Rhino pour Windows, vous retrouverez toutes ses fonctions dans Rhino pour Mac.  Pour activer la ligne de commandes, commencez simplement à taper. Une fenêtre intitulée **Command Search** apparaît.  Vous pouvez appuyer sur la touche Entrée ou sur la barre d'espace pour lancer la commande.

Une boîte de dialogue s'ouvre avec toutes les options qui apparaissent dans la ligne de commandes de Rhino pour Windows. Vous pouvez déplacer cette boîte de dialogue n'importe où à l'écran, Rhino conservera cette position pour la prochaine fois.

Si vous aimez également taper les valeurs des options, cochez l'option **Always provide an input field in command options dialog for typed values** dans le panneau Legacy des préférences. Une lettre de chaque option de la commande sera soulignée. Vous pouvez taper cette lettre et appuyez sur Entrée pour sélectionner cette option. 

Vous n'avez pas besoin de déplacer le curseur lorsque vous tapez des valeurs dans la boîte de dialogue des options de la commande,  tapez tout simplement et Rhino entrera vos valeur automatiquement.  Toutes vos habitudes d'utilisation de la ligne de commandes dans Rhino pour Windows fonctionnent exactement pareil dans Rhino pour Mac.

-->Barres d’outils et palettes#

  * Rhino pour Mac dispose de //boutons// dans la barre d'outils Macintosh standard afin d'exécuter les commandes. 
  * Rhino pour Mac dispose également de //palettes d'outils// ; ce sont des fenêtres contenant une collection de boutons. \\ Une palette d'outils, contenant les boutons les plus souvent utilisés, s'ouvre à gauche du document au démarrage de Rhino pour Mac.
  * Les boutons des palettes d'outils peuvent avoir une ou deux commandes et un menu.
  * Placez la souris sur un bouton dans une palette pour afficher l'info bulle.\\  Si l'info bulle comprend deux lignes de texte, le bouton contient une commande principale et une commande secondaire. 
    * **Cliquez** sur le bouton de la palette pour lancer la commande **principale**.
    * Pour lancer la commande **secondaire**, maintenez la touche **Option** et **cliquez** sur le bouton. 
  * Les boutons des palettes contenant un menu présent un petit triangle dans le coin inférieur gauche.
    * Pour ouvrir le menu, cliquez et maintenez enfoncé le bouton de la souris sur le bouton de la palette. 
  * Les menus de la palette peuvent aussi être affichés sous forme de palette flottante. 
    * Pour afficher une palette flottante, maintenez la touche **Option** puis cliquez tout en maintenant le bouton de la souris. 

<--

-->Clavier#

  * Rhino pour Windows utilise la touche **Ctrl** pour de nombreuses opérations. Dans la version Mac, utilisez la touche **Commande**. 
    * Par exemple, **Commande** + clic remplace **Ctrl** + clic pour désélectionner des objets.
    * La touche **Commande** remplace également la touche **Ctrl** pour les raccourcis de navigation:

| | | |**Touche** | **Action** | **+ touche Commande** |
| | | |**Flèche gauche** | Rotation gauche | Panoramique gauche |
| | | |**Flèche droite** | Rotation droite | Panoramique droite |
| | | |**Flèche vers le haut** | Rotation vers le haut | Panoramique vers le haut |
| | | |**Flèche vers le bas** | Rotation vers le bas | Panoramique vers le bas |

-->Problèmes sur un ordinateur portable#


**Souris Bluetooth**\\ Les ordinateurs portables étant dotés par défaut d'une souris à un seul bouton, il est utile d'ajouter une souris Bluetooth.\\ \\

**Trackpad**\\ Faire glisser avec deux doigts verticalement sur le trackpad zoome dans la vue actuelle. \\ Pour changer ce comportement, dans **Rhinoceros** > **Preferences** > **Trackpad**, vous pouvez cocher **Use two fingers to pan and rotate views**.\\

**Touches manquantes**\\ Un clavier d'ordinateur portable n'a pas de touche **Début**, **Fin**, **PgSuiv** ou **PgPréc**, mais la touche **fn**, utilisée en combinaison avec d'autres touches permet de réaliser les fonctions de ces touches. \\

| **Touche** | **+ touche fn** |
| **Flèche vers le haut** | Page Préc |
| **Flèche vers le bas** | Page Suiv |
| **Flèche vers la gauche** | Début |
| **Flèche vers la droite** | Fin |
| **Supprimer** | Suppr |
