====== 30 사용자 교육 실습실용(Lab) 라이선스  ======


===하나를 선택하세요:===
    * **[[ko:rhino:install:wizard:start:windows:6:commercial:zoo|학교 LAN을 사용하는 단일 컴퓨터 실습실의 30대의 컴퓨터]] - Zoo (권장 방식)**
        * **사용자 관리가 필요하지 않으며, 관리 유지/보수 작업은 최소화됩니다.**
        * 학교 교실/실습실에서 라이선스를 제한할 때 이 옵션을 사용하세요.
        * 학교 LAN을 사용합니다.
        * 무료 플로팅 라이선스 관리자인 **[[https://wiki.mcneel.com/ko/zoo/home|Zoo]]** 가 필요합니다.
        * 인터넷 연결이 필요하지 않습니다.
        * Rhino 계정이 필요하지 않습니다.
        * Rhino 설치는 쉽게 **배포할 수 있습니다**.
          * [[https://wiki.mcneel.com/ko/rhino/installingrhino/7|Windows]]
          * [[https://wiki.mcneel.com/ko/zoo/rhinomacdeployment|Mac]]
        * [[https://www.rhino3d.com/7/eula|최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[https://wiki.mcneel.com/ko/rhino/6/cloud_zoo/setup|인터넷을 사용하는 팀 구성원 30명]] - Cloud Zoo 팀**
        * **상당한 팀 관리 및 지속적인 관리 작업이 필요합니다.**
        * 귀하가 여러 명의 팀 구성원들과 귀하의 라이선스를 공유할 때 이 옵션을 사용합니다.
        * 캠퍼스 외의 라이선스 액세스를 허용합니다.
        * 가끔 인터넷 연결이 필요합니다.
        * 각 학생이 수동으로 Rhino를 설치합니다.
        * 학교 관리자가 Cloud Zoo 팀을 생성하고 관리합니다.
        * 각 학생은 각자의 [[https://accounts.rhino3d.com/help|Rhino Account]] 로그인이 필요합니다.
        * [[https://www.rhino3d.com/7/eula|최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[ko:rhino:install:wizard:start:windows:6:commercial:stand_alone|30대의 단일 컴퓨터 라이선스 방식]] (권장하지 않습니다)**
      * **IT 담당자의 지속적인 관리 작업(설정, 교체, 이미지로 교실 컴퓨터 다시 설치 등)이 필요합니다.**
      * 학교에서 소유한 30대의 개별 컴퓨터에서 사용하는 경우, 이 옵션을 사용합니다.
      * 각 컴퓨터에서 유효성 검사를 받기 위한 인터넷 연결이 1회 필요하거나, 또는 [[https://wiki.mcneel.com/ko/zoo/rhinovalidateoffline|오프라인 유효성 검사]]가 필요합니다.
      * 업데이트할 때를 제외하고, 인터넷 연결이 필요하지 않습니다.
      * LAN 이 필요하지 않습니다.
      * 학교에서 소유한 30대의 컴퓨터에 Rhino를 수동으로 설치합니다. 
      * [[https://www.rhino3d.com/7/eula|최종 사용자 사용권 계약 - End User License Agreement]]

>이 위키 페이지는 Rhino 설치 마법사의 일부입니다. [[ko:rhino:install:wizard:start |처음부터 시작하려면 여기를 클릭하세요]].
