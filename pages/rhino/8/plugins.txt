==== Plugin Support for Rhino 8 ====

Some plug-ins were written to support .NET Framework and do not run in Rhino 8 when running in .NET Core (the default). Here are a list of plugins that we know about:

=== Rhino Plugins ===

== Enscape ==
Status: In Development (Nov 1, 2023)

Chaos group is aware of the situation and is actively developing a Rhino 8 compatible plugin. Release is expected before beginning of 2024.
