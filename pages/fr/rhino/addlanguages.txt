====== Ajouter une langue dans Rhino 4. ======

> **Sommaire:** //Comment ajouter des langues dans Rhino//
Pour ajouter plusieurs langues dans Rhino alors qu'il est déjà installé, suivez ces instructions :


\\

**1.** Placez le CD d'installation de Rhino dans le lecteur CD.

<color White> .</color>

<color White> .</color>

<color White></color>

{{http://fr.wiki.mcneel.com/content/upload/images/PanneauConfiguration.png}}

**2.** Ouvrez le panneau de contrôle de Windows **Ajout/Suppression de programmes**.

**3.** Trouvez **Rhinoceros 4** dans la liste et cliquez sur **Modifier**

<color White> .</color>

<color White></color>

{{http://fr.wiki.mcneel.com/content/upload/images/ModifierInstallation.png}}
**4.** Dans la boîte de dialogue suivante, cliquez sur le bouton **Modifier**.

<color White> .</color>

<color White></color>

{{http://fr.wiki.mcneel.com/content/upload/images/AjouterLangue.png}}

**5.** Sélectionnez les langues dans la boîte de dialogue suivante, cliquez sur **Suivant** et suivez les invites jusqu'à la fin de l'installation.
