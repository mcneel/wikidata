====== How to Create a GUID ======
> **Developer:** //[[developer:rhinoscript|RhinoScript]]//
> **Summary:** //Demonstrates how to create a Globally Unique Identifier (GUID) in RhinoScript//

=====Question=====
We have items that we need to track using unique identification numbers. My boss suggested that we use GUIDs for this, just like Rhino does. Is there any way to create a GUID using a script?

=====Answer=====
There is actually an easy way to generate GUIDs. The **Scriptlet.TypeLib** object includes a method that generates GUIDs. If you need a GUID, here is a short script that will supply you with one:

<code vb>
' Creates a Registry-formatted GUID string
' Ex: {xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx}
Function CreateGuidRegistryFormat
  Dim objTypeLib
  Set objTypeLib = CreateObject("Scriptlet.TypeLib")
  CreateGuidRegistryFormat = Left(objTypeLib.Guid, 38)
End Function
</code>
 
If you want to create a plain GUID -- one without the surrounding curly brackets -- then you can do something like this:

<code vb>
' Creates a plain-formatted GUID string
' Ex: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
Function CreateGuidPlainFormat
  Dim objTypeLib
  Set objTypeLib = CreateObject("Scriptlet.TypeLib") 
  CreateGuidPlainFormat = Mid(objTypeLib.Guid, 2, 36)
End Function
</code>
