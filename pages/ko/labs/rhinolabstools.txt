====== Rhino Labs Tools ======
> **Summary:** //가장 까다로운 Rhino 4.0 디자이너를 위한 여러 툴을 소개합니다.//
> **Updated:** //2009년 10월 13일//

{{:legacy:en:RhinoLabsToolsBig.jpg}}

=====설명=====

RhinoLabsTools 플러그인에는 여러분께서 사용하시기 편리한 기능들이 포함되어 있습니다. 이러한 툴 중 일부는 다음 릴리스에 추가될 수도 있습니다...

=====명령=====
다음은 현재 RhinoLabsTools 플러그인에 있는 명령들입니다:

  - **ArrayLinear** - 개체를 1차원적으로 배열시킵니다.
  - **ConvertDots** - 텍스트 도트를 점 또는 텍스트 개체로 변환합니다.
  - **DupDimStyle** - 치수 스타일을 복제합니다.
  - **ExplodeLinetype** - 비연속적인 선 종류를 세그먼트로 분해합니다. **Rhino 4.0 에서만 실행됩니다**
  - **FindText** - 문서에서 텍스트를 찾습니다.
  - **HideLayersInDetail** - Rhino의 HideInDetail 명령과 유사하지만, 오직 한 레이어에 있는 모든 개체를 숨깁니다.
  - **ImportDimStyles** - 다른 .3dm 파일에서 치수 스타일을 가져옵니다.
  - **MatchCrvDir** - 선택된 커브의 방향을 일치시킵니다.
  - **MatchProperties** - 개체, 재질, 텍스처 속성을 일치시킵니다.
  - **ModifyRadius** - 호와 원의 반지름을 변경합니다.
  - **MoveGripsInside** - 닫힌 평면 경계의 안쪽에 있는 평면 커브의 그립을 이동합니다.
  - **MoveInside** - 닫힌 평면 경계의 안쪽에 있는 평면 커브를 이동합니다.
  - **ReplaceBlock** - 블록을 다른 블록 정의로 대체합니다.
  - **RevCloud** - 구름형 폴리선 (revision cloud) 커브를 만듭니다.
  - **Revert** - 이전에 저장된 문서로 복귀합니다.
  - **SelDimStyle** - 스타일을 기준으로 치수를 선택합니다.
  - **SelHatch** - 해치 개체를 선택합니다.
  - **SelNonManifold** - 비다양체 (non-manifold) 서페이스와 폴리서페이스를 선택합니다.
  - **SelRenderColor** - 확산 렌더링 색으로 개체를 선택합니다 (Rhino renderer 에서만 가능).
  - **ShowLayersInDetail** - **HideLayersInDetail** 로 숨겨진 레이어 개체를 표시합니다.
  - **SplitCrv** - 한 개의 커브를 여러 개의 조각으로 분할합니다.
  - **SuperExplodeBlock** - 블록을 구성요소 개체 (중첩된 블록 포함) 로 분해합니다.
  - **SynchronizeRenderColors** - 개체의 재질 색상과 표시 색상을 동기화합니다.
  - **UnjoinEdge** - 선택된 폴리서페이스 가장자리를 결합 해제합니다. 닫힌 서페이스의 심 (seam) 은 결합 해제하지 않습니다.
  - **VertexColorCreator** - 메쉬 텍스처를 기준으로 메쉬 정점 색을 생성합니다.
  - **VPoint** - 평행 투영된 뷰를 보는 방향을 설정합니다.

=====설치=====
  - **[[http://s3.amazonaws.com/mcneel/rhino/4.0/misc/vcredist_x86.exe|중요: 최신 Microsoft Runtimes]]** 를 다운로드하여 설치하십시오.
  - **[[http://download.mcneel.com/download.asp?id=rhinolabstools|플러그인 다운로드 클릭]]**
  - **RhinoLabsTools.rhp** 파일을 **<drive:>\Program Files Rhinoceros 4.0\Plug-ins** 폴더에 복사합니다.
  - Rhino 4.0 을 시작합니다.
  - **PluginManager** 명령을 실행하고, Install (설치) 단추를 누릅니다. **<drive:>\Program Files Rhinoceros 4.0\Plug-ins** 디렉터리에서 **RhinoLabsTools.rhp** 를 선택합니다.
