====== Rhino 5 라이선스 유형 ======


===하나를 선택하세요:===
    * **[[ko:rhino:install:wizard:start:mac:5:commercial|단일 사용자 상업용 및 90일 평가판 라이선스]]**
      * 한 라이선스당 한 명의 사용자가 허용됩니다.
      * 독립실행형으로 사용합니다. 또는, 무료 플로팅 라이선스 관리자인 Windows용 [[https://wiki.mcneel.com/ko/zoo/home|Zoo]]를 사용하여 LAN 상에서 사용할 수 있습니다.
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|최종 사용자 사용권 계약 - End User License Agreement]]


    * **[[ko:rhino:install:wizard:start:mac:5:educational|단일 사용자 교육용 라이선스]]**
      * 한 라이선스당 한 명의 사용자가 허용됩니다.
      * 독립실행형 설치만 가능합니다.
        * 사용자가 소유한 컴퓨터여야 합니다.
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|최종 사용자 사용권 계약 - End User License Agreement]]


    * **[[ko:rhino:install:wizard:start:mac:5:lab|30 사용자 교육 실습실용 라이선스]]**
      * 30명의 동시 사용자가 허용됩니다.
      * 무료 플로팅 라이선스 관리자인 Windows용 [[https://wiki.mcneel.com/ko/zoo/home|Zoo]]를 사용하여 학교 LAN 상에서 사용할 수 있습니다. 또는 독립실행형으로 사용할 수 있습니다.
      * Rhino 설치는 쉽게 [[https://wiki.mcneel.com/ko/zoo/rhinomacdeployment|배포]]할 수 있습니다.
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|최종 사용자 사용권 계약 - End User License Agreement]]




>이 위키 페이지는 Rhino 설치 마법사의 일부입니다. [[ko:rhino:install:wizard:start |처음부터 시작하려면 여기를 클릭하세요]].
