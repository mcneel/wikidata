====Errore di installazione 1603 - Rhino 6 e Rhino 7====

Errore irreversibile durante l'installazione. (-2147023293)

> **Sommario:** //Come risolvere l'errore di installazione 1603//

=====Problema=====
Gli errori 1603 possono essere causati da vari problemi. Segui tutti i passi in basso.

=====Risposta=====
**Passo 1: Esegui gli aggiornamenti di Windows**
  - Individua ed esegui il [[http://windowsupdate.microsoft.com/|servizio Windows Update]].
  - Se esistono aggiornamenti importanti da installare, fai clic sul pulsante di aggiornamento per installarli.  Questo processo può richiedere molto tempo.

\\
**Passo 2: Installa .NET 4.8 Framework**
  - Scarica ed esegui il [[https://dotnet.microsoft.com/download/dotnet-framework/net48|programma di installazione di .NET 4.8 framework]].
  - Nessun errore al passo 4.
  - Se il programma di installazione di Microsoft .NET **mostra un errore: 
    - Annotare l'errore, quindi andare al punto 3.
\\
**Passo 3: Riparazione del framework .NET 4.8**
  - Scaricare ed eseguire [[https://www.microsoft.com/en-us/download/details.aspx?id=30135|.Net Framework Repair Tool]]
  - Se il programma di installazione di Microsoft .NET **mostra un errore: 
    - Prendi nota dell'errore. 
    - Vai alla sezione **Se il problema persiste**, in basso.

\\
**Passo 4: Installa Rhino**
  - Apri **Windows Explorer**.
  - Scarica la versione più recente di [[https://www.rhino3d.com/download/rhino/7/latest|Rhino 7]] o il programma di installazione di [[https://www.rhino3d.com/download/archive/rhino/6/latest|Rhino 6]].
  - Esegui il file scaricato per avviare il processo di installazione.
  - Se riscontri problemi di installazione, salva una copia del file di registro sul desktop.
\\
**Se il problema persiste**

È frustrante. Ci dispiace! Invia per e-mail il file di registro dal passo 3 a tech.eu@mcneel.com.
