==============
====Rhino for Mac is nearly production ready.====

  * Features we are still working on for the first version:
    * **Import and/or export** for a few file formats
    * **User interface** polish
    * **Localization** Proper translation for all supported languages
    * **Documentation** and tutorials

  * These features will **NOT** be in the first version. **//<- Important//**
    * **Grasshopper**
    * **Layouts**
    * **Worksessions**
    * **SolidWorks** file import
    * **GHS** support
    * **KML, WMF, ZPR, and SCN** file I/O
    * **Third-party Plug-ins**

A new WIP build will be available **every week or two**. Each build will run for about **60 days**.

====System Requirements====
  * 64-bit Core 2 Duo Intel processors or later
  * OS X 10.8.5, or any version of OS X 10.9.  Bugs in earlier versions 10.8 could cause problems in Rhino so the latest version of 10.8 is required.
  * 4 GB RAM (more is recommended)
  * A multiple-button mouse (a scroll wheel is recommended)
  * The [[http://www.3dconnexion.com/products/spacenavigator.html|SpaceNavigator]] is supported. (The driver from 3DConnexion is required.)

--> Not Supported Hardware#
    * OS X 10.7 or earlier versions
    * The **Spaceball** and **SpaceExplorer**
