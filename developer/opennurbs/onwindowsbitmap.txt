====== Convert OnWindowsBitmap to System.Drawing.Bitmap ======
> **Developer:** //[[developer:opennurbs:home|openNURBS]]//
> **Summary:** //Demonstrates how to convert an OnWindowsBitmap to a System.Drawing.Bitmap using the openNURBS toolkit.//

===== Question =====
Using the .Net (C#) SDK, what's the easiest way to convert an OnWindowsBitmap to a System.Drawing.Bitmap?

===== Answer =====
<code c#>
static public Bitmap Create(OnWindowsBitmap on_bitmap)
{
  if (null == on_bitmap || on_bitmap.Width() < 1 || on_bitmap.Height() < 1)
    return null;
  Bitmap result = null;
  try
  {
    result = new Bitmap(on_bitmap.Width(), on_bitmap.Height(), System.Drawing.Imaging.PixelFormat.Format24bppRgb);
    if (null != result)
      for (int y = 0; y < result.Height; y++)
        for (int x = 0; x < result.Width; x++)
          result.SetPixel(x, y, on_bitmap.Pixel(x, y));
  }
  catch (Exception exception)
  {
    System.Diagnostics.Debug.WriteLine(exception.Message + "\n" + exception.StackTrace);
  }
  return result;
}
</code>
