====== Rhino pour Mac sera-t-il compatible avec mon module .NET ? ======
> **Résumé :** //Les modules .NET peuvent pour certains fonctionner sur Windows et Mac.//

Il est possible de créer des modules qui fonctionnent aussi bien sur Rhino pour Windows que sur Rhino pour Mac.  Le développeur devra utiliser RhinoCommon (C# .NET) ; les modules développés en C/C++ ne sont pas compatibles avec Rhino pour Mac.  

Pour obtenir une liste de modules pris en charge sur Rhino pour Mac, veuillez consulter [[http://www.food4rhino.com/browse?sort_by=ds_changed&f[0]=im_field_unified_type%3A773&f[1]=im_field_platform_app%3A726|Food4Rhino]].

Les développeurs trouveront plus d'informations dans la [[http://developer.rhino3d.com/guides/rhinocommon/|section RhinoCommon de nos guides pour développeur]].
