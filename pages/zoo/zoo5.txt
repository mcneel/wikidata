======Zoo 5======

===== Getting Started =====
===Setting up the Zoo===
  - **[[http://download.rhino3d.com/Zoo/5.0/release/download/|Download]]** and install the Zoo on a Windows-based server or workstation.
  - Make sure **[[http://wiki.mcneel.com/zoo/window7firewall|TCP Port 80]]** is open in your system's firewall software.
  - Run **ZooAdmin** and **[[rhino5validate|add your Rhino CD-Keys]]**.
===Setting up Rhino to use the Zoo===
  - If Rhino is already installed, then run Rhino and click **Tools -> Options -> Licenses**. Select your Rhino license and click **Convert**. Close and restart Rhino. During restart, select **Network Node** and then specify the **[[http://wiki.mcneel.com/zoo/determinezoohost|name]]** of your Zoo server.
  - When installing Rhino, select **Network Node** and then specify the **[[http://wiki.mcneel.com/zoo/determinezoohost|name]]** of your Zoo server.

=====Compatibility=====
If you have Rhino 4.0, you need [[http://www.rhino3d.com/download/zoo/4.0/release|Zoo 4.0]].
