====== LAN Zoo läuft nicht auf Windows XP ======
> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //Betrieb des LAN Zoo auf Windows XP//

===== Frage=====
Ich versuche, den LAN Zoo auf Windows XP auszuführen. Sobald ich den Zoo auf dem entsprechenden Computer ausführe, blockiert sich der LAN Zoo und nichts funktioniert mehr. Unterstützt Windows XP den LAN Zoo nicht?

===== Antwort=====
  - Der Zoo benötigt das Microsoft .NET Framework 4.8
  - Das .NET Framework 4.8 und spätere Versionen werden __nicht__ auf Windows XP unterstützt.
