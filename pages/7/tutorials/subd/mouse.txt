====== MX Master 3 Mouse SubD Tutorial ======

> **Summary:** Learn how to create a mouse using Rhino 7's new SubD surfacing tools. Tutorial is for intermediate level

=====Overview=====


The tutorial uses a simple format of GIFs and command names to show us the overall process of making the MX Master 3 mouse casing. A great way to discover the new SubD tools in Rhino 7. If you want to learn more about SubD, keep tuned to our [[https://www.youtube.com/watch?v=Ipb84RImOX8&list=PLWIvZT_UEpWX25WrL19LHhD_qgemaS9ai | Youtube channel]], where you'll find the latest on our SubD tools
