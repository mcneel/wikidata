====== Rhino C++ Plug-in SDK ======
[[http://wiki.mcneel.com/developer/cplusplusplugins|{{:developer:rhino_plugin.png?128|Rhino C++ Plug-in SDK}}]]
=====Overview=====
The Rhino C++ Software Development Kit (SDK) provides the tools to customize and extend Rhino with direct access to its database structures, geometry, graphics system, and command definitions, and much more.

All of our development tools are free, royalty free, and include free support!

> **Requirements:** 
>
> Microsoft Visual C++ 2010 for Rhino 5 64-bit.
> Microsoft Visual C++ 2005 for Rhino 5 32-bit and Rhino 4.
>
> //The Express Editions of Microsoft Visual Studio will **not **work.//
