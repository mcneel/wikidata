======Rhino 5 pour Mac est presque là !======

[[http://wiki.mcneel.com/rhino/mac/home|{{:rhino:mac:osx.png?112|Page principale de Rhino pour Mac}}]][[http://wiki.mcneel.com/rhino/mac/home|{{:rhino:rhino_symbol.png?164|Page principale de Rhino pour Mac}}]]

 La Release Candidate 1 (RC1) est un produit complet qui fonctionne avec les clés de licence pour une période d'évaluation de 90 jours, commerciale et éducation (mais pas les clés WIP). Après avoir téléchargé la RC1, la période d'évaluation de 90 jours commence.

Cette version RC1 sera la version définitive de Rhino pour Mac  [[http://discourse.mcneel.com/t/reporting-problems-with-mac-rhino/6078| sauf si vous nous indiquez un problème important]].

**Prochaines étapes :**

  - Votre version WIP de Rhino pour Mac ne fonctionnera plus à partir du 3 juin 2015.
  - Avant le 3 juin, visitez [[http://www.rhino3d.com/download/rhino-for-mac/5/evaluation|http://www.rhino3d.com/download/rhino-for-mac/5/evaluation]] pour télécharger la RC1. 
  - Nous vous enverrons un message avec votre clé de licence d'évaluation de 90 jours pour Rhino pour Mac ainsi qu'un lien de téléchargement.
  - Dans les semaines à venir, nous annoncerons la sortie de Rhino pour Mac avec une offre de lancement et les indications pour acheter.

**Important :**

Certaines fonctions de la WIP [[http://discourse.mcneel.com/t/rhino-for-mac-removed-features/19947|ont été supprimées]] de Rhino pour Mac.

Rhino pour Mac n'est [[http://www.rhino3d.com/mac-feature-compare/|PAS pareil]] que Rhino pour Windows.
[[http://discourse.mcneel.com/t/reporting-problems-with-mac-rhino/6078|Merci de nous prévenir]] si vous trouvez une erreur qui devrait être corrigée avant la sortie du logiciel.

-----
====Configuration requise====
  * Apple Mac avec processeur Intel Core 2 Duo 64 bits ou supérieur.
  * OS X 10.8.5, 10.9.5, 10.10 ou supérieur.
  * 4 Go ou plus de RAM est recommandé.
  * Un processeur graphique NVIDIA ou AMD est recommandé.
  * Une souris à plusieurs boutons avec molette est recommandée.
  * Le [[http://www.3dconnexion.com/products/spacenavigator.html|SpaceNavigator de 3DConnexion ]] peut être utilisé (un pilote est nécessaire). 
  * [[http://wiki.mcneel.com/fr/zoo/home|Le Zoo]] (Nécessaire uniquement pour le partage de licence en réseau).
