====== Alias d'AutoCAD pour Rhino ======

==== Question ====
AutoCAD possède plusieurs alias de commandes utiles. Par exemple, si je tape "L" dans la ligne de commande d'AutoCAD, la commande Ligne s'exécute. Ces alias peuvent-il être intégrés dans les alias de Rhino ?

==== Réponse ====
Nous avons préparé un ensemble d'alias AutoCAD que vous pouvez importer dans Rhino. 

=== Configurer Rhino pour Windows===
  - **{{:rhino:acadaliasesforrhino.zip|Téléchargez}}** et décomprimez le fichier texte comprenant les alias AutoCAD pour Rhino. 
  - Lancez Rhino pour Windows puis sélectionnez **Outils > Options**.
  - Ouvrez la section **Alias**. 
  - Cliquez sur le bouton **Importer**.
  - Sélectionnez le fichier de texte que vous venez de décomprimer. 

{{:rhino:acadaliases_win.png?500}}

===Configurer Rhino pour Mac===
  - **{{:rhino:acadaliasesforrhino.zip|Téléchargez}}** et décomprimez le fichier texte comprenant les alias AutoCAD pour Rhino.
  - Lancez Rhino pour Mac puis sélectionnez **Rhinoceros > Préférences**.
  - Cliquez sur l'option **Alias**. 
  - Cliquez sur le bouton déroulant représentant un **'Engrenage'** puis cliquez sur l'élément **Importer des alias à partir d'un fichier...**.
  - Sélectionnez le fichier de texte que vous venez de décomprimer.

{{:rhino:acadaliases_mac.png?500}}
