====== Risoluzione dei problemi relativi all'installazione di Rhino ======

> Questo articolo si riferisce a Rhino 6 e Rhino 7. Articoli simili sono disponibili per [[http://wiki.mcneel.com/rhino/installproblems|Rhino 5]] o [[http://wiki.mcneel.com/rhino/installproblems_legacy|Rhino 3 e 4]].

**Nota importante:** È possibile: evitare quasi tutti i problemi e le correzioni elencati in questa pagina se, prima dell'installazione di Rhino, sono stati eseguiti tutti gli **aggiornamenti importanti** di Windows utilizzando **[[http://windowsupdate.microsoft.com/|Windows Update]]**. Per verificare lo stato di Windows Update sul computer, consulta la [[http://windowsupdate.microsoft.com/|pagina del localizzatore di Windows Update sul sito web di Microsoft]].\\

Una volta aperto Windows Update sul computer, fai clic sul pulsante di **verifica degli aggiornamenti**. Dopo aver installato i nuovi aggiornamenti, prova nuovamente a installare Rhino.

Se Windows Update è stato installato, mostra un errore o non ha completato l'installazione degli aggiornamenti, consulta l'articolo sul [[https://support.microsoft.com/en-us/kb/2700567|blocco di Microsoft Update]].

**Segnalazione dei problemi di installazione**

Se riscontri un errore durante il processo di installazione, il programma di installazione di Rhino 6 visualizzerà un file di registro.  Questo file risulta molto utile ai nostri tecnici per la risoluzione di eventuali problemi riscontrati dagli utenti. 
  - Nella finestra in cui appare il messaggio di errore, appare anche un link. 
  - Fai clic sul link per aprire il file di registro e salvarlo in formato txt.
  - Invia un'e-mail a [[tech.eu@mcneel.com]] con il file di testo in allegato.
