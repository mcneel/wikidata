====== Les fichiers 3DM de Rhino pour Mac sont-ils compatibles avec Rhino pour Windows ? ======
> **Résumé:** //Les fichiers 3DM sont-ils compatibles entre les versions Mac et Windows de Rhino ?//

Oui, les fichiers de Rhino 5 sont identiques. Vous pouvez également exporter des fichiers version la version 4 de Rhino pour Windows.
