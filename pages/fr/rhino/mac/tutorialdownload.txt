=====Télécharger les fichiers de tutoriel de Rhino pour Mac sur OS X 10.7 et versions suivantes =====

**Remarque:** Les tutoriels ont été écrits pour la version Windows de Rhino. La version Mac n'étant pas terminée, les tutoriels n'ont pas été convertis pour refléter les différences.

**Pour télécharger les fichiers de tutoriel**

  * Dans le menu **Rhino**, cliquez sur **? > Apprendre à utiliser Rhino** puis sur **Ouvrir les modèles des didacticiels...**. 

Si les fichiers des tutoriels ne sont pas installés sur votre ordinateur, Rhino vous demandera de les installer. Le fichier **rh40mac_wip_tutorials_20090724.dmg** sera téléchargé dans votre dossier **Téléchargements**.

Ouvrez le fichier **rh40mac_wip_tutorials_20090724.dmg** pour afficher une nouvelle fenêtre du Finder avec le fichier **RhinocerosTutorials.mpkg**.

Double-cliquez sur le fichier **RhinocerosTutorials.mpkg** pour installer les fichiers des tutoriels. 

Il se peut que OS X vous avertisse que le fichier a été créé par un développeur non identifié et qu'il refuse de l'ouvrir. Si cela se produit, **cliquez avec le bouton de droite** (ou **ctrl+clic**) sur le fichier **RhinocerosTutorials.mpkg** et dans le menu contextuel, cliquez sur **Ouvrir**. Dans la boîte de dialogue, cliquez sur le bouton **Ouvrir**. 

L'installation du tutoriel démarrera.  Suivez les instructions à l'écran. 

Lorsque l'installation est terminée, cliquez sur **? > Apprendre à utiliser Rhino > Ouvrir les modèles des didacticiels** à nouveau pour ouvrir le modèle d'un tutoriel.
