==== Error de instalación 1601 - Rhino 6 ====

Error del servicio de instalación (-2147023295)

> **Resumen:** //Cómo resolver el error del instalación 1601//

=====Problema=====
Varios problemas pueden causar el error 1601. Siga todos los pasos que se describen a continuación.

=====Respuesta=====
**Paso 1: Ejecute Windows Update**
  - Localice y ejecute el [[http://windowsupdate.microsoft.com/|servicio de Windows Update]].
  - Si hay actualizaciones importantes para instalar, haga clic en **Actualizar ahora** para instalarlas.  Este proceso puede tardar un poco.
  - Si se produce un error al ejecutar el instalador de Windows Update, lea el artículo [[https://support.microsoft.com/es-es/help/10164/fix-windows-update-errors|Reparar los problemas de Windows Update]].
  - Si soluciona el problema de Windows Update y termina con el [[windowsupdate.microsoft.com|servicio de Windows Update]], siga con el paso 2]].
\\
**Paso 2: Instale .NET 4.6 Framework**
  - Descargue y ejecute el [[https://www.microsoft.com/download/details.aspx?id=49981|instalador de .NET 4.6 Framework]].
  - Si el instalador Microsoft.NET muestra un error: 
    - Anote el error 
    - Continúe con el paso **¿Todavía no funciona?**, más abajo.

\\
**Paso 3: Instale Rhino**
  - Abra el **Explorador de Windows**.
  - Descargue el instalador más reciente de Rhino 6 aquí: [[http://www.rhino3d.com/download/rhino/6/latest|descarga completa de Rhino 6]]
  - Ejecute el archivo descargado para iniciar el proceso de instalación.
  - Si el instalador falla, guarde una copia del archivo de registro en el escritorio.
\\
**¿Todavía no funciona?**

Lo sentimos. Envíe los archivos de registro desde el paso 3 a tech.eu@mcneel.com.
