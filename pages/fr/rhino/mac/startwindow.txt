====== Fenêtre de départ de Rhino pour Mac ======

Lorsque Rhino pour Mac démarre, la fenêtre de démarrage affiche les derniers modèles ouverts sur le côté droit. 

Le dernier modèle est déjà sélectionné et il suffit d'appuyer sur Entrée pour l'ouvrir. Vous pouvez sélectionner un modèle différent en appuyant sur les flèches. Vous pouvez aussi taper les premières lettres d'un modèle pour le sélectionner à partir de son nom. Lorsque le modèle désiré est sélectionné, appuyez sur Entrée pour l'ouvrir. 

Cette même fenêtre permet aussi d'accéder aux fichiers modèles. Cliquez sur le bouton //Fichiers modèles// en bas à droite de la fenêtre pour afficher la liste des fichiers modèle. Vous pouvez aussi appuyer sur Commande-T pour passer en mode fichiers modèles. Le bouton //Derniers modèles// ou la combinaison Commande-R permet de revenir à la liste des derniers fichiers. 

Créez vos propres fichiers modèles dans Rhino avec le menu Fichier > Enregistrer sous un fichier modèle...  Vos fichiers modèles seront affichés en haut de la liste. Vous pouvez choisir un fichier modèle comme fichier par défaut en cliquant avec le bouton droit sur un modèle et en sélectionnant //Choisir "XXX" comme modèle par défaut// dans le menu contextuel. Vos fichiers modèles personnalisés disposent également du choix de menu //Afficher dans le Finder//.  Cette option permet de localiser le fichier modèle et modifier son contenu, de changer son nom ou de le supprimer. 

Le bouton //Ouvrir un autre...// (Commande-O) de cette fenêtre lance la boîte de dialogue Ouvrir un fichier. Le bouton //Nouveau modèle// (Commande-N) crée une nouvelle fenêtre de modélisation en utilisant le fichier modèle par défaut actuel comme fichier de démarrage.
