====== Python-Skripting in Mac Rhino ======
Mac Rhino unterstützt ein Test-Plug-in für Python-Skripting in Rhino. Zum Hinzufügen dieser Unterstützung:

  - Installieren Sie die [[http://www.rhino3d.com/download/wenatchee/5.0/wip|neueste Version von Rhino Mac]]
  - Führen Sie diese Installation aus. \\ Dadurch werden MACRHI-Dateien mit Rhino verknüpft. Wenn Sie nun doppelt auf eine MACRHI-Datei klicken, wird Rhino gestartet.
  - [[http://download.rhino3d.com/IronPython/5.0/wip/download/|Download des IronPython-Plug-ins]]. \\ Dies ist ein Mac-Rhino-Installer des Plug-ins. 
  - Doppelklicken Sie auf die Datei, um das Plug-in zu installieren. \\ Mac Rhino wird gestartet und das Plug-in installiert.
  - Schließen Sie Rhino und starten Sie es neu. \\ Ihnen sollte nun der Befehl **RunPythonScript** zur Verfügung stehen.

==== Befehl RunPythonScript ====
Es gibt zwei Versionen dieses Befehls:

**RunPythonScript** - Öffnet einen Datei-öffnen-Dialog, in dem Sie ein Python-Skript (PY-Datei) wählen können. Das Plug-in beinhaltet einige Beispiel-Skripts zur Ausführung, deren Ordner beim ersten Öffnen des Dialogs angezeigt werden sollte.

**-RunPythonScript** - Bei Eingabe des Befehls mit Bindestrich können Sie einen Dateinamen in der Befehlszeile eingeben. Wenn das Skript im Skriptordner Ihres Computers vorhanden ist, wird es ausgeführt. Durch Eingabe eines Punktes können Sie ein Skript in einem Unterverzeichnis des Skriptordners ausführen. Wenn Sie beispielsweise die Datei KurvenLänge.py im Unterverzeichnis "Beispiele" ausführen möchten, geben Sie einfach in der Befehlszeile "beispiele.kurvenlänge" ein.

**Hinweis**: Alle Beispiele, die diesem Plug-in beigelegt sind, sollten sowohl unter [[http://mac.rhino3d.com|Mac Rhino WIP]] als auch [[http://www.rhino3d.com/|Windows Rhino 5]] laufen.
