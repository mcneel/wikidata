===== Décomprimer un fichier EXE de Rhino 4 et l'installer manuellement =====

Certains systèmes ne prennent pas en charge le format de fichier png que nous utilisons dans l'écran d’installation de Rhino 4 et ne peuvent pas montrer correctement les options d'installation. Si c'est votre cas, vous pouvez essayer la solution suivante :

  - Téléchargez et installez l'application [[http://www.7-zip.org/|7-Zip]] sur votre système.
  - Cliquez avec le bouton de droite sur le fichier d'installation de Rhino 4.
  - Dans le menu, sélectionnez "**7-Zip**" et "**Extraire les fichiers...**".
  - Sélectionnez l'emplacement où extraire le contenu et notez cet emplacement.
  - Naviguez jusqu'au dossier indiqué et rechercher le sous-dossier **Rhinoceros**.
  - Cherchez **Rhino4_Setup.msi**. Double cliquez sur le fichier pour le lancer et commencer l'installation de Rhino 4. 

Le même procédé est décrit pour la **version 5** [[http://wiki.mcneel.com/fr/rhino/rhino5/manual_extract_from_file|ici]].
