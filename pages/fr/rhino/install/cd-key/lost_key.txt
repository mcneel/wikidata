====== Je ne retrouve pas clé de licence de Rhino ======

La clé de licence de Rhino est très importante pour déterminer à qui appartient une licence de Rhino.  Conservez les clés de licence dans un lieu sûr. 

Si vous ne retrouvez pas votre clé de licence, voici quelques indices où chercher. 

Si vous possédez Rhino version 1, 2 ou 3, le CD était envoyé dans une pochette collée sur la couverture intérieure du manuel de l'utilisateur. Le manuel de l'utilisateur était un livre d'environ 2,5 cm d'épaisseur avec une couverture brillante noire. La version 4 de Rhino était envoyée directement dans un boîtier de DVD avec une étiquette noire. 

Si vous ne trouvez pas le boîtier DVD contenant le numéro, nous pourrons peut-être vous aider si vous avez enregistré le produit. Si vous avez sauté l'étape de l'enregistrement, nous n'aurons aucune information. Les numéros de série des produits n'étaient pas suivis lors de l'envoi, votre numéro de commande ne nous aidera donc pas. 

Pour information, nous ne pouvons pas envoyer de CD de remplacement pour les versions 1 et 2 de Rhino car le noyau de programmation était protégé par une licence tierce dont le contrat est maintenant terminé. 

Si vous avez épuisé toutes les options de recherche du DVD, il existe des options de remplacement pour la version commerciale.  Contactez votre bureau régional pour plus d'informations :
