====== Windows XP에서 Zoo 실행 ======
> **제품:** //[[ko:zoo:home|Zoo]]//
> **요약:** //Windows XP에서 Zoo 5 사용 방법.//

===== 문제 =====
Windows XP에서 Zoo를 실행하려고 합니다. 컴퓨터에서 Zoo를 시작하면 갑자기 Zoo가 멈추고 아무런 반응이 없습니다. Windows XP에서 Zoo 5가 지원됩니까?

===== 해결책 =====
  - Zoo 5는 [[http://support.microsoft.com/kb/322389|Microsoft Windows XP Service Pack 3]]에서 지원됩니다. 
  - Zoo 5에는 [[http://www.microsoft.com/ko-kr/download/details.aspx?id=17718|Microsoft .NET Framework 4]]가 필요합니다.
