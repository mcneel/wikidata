====== Pourquoi ne puis-je pas cliquer avec le bouton de droite pour répéter la dernière commande ? (Rhino 5 pour Mac) ======
> **Résumé:** //Il est possible d'utiliser le bouton droit pour répéter la dernière commande. Il suffit de modifier quelques paramètres.//

=====Problème=====
Contrairement à la version Windows, Rhino pour Mac ne répète pas la dernière commande en cliquant avec le bouton droit. 

=====Réponse=====
Vous pouvez changer cela en désactivant le menu contextuel, ce qui activera la répétition.

  - Dans le menu, cliquez sur **Rhinoceros** puis sur **Préférences**.
  - Sélectionnez le panneau **Souris**.
  - Dans la section **Bouton droit de la souris**, désactivez la case **Activer les menus contextuels**.
