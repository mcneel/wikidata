======Utilisation des licences avec le Cloud Zoo======

Cloud Zoo est le nouveau mode de licence pour Rhino 6. Consultez cette page pour un aperçu des [[fr:rhino:6:license|modes de licence pour Rhino 6]]

=====Configuration des licences personnelles sur le Cloud Zoo=====

  - [[https://accounts.rhino3d.com/help#create_account|Créez un compte Rhino]]
  - [[https://accounts.rhino3d.com/help#licenses|Ajoutez une licence dans votre Cloud Zoo]]
  - Connectez-vous à votre compte Rhino au démarrage de Rhino


=====Équipes du Cloud Zoo pour les bureaux et les écoles=====
Le Cloud Zoo permet de créer des équipes. Les équipes sont des groupes de personnes qui peuvent utiliser un ensemble de licences dans le Zoo.

**Configuration d'équipes sur le Cloud Zoo**

  - [[https://accounts.rhino3d.com/help#create_account|Créez un compte Rhino]]
  - [[https://accounts.rhino3d.com/help#create_group|Créez une équipe dans votre Cloud Zoo]]
  - [[https://accounts.rhino3d.com/help#licenses|Ajoutez une ou plusieurs licences dans votre équipe]]
  - Invitez des membres à rejoindre votre équipe en utilisant une de ces deux méthodes :
      - [[https://accounts.rhino3d.com/help#invite_group_members|Invitez des membres individuellement]]
      - [[https://accounts.rhino3d.com/help#link_domain|Liez le domaine de votre organisme avec une équipe]]*

===Action des membres de l'équipe===

  - Les membres de l'équipe acceptent l'invitation
  - Les membres de l'équipe peuvent utiliser Rhino en se connectant au compte Rhino au démarrage


*Équipes liées à un domaine

Pour une administration plus facile des grandes équipes, des comptes liés à travers un domaine peuvent être définis. Toute personne possédant une adresse e-mail correspondant à ce domaine pourra rejoindre automatiquement votre équipe et aura accès à vos licences - aucune vérification par e-mail n'est nécessaire. Les équipes liées au domaine sont compatibles avec les fournisseurs d'authentification personnalisés (Google, Active Directory ou One Login) afin de garder centralisé votre gestion d’utilisateurs. 

===Informations supplémentaires sur le Cloud Zoo===

  - Vous pouvez passer d'un mode de licence à un autre mais ce n'est pas une opération automatique. Pour plus d'informations, consultez l'article : [[fr:rhino:6:change-license|Changer de licence dans Rhino 6 pour utiliser le Cloud Zoo]].
  - Vous trouverez ici les réponses aux [[fr:rhino:faq:licensing|Questions souvent posées]] sur les licences.
