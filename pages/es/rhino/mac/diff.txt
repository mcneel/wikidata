====== ¿Viene de Rhino para Windows? ======

¿Viene de Rhino para Windows?  ¿Se plantea dar el salto a OS X?  ¿O solamente tiene curiosidad?  Aquí tienen una guía breve para hacerse a la idea...


----

===== ¿Dónde están las barras de herramientas? =====
De manera predeterminada, Rhino para Mac no considera que se viene de Rhino para Windows.  Si desea ver las barras de herramientas al estilo de Windows, vaya a **Rhinoceros** > **Preferencias** > **Temas** y seleccione **Rhino 5 para Windows**.  Deberá usar una nueva ventana de modelado para que los cambios surtan efecto.  Para obtener más información sobre el tema de Rhino para Mac, consulte la siguiente sección...

----

===== Barras de herramientas y paletas de herramientas ===== 
  * Rhino para Mac tiene //botones de herramientas// que ejecutan los comandos de Rhino en la barra de herramientas estándar del Mac.
  * Rhino para Mac también tiene //paletas de herramientas//, que son ventanas que contienen un conjunto de botones de herramientas. \\ Una paleta de herramientas de botones de herramientas frecuentes se abre a la izquierda de la ventana del documento cuando se inicia Rhino para Mac.
  * Los botones de las paletas de herramientas pueden tener uno o dos comandos y un menú.
  * Pase el ratón sobre un botón de la paleta de herramientas para mostrar el tooltip del botón. \\ Si el tooltip tiene dos líneas de texto, el botón contiene un comando principal y un comando secundario.
    * **Haga clic** en el botón de la paleta de herramientas para ejecutar el comando **principal**.
    * Para ejecutar el comando **secundario**, mantenga pulsada la tecla **Opción** y **haga clic** en el botón de la herramienta.
  * Los botones de la paleta de herramientas que contienen un menú tienen un pequeño triángulo en la esquina inferior izquierda.
    * Para abrir el menú, haga clic y mantenga pulsado brevemente el botón de la paleta de herramientas.
  * Los menús de la paleta de herramientas también pueden mostrarse como paletas de herramientas flotantes.
    * Para mostrar una paleta de herramientas flotante, mantenga pulsada la tecla **Opción** y haga clic y mantenga pulsado el botón del ratón.

===== Ratón ===== 

**Recomendaciones**

  * Para facilitar la navegación con el botón derecho del ratón, se recomienda usar un ratón de dos botones con rueda de desplazamiento, aunque todas las operaciones se pueden realizar usando un ratón de un solo botón.
  * Para duplicar las acciones del botón derecho del ratón con un ratón de un solo botón, mantenga pulsada la tecla **Control** y haga clic en el botón del ratón.

**Navegación**

  * Las operaciones de **arrastre con el botón derecho del ratón** y **arrastre con el botón derecho del ratón+Mayús** funcionan igual que en Rhino para Windows. Consulte la [[[http://docs.mcneel.com/rhino/mac/help/en-us/user_interface/shortcuts.htm|Ayuda de Rhino - Accesos directos]]. \\ Independientemente del estilo de proyección de la vista, pulse **Mayús** para encuadrar, pulse **Comando** para el zoom y pulse **Comando+Mayús** para rotar la vista.
  * Utilice el rueda central para ampliar y reducir el plano en una vista.
  * Si tiene un ratón de un botón, pulse las teclas **Control**, **Comando** y **Mayús** para rotar una vista.

----

===== Menú contextual mediante clic con el botón derecho ===== 

  * Haga clic con el botón derecho del ratón en una vista para ver el menú contextual.
  * De manera predeterminada, un clic con el botón derecho **no** actúa como un **Intro** para repetir el último comando. \\ Para cambiarlo, en **Rhinoceros** > **Preferencias** > **Ratón** > **Ratón**, desactive la opción **Activar menús contextuales**.

----

===== Línea de comandos ===== 

Si está acostimbrado a usar la línea de comandos en Rhino para Windows, toda la funcionalidad de la línea de comandos de Rhino para Windows también está disponible en Rhino para Mac.  Pa ejecutar un comando, empiece a escribir. El texto aparecerá en el campo de comandos de la barra lateral izquierda.  Puede pulsar Intro o la barra espaciadora para ejecutar el comando.  Aparecerá un diálogo de opciones de comando con las mismas opciones de comando que las que aparecen en Rhino para Windows. \\ \\

Si también quiere introducir los valores de las opciones del comando, cada opción tiene unas letras subrayadas. Puede escribir la letra y pulsar Intro para seleccionar esa opción. 

No es necesario mover el cursor cuando se introducn los valores en el diálogo de opciones de comando.  Escriba y Rhino introducirá los valores en el diálogo automáticamente.  El funcionamiento de la línea de comandos en Rhino para Windows es exactamente igual en Rhino para Mac.

----

===== Teclado =====

  * Rhino para Windows utiliza la tecla **Control** para muchas operaciones. En la versión para Mac, se utiliza la tecla **Comando** en su lugar.
    * Por ejemplo, **Comando** + clic reemplaza **Ctrl** + clic para cancelar la selección de objetos.
    * Además, la tecla **Comando** reemplaza la tecla **Ctrl** para los accesos directos de navegación:

| | | |**Tecla** | **Acción** | **+ tecla Comando** |
| | | |**Flecha izquierda** | Rotar hacia la izquierda | Encuadrar hacia la izquierda |
| | | |**Flecha derecha** | Rotar hacia la derecha | Encuadrar hacia la derecha |
| | | |**Flecha arriba** | Rotar hacia arriba | Encuadrar hacia arriba |
| | | |**Flecha abajo** | Rotar hacia abajo | Encuadrar hacia abajo |

----

===== Problemas con equipos portátiles ===== 


**Ratón Bluetooth**\\ Puesto que los equipos portátiles solo tienen un trackpad, es útil añador un ratón Bluetooth como, por ejemplo, el Logitech V270.

**Faltan teclas del teclado**\\ Un teclado de un portátil no tiene las teclas **Home**, **End**, **PageDn** o **PageUp** separadas, pero la tecla **fn** en combinación con otras teclas duplica la función de las mismas. \\

| **Tecla** | **+ tecla fn** |
| **Flecha arriba** | Page Up |
| **Flecha abajo** | Page Down |
| **Flecha izquierda | Home |
| **Flecha derecha | End |
| **Eliminar** | Delete |
