=====Historique=====
Le module PanelingTools permet de générer des motifs de cellules 2D et 3D et de les propager sur des grilles rectangulaires.

Il permet également le dessin intuitif de concepts de pannellisation ainsi que la rationalisation d'une géométrie complexe dans un format adapté à l'analyse et à la fabrication. Le module est étroitement intégré dans Rhino 4 et largement utilisé dans l'architecture et autres métiers de la construction. 

=====Télécharger=====

===Pour Rhino 5 sur Windows 32 et 64 bits===
  * Téléchargez **[[http://download.rhino3d.com/Rhino/5.0/PanelingToolsV5/| le programme d'installation de PanelingTools]] et double-cliquez sur le fichier.** Une fois installé, PanelingTools se chargera à l'ouverture de Rhino (pas besoin de déplacer le module dans Rhino ni utiliser le gestionnaire de modules). Les barres d'outils sont aussi installées mais vous devez les charger la première fois avec l'outil de Configuration des barres d'outils. 
  * PanelingTools pour Grasshopper est sorti. **[[http://www.grasshopper3d.com/group/panelingtools|Cliquez pour plus d'informations...]]**

===Pour Rhino Mac WIP===
  * Le module PanelingTools fait partie de **[[http://www.irhino3d.com/|Rhino pour Mac WIP...]]**. Le menu PanelingTools n'est pas implanté car les modifications de l'interface ne fonctionnent pas encore sur la version Mac, mais toutes les commandes sont disponibles. Tapez "pt" pour voir toutes les commandes de PanelingTools dans la liste de saisie semi-automatique. Donnez-nous votre avis !
