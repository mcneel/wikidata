======Convertir un nodo de red en un nodo independiente======
> **Producto:** //[[zoo:zoo5|Zoo 5]]//
> **Resumen:** //Cómo convertir una instalación de Rhino 5 de un nodo de red en un nodo independiente.//

=====Pregunta=====
He instalado por error una de mis licencias de Rhino 5 como nodo de red, cuando en realidad quería instalarla como nodo independiente. Cuando inicio Rhino y voy a la página **Licencias** en el diálogo de **Opciones**, el botón **Convertir** está desactivado. He iniciado sesión como administrador y he vuelto a intentarlo, pero el botón sigue desactivado. ¿Cómo puedo cambiar esta instalación a un nodo independiente?


> **Nota:** //Estas instrucciones también pueden usarse para cambiar de un nodo independiente a una licencia flotante invirtiendo los datos introducidos.//
=====Respuesta=====
Para convertir una instalación de Rhino 5 de un nodo de red en un nodo independiente, siga estos pasos:

====Windows Vista, Windows 7 o Windows 8====

1. Inicie sesión en la estación de trabajo como administrador o como usuario con derechos de administrador.

2. Ejecute el Explorador de Windows.

3. Abra la carpeta siguiente:
  C:\ProgramData\McNeel\Rhinoceros\5.0\License Manager\Licenses

> **Nota:** //De manera predeterminada, Microsoft oculta la carpeta ProgramData. También puede usar las **Opciones de carpeta** para permitir la visualización de los archivos y carpetas ocultos, o puede escribir **%ProgramData%** en la barra de direcciones del Explorador de Windows para ir a la carpeta.//

4. Elimine el archivo siguiente (si se encuentra):
  60515f84-8f7f-41da-801d-1c87e32f88f5.lic
  
5. Abra la carpeta siguiente:
  C:\Users\<user_name>\AppData\Roaming\McNeel\Rhinoceros\5.0\License Manager\Licenses

> **Nota:** //De manera predeterminada, Microsoft oculta la carpeta AppData. También puede usar las **Opciones de carpeta** para permitir la visualización de los archivos y carpetas ocultos, o puede escribir **%AppData%** en la barra de direcciones del Explorador de Windows para ir a la carpeta.//

6. Elimine el siguiente archivo (si se encuentra):
  60515f84-8f7f-41da-801d-1c87e32f88f5.lic

7. Haga clic con el botón derecho del ratón en el icono de Rhino 5 y seleccione **Ejecutar como administrador**.

8. Introduzca y valide el código de CD de Rhino 5.
====Windows XP====

1. Inicie sesión en la estación de trabajo como administrador o como usuario con derechos de administrador.

2. Ejecute el Explorador de Windows.

3. Abra la carpeta siguiente:
  C:\Documents and Settings\All Users\Application Data\McNeel\Rhinoceros\5.0\License Manager\Licenses

4. Elimine el siguiente archivo (si se encuentra):
  60515f84-8f7f-41da-801d-1c87e32f88f5.lic

5. Abra la carpeta siguiente:
  C:\Documents and Settings\<nombre_de_usuario>\Application Data\McNeel\Rhinoceros\5.0\License Manager\Licenses

6. Elimine el siguiente archivo (si se encuentra):
  60515f84-8f7f-41da-801d-1c87e32f88f5.lic
  
7. Ejecute Rhino 5 haciendo doble clic en el acceso directo de Rhino 5.

8. Introduzca y valide el código de CD de Rhino 5.
