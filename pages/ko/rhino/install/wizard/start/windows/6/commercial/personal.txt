====== Rhino 6를 여러 대의 컴퓨터에 설치하기 ======

>안내: 귀하의 단일 사용자 상업용 라이선스를 여러 대의 컴퓨터 중 한 번에 한 대에서 실행할 때 이 방식을 사용하세요. 이 라이선스 배포 유형을 사용하시려면 가끔씩 인터넷 연결과 Rhino 계정 로그인이 필요합니다. 귀하의 Rhino 계정 자격 증명을 타인과 공유하지 마십시오.

  - Rhino 설치 관리자 패키지를 **[[https://www.rhino3d.com/download/Rhino/6.0/release|다운로드]]**합니다.
    - 귀하의 이메일 주소를 입력하고 "다음"을 클릭합니다.
    - "한국어 지금 다운로드"를 클릭하거나 원하시는 다른 언어를 선택하세요.
  - **설치 관리자 실행**합니다. 로컬 관리자 권한이 필요합니다.
    - 사용 약관과 개인 정보 취급 방침에 동의합니다.
      - 링크를 클릭하여 읽습니다. - 권장 사항입니다.
    - "지금 설치"를 클릭합니다.
      - 선택 사항 - 고급 옵션을 보려면 톱니 바퀴 모양의 아이콘을 클릭합니다.
    - 설치가 완료되면 "닫기"를 클릭합니다.
  - **라이선스 설치 배포하기**
    - Rhino를 시작합니다.
      - Rhinoceros 6 라이선스가 필요합니다:
        - {{:rhino:install:wizard:start:windows:6:commercial:v6_license.png?400|}}
        - "**로그인**" 을 선택합니다. 
          - 웹 브라우저가 열리고 귀하의 Rhino 계정 로그인 화면이 표시됩니다.
          - 귀하의 로그인 자격 증명을 제공합니다. 
            - 아직 Rhino 계정이 없다면 새로운 계정을 만듭니다. 
          - 귀하의 라이선스를 **개인용** 라이선스에 추가합니다. 
          - 로그인을 완료하면 브라우저 창을 닫습니다. 
        - Rhino가 시작합니다.

  - **다른 컴퓨터에서 Rhino를 시작합니다.**
    - Rhino를 시작합니다.
      - Rhinoceros 6 라이선스가 필요합니다:
        - "로그인" 을 선택합니다.
        - 귀하의 Rhino 계정 로그인 자격 증명을 제공합니다.
        - Rhino가 시작합니다.

=== Rhino 설치에 실패하셨습니까? ===
  * Rhino 설치 관리자가 실패할 경우, 오류에 대한 설명과 해결 방법이 안내되어 있는 대화상자가 표시됩니다.
  * [[https://wiki.mcneel.com/ko/rhino/6/installproblems|MSI 오류 코드]]가 대화상자에 표시되었나요?
  * 필요하다면, 오류 로그를 열어 파일에 저장하고 McNeel 기술 지원 <tech@mcneel.com> 에게 이메일로 보내주시기 바랍니다.
