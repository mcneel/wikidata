===== Rhino 4 sur Windows 10 =====

Rhino 4 a été conçu pour s'exécuter sur Windows XP et ne fonctionne pas sur Windows Vista et toute version supérieure.

Si vous essayez d'exécuter Rhino 4 sur une version plus récente de Windows, vous rencontrerez ces problèmes :


==== Limitations et configurations requise ====
  * **Mémoire :** - Votre ordinateur 64 bits est doté de 16Go de mémoire RAM, voire plus. Rhino 4 est une application 32 bits et ne peut accéder qu'à 4 Go de RAM. Rhino 4 ne prend pas en charge les très grands modèles comme peuvent le faire les nouvelles version de Rhino. 
  * **Contrôle de compte d'utilisateur** - Rhino 4 est sorti avant que Microsoft n'ajoute la fonction de **Contrôle des comptes d'utilisateur** dans Windows. Rhino 4 doit être exécuté en **mode de compatibilité Windows XP SP3.** De cette façon, Rhino peut enregistrer à sa fermeture les modifications réalisées sur les barres d'outils et les paramètres. Les paramètres du mode de compatibilité Windows font partie des propriétés du raccourci Bureau.
  * **Basse résolution :** Votre ordinateur est probablement compatible avec les écrans haute résolution. Vous devez configurer Windows pour qu'il s'exécute avec une résolution écran plus faible (comme par exemple 1024 x 768). Si vous ne modifiez pas la résolution, certaines parties de l'interface de Rhino - comme le **panneau des calques** - seront vides et les icônes des barres d'outils de Rhino 4 seront trop petits pour être distingués correctement.
