======Mehrere IP-Adressen======
> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //Anleitung zum Konfigurieren des LAN Zoo auf Systemen mit mehreren IP-Adressen.//

=====Frage=====
Ist es für den Zoo möglich, einen anderen Port zu verwenden als TCP Port 80?

=====Antwort=====
Nein, für den LAN Zoo besteht keine Möglichkeit, nach Verbindungen auf einem sonstigen TCP- oder UDP-Port zu lauschen.

Andererseits kann der LAN Zoo den Zoo Service an eine bestimmte IP-Adresse binden. 

{{:zoo:zoo_ip.png|Zoo Options}}

Das Feld **An IP-Adresse binden** ermöglicht es Ihnen den Zoo-Server so einzustellen, dass er nur auf Verbindungen von einer bestimmten Netzwerk-Interface lauscht.  

Wenn Sie zum Beispiel zwei Netzwerkkarten in Ihrem Computer haben, dann könnten Sie von beiden Benutzeroberflächen aus auf den Computer zugreifen.  Mit dieser Option können Sie vom Dropdown-Menü aus die IP-Adresse der entsprechenden Netzwerkkarte auswählen, damit der Zoo nur über diese Adresse erreicht werden kann.

Wenn Ihr Server also mit mehreren IP-Adressen konfiguriert ist, ist es möglich den Zoo-Dienst auf einer IP-Adresse auszuführen und einen weiteren Dienst, wie z.B. IIS, auf einer anderen.
