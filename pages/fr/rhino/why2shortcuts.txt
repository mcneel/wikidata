======Pourquoi deux raccourcis ?======
**Vous avez terminé d'installer Rhino 5 sur votre ordinateur Windows 64 bits et vous avez deux raccourcis sur le bureau.**

Le programme d'installation de la version 5 installe Rhino 32 bits et Rhino 64 bits automatiquement. Chaque version possède des limites et des possibilités propres. 

**Rhino V5 32 bits :**\\
  * **Avantage** - Rhino V5 32 bits peut faire tourner des modules conçus pour la version 5 et la version 4 de Rhino. 
  * **Inconvénient** - Les applications 32 bits sont limitées à utiliser un total de 4Go de mémoire, partagée entre Windows et l'application.

**Rhino V5 64 bits :**\\
  * **Avantage** - Rhino V5 64 bits n'est pas limité à 4Go de mémoire. Il peut utiliser toute la mémoire installée sur votre ordinateur. 
  * **Inconvénient** - La version 5 64 bits ne peut pas faire tourner des modules conçus pour la version 4. Le module doit être conçu spécifiquement pour la version 5 64 bits. 

**Ai-je besoin des deux versions ?**\\
  * Peut-être, ou pas.
  * Si vous ne savez pas, laissez les deux
  * Si l'espace sur le disque est limité, comme sur un disque SSD, vous pouvez désinstaller celle que vous n'utilisez pas.
    * Il est possible de désinstaller une seule version de Rhino, 32 ou 64 bits.
    * **Ne désinstallez pas les éléments "Rhinoceros 5 Help Media" ou "Rhinoceros 5 Language Pack Installer**
