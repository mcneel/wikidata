====== LAN Zoo ne peut pas être exécuté sur Windows XP ======
> **Produit :** //[[fr:zoo:home|Zoo]]//
> **Résumé:** //Utilisation de LAN Zoo sur Windows XP//

===== Question=====
J'essaie d'utiliser le LAN Zoo sur Windows XP. Dès que je lance le Zoo sur cette machine, il se bloque et rien ne se passe. N'est-il pas possible d'utiliser le LAN Zoo sur Windows XP ?

===== Réponse=====
  - LAN Zoo a besoin de Microsoft .NET Framework 4.8.
  - Le .NET Framework 4.8 et les versions suivantes ne peut __PAS__ être installées sur Windows XP.
