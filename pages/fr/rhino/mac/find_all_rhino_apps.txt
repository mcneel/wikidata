====== Comment trouver toutes les copies de Rhino pour Mac sur mon ordinateur ? ======

**Réponse** : Lancez la commande ''TestFindAllRhinoApps'' dans Rhino pour Mac...et lisez les instructions.

===== Problème =====

macOS simplifie la gestion et l'exécution de plusieurs versions d'une seule application. La meilleure solution consiste à conserver toutes vos applications dans le dossier ''/Applications'', mais, étant donné que les applications macOS sont livrées sous forme de Packs d'application - vous pouvez facilement renommer ou déplacer ces packs.  Parfois, afin de faire des tests, vous pouvez avoir besoin de renommer ou déplacer ces applications. Si vous utilisez Rhino pour Mac, en particulier avec RhinoWIP pour Mac, ou si vous testez différentes versions de Rhino pour Mac, vous pouvez ne plus retrouver toutes ces applications et ne plus savoir lesquelles sont obsolètes. Cela peut entraîner des problèmes dans certains cas. 

===== Solution =====

Rhino pour Mac possède une commande cachée que vous pouvez utiliser pour trouver toutes les instances de l'application Rhino se trouvant sur votre ordinateur et vous permettre de décider quoi en faire : ''TestFindAllRhinoApps''. 

{{:fr:rhino:mac:testfindallrhinoapps.png|}}

Vous pouvez facilement naviguer parmi ces applications installées en les sélectionnant et en cliquant sur le bouton "Trouver dans le Finder. Si vous en voyez une qui //ne devrait normalement pas être installée//, sélectionnez-la et cliquez sur "Déplacer dans la corbeille."
