====== QuadMesh ======
> **Summary:** //Plugin to mesh triangles into quads//

=====Description=====

This plugin for Rhino4 will allow you to make quads out of triangle meshes.

----

\\

\\

[[http://download.rhino3d.com/download.asp?id=QUADMESH&language=|Click here to download plugin]]

=====Installation instructions=====

  - Copy QuadMesh.rhp to your "[[rhino:rhinoceros|RhinoCeros]] 4.0\Plug-ins" directory.
  - Start Rhino.
  - Go to the Tools - Options - Plug-ins page.
  - Click on the Install button.
  - Browse to the folder where you copied the file in step 1 above.
  - Select QuadMesh.rhp and click the Open button.
  - Click the OK button to close the Rhino Options page.

===== Use Instructions=====

  - Run QuadMesh
