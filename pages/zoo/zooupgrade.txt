====== Upgrading a LAN Zoo Server ======
> **Product:** //[[zoo:home|Zoo]]//
> **Summary:** //Upgrading a Zoo server.//

To support Rhino 7, you need to either upgrade to **LAN Zoo 7** or migrate to the **[[https://wiki.mcneel.com/rhino_accounts/home|Cloud Zoo]]**. 


===== Download and Install =====
  * [[http://www.rhino3d.com/download/zoo/7/latest|Download the LAN Zoo 7 here...]]

  * Install in the same location as your current LAN Zoo


===== FYI =====
  * [[zoo:zoodotnet|Microsoft .NET Framework 4.5 is required]].

  * Zoo 7 replaces your older Zoo.

  * Your license will be automatically be moved to Zoo 7. 

  * Users running Rhino at the time of installation will not be affected.

  * The installation takes about 15 seconds. During that time, users will not be able to get a license from the Zoo.
