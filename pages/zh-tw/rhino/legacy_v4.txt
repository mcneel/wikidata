
===== 在 Windows 10 或 Windows 11 執行 Rhino V4 =====

Rhino V4 設計用來在 Windows XP 下執行，但也可以在新版 Windows 中最初設計的那樣運作，以下是在較新版本 Windows 下運作的 Rhino V4 限制與要求。

==== 限制與要求 ====
  * **記憶體** - 新的64位元電腦系統可能配備16GB或更高的RAM。Rhino V4 是一款32位元的應用程式。這意味著無論新系統有多少RAM，都只會用到最多4GB的RAM。 
  * **舊有應用系統** -  Microsoft 在Windows加入**使用者帳戶控制**之前發佈Rhino V4。要解決這個問題，您需要在**Windows XP SP3兼容模式**中執行Rhino V4，否則Windows將阻止Rhino關閉時保留對工具列及其他設定的更改。 Windows兼容模式設定在"桌面捷徑"屬性的內容裡。
  * **低解析度** - 您的新電腦能夠執行的螢幕解析度，可能比舊windows XP系統高出很多。您需要限制Windows並以較低的解析度(如：1024X768)來執行，這點在Rhino V4發佈時是受到支援的，要不然像**圖層面板**這樣的工具就會呈現空白，而且因為尺寸太小無法閱讀。

==== 考慮嘗試Rhino V8 ====
除了Rhino V8[[https://www.rhino3d.com/features|的新功能與特性之外]] ，要充分利用新的64位元Windows系統的功能。\\
