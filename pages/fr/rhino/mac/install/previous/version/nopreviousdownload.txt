====== Puis-je télécharger une version précédente de Rhino pour Mac ? ======
> **Résumé:** //La version actuelle de Rhino pour Mac a un problème. Est-il possible de télécharger la version précédente ?//

Non, les versions précédentes de Rhino pour Mac ne sont plus disponibles. 

Si vous rencontrez un bogue important dans la version actuelle, merci de laisser un message sur le [[http://discourse.mcneel.com/c/rhino-for-mac|forum Discourse dans la catégorie Rhino for Mac]].
