==== Resetting Rhino's Preferences ====
Follow this procedure to remove all of your changed Rhino preferences and reset all of them to the original values.

  - Quit Rhino.
  - In the Finder, hold down the Option key and select Library from the Go menu.  The Library folder appears.
  - Select the Preferences folder inside the Library folder.
  - Find the file com.mcneel.rhinoceros.plist and drag it to the Trash.  Alternatively, you could drag the file to your Desktop if you want to restore these preferences in the future.
