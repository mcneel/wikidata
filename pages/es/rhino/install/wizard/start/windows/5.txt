====== Opciones de licencia de Rhino 5======

===Elija una:===
    * **[[rhino:install:wizard:start:windows:5:commercial|Licencias Comercial monopuesto y Evaluación de 90 días]]**
      * Permite un usuario por licencia
      * Puede usarse como licencia independiente o en una red (excepto la versión Evaluación) utilizando el [[https://wiki.mcneel.com/zoo/home|Zoo]] para Windows, el gestor de licencias flotantes gratuito
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Contrato de licencia de usuario final]]

    * **[[rhino:install:wizard:start:windows:5:educational|Licencia Educacional monopuesto]]**
      * Permite un usuario por licencia
      * Solo instalación independiente
        * El usuario debe ser el propietario del equipo
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Contrato de licencia de usuario final]]

    * **[[rhino:install:wizard:start:windows:5:lab|Licencia Lab EDU para 30 usuarios]]**
      * Permite 30 usuarios al mismo tiempo
      * Puede usarse en una red de centro docente utilizando el [[https://wiki.mcneel.com/zoo/home|Zoo]] para Windows, el gestor de licencias flotantes gratuito
      * Instalación de Rhino con [[https://wiki.mcneel.com/rhino/installingrhino/5.0|fácil implementación]]
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Contrato de licencia de usuario final]]
