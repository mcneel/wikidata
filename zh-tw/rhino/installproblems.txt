-->在新電腦上安裝最新更新版本#
**//問：//** //我買了一台新電腦並且想安裝Rhino版本**X** SR **Y**。我有購買升級版本的Rhino，我需要安裝所有的舊版及修正版嗎？ //

**//答：//**  您只要安裝您最近版本的安裝光碟。在安裝時，將會檢查是否會有舊版，您只要將光碟片放入檢查，並不需要實際安裝。當您安裝後，檢查並升級到最新的修正版即可。(Rhino 5則是會要求您輸入舊版序號)\\

<--

-->什麼是修正版？#

修正版是針對現存版本進行錯誤修正，維護性更新，並且為免費提供。這些安裝程式可經由網路下載、並可安裝在已安裝程式的電腦上安裝。修正版可由兩種方式確認版本，一是SR編號，例如：SR3 或 SR4b，另一為日期，可由下載檔案名稱上看到例如：20050729，則代表是西元2005年7月29日版本。
<--

-->如何知道我目前安裝的版本為？#
您可由程式中的說明>關於查看您的軟體程式版次，會出現Rhino開啟的畫面，您可以由”關於”了解目前的版次。 [[rhino:rhinoceros|Rhinoceros]].
\\
若要查Flamingo的版次，請在指令列輸入 FlamingoAbout 指令，Bongo則輸入BongoAbout 指令。

<--

-->如何取得修正版？#

您可以在Rhino的說明>檢查是否有新版本來快速了解是否有新的修正版。若您的電腦有網路連線，McNeel的網路伺服器將會提供您最新版的下載連結。若您使用Flamingo，請在Raytrace或Photometric下找檢查是否有新版本。若您使用Bongo，請到Bongo>找檢查是否有更新。
\\
在Rhino 5，請到**工具>選項>更新與統計**並設定更新頻率為"修正版" 或者是您可以嘗試改到"修正版候選版"。 當您關閉Rhino，您將會收到安裝最新修正版的提示，您可以選擇是否要立即安裝。

<--

-->我可以在新增移除程式這邊直接移除較舊的版本且不會有任何問題？#
您在系統控制台的新增移除程式可以看到所有已安裝的修正版本，您可以發現每個更新版本皆使用約250MB的硬碟空間，這部分有些誤導，微軟系統回報所有已安裝的檔案被添加到現有的安裝檔案中。請**不要**任意自行移除較舊版本的Rhino 5修正版本。

-->Rhino 5.0安裝不完整，缺少語言包#
您是否缺少了範本檔、工具列或外掛程式？這些是語言包沒有安裝的徵狀，這可能是某些安裝過程中產生的錯誤或不完整安裝。

若您在安裝過程中缺少了些語言包，您可以由Rhino 5安裝光碟中找群Installer資料匣，您將需要手動自行完成安裝。

若要確認是否有安裝好語言支援給外掛及介面，您可以找尋下列路徑：
C:\Users\<login>\AppData\Roaming\McNeel\Rhinoceros\5.0. 

若資料匣不見貨您已經開啟Rhino，您將會發現沒有工具列、範本檔及說明檔。
請依照下列步驟修復：
     - 執行光碟中的繁體中文版的LuanguaPack_zh-tw.msi檔案。
     - 開啟Rhino並輸入**ToolbarReset**。
     - 關閉Rhino並重新開啟。這將會將遺漏的元件都重新設定。

-->Error 1327 - Invalid Drive:#
**//問：//** //安裝時有一個錯誤訊息為 "Error 1327. Invalid Drive (E:, F:, etc)"的錯誤 //

**//產生原因：//** //您的使用者帳戶設定有幾個虛擬槽用來指定為特定資料匣，例如”我的文件” //

**//解決方式：//**
  - 微軟的開始鍵
  - 輸入 RegEdit 輸入enter
  - 尋找下列路徑HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders
  - 由安裝檔案尋找錯誤提示錯誤的資料槽，變更至有效的槽或者資料匣即可。  - 尋找下列路徑 HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders
  - Look for directories listed that contain the drive letter mentioned in the error from 

**注意：** 這個變更登錄資料的解決方式，若操作不正卻很可能會造成您電腦嚴重的問題。因此，請您務必小心執行這些步驟。此外，您可以在修改之前，先備份登錄檔。如果發生問題，您可以還原回復。關於備份及還原登錄檔的更多訊息，您可以參考下列微軟技術支援文章：
[[http://support.microsoft.com/kb/322756|如何在 Windows XP 中備份及還原登錄]]

-->Error 1402 or 1406 (Could not open key: UNKNOWN\Components)#
** //問題：//** //若您看到錯誤訊息 "Could not open key: UNKNOWN\Components\(lots of letters and numbers).Error 1402. 確認您有足夠權限，請聯絡您的資訊網管人員"//

**//造成原因//** 一個或多個註冊碼無法寫入 HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Installer\UserData\S-1-5-<numbers>\Components沒有足夠權限。

**//解決方案1：//**
在微軟的開始表，在搜尋欄輸入"CMD"。在"cmd.exe"上按右鍵然後選擇"以使用者管理員權限執行"。複製下列內容並複製至cmd，按Enter。然後Rhino將可安裝。

<code>
secedit /configure /cfg %windir%\inf\defltbase.inf /db defltbase.sdb /verbose
</code>

**//解決方案2：//**
//警告：下列方式將編輯微軟機碼，若有錯誤可能會造成電腦無法開機，若您擔心有相關問題產生，請直接聯絡McNeel技術人員服務。//
  - 由微軟的TechNet下載 [[http://technet.microsoft.com/en-us/sysinternals/bb897553|psexec]]並解壓縮放在電腦桌面。
  - 開啟命令提示字元(Windows command shell)
  - 將路徑改到您的桌面
  - 在指令列輸入： **psexec -i -d -s c:\windows\regedit.exe**這會導致 Windows 運行登錄編輯程式使用系統帳戶將有比帳戶管理員更強大的許可權。
  - 瀏覽HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Installer\UserData\
  - 在**UserData**資料匣上按右鍵，然後點選**Permissions...**
  - 在Permissions對話框選擇 **Advanced**
  - 確認在這個資料匣"系統管理者身分(Administrators)"有"完全控制" 權限，並且設定使用者有可讀的權限。
  - 選擇 **Replace all child object permissions with inheritable permissions from this object**
  - 點選兩次 **OK**以套用新的權限設定。
  - 再次安裝Rhino。

-->安裝程式不支援Windows 7/8？#

**問：**\\ 我目前使用Windows 7/8，但我在使用Rhino安裝光碟安裝時系統卻提示不支援？

**答：**\\ 
  - 在setup.exe檔案上按右鍵，選擇**內容**。
