======Convertire un nodo di gruppo in un nodo standalone======
> **Prodotto:** //[[zoo:zoo|Zoo]]//
> **Sommario:** //Spiega come convertire un'installazione di Rhino 5 da nodo di gruppo a nodo standalone.//

=====Domanda=====
Per sbaglio ho installato una delle mie licenze di Rhino 5 come nodo di gruppo, anziché come nodo standalone. Quando lancio Rhino e vado nella scheda **Licenze** della finestra di dialogo **Opzioni**, il pulsante **Converti** è disattivato. Ho eseguito l'accesso come amministratore e riprovato, ma il pulsante continua ad apparire disattivato. Come faccio a cambiare questa installazione in un nodo standalone?


> **Nota:** //Queste istruzioni si possono usare anche per passare da standalone a libera.//
=====Risposta=====
Per convertire un'installazione di Rhino 5 da nodo di gruppo a nodo standalone, seguire i seguenti passi:

====Windows Vista, Windows 7 o Windows 8====

1. Accedere alla stazione di lavoro come Amministratore oppure come utente con diritti di amministrazione.

2. Eseguire Windows Explorer.

3. Accedere alla seguente cartella:
  C:\ProgramData\McNeel\Rhinoceros\5.0\License Manager\Licenses

> **Nota:** //Di default, Microsoft nasconde la cartella ProgramData. Per consentire la visualizzazione dei file nascosti e delle cartelle nascoste, usare le **Opzioni delle cartelle** oppure digitare **%ProgramData%** nella barra degli indirizzi di Windows Explorer per accedere a questa cartella.//

4. Eliminare il seguente file (se trovato):
  60515f84-8f7f-41da-801d-1c87e32f88f5.lic
  
5. Accedere alla seguente cartella:
  C:\Utenti\<nome_utente>\AppData\Roaming\McNeel\Rhinoceros\5.0\License Manager\Licenses

> **Nota:** //Di default, Microsoft nasconde la cartella AppData. Per consentire la visualizzazione dei file nascosti e delle cartelle nascoste, usare le **Opzioni delle cartelle** oppure digitare **%AppData%** nella barra degli indirizzi di Windows Explorer per accedere a questa cartella.//

6. Eliminare il seguente file (se trovato):
  60515f84-8f7f-41da-801d-1c87e32f88f5.lic

7. Cliccare con il tasto destro sull'icona di Rhino 5 e selezionare **Esegui come amministratore**.

8. Immettere e convalidare il codice di licenza di Rhino 5.
====Windows XP====

1. Accedere alla stazione di lavoro come Amministratore oppure come utente con diritti di amministrazione.

2. Eseguire Windows Explorer.

3. Accedere alla seguente cartella:
  C:\Documents and Settings\Tutti gli utenti\Dati applicazioni\McNeel\Rhinoceros\5.0\License Manager\Licenses

4. Eliminare il seguente file (se trovato):
  60515f84-8f7f-41da-801d-1c87e32f88f5.lic

5. Accedere alla seguente cartella:
  C:\Documents and Settings\<nome_utente>\Application Data\McNeel\Rhinoceros\5.0\License Manager\Licenses

6. Eliminare il seguente file (se trovato):
  60515f84-8f7f-41da-801d-1c87e32f88f5.lic
  
7. Eseguire Rhino 5 con un doppio clic sull'icona di collegamento a Rhino 5.

8. Immettere e convalidare il codice di licenza di Rhino 5.
