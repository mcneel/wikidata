===== Supprimer une licence du Zoo ou d’un ordinateur =====

Pour ajouter une licence en cours d’utilisation à votre compte Rhino, vous devez tout d’abord la supprimer du Zoo ou de l’appareil sur lequel elle est utilisée.


==== Retirer une licence du Zoo ====
  - **Lancez l’application Zoo Admin**.
  - **Sélectionnez la licence**.
  - Cliquez sur le bouton **Delete**.

==== Retirer une licence d’un ordinateur ====
  - Connectez votre ordinateur à Internet
  - **Lancez Rhino**
    - Windows : Dans le menu **Outils**, cliquez sur **Options**
    - Mac : Dans le menu **Rhinoceros** (pour la V6), **Rhino 7** ou **Rhino 8**, cliquez sur **Préférences**
  - Cliquez sur **Licences**
  - Cliquez sur **Changer de clé de licence**
  - Cliquez sur **Options**
  - Cliquez sur **Retirer la licence**

==== Ajouter la licence sur le Cloud Zoo ====
  - Visitez la page [[https://www.rhino3d.com/licenses/?controller=add_licenses|Ajouter des licences]].
  - Ajoutez la licence à votre compte ou à l’une des équipes à laquelle vous appartenez.
