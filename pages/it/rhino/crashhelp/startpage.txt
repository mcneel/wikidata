======Blocco dopo aver fatto clic su un file .3dm di Rhino====== 

\\

**Problema:**

Rhino si blocca quando faccio doppio clic su un file .3dm di Rhino.


**Prova:**

Avvia Rhino da un **tasto di scelta rapida** o dal manu **Avvia**.

**Se Rhino si avvia:**
  * Apri un file dal menu **File** > **Apri** o dal pulsante della barra degli strumenti **Apri**.\\ 
  * Se il file non si apre, probabilmente il problema è dovuto al file. Prova il comando **VerificaFile3dm**.

**Si è risolto il problema?**
  * **Sì**. 
  * **No**. Consulta //[[rhino:crashhelp:fail_shortcut|Blocco all'avvio di Rhino]]//.
