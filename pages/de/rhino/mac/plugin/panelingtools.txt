====== Kann ich die PanelingTools verwenden? (Rhino 5 für Mac) ======
> **Zusammenfassung:** //Gibt es das Plug-in PanelingTools auch für Rhino 5 für Mac?//

Ja. Die PanelingTools sind **kein** Bestandteil der veröffentlichten Version von Rhino 5 für Mac, aber in der [[http://www.rhino3d.com/go/download/rhino-for-mac/wip/latest|WIP-Version von Rhino für Mac]] verfügbar. Wenn Sie eine Lizenz für Rhino für Mac besitzen, können Sie die WIP-Version von Rhino für Mac kostenlos herunterladen und installieren, um so Zugriff auf die PanelingTools zu erhalten. Bitte beachten Sie, dass Rhino für Mac WIP eine Work-in-Progress-Version ist. Dies bedeutet, dass sie zwar die neuesten Funktionen enthält, aber unter Umständen auch instabil sein bzw. Fehler enthalten kann. Neue WIPs werden regelmäßig veröffentlicht, wenn neue Funktionen hinzukommen oder Fehler ausgebessert werden. 

Hinweis: In der WIP-Version gibt es kein PanelingTools-Menü. Die Befehle sind vielmehr alle über die Befehlszeile verfügbar. Um alle PanelingTools-Befehle anzuzeigen, geben Sie in der AutoVervollständigen-Liste "pt" ein.
