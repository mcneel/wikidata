======Usare lo Zoo 5 con un server proxy======
> **Prodotto:** //[[zoo:zoo5|Zoo 5]]//
> **Sommario:** //Come aggirare un server proxy per accedere ad un server dello Zoo 5 locale.//

=====Informazioni generali=====
Molte aziende oggi usano proxy per proteggere i loro utenti da eventuali intrusioni e per usare un numero minore di indirizzi IP instradati pubblicamente. Un proxy è un server che si interpone tra l'utente ed Internet. Esso riceve tutte le richieste del client ad Internet, le soddisfa e quindi manda le informazioni al client.

=====Aggirare un server proxy=====
In alcuni casi, è preferibile aggirare il proxy ed accedere direttamente ad un sito o ad una risorsa. Per esempio, una risorsa disponibile localmente, come un server dello Zoo 5, che è disponibile tramite un accesso diretto alla rete.

Windows supporta l'aggiunta e la configurazione di proxy dalle impostazioni di Internet Explorer.

Si può attivare l'opzione **Ignora server proxy per indirizzi locali** per aggirare gli indirizzi locali, seguendo i seguenti passi:

  - Fare clic su Start e quindi su Pannello di controllo.
  - Fare clic su Rete e Internet e quindi su Opzioni Internet.
  - Nella finestra di dialogo Opzioni di Internet, cliccare sulla scheda Connessioni.
  - Fare clic sul pulsante Impostazioni LAN.
  - Selezionare la casella "Ignora server proxy per indirizzi locali".
  - Fare clic su OK per completare il processo di configurazione del proxy.
