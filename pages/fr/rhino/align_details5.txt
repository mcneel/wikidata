Aligner les vues détaillées sur une mise en page

Les vues détaillés sont les fenêtres qui composent la mise en page. Vous pouvez définir un grand nombre de vues détaillées montrant des parties de votre modèle avec différentes vues, échelles, couleurs et largeurs d'impression. 

Lorsque les vues détaillées sont crées et leur échelle définie, il peut être nécessaire de les aligner sur la mise en page. 
Deux options sont possibles. 
  * La première utilise la commande **Déplacer** pour déplacer les vues détaillées à partir d'un point sur le modèle vers une ligne de référence sur la mise en page. 
  * La deuxième utilise le **Repérage intelligent** pour changer la position de la vue détaillée en alignant les vues détaillées dans le modèle sur une ligne de référence générée à partir de plusieurs points dans d'autres vue détaillées.  

Il existe également deux méthodes permettant d'ajuster la taille des vues détaillées utilisées dans ces vidéos. 
  * Vous verrez comment modifier la taille des vues détaillées avec les points de contrôle.
  * Et vous verrez le manipulateur réalisant une échelle uniforme sur la taille générale des vues détaillées. Remarque : pour modifier l'échelle uniformément avec le manipulateur, appuyez sur la touche Maj et déplacez le marqueur de l'échelle sur le manipulateur.
