======Flamingo의 설치======
%#003366%**Rhino 2 에서 Rhino 3 으로 업그레이드했습니다. 제가 갖고 있는 Flamingo 1.0 CD 로 설치가 안 됩니다. Rhino 3 에서 어떻게 하면 Flamingo 가 실행됩니까?**

  * Rhino 3.0 용 Flamingo 1.1 이 필요합니다. 갖고 계신 **Rhino 3.0 설치 CD**를 보시면 Flamingo 폴더에 Flamingo 1.1 설치 파일이 있습니다. 이 설치 파일을 실행한 후. 사용자의 원래 Flamingo 1.0 CD 키를 입력하시면 됩니다. Flamingo 1.1 은 Flamingo 1.0 과 완전히 동일하며, Flamingo 1.0 소유자들에게 무료로 제공됩니다. www.flamingo3d.com/download.htm 에서 Flamingo 의 서비스 릴리스를 꼭 확인하세요.

%#003366%**Rhino 4로 업그레이드했습니다.  제가 갖고 있는 Flamingo 1.1 CD 로 설치가 안 됩니다. Rhino 4 에서 어떻게 하면 Flamingo 가 실행됩니까?**

  * Rhino 4.0 DVD 케이스 안에 들어 있는 빨간색의 **Rhino 4.0 CD2**로 Flamingo 를 설치하실 수 있습니다. 또는 Flamingo 1.1 SR6 또는 그 이후의 새로운 CD 를 사용하시면 됩니다.

> **Exception:** //갖고 계신 빨간색 Rhino CD2의 표면에 **RH40PRT-CD2 0107** 또는  **RH40PRT-CD2 0207** 라고 찍혀 있는 경우, Flamingo 1.1 설치관리자에서 Flamingo 1.0 (Rhino 2 용) 또는 1.1 (Rhino 3 용) 이 이미 설치되어 있는지 확인하게 됩니다. 이러한 빨간색 CD2 의 초기 버전을 갖고 있고, Flamingo 예전 버전이 설치되어 있지 않다면, 새롭게 업데이트된 빨간색 CD2 를 McNeel 기술지원 담당자 - **<tech@mcneel.com>** 에게 요청하시기 바랍니다.//

  * [[http://download.mcneel.com/whatsnew.asp?product=flamingo&version=2.0&release=beta|%#006699%Flamingo 2.0 Beta 다운로드]]

%#003366%**Vista 사용자입니다. Vista 에서 Flamingo 1.1 도움말을 볼 수 없습니다.**

%#003366%**Flamingo 1.1 재질과 플랜트 라이브러리는 어디에 설치되어 있나요?**
  * C:\Program Files\Rhinoceros 3.0\Plug-ins\Flamingo\English\Libraries

%#003366%**Flamingo 2.0 WIP/Beta 의 라이브러리는 어디에 설치되어 있나요?**
  * C:\Documents and Settings\All Users\Application Data\McNeel\Flamingo\2.0\English\Libraries

> **Note:** //"Application Data" 폴더를 보려면 "숨겨진 파일과 폴더"가 표시되도록 설정해야 합니다. 설정은 Windows 탐색기 (Explorer) - 도구 - 폴더 옵션 - 보기 탭의 고급 설정에서 변경할 수 있습니다.//

%#003366%**사용자가 지정한 재질 정의는 어떻게 관리하나요?**
  * [[Flamingo_Custom_Materials|%#006699%사용자가 지정한 Flamingo 재질의 관리법(영어)]]
%#003366%**점프 드라이브 또는 이동식 디스크에서 재질 라이브러리를 사용할 수 있나요?**
  * [[http://wiki.mcneel.com/_media/legacy/en/Flamingo_User_Libraries3.pdf|%#006699%이동식 미디어에서 Flamingo 1.1 재질 라이브러리를 사용하는 방법 - PDF 파일(영어)]]
    * 많은 Flamingo 사용자들께서 이동식 디스크에 라이브러리를 보관하는 방법에 대해 질문해 주셨습니다. 학생들의 경우, 작업 중에 컴퓨터가 갑자기 멈추거나, 재설정 등의 이유로 라이브러리를 잃게 되는 일을 방지할 수 있는 방법을 자주 문의하기도 합니다. 때로는 하드 드라이브에 남겨둔 라이브러리를 다른 사용자가 수정하는 일도 발생할 수 있습니다. 여기에 소개된 방법으로 이동식 플래시 드라이브나 네트워크 드라이브에 사용자 라이브러리를 보관할 수 있습니다. 단, 이 방법은 Flamingo 2 에서는 적용되지 않습니다.

%#003366%**Flamingo 2 에서 어떻게 재질과 플랜트 라이브러리를 찾을 수 있나요?**
  * Flamingo 2.0 에서 설치되는 재질과 플랜트 라이브러리는 모든 사용자가 사용/수정할 수 있는 특정한 위치에 배치됩니다. Flamingo 의 브라우저의 Material 탭에 있는 기본 라이브러리는 다음 위치에 저장됩니다.
    * **Windows Vista** -  C:\ProgramData\McNeel\Flamingo\2.0\<language>\<libraryfolder>
    * **Windows XP** - C:\Documents and Settings\All Users\Application Data\McNeel\Flamingo\2.0\<language>\<libraryfolder>
  * Flamingo 2.0 을 시작하면 다음 위치에서 라이브러리 목록을 읽어옵니다:
    * **System registry** - HKEY_LOCAL_MACHINE\SOFTWARE\McNeel\Flamingo\2.0\<localeID>    valuename=Libraries
  * 라이브러리를 다른 위치로 옮기면, Tools > Options > Flamingo (Rhino 에서 옵션 Options. (문서 속성 Document properties 이 아닙니다)) 에서 표시되는 "Library locations (show in browsers)" 에 새 위치를 추가해야 합니다.
  * 해당 대화 상자의 상단 부분에 있는 BaseFolder 설정은 비트맵 등을 찾는 데 사용됩니다. 기본적으로 설치하게 되면, Base Folder 는 다음 위치를 말합니다:
    * **Windows Vista** - **C:\ProgramData\McNeel\Flamingo\2.0\Resources**
    * **Windows XP** - C:\Documents and Settings\All Users\Application Data\McNeel\Flamingo\2.0\Resources
    * **System registry** - HKEY_LOCAL_MACHINE\SOFTWARE\McNeel\Flamingo\2.0\Paths    valuename=Support


  * 세 가지 힌트: Library search 는 1 단계 깊게 실행되고, Support search 는 재귀적이며, hidden libraries 도 사용 가능합니다.
    * Library search 폴더의 MLIB 파일은 Flamingo 브라우저의 **Materials tab**에 표시됩니다.
    * Support search 폴더의 내부 및 하위에 있는 MLIB 파일은 Flamingo 브라우저의 Materials 탭에 표시되지 않지만, 누락된 재질 정의로 찾을 수 있습니다. 예를 들어, Flamingo 브라우저의 Material 탭에 영어로 된 재질 라이브러리만이 표시되는 경우에도, 독일어로 된 .3DM 파일에서 표준 독일어 재질로 렌더링이 이루어지게 됩니다.
