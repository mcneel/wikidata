====== Will Rhino 5 for Mac support my .NET plug-in? ======
> **Summary:** //.NET plug-ins that run on both Windows and Mac are possible and in development.//

We are working on tools that will let Rhino plug-in developers write plug-ins that will run on both OS X and Windows.  [[http://wiki.mcneel.com/rhino/mac/release/updates/service/release|We are planning on releasing .NET plugin support in our 5.1 update]].
