====== Importer des RUI et des configurations personnalisées dans Rhino 8 ======
{{ :rhino:home:rui_open.jpg?200|}}

==== 1. Importer des fichiers de menu de RUI ====
Si vous souhaitez importer une //RUI de Rhino 6 ou 7//, vous pouvez le faire ici dans Rhino 8 :

  * Ouvrez le menu **Fenêtre -> Barres d’outils... -> **, puis cliquez sur  **Fichier -> Ouvrir... **, pour ouvrir la RUI.

{{ :rhino:home:custom8-2.png?200|}}





==== 2. Raccourcis Alias ====
Les alias sont enregistrés dans leur propre fichier. Dans //Rhino 6 ou 7// vous pouvez aller dans **Options -> Alias -> Exporter** pour exporter un fichier ini.
Dans //Rhino 8//, allez dans **Options -> Alias -> Importer** pour importer ce fichier.



==== 3. Raccourcis clavier ====

Vous pouvez enregistrer des options du système de Rhino comme les raccourcis clavier, les paramètres des aides à la modélisation, les paramètres des fichiers et plus encore, dans //Rhino 6 ou 7// avec la commande **ExporterOptions**.

{{ :rhino:home:custom8-1.png?150|}}

Dans Rhino 6 ou 7, tapez **ExporterOptions** ou sélectionnez cette fonction dans le menu Outils, et exportez le fichier ini. 
Dans Rhino 8, tapez **ImporterOptions** ou sélectionnez cette fonction dans le menu Outils, et importez le fichier ini.

==== 4. Anciens fichiers de barres d'outils de Rhino 5 ====

Le format de fichier de barres d'outils .TB ne peut pas être importé directement dans Rhino 8.
Il s'agit d'un format ancien qui n'est plus pris en charge par Rhino 8 ou les versions plus récentes. 

Vous pouvez contourner ce problème avec Rhino 6 ou 7 pour Windows, en important la barre d'outils et en l'enregistrant en tant que fichier RUI.
Pour cela, allez dans Options -> Barres d'outils -> Fichier.
{{:rhino:home:rhino7_save_rui.png?400|}}

  * Ouvrez le menu **Fenêtre -> Barres d’outils... -> **, puis cliquez sur  **Fichier -> Ouvrir... **, pour ouvrir la RUI.

Si vous //n'avez pas// **les versions 6 ou 7 de Rhino pour Windows** et que vous avez Rhino 8, vous pouvez télécharger une **clé de licence pour une version précédente** [[https://www.rhino3d.com/support/|ici]] et installer Rhino 6 ou 7. Téléchargez Rhino à partir d'Archives [[https://www.rhino3d.com/support/|ici]].

Toutefois, si vous n'avez qu'un ou deux fichiers .TB que vous souhaitez convertir en fichiers .RUI, vous pouvez nous envoyer un e-mail à //tech@mcneel.com// en joignant vos fichiers.  Nous sommes là pour vous aider.
