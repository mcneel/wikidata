====== Fehler: Server ist ausgelastet ======
Dieser Fehler kann beim Öffnen oder Schließen jeglicher Version von Rhino für Windows auftreten.

{{ :rhino:server_busy.png?400|}}

Aus den wenig hilfreichen und umständlichen Windows-Dialogfenstern ist sehr schwierig zu deuten, worin genau der Fehler besteht.
Der Server im Dialog **//Server ausgelastet//** bezieht sich nicht auf unseren Lizenzserver.

Es handelt sich um einen ziemlich allgemein gefassten Windows-Fehlerdialog
Dies bezieht sich auf einen direkt auf Ihrem Computer laufenden **COM-Server**.

[[https://learn.microsoft.com/de-de/windows/win32/com/com-clients-and-servers|Hier]] erfahren Sie mehr über Microsoft-COM-Kunden und Server.

==== Empfehlungen zur Fehlerbehebung: ====
  - Führen Sie alle Microsoft-Aktualisierungen aus.
  - Laden Sie [[https://www.rhino3d.com/download/|hier]] die neueste Rhino-Version herunter und installieren Sie sie.
  - Starten Sie den Copmputer neu.
  - Öffnen Sie den Windows-**TaskManager** und beenden Sie den Vorgang für alle noch laufenden Rhino-Vorgänge.
  - Im Rahmen Ihrer Computerwartung führen Sie wenigstens einmal in der Woche einen Neustart aus.

In unserem [[https://discourse.mcneel.com/|Support-Forum]] können Sie sehen, wenn andere über diese Meldung berichten.
Hier gibt es einen Bericht, der sich auch auf das Enscape.Plug-in bezieht.
