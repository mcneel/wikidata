====== Installation des Microsoft .NET Framework 4.0 schlägt fehl (Fehlercode 5100) ======
> **Zusammenfassung:** //Behebung des Installationsfehlers 5100.//
> **Weitere Infos zu Installationsproblemen:** //[[http://wiki.mcneel.com/de/rhino/installproblems|Problembehandlung bei der Installation von Rhino 5]]//

**Problem**\\ 

Laut Microsoft hängt der Fehlercode 5100 mit den Systemanforderungen zusammen.


**Lösung 1**\\ 

Starten Sie den Copmputer neu. 

Wenn der vorherige Setup einen Neustart benötigt, wurde er vielleicht nicht korrekt abgeschlossen. Nach dem Neustart sollte das .NET Framework 4 korrekt installiert sein.

**Lösung 2**\\ 

Verfügt das Zielsystem über genügend freien Speicherplatz? Das .NET Framework 4.0 benötigt mindestens 2 GB freien Speicherplatz auf dem Datenträger. Geben Sie Speicherplatz auf dem Ziellaufwerk frei, damit die Installation abgeschlossen werden kann.
