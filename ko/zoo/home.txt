======Zoo - 라이선스 관리자======

=====개요=====
Zoo를 사용하시면 동일한 네트워크 작업그룹에 속한 사용자들이 Rhino, Flamingo, Penguin, Brazil, Bongo 라이선스 (CD 키) 를 공유할 수 있습니다. 이 제품들을 독립실행형 (stand-alone) 노드가 아닌, 작업그룹 노드로 설치하면 다음과 같이 실행됩니다: 

  * 작업그룹 노드가 시작되면 Zoo에게 라이선스를 요청합니다. 
  * 사용되지 않은 라이선스가 해당 노드로 배정됩니다. 
  * 노드가 종료되면 라이선스가 Zoo의 라이선스 풀(pool)로 반납됩니다. 

** Zoo는 무료입니다!**

**[[http://www.rhino3d.com/download/zoo/5.0/latest|Zoo 5 다운로드]]** - Rhino 5, Bongo 2.0 용 (//Brazil 2.0, Flamingo nXt, Penguin 2.0 은 [[ko::zoo::rhinolm5|Rhino 5 SR6 이후 버전]]//)을 사용하셔야 합니다.

**[[http://www.rhino3d.com/download/zoo/4.0/commercial|Zoo 4.0 다운로드 ]]** - Rhino 4, Flamingo, Penguin, Brazil, Bongo 1.0 용 (//Rhino 5 와 Bongo 2.0 은 지원되지 않습니다//).

=====기능=====
-->추가 비용이 없습니다#
Rhino, Flamingo, Penguin, Brazil, Bongo의 특별한 버전이 필요하지 않습니다. 
<--
-->특수 장비가 필요 없습니다#
Zoo는 같은 네트워크 워크그룹에 속한 어떤 시스템에서도 실행됩니다. 
<--
-->특별한 소프트웨어가 필요 없습니다#
Zoo는 Microsoft .Net Framework 4가 지원되는 모든 Windows 시스템 (Windows XP 를 비롯한 그 이후 버전) 에서 실행됩니다.
<--
-->간단한 설정#
작업그룹에 속한 컴퓨터에 Zoo를 설치하고 라이선스 키를 각 컴퓨터에서가 아닌, Zoo에 입력합니다. 
<--
-->체크아웃#
라이선스 키는 체크아웃이 가능하므로 노트북 컴퓨터 사용자가 네트워크에서 연결을 해제할 수 있습니다. 해당 노트북 컴퓨터가 네트워크에 다시 연결되면 키가 체크인됩니다. 
<--
-->안정성#
네트워크 연결이 끊어지거나 서버가 다운되더라도 작업그룹 노드는 계속해서 실행됩니다.
<--
-->라우팅이 가능합니다#
Zoo는 단일 세그먼트 네트워크 및 라우터로 시스템이 구분된 네트워크에서도 모두 실행됩니다. 
<--
-->혼재된 환경#
같은 Windows 도메인 또는 작업그룹에 작업그룹 노드와 독립 실행형 노드를 혼합하여 사용할 수 있습니다. 
<--
-->서비스로 실행됩니다#
Zoo가 있는 시스템을 다시 부팅하면 Zoo가 자동으로 시작합니다. 
<--

=====Zoo 5의  새 기능====
-->새 버전!#
Zoo 5는 Zoo의 새로운 버전입니다. 완전히 새롭게 쓰여진 Zoo 5로 기존 Zoo에는 없었던 몇 가지 기능을 사용해 보세요
<--
-->표준 인터넷 프로토콜 지원#
클라이언트는 HTTP (포트 80) 프로토콜을 사용하여 Zoo 5와 연결됩니다. 시스템 관리자는 이러한 점을 사용하여 복잡한 네트워크나 인터넷을 통해 쉽게 Zoo 요청을 라우팅할 수 있습니다.
<--
-->방화벽 친화성#
Zoo 5에서는 클라이언트가 서버를 주기적으로 체크하도록 되어 있으나, 클라이언트 쪽으로 다시 연결을 시도하지 않습니다. 관리자는 더 이상 클라이언트 컴퓨터 상의 방화벽 포트를 열어둘 필요가 없습니다.
<--
-->제한된 라이선스 체크아웃#
예전 Zoo 버전에서는 업무용 출장이나 노트북 컴퓨터 사용자를 위한 라이선스 체크아웃이 허용되었습니다. 단, 체크아웃이 허용되는 기간에는 제한이 없었습니다. Zoo 5에서는 관리자가 라이선스 체크아웃 기간을 지정할 수 있습니다.
<--
-->서드 파티 (3rd Party) 플러그인 지원#
예전 Zoo 버전에서는 McNeel 제품 (예: Rhino, Flamingo, Bongo, Brazil, Penguin) 의 라이선스만을 관리할 수 있었습니다. Zoo 5에서는 서드 파티 플러그인 개발자가 자신이 개발한 제품을 Zoo에 추가할 수 있습니다.
<--

=====시스템 요구 사항=====
  * Windows 기반 시스템.
  * [[http://www.microsoft.com/ko-kr/download/details.aspx?id=17718|Microsoft .NET Framework 4]] (Client Profile만이 아니라 전체 Full Framework)
  * TCP/IP 용으로 설정된 네트워크 연결.
