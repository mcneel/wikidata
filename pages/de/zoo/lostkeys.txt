======Wiederherstellen verlegter Lizenzen======
> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //Erklärung, wie Sie mit dem Zoo verloren gegangene Lizenzen wiederherstellen können.//

=====Frage=====
Wir möchten mit unserem Zoo auf einen anderen Server umziehen. Dazu müssen wir die Lizenzschlüssel auf dem neuen Server erneut eingeben. Leider finden wir nicht mehr alle unserer Rhino-Lizenzschlüssel. Gibt es eine Möglichkeit, meine Lizenzen einfach von einem Server auf einen anderen zu migrieren?

=====Antwort=====
[[de:zoo:movezoo|Der Umzug des Zoo-Lizenzmanagers]] von einem Server auf einen anderen ist im Grunde nichts anderes, als ihn neu zu installieren.

Es ist leider nicht möglich, Lizenzschlüssel vom McNeel-Produkten von einem Zoo-Server auf einen anderen zu "verschieben" oder zu "kopieren". Produkt-Lizenzschlüssel müssen zum neuen Zoo-Server genauso hinzugefügt werden wie bei der Erstinstallation.

**Hinweis:** Mit dem Zoo 5 Service Release 8 (SR8) oder neuer können Sie eine CSV-Datei exportieren, die die Seriennummern (nicht die Lizenzschlüssel) der mit diesem Zoo verwalteten Lizenzen enthält. Dies kann hilfreich sein, wenn Sie Ihre Lizenzschlüssel nicht mehr finden. Sie können dann die Seriennummer exportieren und per E-Mail an [[sales@mcneel.com|McNeel]] senden, wo wir Ihre verloren gegangene Lizenz suchen können.

Zum Exportieren der Seriennummern führen Sie den **Zoo Administrator** (ZooAdmin.exe) aus und wählen Sie **File -> Export**.
  
Die Versionsnummer Ihres Zoos können Sie im **Zoo Administrator** (ZooAdmin.exe) unter **Help -> About** ablesen.
