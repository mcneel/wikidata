====== Rhino 5 단일 사용자 상업용 라이선스 ======


===하나를 선택하세요:===
    * **[[https://wiki.mcneel.com/ko/rhino/5/macinstall|단일 사용자가 한 대 이상의 컴퓨터를 소유한 경우]]**
      * 사용자의 여러 개인용 컴퓨터에서 라이선스를 사용하려면 이 옵션을 사용하세요.
        * 한 번에 한 대의 컴퓨터에서 Rhino를 실행하는 것으로 제한됩니다.
      * 인터넷 연결이 필요하지 않습니다.
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[https://wiki.mcneel.com/ko/zoo/rhinomacdeployment|회사 LAN에서의 여러 사용자]]**
      * LAN을 통해 여러 사용자들이 라이선스를 공동으로 사용하는 경우 이 옵션을 사용하세요.
      * Windows **[[https://wiki.mcneel.com/ko/zoo/home|Zoo]]** 서버가 필요합니다.
        * 동시 사용할 수 있는 사용자의 수는 [[https://wiki.mcneel.com/ko/zoo/home|Zoo]]의 개별 라이선스 수로 제한됩니다.
      * 자동 [[https://wiki.mcneel.com/ko/zoo/rhinomacdeployment|설치 배포 ]] 방법 안내
      * 인터넷 연결이 필요하지 않습니다.
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|최종 사용자 사용권 계약 - End User License Agreement]]

>이 위키 페이지는 Rhino 설치 마법사의 일부입니다. [[ko:rhino:install:wizard:start |처음부터 시작하려면 여기를 클릭하세요]].
