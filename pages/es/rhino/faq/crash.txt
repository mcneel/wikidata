======Cómo enviar un informe de error de Rhino======

Si se produce un error en Rhino, envíe un informe de error para que se pueda incluir en los //problemas para solucionar//.

  *   ¿Puede repetir el error? Si es así, los desarrolladores necesitan los detalles específicos para poder reproducir el error. Un problema reproducible un buen modo de detectar y solucionar los problemas de software.
  *   Si hay suficiente memoria disponible cuando se produce el error en Rhino, se inicia el sistema de informe de errores. Introduzca su dirección de correo electrónico y describa qué estaba intentando hacer cuando Rhino se bloqueó. Puede resultar útil incluir una lista breve de los comandos que estaba usando antes del error.
  *   Si es posible, marque la casilla para enviar el archivo 3DM con el informe.
  *   Si incluye su dirección de correo electrónico y una descripción, **su informe será revisado por una persona**. Si no incluye su dirección de correo electrónico y la descripción, se utilizará un proceso automatizado para analizar el informe de error.
