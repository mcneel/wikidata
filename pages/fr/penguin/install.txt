** Installation de [[penguin:home|Penguin]] **
> **Résumé :** //Comment scripter l'installation de [[penguin:home|Penguin]].//

=====Installation=====

L'installation de [[penguin:home|Penguin]] avec des scripts est similaire à celle de Rhino et Flamingo.

Format de la ligne de commande que vous exécuterez normalement à partir d'un fichier batch (*.bat) :

   <chemin vers le fichier d'installation .exe> <paramètres>

Paramètres de la ligne de commande :

  * **/RhinoVer=[2|3|4]**    Définit la version de Rhino où installer Penguin. Ce paramètre est obligatoire.
  * **/CDKey=XXXXXXXXXXXX**    Utilisez le code CD à 12 caractères de [[penguin:home|Penguin]], sans tirets ni la partie "PE10". Ce paramètre est obligatoire sauf si vous installez [[penguin:home|Penguin]] en tant que nœud de groupe de travail. 
  * **/WorkgroupNode=[Y|N]**    N signifie que Penguin sera installé en autonome et vous devrez utiliser le paramètre CDKey, Y signifie que vous voulez que [[penguin:home|Penguin]] obtienne sa licence du Zoo. Ce paramètre est obligatoire.
  * **/Workspace=[Y|N]**    Indique si la barre d’outils de [[penguin:home|Penguin]] doit être ouverte ou non dans Rhino.
  * **/InstallPath=<chemin complet vers le dossier>**   Ignorez ce paramètre si vous voulez réaliser l'installation dans le dossier défini par défaut ...Program Files\Rhinoceros\Plug-ins\[[penguin:home|Penguin]]

=====Exemples=====

   "c:\mon dossier d'installation\penguin_en_20050726.exe" /RhinoVer=3 /CDKey=123456789ABC WorkgroupNode=N /Workspace=Y

   penguin_en_20050726.exe /RhinoVer=2 /WorkgroupNode=Y /Workspace=N /InstallPath="C:\Program Files\Renderers\Penguin"

=====Remarques=====

Certains utilisateurs ont rencontré des problèmes d'installation de [[penguin:home|Penguin]]  en suivant ces instructions. Ils ont pu les résoudre en ajoutant les options **/s /autoinstall**.
