====== Exportation de fichiers XAML ======
> **Sommaire:** //Module pour exporter des fichiers XAML//

=====Description=====

Ce module pour Rhino4 vous permet d'exporter des fichiers XAML.

----

\\

\\

[[http://download.rhino3d.com/download.asp?id=XAML_EXPORT&language=|Cliquez ici pour télécharger le module]]

=====Instructions d'installation=====

  - Copiez le fichier export_xaml.rhp dans votre dossier "Rhinoceros 4.0\Plug-ins".
  - Lancez Rhino.
  - Ouvrez la section Outils - Options - Modules.
  - Cliquez sur le bouton Installer.
  - Cherchez le dossier où vous avez copié le fichier lors de l'étape 1 ci-dessus.
  - Sélectionnez export_xaml.rhp et cliquez sur le bouton Ouvrir.
  - Cliquez sur le bouton Accepter pour fermer les Options de Rhino.

===== Instructions d'utilisation=====

  - Dans le menu Fichier, utilisez la commande Enregistrer sous ou Exporter.
  - Dans la case Type de fichiers, sélectionnez .xaml

Qu'est-ce que XAML ?  Je ne le savais pas non plus avant de commencer ce projet. Ce format ne sera pas très utile à la plupart des utilisateurs de Rhino, sauf si vous utilisez les modèles de Rhino pour développer des interfaces pour de nouvelles applications. Voici un lien intéressant qui vous donnera un aperçu du format XAML. http://msdn.microsoft.com/fr-fr/library/cc295302(en-us).aspx  Si vous utilisez Windows Vista vous pouvez voir ces fichiers graphiquement dans Internet Explorer (ils sont au format ascii, vous pouvez donc voir la structure ou les modifier dans Notepad ou votre éditeur de texte habituel). Si vous êtes sous XP, vous aurez besoin de SilverLight 2 de MS pour les voir dans IE.  Voici un lien.  http://www.microsoft.com/SILVERLIGHT/resources/install.aspx?v=2.0
