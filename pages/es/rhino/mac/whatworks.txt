====== ¿Qué funciona y qué no funciona en Rhino para Mac? ======

Esta no pretende ser una lista exhaustiva de todas las funciones disponibles o que faltan, ni una lista de errores pendientes. Simplemente se trata de ofrecer información general sobre el estado actual de Rhino para Mac.

==== ¿Qué funciona? ====

Puede abrir, crear, modificar y guardar documentos de Rhino, incluidos documentos de Rhino para Windows.

Puede tener más de un documento de Rhino abierto a la vez y trabajar en todos ellos simultáneamente.

Puede cortar, copiar y pegar, incluso entre diferentes documentos de Rhino.

Casi todos los comandos de Rhino funcionan.

La mayor parte del diálogo Propiedades de objeto está implementado. Puede encontrarlo en Editar/Propiedades de objeto...

Parte del diálogo Opciones está implementado, pero no igual que en Windows. Puesto que se puede tener más de un documento abierto al mismo tiempo, las Propiedades de documento se han dividido desde las Propiedades de la aplicación. Propiedades de aplicación se encuentra en Rhino/Preferencias. Propiedades de documento se encuentra en Archivo/Configuración. El diálogo Propiedades de documento existe y los paneles Rejilla y Malla funcionan.

Hay importadores de archivos .DWG, .PDF, .AI, .FBX, .OBJ, .STL e .IGES. Hay exportadores de archivos .DWG, .AI, .FBX, .OBJ, .SAT, .STL e .IGES.

==== ¿Qué no funciona? ====

En Rhino para Mac solo funicona el renderizador Toucan. No se implementarán otros renderizadores a corto plazo.

La vista de página no funciona.

==== Limitaciones de Rhino para Mac ====

Algunas funciones solo están implementadas parcialmente. Esta sección describe algunas de las limitaciones.

Hay exactamente cuatro vistas. Puede cambiar el tamaño de las vistas existentes y reducir el ancho a cero, pero no es posible hacerlas desaparecer.

La impresión no funciona correctamente. Se //puede// imprimir un documento con Rhino para Mac, pero las funciones de impresión son limitadas. Rhino para Mac imprimirá la vista seleccionada tal como se vea. Si aumenta la vista, la imagen impresa se verá más grande.

Como alternativa, utilice el renderizador integrado en Rhino para crear una imagen para imprimir.
