======Umwandlung einer Rhino-Einzelnutzerlizenz zur Verwendung mit dem Zoo======

> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //So wandeln Sie eine Installation von Rhino von einer Einzelplatzlizenz in eine Zoo-Lizenz um.// 

=====Frage=====
Ich habe den **Zoo** installiert und eine oder mehrere meiner Lizenzen hinzugefügt. Wie kann ich meine vorhandenen Rhino-Einzelplatzsysteme so **umwandeln**, dass ich Sie mit dem Zoo verwenden kann?


=====Antwort=====
Um eine Rhino-Einzelnutzerlizenz so umzuwandeln, dass sie mit dem Zoo verwendet werden kann, folgen Sie dieser Anleitung:

====Rhino 6 für Windows ====

  - Starten Sie **Rhino 6 für Windows**.
  - Klicken Sie auf das Menü **Werkzeuge** und dann auf das Menüelement **Optionen**.
  - Klicken Sie auf die Seite **Lizenzen**.
  - Wählen Sie die Rhino-Lizenz und klicken Sie auf **Lizenzschlüssel ändern**.

{{zoo:rhino6win_convert.png?640}}

====Rhino 5 für Windows ====

  - Starten Sie **Rhino 5 für Windows**.
  - Klicken Sie auf das Menü **Werkzeuge** und dann auf das Menüelement **Optionen**.
  - Klicken Sie auf die Seite **Lizenzen**.
  - Wählen Sie die Rhino-Lizenz und klicken Sie auf **Umwandeln**.

{{zoo:rhino5win_convert.png?640}}

====Rhino 5 für Mac====

  - Starten Sie **Rhino 5 für Mac**.
  - Klicken Sie auf das Menü **Rhinoceros** und dann auf das Menüelement **Optionen...**.
  - Klicken Sie auf die Option **Lizenzen**.
  - Wählen Sie die Rhino-Lizenz und klicken Sie auf **Lizenzschlüssel ändern**.

{{zoo:rhino5mac_convert.png?640}}
