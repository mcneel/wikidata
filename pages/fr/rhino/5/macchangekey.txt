====== Changer une clé de licence de Rhino pour Mac ======
Il existe deux raisons principales pour lesquelles vous pourriez vouloir changer de clé de licence de Rhino pour Mac :
  * Vous utilisez une licence d'évaluation de 90 jours et vous avez acheté une licence permanente
=== -ou- ===
  * Vous avez entré une clé de licence erronée et vous devez la changer. 

----
===== Remplacer une licence d'évaluation par une licence permanente =====
==== Méthode 1 ====
  * Lorsque vous lancez Rhino pour Mac vous voyez cette boîte de dialogue _
{{ :rhino:5:eval.png |}}
  * Cliquez sur le bouton **Changer de clé de licence**.
  * Entrez votre **Nom**, **Société** (optionnel), la **Clé de licence** et cliquez sur **Terminé**. 
{{ :rhino:5:name-org-key.jpg?500 |}}
Lorsque vous redémarrez Rhino, suivez les instructions pour **Valider votre licence**. 

----

==== Méthode 2 ====
**Remarque :** Utilisez cette méthode lorsque vous souhaitez changer une clé entrée par erreur. 
Lorsque vous êtes dans Rhino, cliquez sur le menu déroulant **Rhinoceros**. 
  * Cliquez sur **Préférences**.
  * Cliquez sur l'outil **Licences**.
{{ :rhino:5:rhinopref.png?500 |}}
  * Cliquez sur le bouton **Changer de clé de licence**.
{{ :rhino:5:changeeval.png |}}
  * Entrez votre **Nom**, **Société** (optionnel), la **Clé de licence** et cliquez sur **Terminé**.
{{ :rhino:5:name-org-key.jpg?500 |}}

Lorsque vous redémarrez Rhino, suivez les instructions pour **Valider votre licence**.
