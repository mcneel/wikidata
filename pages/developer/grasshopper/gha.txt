====== Grasshopper Components ======
[[http://wiki.mcneel.com/developer/grasshopper/gha|{{:developer:grasshopper:gh_assembly_extension.png?120|Grasshopper Components}}]]

=====Overview=====
For designers explore new shapes using generative algorithms, Grasshopper® is a graphical algorithm editor tightly integrated with Rhino’s 3-D modeling tools. Grasshopper requires no knowledge of programming or scripting, but still allows designers to build form generators from the simple to the awe-inspiring.

{{ :developer:grasshopper:grasshopperdefinition.jpg?direct&600 |}}

With the Grasshopper Component SDK, you can extend Grasshopper by creating custom components with direct access to Rhino and Grasshopper  structures, geometry, graphics system, and much more.

=====Requirements===== 

  - [[http://www.rhino3d.com|Rhinoceros 5]]
  - [[http://www.grasshopper3d.com|Grasshopper 1.0]]
  - Microsoft Visual Studio 2010 (C# or Visual Basic) or greater*

 //* Grasshopper 1.0 Components are .NET Framework 4 Assemblies//
