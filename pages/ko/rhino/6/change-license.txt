====Rhino 7 또는 6 버전에서 Cloud Zoo를 사용하도록 변경하기====

사용자의 컴퓨터에 설치된 독립실행형(Stand-alone) 라이선스를 Rhino Account 라이선스 인증 방식으로 변경하려면 다음 안내에 따라 모두 실행하세요:

**1. 독립실행형 라이선스 유효성을 반환하기:**
  - Rhino를 시작합니다.
  - 도구 메뉴 > 옵션 > 라이선스에서 “라이선스 변경”을 클릭합니다.
  - 대화상자의 왼쪽 아래에 있는 “라이선스 제거”를 클릭합니다.
    (라이선스가 Zoo 서버에 있다면 Zoo admin 콘솔에서 해당 라이선스를 선택하고 삭제 도구 아이콘을 누릅니다)

**2. 라이선스를 Cloud Zoo에 추가하기:**
  - “[사용자의 이름]으로 로그인”을 클릭합니다.
  - 개인용 라이선스가 없다는 메시지가 표시됩니다. 
  - “라이선스 추가”를 클릭합니다.
  - 인터넷 브라우저가 실행되고 안내가 적힌 웹페이지가 열립니다.
  - “라이선스 추가 페이지로 갑니다”를 클릭합니다.
  - Rhino 7 라이선스 키를 입력하고 “라이선스 추가”를 클릭합니다.

**3. Cloud Zoo를 사용하여 Rhino 시작하기:**
  - 라이선싱 대화상자에서 "다시 시도합니다"를 클릭합니다.
  - Cloud Zoo에 있는 사용자의 라이선스를 사용하여 Rhino가 시작합니다.

**다른 컴퓨터에서 Rhino 사용하기**
  - 다른 컴퓨터에서 Rhino를 시작합니다.
  - 사용자의 Rhino 계정에 로그인합니다.
