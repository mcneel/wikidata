======Fernüberwachung von LAN-Zoo-Lizenzen======
> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //Überwachung von LAN-Zoo-Lizenzen über einen Webbrowser.//

=====Frage=====
Gibt es eine Möglichkeit zur Fernüberwachung der Lizenzen in Echtzeit über ein anderes Gerät als den Server selbst? Wir würden Anwendern gerne die Möglichkeit geben, nachzusehen, wer gerade eine Lizenz verwendet, damit sie sich untereinander selbst organisieren können. Viele andere Anwendungen bieten eine Webseite oder eine andere Anwendung, in der der aktuelle Verwendungsstatus von Lizenzen angezeigt werden kann.

=====Antwort=====
Mit dem LAN Zoo können Anwender die Verwendung der Lizenz von ihrem Webbrowser aus nachverfolgen. Dazu öffnen Sie einfach die **Statusseite** Ihres LAN-Zoo-Servers, indem Sie

  http://<zoo_server_host_name>/status

in Ihren Browser eingeben. Wenn Ihr LAN-Zoo-Server also beispielsweise //zooserver.meinefirma.de// lautet, geben Sie ein:

  http://zooserver.meinefirma.de/status
  
Die Versionsnummer Ihres LAN-Zoos können Sie im **Zoo Administrator** (ZooAdmin.exe) unter **Help -> About** ablesen.
