====== Lo Zoo richiede Microsoft .NET Framework 4.5 ======
Windows Server 2008 può non avere il .NET Framework aggiornato a 4.5
  * Sarete in grado di installare lo Zoo 6, ma non vi sarà possibile avviare il servizio.
  * Windows Server 2012 o successivi dovrebbero già avere questo aggiornamento.
In questa [[https://msdn.microsoft.com/en-us/library/8z6watww(v=vs.110).aspx|pagina]], troverete la storia delle release di .net e le versioni di Windows.

> **Prodotto:** //[[it:zoo:home|Zoo]]//
> **Sommario:** //Lo Zoo e Microsoft .NET Framework 4.5//


Lo Zoo richiede **[[https://www.microsoft.com/en-us/download/details.aspx?id=30653|Microsoft .NET Framework 4.5]]**. Potete assicurarvi che Microsoft .NET Framework 4.5 sia installato nel vostro sistema usando [[http://windows.microsoft.com/en-us/windows/windows-update|Windows Update]].
