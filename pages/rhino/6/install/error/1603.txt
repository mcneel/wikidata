====Install Error 1603 - Rhino 6 & 7====

Fatal error during installation. (-2147023293)

> **Summary:** //How to fix the installation Error 1603//

=====Problem=====
A wide variety of problems can cause 1603 errors. Please follow all the steps below.

=====Answer=====
**Step 1: Perform Windows updates**
  - Locate and run the [[http://windowsupdate.microsoft.com/|Windows Update service]].
  - If there are any important updates to install, click **Update Now** to install the updates.  This process can take a long time.

\\
**Step 2: Install .NET 4.8 framework**
  - Download and run the [[https://dotnet.microsoft.com/download/dotnet-framework/net48|.NET 4.8 framework installer]].
  - If the Microsoft .NET installer shows an error: 
    - Write down the error 
    - Proceed to **Still not working?**, below.
\\
**Step 3: Repair the .NET 4.8 Framework**
  - Download and run the [[https://www.microsoft.com/en-us/download/details.aspx?id=30135|.Net Framework Repair Tool]]
  - If the Microsoft .NET Framework Repair Tool shows an error: 
    - Write down the error 
    - Proceed to **Still not working?**, below.

\\
**Step 4: Install Rhino**
  - Open **Windows Explorer**.
  - Download the most recent [[https://www.rhino3d.com/download/rhino/7/latest|Rhino 7]] or [[https://www.rhino3d.com/download/archive/rhino/6/latest|Rhino 6]] installer.
  - Run the file you downloaded to start the installation process.
  - If the installer fails, save a copy of the log file to your desktop.
\\
**Still not working?**

That's frustrating. We're sorry! Please email the log file from Step 3 to tech@mcneel.com
