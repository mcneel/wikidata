====== Rhino for Macの以前のバージョンはダウンロードできますか? ======
> **概要:** //現在のバージョンのRhino for Macに問題があります。以前のバージョンはダウンロードできますか?//

No, previous versions of Rhino for Mac are not currently available.  

If there is a major bug in the current stable release, please report it on [[http://discourse.mcneel.com/c/rhino-for-mac|Discourse in the Rhino for Mac category]].
