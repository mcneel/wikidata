====== Fehlerberichte für Mac Rhino ======


Klicken Sie [[http://discourse.mcneel.com/category/rhino-for-mac|hier]], um zum Mac-Forum zu gelangen. Sie können das Forum nach Ihrem Problem durchsuchen und einen eigenen Kommentar hinzufügen oder durch Klick auf **Create Topic** ein neues Thema erstellen.

Wenn Sie Ihrem Post eine Datei wie z.B. einen Screenshot hinzufügen möchten, klicken Sie oben im Fenster **Create a new topic** auf die Upload-Schaltfläche{{:rhino:mac:9-16-2013_2-11-02_pm.png?nolink|}}.

==== Senden einer großen Datei an uns ====

Wenn Sie eine kleine Datei haben, die uns bei der Fehlerbehebung hilfreich sein kann, können Sie sie einfach an Ihren Post im [[http://discourse.mcneel.com/category/rhino-for-mac|Discourse-Forum]] anhängen.\\

Falls Ihre Datei sehr groß ist:
  - Gehen Sie auf unsere [[http://www.rhino3d.com/upload|Webseite zum Upload einer Datei]]. 
  - Geben Sie Ihre Informationen ein und geben Sie **<mactech@mcneel.com>** als Empfängeradresse bei McNeel an. Versuchen Sie, Ihr Problem so detailliert wie möglich zu beschreiben.

==== Ein paar Regeln für einen guten Fehlerbericht ====
Wir möchten die Fehler beseitigen, die bei Ihnen auftreten, doch dazu müssen wir diese Fehler zuerst einmal reproduzieren können. Ein guter Fehlerbericht hilft uns dabei. Fügen Sie deshalb so viele Details wie möglich hinzu, auch wenn sie Ihnen offensichtlich erscheinen. Ein guter Fehlerbericht enthält: 

  - Alle Schritte, die zum Auftreten des Fehlers führen. Wenn Sie Befehle ausführen, notieren Sie, ob Sie diese über die Schaltflächen oder als Eingabe in der Befehlszeile eingegeben haben. Seien Sie bei der Beschreibung des Befehls spezifisch.
  - Was nach jedem Schritt passierte.
  - Welches Verhalten Sie eigentlich erwartet haben. 
  - Ein Modell, das das Problem vor Augen führt.
  - Einen Screenshot des Problems.
  - Eine Beschreibung Ihrer Hardware und der von Ihnen verwendeten Rhino-Version:
    - Öffnen Sie dazu das Panel der Rhino-Eigenschaften.
    - Wählen Sie den Reiter **OpenGL**. Hier finden Sie eine Beschreibung Ihres Computers.
    - Klicken Sie auf die Schaltfläche **In die Zwischenablage kopieren**. Dies kopiert den Inhalt des Reiters in Ihre Zwischenablage.
    - Fügen Sie die kopierte Computer-Beschreibung in Ihren Post ein.
  - Tritt dieses Problem bei jedem Modell oder nur bei einem bestimmten auf?  Wenn das Problem nur bei einigen Modellen auftritt, benötigen wir eine Kopie dieser Modelle, um das Problem zu reproduzieren.

Gehen Sie bei Ihrer Beschreibung immer davon aus, dass Sie als //Einziger// dieses Problem haben.
