====== Rhino V5 Single Commercial License ======


===Choose one:===
    * **[[rhino:install:wizard:start:windows:5:commercial:install|One or more computers owned by single user]]**
      * Use this option when you want to share you license between your personal computers
        * Limited to running on one computer at a time
      * Does not require Internet access
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:5:commercial:install|Multiple users on office LAN]]**
      * Use this option when you are sharing your license(s) between multiple users over a LAN
      * Requires Windows **[[https://wiki.mcneel.com/zoo/home|Zoo]]** server
        * Concurrent use limited to the number of individual licenses in [[https://wiki.mcneel.com/zoo/home|Zoo]]
      * Does not require Internet access
      * Silent [[https://wiki.mcneel.com/rhino/installingrhino/5.0|deployment]] instructions
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|End User License Agreement]]
