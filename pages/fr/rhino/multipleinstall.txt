====== Installer Rhino sur plusieurs ordinateurs ======

Le **[[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|contrat de licence]]** vous permet d'installer Rhino sur tous les ordinateurs que vous contrôlez directement, du moment que vous ne démarrez Rhino que sur un seul ordinateur à la fois. La licence de Rhino se base sur une "utilisation simultanée", non sur un "nombre d'installations". 

Voici les termes exacts du contrat de licence :\\
//"Robert McNeel & Associates vous concède une licence non-exclusive pour utiliser le logiciel sur tout 
ordinateur que vous possédez à condition que le nombre d'utilisateurs simultanés ne dépasse pas le 
nombre de licences que vous possédez. Si vous exécutez le logiciel sur un serveur, le nombre d'utilisateurs simultanés 
ne doit pas dépasser le nombre de licences que vous possédez."// 

  * Vous ne pouvez pas "prêter" Rhino à un ami ou un membre de votre famille.
  * Vous pouvez utiliser le Zoo, un gestionnaire de licences gratuit, pour partager un petit nombre de licences avec un grand nombre d'utilisateurs. 
  * Le Zoo n'est **[[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|pas nécessaire]]** pour un utilisateur individuel possédant plusieurs ordinateurs.\\
