====== Come disinstallare Rhino per Mac ======
> **Riepilogo:** //Seguite i seguenti passi per rimuovere completamente Rhino per Mac dal computer.//

Seguite i seguenti passi per rimuovere completamente Rhino per Mac dal computer:

  - Rimuovere l'applicazione dalla cartella "Applicazioni".
  - Rimuovere la directory **/Libreria/Application Support/McNeel**. Questa directory contiene i modelli dei tutorial ed i manuali scaricati; può non esistere nel computer.  Questa cartella contiene anche il vostro file di licenza.
  - Rimuovere la directory  **~/Libreria/Application Support/McNeel**  dalla home directory. Questa directory viene usata opzionalmente da Rhino ed è possibile che non esista nel vostro computer. Può contenere i materiali creati da voi, gli script di Python ed altre risorse.
  - Rimuovere il file **~/Libreria/Preferenze/com.mcneel.rhinoceros.plist** dalla home directory. Questo file contiene tutte le vostre preferenze di Rhino.
