====== Rhino para Mac y el Zoo ======
> **Producto:** //[[zoo:home|Zoo]]//
> **Resumen:** //Licencias flotantes de Rhino 5 para Mac con el Zoo 6 para Windows.//

===== Requisitos =====
  * Zoo 6 para Windows. //**[[http://www.rhino3d.com/download/zoo/6/latest|Descargar...]]**//
  * Un equipo con Windows en la red con sus Macs. (Puede ejecutar Windows en un Mac utilizando [[http://www.vmware.com/products/fusion/|VMWare]].)


===== Notas =====

  * Zoo 6 para Windows es compatible con [[http://www.rhino3d.com/mac|Rhino para Mac]], además de con todos nuestros últimos [[:zoo:products|productos para Windows]]. 
  * //Los productos antiguos para Windows requieren una versión antigua del Zoo para Windows//.
  * [[http://wiki.mcneel.com/zoo/zooosx|Zoo para OS X]] está en desarrollo.
