====== Rhino pour Mac et le Zoo ======
> **Produit :** //[[fr:zoo:home|Zoo]]//
> **Résumé:** //Partager des licences de Rhino 5 pour Mac à l'aide de Zoo 6 pour Windows.//

===== Conditions requises =====
  * Le Zoo 6 pour Windows. //**[[http://www.rhino3d.com/download/zoo/6/latest|Télécharger...]]**//
  * Un ordinateur Windows en réseau avec vos Mac. (Vous pouvez exécuter Windows sur un Mac avec [[http://www.vmware.com/products/fusion/|VMWare]].)


===== Remarques=====

  * Zoo 6 pour Windows peut gérer des licences de [[http://www.rhino3d.com/mac|Rhino pour Mac]], ainsi que toutes les licences de nos derniers [[fr:zoo:products|produits pour Windows]]. 
  * //Pour gérer des licences de logiciels pour Windows plus anciens, une autre version du Zoo pour Windows est nécessaire//.
  * [[http://wiki.mcneel.com/fr/zoo/zooosx|Zoo pour OS X]] est en cours de développement
