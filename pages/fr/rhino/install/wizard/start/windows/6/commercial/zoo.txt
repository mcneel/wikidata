====== Installation de Rhino 6 afin d'utiliser le gestionnaire de licences sur réseau local Zoo  ======

>Remarque : Utilisez cette option lorsque vous partagez une ou plusieurs licences entre plusieurs utilisateurs sur un réseau local. Aucune connexion à Internet n'est nécessaire. L'installation et la configuration du gestionnaire de licences Zoo gratuit est nécessaire. Rhino peut être déployé silencieusement sur les postes de travail. 

  - **[[https://www.rhino3d.com/download/rhino/6.0/release|Téléchargez]]** le pack d'installation de Rhino
    - Entrez votre adresse e-mail et cliquez sur "Suivant"
    - Cliquez sur "Télécharger maintenant en français" ou sélectionnez une autre langue

**Type d'installation :** Choisir
    - **Déployée** - Pour les administrateurs informatiques
      - Instructions de [[https://wiki.mcneel.com/fr/rhino/installingrhino/6|déploiement]] silencieux
      - Lancez Rhino
    - **Manuel**
      - **Lancez le programme d'installation** - Des droits d'administrateur sont nécessaires
        - Acceptez les conditions et la politique de confidentialité
          - Cliquez sur les liens pour les lire - recommandé
        - Cliquez sur "Installer maintenant"
          - Optionnel - Cliquez sur l'icône de l'engrenage pour voir les options avancées
        - Cliquez sur "Fermer" lorsque l'installation est terminée
        - Lancez Rhino - Rhinoceros 6 a besoin d'une licence : une boîte de dialogue est affichée
        - Choisissez **[[https://wiki.mcneel.com/fr/zoo/home|Utiliser le Zoo]]**
        - Cochez **Utiliser le serveur Zoo avec le nom d'hôte ou l'adresse IP suivant**
        - Tapez le FQDN (nom de domaine) ou l'adresse IP du serveur Zoo
        - Cliquez sur **Continuer**
        - Rhino démarrera

=== L'installation de Rhino a-t-elle échoué ? ===
  * Si l'installation de Rhino a échoué, vous verrez une boîte de dialogue indiquant l'erreur et des instructions.
  * Vous avez obtenu un [[https://wiki.mcneel.com/fr/rhino/6/installproblems|code d'erreur MSI ?]]
  * Nous vous demanderons peut-être d'ouvrir le journal d'erreur, de l'enregistre dans un fichier et de l'envoyer à l'équipe d'assistance technique de McNeel. <tech.eu@mcneel.com>
