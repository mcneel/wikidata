====== Rhino V6 Single EDU License ======


Select license deployment type:
    * **[[rhino:install:wizard:start:windows:6:edu:stand_alone|Single stand-alone computer]]**
      * Use this option when only one computer will be used
      * Does not require Internet access
      * Locks license to one computer
    * **[[rhino:install:wizard:start:windows:6:edu:personal|Multiple computers, single user]]**
      * Use this option when you want to share you license between your personal computers
      * Requires occasional Internet access

>This wiki page is part of a Rhino Install Wizard. [[rhino:install:wizard:start |Start from the beginning]].
