
===== Switch to standalone (Single-Computer) Licensing =====
This can be helpful if you are going to have extended time off the internet.
Or if you have an expired eval that you want to run as a viewer.
Here's how to add a standalone (Single-Computer) license to Rhino:

==== From Inside of Rhino ====

If Rhino is already installed and is getting a license from a CloudZoo, you can:
  - In Rhino, go to **Tools** > **Options**.
  - In the **Options** dialog, click the **Licenses** page.
  - Click **Change your license key**.
  - In the Licensing dialog, enter your license key.
  - In the lower-left, select **Options** and **Use only on this computer**, then click **Continue**.

Rhino will then enter the Rhino key and validate the license online if you have an internet connection.

====New Rhino Install====
If Rhino is not licensed:
  - Open Rhino and display the dialog //"Rhino needs a license to run."//
  - In the lower-left, select **Options** and **Use only on this computer**, then click **Continue**.

====Troubleshooting====
If Rhino does not open, and allow you to pick a license method, you can reset the license keys manually.
Then Rhino will open with the dialog "Rhino needs a license to run."
To reset your Rhino keys, go here.
