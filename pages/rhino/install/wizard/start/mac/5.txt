====== Rhino V5 license type ======


===Select One:===
    * **[[rhino:install:wizard:start:mac:5:commercial|Single Commercial and 90-day evaluation Licenses]]**
      * Allows one user per license
      * May be used stand-alone or over a LAN using the Windows [[https://wiki.mcneel.com/zoo/home|Zoo]] free floating license manager
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|End User License Agreement]]


    * **[[rhino:install:wizard:start:mac:5:educational|Single Educational License]]**
      * Allows one user per license
      * Stand-alone deployment only
        * Computer must be owned by the user
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|End User License Agreement]]


    * **[[rhino:install:wizard:start:mac:5:lab|30-user EDU Lab License]]**
      * Allows 30 concurrent users
      * May be used over a school LAN using the Windows [[https://wiki.mcneel.com/zoo/home|Zoo]] free floating license manager (recommended), or stand-alone
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|End User License Agreement]]
