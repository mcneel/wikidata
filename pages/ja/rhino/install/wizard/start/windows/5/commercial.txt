====== Rhino V5 単一ユーザー商用版ライセンス ======


===以下のうち1つを選んでください:===
    * **[[jp:rhino:install:wizard:start:windows:5:commercial:install|ユーザーお1人で所有されている1台または複数のコンピューター]]**
      * ライセンスをユーザーお1人で所有の複数のコンピューターで共有されたい場合はこのオプションをご利用ください
        *同時にお使いいただけるのは1台のコンピューターに限られます
      * インターネットアクセスは不要です
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|エンドユーザー使用許諾契約書]] （※日本語は15-16ページをご覧ください）

    * **[[jp:rhino:install:wizard:start:windows:5:commercial:install|社内LANでの複数のユーザー]]**
      * LAN経由で複数のユーザーがライセンスを共有している場合はこのオプションをご利用ください
      * Windows **[[https://wiki.mcneel.com/ja/zoo/home|Zoo]]** サーバーが必要です
        * 同時に使用できるのは[[https://wiki.mcneel.com/ja/zoo/home|Zoo]]に登録されているライセンス数が上限です
      * インターネットアクセスは不要です
      * サイレントに[[https://wiki.mcneel.com/ja/rhino/installingrhino/5.0|展開]]する手順
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|エンドユーザー使用許諾契約書]] （※日本語は15-16ページをご覧ください）
