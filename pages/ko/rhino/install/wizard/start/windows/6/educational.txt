====== Rhino 단일 사용자 교육용 라이선스 ======


===하나를 선택하세요:===
    * **[[ko:rhino:install:wizard:start:windows:6:commercial:czpersonal|여러 컴퓨터, 단일 사용자]] - Cloud Zoo 개인용**
      * 사용자의 여러 개인용 컴퓨터에서 라이선스를 사용하려면 이 옵션을 사용하세요.
      * 가끔 인터넷 연결이 필요합니다.[[https://wiki.mcneel.com/ko/rhino/install/wizard/start/windows/6/commercial/czpersonal/firewall|(HTTPS 443 TCP 포트 연결)]]
      * 단일 사용자 교육용 라이선스는 Cloud Zoo 팀이 지원되지 않습니다. 
      * [[https://www.rhino3d.com/6/eula|최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[ko:rhino:install:wizard:start:windows:6:commercial:stand_alone|단일 컴퓨터 라이선스 방식]]**
      * 1대의 컴퓨터에서만 Rhino 6를 사용할 경우 이 옵션을 사용합니다.
      * 라이선스는 1대의 컴퓨터에서 **독점 사용**하도록 제한됩니다.
      * 인터넷 연결이 필요하지 않습니다.
      * [[https://www.rhino3d.com/6/eula|최종 사용자 사용권 계약 - End User License Agreement]]


>이 위키 페이지는 Rhino 설치 마법사의 일부입니다. [[ko:rhino:install:wizard:start |처음부터 시작하려면 여기를 클릭하세요]].
