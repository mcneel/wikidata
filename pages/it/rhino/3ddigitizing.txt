====== Digitalizzazione 3D ======
Gli strumenti di **digitalizzazione 3D** possono essere utili per **[[http://en.wikipedia.org/wiki/Reverse_engineering|ingegneria inversa]].** Con un digitalizzatore, è possibile creare un computer da un oggetto fisico, provando coordinate 3D contemporaneamente per creare curve di input per comandi di superficie. Poiché ciascun comando di superficie richiede curve di input specifiche, un modellatore di Rhino con esperienza deve decidere i migliori strumenti di creazione di superfici da usare per duplicare la forma di un'oggetto. Con un digitalizzatore 3D, puoi provare i punti necessari per creare le curve di input richieste. 

I digitalizzatori 3D possono inoltre essere usati per finalità di analisi. Provando punti in aree critiche sul modello fisico e confrontando tali punti con un modello digitale, è possibile valutare la precisione del modello digitale.
