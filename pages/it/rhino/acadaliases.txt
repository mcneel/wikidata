====== Alias di AutoCAD per Rhino ======

==== Domanda ====

AutoCAD presenta alias di comando utili. Ad esempio, se inserisci "L" nel prompt dei comandi di AutoCAD, verrà eseguito il comando Linea. Questi alias possono essere creati nel gruppo di alias standard di Rhino?

==== Risposta ==== 

Anziché creare alias di AutoCAD in Rhino, abbiamo fornito un gruppo standard di alias di AutoCAD da importare in Rhino. 

=== Configurare Rhino per Windows === 

- **{{:rhino:acadaliasesforrhino.zip|Scarica}}** e decomprimi gli alias di AutoCAD per il file di testo di Rhino. 
- Apri Rhino per Windows, quindi fai clic su **Strumenti -> Opzioni**. 
- Naviga alla pagina **Alias**. - Fare clic sul pulsante  **Import**. 
- Seleziona il file di testo appena decompresso. 

{{:rhino:acadaliases_win.png?500}} 

===Configurare Rhino per Mac=== 

- **{{:rhino:acadaliasesforrhino.zip|Scarica}}** e decomprimi gli alias di AutoCAD per il file di testo di Rhino. 
- Apri Rhino per Mac, quindi fai clic su **Rhinoceros -> Preferenze**. 
- Fai clic sull'opzione **Alias**. 
- Fai clic sul pulsante a discesa a forma di **ingranaggio**, quindi fai clic su **Importa alias da file...**. 
- Seleziona il file di testo appena decompresso.
