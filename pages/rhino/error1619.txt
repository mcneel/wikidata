====== Install Error 1619 (Rhino 5 for Windows) ======
> **Summary:** //How to fix the installation Error 1619.//
> **Related Installation Help:** //[[installproblems|Troubleshooting Rhino 5 Installation]]//

=====Problem=====
The Windows Installer returns error code 1619 when the installation package could not be opened.  Most likely, the reference to the .msi file in the command line is in error.

=====Answer=====
To fix this problem, restart the computer and then run the installer in steps.  The steps are included in the [[rhino/error1603|Rhino Error 1603 Steps for Manual install]]
