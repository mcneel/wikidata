=====Download dei file dei tutorial di Rhino per Mac su OS X 10.7 e successivi=====

**Nota:** I tutorial sono scritti per la versione di Rhino per Windows. Visto che la versione per Mac non è completa, i tutorial non sono ancora stati adattati per rispecchiare le differenze tra le due versioni.

**Per scaricare i file dei tutorial**

  * Nel menu di **Rhino**, fare clic su **Aiuti > Apprendere Rhino > Apri modelli dei tutorial...**. 

Se non avete i file dei tutorial nel computer, vi verrà richiesto di scaricarli. Il file **rh40mac_wip_tutorials_20090724.dmg** verrà scaricato nella cartella Download.

Aprire il file **rh40mac_wip_tutorials_20090724.dmg** per visualizzare una nuova finestra del Finder contenente il file **RhinocerosTutorials.mpkg**.

Fare doppio clic sul file **RhinocerosTutorials.mpkg** per installare i file dei tutorial.

È possibile che OS X mostri un avviso che dice che il file è stato creato da uno sviluppatore non identificato e quindi non lo esegua. In tal caso, **fare clic destro** (o **ctrl+clic**) su **RhinocerosTutorials.mpkg** e, nel menu contestuale, fare clic su **Apri**. Nella finestra di dialogo, fare clic sul pulsante **Apri**. 

Si avvierà il programma di installazione dei tutorial.  Seguire le indicazioni del programma di installazione.

Al termine, fare clic su **Aiuti > Apprendere Rhino > Apri modelli dei tutorial...** di nuovo per aprire un modello.
