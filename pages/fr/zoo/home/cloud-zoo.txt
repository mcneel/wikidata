====Qu'est-ce que le Zoo ?====

Une système de gestion de licences basé sur le cloud qui :
   - Vous permet d'utiliser Rhino en vous connectant au Cloud Zoo avec votre compte Rhino. 
   - Assure le suivi des licences que vous possédez. www.rhino3d.com/licenses
   - Vous permet de suivre et gérer les membres d'une équipe. www.rhino3d.com/licenses
   - Vous permet d'utiliser Rhino sur plusieurs ordinateurs tant que vous respectez [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|le contrat de licence]].

Le Cloud Zoo est une nouvelle option pour activer la licence de Rhino. Il peut remplacer le système de clé de licence existant ou le Zoo, ou vous pouvez continuer la méthode de licence précédente. 

===Configuration requise===

  * Rhino 6.0.
  * Un compte Rhino (créez un compte sur [accounts.rhino3d.com](https://accounts.rhino3d.com)).
  * Une licence de Rhino 6 validée avec n'importe quelle adresse électronique liée à votre compte Rhino.
  * Une connexion à Internet lorsque vous choisissez de vous connecter  (vous pouvez vous déconnecter après).

===Avantages===
  - Pas besoin de s'occuper des clés de licence lors de l'utilisation de Rhino.
  - Possibilité d'utiliser Rhino sur plusieurs ordinateurs facilement. 
  - Rhino continue à fonctionner lorsque vous n'êtes pas connecté à Internet (une connexion non fréquente est nécessaire).

====Comment utiliser le Cloud Zoo :====

  - Dans la boîte de dialogue **Licences**, sélectionnez **Se connecter**.
  - Connectez-vous à votre compte Rhino lorsque Rhino vous le demande et donnez l'autorisation à Rhino d'accéder aux informations nécessaires sur votre compte.

Vous saurez que vous êtes connecté en cliquant sur le menu **?** puis en cliquant sur **À propos de Rhinoceros** - votre avatar et votre nom d¡utilisateurs apparaîtront sur l'écran de présentation.

**Pour convertir une licence autonome :**

  - Dans le menu **Outils**, cliquez sur **Options**.
  - Dans la boîte de dialogue Options, ouvrez la section **Licences**.
  - Cliquez sur le bouton **Changer de clé de licence**.
  - Dans la boîte de dialogue **Licences**, sélectionnez **Se connecter**.
  - Connectez-vous à votre compte Rhino lorsque Rhino vous le demande et donnez l'autorisation à Rhino d'accéder aux informations nécessaires sur votre compte.

**Et si je veux qu'un autre utilisateur de Cloud Zoo utilise Rhino sur mon ordinateur ?**

  - Dans la ligne de commande de Rhino, tapez **Déconnexion** pour vous déconnecter de votre compte Rhino.
  - Suivez les étapes de la section ci-dessus **Comment utiliser une licence avec un compte Rhino**.

**Et si je veux repasser du Cloud Zoo à une licence autonome ?** 

  - Retirez votre licence du Cloud Zoo.
  - Dans Rhino, dans le menu **Outils**, cliquez sur **Options**.
  - Dans la boîte de dialogue Options, ouvrez la section **Licences**.
  - Cliquez sur le bouton **Changer de clé de licence**.
  - Dans la boîte de dialogue **Licences**, cliquez sur **Entrer votre clé de licence**.
  - En bas à gauche, **Utiliser uniquement sur cet ordinateur** puis sur **Continuer**.
Plus d'informations [[https://wiki.mcneel.com/rhino_accounts/remove_license|ici.]]

**Et si je veux utiliser Rhino 6 comme licence d'évaluation expirée qui ne peut pas enregistrer ni exporter ?** 
Vous devez configurer Rhino comme licence autonome pour cela. 
  - Retirez votre licence du Cloud Zoo.
  - Dans Rhino, dans le menu **Outils**, cliquez sur **Options**.
  - Dans la boîte de dialogue Options, ouvrez la section **Licences**.
  - Cliquez sur le bouton **Changer de clé de licence**.
  - Dans la boîte de dialogue **Licences**, **entrez les informations sur la licence d'évaluation**.
  - En bas à gauche cliquez sur, **Utiliser uniquement sur cet ordinateur** puis sur **Continuer**.
Plus d'informations [[https://wiki.mcneel.com/rhino_accounts/remove_license|ici.]]
