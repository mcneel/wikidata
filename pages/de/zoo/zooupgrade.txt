====== Upgrade eines LAN-Zoo-Servers ======
> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //Upgrade eines LAN-Zoo-Servers.//

Zur Unterstützung der neuesten Rhino-Version können Sie entweder ein Upgrade Ihres **LAN Zoo** durchführen oder auf **[[https://wiki.mcneel.com/de/rhino_accounts/home|Cloud Zoo]]** umziehen (was oft die bessere Wahl ist). 

===== Voraussetzung  =====

[[https://wiki.mcneel.com/de/zoo/zoodotnet|Der LAN-Zoo benötigt Microsoft .NET Framework 4.8]]

===== Download und Installation =====
  * [[https://www.rhino3d.com/download/zoo/latest|Laden Sie hier den LAN Zoo herunter.]]

  * Installieren Sie ihn auf dem selben Server wie Ihren aktuellen LAN Zoo.

===== INFO =====

  * Upgrades von LAN Zoo ersetzen Ihren vorherigen LAN Zoo.

  * Ihre Lizenzen werden automatisch in den neuen LAN Zoo verbracht. 

  * Anwender, die Rhino zum Zeitpunkt der Installation verwenden, sind nicht betroffen.

  * Die Installation dauert etwa 15 Sekunden.
