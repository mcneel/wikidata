===== Rechercher de problèmes éventuels sur les polices===== 

1. Dans **Applications**, lancez **Livre des polices**. 

2. Dans la colonne **Collection**, cliquez sur **Toutes les polices** (en haut). 

{{:rhino:mac:checkfonts1.png?direct&300|}}

3. Dans la colonne **Polices**, cliquez sur une police. 

4. Tapez **Commande+A** pour sélectionner toutes les polices. 

{{:rhino:mac:checkfonts2.png?direct&300|}}

5. Dans le menu de l'**Application**, sélectionnez **Fichier > Valider des polices**. Une boîte de dialogue apparaît et le programme vérifie les polices. 

{{:rhino:mac:checkfonts3.png?direct&300|}}

6. Réparez toutes les erreurs de police que vous trouvez.
