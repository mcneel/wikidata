====== Finestra di avvio di Rhino per Mac ======

Quando Rhino per Mac si avvia, sulla destra della finestra di avvio vengono elencati i modelli aperti di recente.

Il modello più recente appare evidenziato e, per aprirlo, basta premere Invio.  È possibile selezionare un modello diverso premendo i tasti di direzione su o giù.  Un modello si può anche selezionare digitando le prime lettere del suo nome.  Una volta selezionato il modello desiderato, premere Invio per aprirlo.

La stessa finestra consente l'accesso ai file modello.  Fare clic sul pulsante //Show Templates// sul vertice inferiore destro della finestra per elencare i file modello.  Oppure, per passare all'elenco dei file modello, si può premere Comando-T.  Premendo il pulsante //Show Recent Models// oppure Comando-R, si ritorna all'elenco dei modelli recenti.

Per creare i propri file modello in Rhino, usare la scelta da menu File > Salva come modello...  I file modello personalizzati verranno visualizzati nella parte superiore dell'elenco dei file modello.  Si può indicare che uno dei file modello sia quello predefinito facendo clic destro sul file e selezionando //Make "XXX" the default template// dal menu contestuale.  I file modello personalizzati presentano anche la scelta da menu //Show in Finder//.  La si può usare per reperire il file modello e modificarne i contenuti, cambiare il nome del file oppure eliminare il file.

Il pulsante //Open Other...// (Comando-O) che appare nella finestra apre la finestra di dialogo Apri file per consentire la selezione di un modello qualsiasi di Rhino.  Il pulsante //New Model// (Comando-N) crea una nuova finestra di modellazione usando il modello predefinito corrente come contenuto iniziale.
