====== Rhino4.NET Project and Command Wizards ======
> **摘要:** //建立Rhino.NET插件和命令的向导//
====== .NET ======

[[http://wiki.mcneel.com/_media/legacy/en/Rhino4DotNetWizards.zip|Rhino4DotNetWizards.zip]] (**updated April 3, 2008**)

上面的zip 文件包含用Visual Studio 2005的VB.NET和C#建立插件和命令的向导

.安装这些向导你需要:
  * 解压 [[http://wiki.mcneel.com/_media/legacy/en/Rhino4DotNetWizards.zip|Rhino4DotNetWizards.zip]] 这个压缩包包含两个文件 (RMA.VisualStudioWizards.dll and RhinoDotNetWizards.vsi)
  * 安装 RMA.VisualStudioWizards.dll 至您的电脑的 global assembly cache (GAC). 这步可以由下面两步完成:
    * 打开控制面板中的管理工具并且启动 Microsoft .NET Framework 2.0 配置应用. 单击 '管理 程序集缓存' 然后 '将程序集添加到程序集缓存'. 然后会显示一个文件打开对话框. 再选择文件 RMA.VisualStudioWizards.DLL
    * 打开资源管理器,将文件复制至程序集目录 (大概是 D:\WINDOWS\assembly)
    * 备注: 当你第一次在GAC中安装一个程序集时, 文件会被复制至特定的程序集目录. 所以你可以删除原来的 RMA.VisualStudioWizards.DLL .
  * 双击 RhinoDotNetWizards.vsi . 这是一个Visual Studio 安装文件,会复制向导至你电脑上的恰当目录. **NOTE:** 如果你是从旧的版本更新RhinoDotNetWizards.vsi, 选择<color blue>覆盖同名文件</color> 选项

=====XML Comments file (updated September 26, 2007)=====
放置一个XML 注释文件在你电脑上,这样就可以显示rhino SDK的方法解释了.

[[http://wiki.mcneel.com/_media/legacy/en/Rhino_DotNET_XML.zip|下载一个xml 注释文件压缩包]]

解压文件并且复制 "Rhino_DotNET.xml" 至Rhino系统目录 (就是使和rhino4.exe同目录)
