====== Licencia comercial monopuesto de Rhino 6 ======


===Elija una:===
    * **[[rhino:install:wizard:start:windows:6:commercial:czpersonal|Varios equipos, un único usuario]] - Cloud Zoo (Personal)**
        * Utilice esta opción cuando quiera utilizar su licencia en varios de sus ordenadores personales
        * Requiere una conexión a Internet puntual
        * [[https://www.rhino3d.com/6/eula|Contrato de licencia de usuario final]]

    * **[[rhino:install:wizard:start:windows:6:commercial:czteam|Varios equipos, varios usuario]] - Cloud Zoo (Grupo)**
        * Utilice esta opción cuando quiera compartir su(s) licencia(s) entre varios usuarios
        * Requiere una conexión a Internet puntual
        * [[https://www.rhino3d.com/6/eula|Contrato de licencia de usuario final]]

    * **[[rhino:install:wizard:start:windows:6:commercial:zoo|Varios usuarios en red de oficina]] - Zoo**
        * Utilice esta opción cuando quiera compartir su(s) licencia(s) entre varios usuarios, se limita a la red de la oficina
        * Requiere el **[[https://wiki.mcneel.com/zoo/home|Zoo]]** para Windows, el gestor licencias flotantes gratuito
        * Instalación de Rhino con **[[https://wiki.mcneel.com/rhino/installingrhino/6|fácil implementación]]**
        * No requiere acceso a Internet
        * [[https://www.rhino3d.com/6/eula|Contrato de licencia de usuario final]]

    * **[[rhino:install:wizard:start:windows:6:commercial:stand_alone|Licencia monopuesto]]**
      * Utilice esta opción cuando se vaya a usar un solo equipo/ordenador
      * Bloquea la licencia **exclusivamente** en un equipo/ordenador
      * Requiere acceso a Internet una sola vez para realizar la validación, aunque también se puede realizar la [[https://wiki.mcneel.com/zoo/rhinovalidateoffline|validación sin conexión.]]
      * Después de la validación, no requiere acceso a Internet excepto para las actualizaciones.
      * [[https://www.rhino3d.com/6/eula|Contrato de licencia de usuario final]]
