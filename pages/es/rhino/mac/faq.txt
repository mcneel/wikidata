====== Preguntas más frecuentes de Rhino para Mac ======

--> ¿Puedo comprar Rhino para Mac?#

No, Rhino para Mac actualmente está en desarrollo. Todavía no está a la venta.  Simplemente [[http://www.rhino3d.com/download/wenatchee/5.0/wip|descargue Rhino para Mac]] y utilícelo durante el periodo de desarrollo.  Puede enviarnos sus comentarios y sugerencias.

<--

-->¿Cuándo saldrá la versión de Rhino para Mac?#

No lo sabemos. McNeel & Associates tiene un ciclo de desarrollo poco común, donde se suelen publicar versiones previas &  no acabadas de nuestro software para obtener comentarios de nuestros usuarios. Rhino para Mac sigue el mismo estilo de desarrollo utilizado para el [[http://www.rhino3d.com/nr.htm|desarollo de Rhino para Windows]].

Para obtener una lista completa de los errores pendientes antes de publicar la versión inicial, consulte [[http://mcneel.myjetbrains.com/youtrack/issues?q=project:MR+release+target:5.0+%23unresolved+sort+by:+State+|esta lista]].

<--
-->Cuando salga a la venta, ¿cuánto costará?#

Todavía no se ha establecido el precio, pero será inferior que el de Rhino 5 para Windows.  Habrá un precio promocional antes de la salida para los usuarios de Rhino 5 para Windows y un precio promocional por tiempo limitado disponible para todo el mundo.

<--

-->Si compro Rhino para Windows, ¿tendré Rhino para Mac gratis? ¿Cuál será la política multiplataforma?#

Los usuarios de Rhino 5 para Windows disfrutarán de un descuento promocional previo a la salida de la versión; una licencia de Rhino 5 para Windows se podrá utilizar para obtener un precio especial en una copia de Rhino para Mac.
<--

-->¿Qué es una WIP?#

WIP significa Work In Progress (Trabajo en progreso).  Es similar a una "beta" y a menudo se publica en paralelo a una verisón comercial estable, pero la WIP contiene funciones experimentales o a medio terminar.  Las versions WIP normalmente están disponibles para cualquier usuario que tenga una versión comercial (o educacional) del programa.  La WIP de Rhino para Mac WIP hace años que está en desarrollo.  Actualmente tenemos una WIP preliminar y también habrá una WIP posterior que estará disponible para aquellos que hayan adquirido la versión completa.
<--

--> Rhino para Mac tiene un límite de tiempo. ¿Qué pasa cuando caduca?#

La fecha de caducidad sirve para asegurarnos de que todo el mundo está probando la última versión y que no se informe de errores que ya se han solucionado. 

Antes de que caduque cada versión WIP de Rhino para Mac, habrá una nueva versión disponible para [[http://www.rhino3d.com/download/wenatchee/5.0/wip|descargar]]. Rhino para Mac comprueba las actualizaciones de WIP cada vez que se inicia y le notifica si hay una nueva versión disponible. También se enviará un mensaje en la [[http://discourse.mcneel.com/category/rhino-for-mac|sección de Rhino para Mac]] del [[http://discourse.mcneel.com/ |foro de Rhino]] cada vez que haya una nueva WIP disponible.

<-- 

-->Cuando salga la versión comercial de Rhino para Mac, ¿qué pasará con la WIP?#
La WIP preliminar caducará unas semanas después de la salida de la versión comercial.  Habrá una WIP posterior que solo estará disponible para los que hayan adquirido una versión de Rhino para Mac.
<--

-->Cuando salga la versión comercial de Rhino para Mac, habrá Service Releases o Actualizaciones?#
Sí.  Las actualizaciones de Rhino para Mac 5 incluirán correcciones de errores y nuevas funciones.  Las funciones como los Diseños y las Sesiones de trabajo no se incluirán en la versión inicial, pero se añadirán en las actualizaciones posteriores. Quienes compren Rhino para Mac 5 recibirán las actualizaciones de forma gratuita.
<--

-->¿En qué se diferencia Rhino para Mac de Rhino para Windows?#
  * Rhino para Mac tendrá casi todas las funciones de modelado que tiene Rhino para Windows.
  * Rhino para Mac tiene un renderizador integrado, objetos de anotación como cotas, texto e impresión simple.

__A diferencia de Rhino para Windows, Rhino para Mac actualmente no es__

  * Extensible con plugins o complementos y, por tanto:
  * No es compatible con RhinoCAM, RhinoGold, Matrix, Grasshopper.
  * No admite las Diseños, Sesiones de trabajo ni Herramientas de animación (se incluirán en actualizaciones posteriores).

...La interfaz de Rhino para Mac es considerablemente diferente a la de Rhino para Windows.
<--

-->¿Rhino para Mac funciona en otros idiomas a parte de inglés?#

Sí. Actualmente, los idiomas adicionales de la interfaz de Rhino son español, francés, alemán, italiano, checo, polaco, chino, japonés y coreano. Este es trabajo también está en progreso y es posible que exista alguna traducción incompleta. 

Rhino para Mac utiliza la selección del idioma predeterminado en Macintosh. Para utilizar un idioma diferente al predeterminado de OS X
  - En el menú, haga clic en **Rhinoceros** y luego en **Preferencias**.
  - En el panel **Preferencia de idioma**, seleccione su idioma.

<-- 

-->¿Qué no funciona? ¿Por qué me aparece el mensaje "Unknown or unimplemented command"?#

Tenemos previsto que la versión para Mac tenga las mismas funciones que la versión de Rhino para Windows, pero todavía no están implementadas todas las funciones en esta versión WIP de Rhino para Mac. Consulte el [[http://www.rhino3d.com/nr.htm|proceso de desarrollo de software]] de Rhino para Windows para ver cómo presentamos nuevos productos. Estamos siguiendo el mismo plan con el desarrollo de Rhino para Mac. Actualmente estamos en la [[http://www.rhino3d.com/nr.htm#phase1|fase 1: trabajo en progreso]]. 

En esta [[rhino:mac:whatworks|página web]] se describe qué funciona y que queda por implementar.

<-- 

-->¿Cómo se desinstala Rhino?#

Los siguientes pasos eliminarán completamente Rhino para Mac de su ordenador:

  - Elimine la aplicación de la carpeta Aplicaciones.
  - Elimine el directorio **/Library/Application Support/McNeel**. Este directorio contiene los modelos de los tutoriales y los manuales si los descargó.
  - Quite el directorio **~/Library/Application Support/McNeel** del directorio principal. Este directorio lo utiliza Rhino y puede que no exista en su ordenador. Puede contener materiales de Toucan que haya creado, scripts de Python y otros plug-ins de Rhino para Mac.
  - Elimine el archivo **~/Library/Preferences/com.mcneel.rhinoceros.plist** del directorio principal. Este archivo contiene todas las preferencias de Rhino.

==== Compatibilidad ====

-->¿Los archivos de datos son compatibles con la versión de Windows?#

Sí, los archivos de Rhino 5 son idénticos. También es posible exportar archivos de Rhino 4 para que sean compatibles con Rhino 4 para Windows.

<-- 

-->¿Puedo usar mi ratón de dos botones / tres botones?#

Sí, todos los Mac permiten ratones con varios botones y rueda de desplazamiento. Pero también puede usar un ratón de Mac de un botón. Para simular el clic con el botón derecho del ratón, pulse la tecla Control mientras hace clic en el único botón del ratón.

<-- 

-->¿Puedo usar un Spaceball/SpaceNavigator/SpaceExplorer?#

Solo es compatible con el Mac el SpaceNavigator.

<--

==== Scripts y plug-ins ====

-->¿Puedo ejecutar un script de comando en la versión para Mac?#

Sí. Para cargar y ejecutar un script de comando, en el menú de Rhino, seleccione **Herramientas > Comandos > Leer desde archivo**.
 
<--

-->¿Puedo ejecutar un programa de RhinoScript en la versión para Mac?#

No. RhinoScript está basado VisualBasic y VisualBasic no es compatible con OS X. Estamos utilizando Python para el scripting en la versión de Mac y Windows, que permite crear y usar scripts para ambas versiones de Rhino. Visite el [[http://discourse.mcneel.com/category/scripting | foro de Rhino de soporte sobre scripting]] para obtener ayuda sobre el lenguaje de scripting Python para Windows y OS X.

<--


-->¿Los plug-ins externos funcionarán en Rhino para Mac?#

Quizás. Los plug-ins para Windows no funcionan en Mac. La decisión de adaptar un plug-in a Rhino para Mac dependerá del desarrollador del plug-in.
<--
 
-->¿La versión para Mac será compatible con mi plug-in C?#

En un futuro. Todavía no podemos ofrecer soporte para plug-ins C, pero tenemos previsto hacerlo en un futuro. Tendrá que recompilar el plug-in utilizando el sistema de desarrollo para Mac. Según la complejidad del plug-in, puede que no sea mucho trabajo crear un plug-in que compile y funcione tanto en Windows como en Macintosh.

<--

-->¿La versión para Mac será compatible con mi plug-in .NET?#

En un futuro. Estamos trabajando en herramientas que permitan a los desarrolladores de plug-ins de Rhino escribir plug-ins que funcionen tanto en OS X como en Windows.

==== Problemas de instalación ====

-->Tengo que desinstalar Rhino antes de instalar una nueva versión?#

No. 

<-- 

-->OS X dice "No se han podido abrir las siguientes imágenes de disco." ¿Qué hago ahora?#

Probablemente ha descargado un archivo .dmg dañado.  Compruebe que el archivo .dmg descargado sea válido. Los archivos a veces pueden dañarse durante la descarga.

  - Inicie el programa Utilidad del disco **/Aplicaciones/Utilidades/Utilidad del disco**
  - Desde el menú, seleccione **Imágenes > Verificar...** Aparecerá el cuadro de diálogo Abrir archivo.
  - Seleccione el archivo .dmg que ha descargado. Normalmente se encuentra en la carpeta Descargas y tiene el nombre rhinoceros_wip_YYYYMMDD.dmg
  - Si el archivo descargado no es válido, vaya la página de [[http://www.rhino3d.com/download/wenatchee/5.0/wip|descargas]] y solicite otro vínculo de descarga.
  - Si vuelve a suceder, tiene una conexión a Internet no segura y debería intentar descargar el archivo desde otra conexión a Internet.  La descarga es de más de 100 MB.

<-- 

-->Instalé la última versión de Rhino y dice que ha caducado. ¿Ahora qué?#

Tiene instalada más de una copia de Rhino y todavía se está ejecutando la versión caducada. Debe eliminar **todas** las copias de Rhino. 

Haga doble clic en cualquier archivo 3DM. Si el archivo se abre en Rhino, seguirá teniendo una copia de Rhino instalada en algún lugar. 
  - Haga clic con el botón derecho en el icono de Rhino en el Dock, seleccione **Opciones > Mostrar en el Finder** desde el menú para saber dónde está.
  -  Arrastre esa copia a la Papelera. 
  - Cuando no haya más copias de Rhino, al hacer doble clic en un archivo 3DM, este se abrirá con TextEdit. Cuando esto suceda, significa que Rhinoceros ya se ha eliminado.
  - Reinstale la versión que acaba de descargar.

-->Me ha aparecido un diálogo misterioso que dice "Program Failure" y  Rhino para Mac ya no funciona.#

Cuando Rhino para Mac se encuentra con una situación no esperada, muestra un diálogo con el mensaje genérico "Program Failure". Después de este error, Rhino para Mac se cerrará y ya no podrá ejecutar comandos ni opciones de menú. Podrá guardar su archivo y salir de Rhino para Mac, pero no podrá hacer nada más. Necesitamos saber cuándo se produce este error. 
Cuando trabaje con Rhino, puede que vea el siguiente diálogo:

Rhino muestra diálogos genérico en distintas situaciones. Todavía no están implementadas todas las funciones en Rhino, por lo que aparecerá este diálogo cuando intente utilizar una función no implementada. También hay código de programa en Rhino que solo puede usarse en la versión de Rhino para Windows. Si alguna acción intenta ejecutar este código, también aparecerá el diálogo "Rhino Error". Por último, también se están llevando a cabo tests de depuración en este código inicial, por lo que verá este diálogo cuando un test interno descubra una situación que no debería producirse.

{{:rhino:mac:notimpl.png?nolink|}}

Cuando aparezca alguno de estos diálogos, escriba el nombre del archivo, el número de línea y la información de la versión del diálogo. Envíenos esta información por correo electrónico, junto con una descripción de lo que estaba haciendo cuando apareció el diálogo, a la dirección <mactech@mcneel.com>. Intentaremos reproducir el problema  y encontrar una solución para que el error no vuelva a producirse en el futuro.

Después de que aparezca este diálogo, Rhino restringe el uso de los comandos a solo unos pocos. Podrá guardar el archivo y podrá salir de Rhino, pero no podrá hacer mucho más.

--> Quiero el clic derecho para repetir el último comando #

Si desactiva el menú contextual, podrá utilizar la función de repetir el comando.

  - En el menú, haga clic en **Rhinoceros** y luego en **Preferencias**.
  - Seleccione el panel **Ratón**.
  - En la sección **Botón derecho del ratón**,  desactive la casilla **Activar menús contextuales**.
