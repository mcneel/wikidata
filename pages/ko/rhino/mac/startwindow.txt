====== Mac용 Rhino 시작 창 ======

Mac용 Rhino를 시작할 때, 사용자가 최근에 열었던 모델이 시작 창의 오른쪽에 목록으로 표시됩니다.

가장 최근 모델은 강조 표시되고, Return 키를 누르면 해당 모델이 열립니다. 위/아래 화살표 키를 눌러 다른 모델을 선택할 수 있습니다. 또한, 모델 이름의 처음 몇 글자를 입력하여 모델을 선택할 수 있습니다. 원하는 모델을 선택하면 Return 키를 눌러 해당 모델을 엽니다.

시작 창에서 템플릿 파일에 액세스할 수 있습니다. 창의 오른쪽 아래 모서리에 있는 //템플릿 표시// 단추를 클릭하면 템플릿 파일 목록으로 변경되어 표시됩니다. 템플릿 목록으로 변경하려면 Command-T를 눌러도 됩니다. //최근 모델 표시// 단추를 클릭하거나 Command-R을 누르면 최근 모델 목록으로 되돌아갑니다.

Rhino에서 자신만의 템플릿을 만들려면 파일 > 템플릿으로 저장... 메뉴를 선택합니다.  사용자가 만든 템플릿은 템플릿 목록의 최상에 위치합니다. 한 템플릿 파일을 기본 템플릿 파일로 설정하려면 해당 템플릿 모델을 오른쪽 클릭하고 상황에 맞는 메뉴에서 //"XXX"을 기본 템플릿으로 만들기//를 선택합니다. 사용자가 지정한 템플릿 모델에서 //Finder에서 표시// 메뉴도 선택할 수 있습니다. 이 기능을 사용하여 템플릿 파일을 찾고 템플릿 컨텐츠를 편집하고, 이름을 변경하거나 해당 템플릿을 삭제할 수 있습니다.

이 창의 //다른 모델 열기...// 단추 (Command-O)는 다른 Rhino 모델을 선택할 수 있는 파일 열기 대화상자를 엽니다. //새 모델// 단추 (Command-N)를 클릭하면, 현재 기본 템플릿을 시작 컨텐츠로 사용하여 새 모델링 창을 만듭니다.
