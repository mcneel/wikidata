======Convertir une licence autonome de Rhino pour utiliser le Zoo======

> **Produit :** //[[fr:zoo:home|Zoo]]//
> **Résumé :** //Explique comment convertir une installation autonome de Rhino afin d'utiliser les licences du Zoo.//

=====Question=====
Nous avons récemment installé le **Zoo** et ajouter nos licences Comment puis-je convertir les licences de Rhino installées sur les postes de travail afin qu'elles utilisent les licences du Zoo ?


=====Réponse=====
Pour convertir une installation autonome de Rhino afin d'utiliser le Zoo, suivez ces étapes :

====Rhino 6 ou 7 pour Windows ====

  - Exécutez **Rhino 6 ou 7 pour Windows**.
  - Ouvrez le menu **Outils** puis cliquez sur **Options**.
  - Ouvrez la section **Licences**. 
  - Sélectionnez la licence de Rhino puis cliquez sur **Modifier votre clé de licence**.

{{zoo:rhino6win_convert.png?500}}

====Rhino 5 pour Windows ====

  - Lancez **Rhino 5 pour Windows**.
  - Ouvrez le menu **Outils** puis cliquez sur **Options**.
  - Ouvrez la section **Licences**.
  - Sélectionnez la licence de Rhino et cliquez sur **Convertir**.

{{zoo:rhino5win_convert.png?500}}

====Rhino pour Mac====

  - Exécutez **Rhino 5, 6 ou 7 pour Mac**.
  - Ouvrez le menu **Rhinoceros** puis cliquez sur **Préférences...**.
  - Cliquez sur l'option **Licences**.
  - Sélectionnez la licence de Rhino puis cliquez sur **Modifier votre clé de licence**.

{{zoo:rhino5mac_convert.png?500}}
