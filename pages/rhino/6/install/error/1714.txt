====== Install Error 1714 (Rhino 6 for Windows) or System Error 1612 ======
> **Summary:** //How to fix the installation Error 1714//
> **Related installation help:** //[[:rhino:6:installproblems|Troubleshooting Rhino Installation]]//

=====Problem=====
The log file shows "Rhino Installer Engine -- Error 1714. The older version of Rhino Installer Engine cannot be removed.  Contact your technical support group.  System Error 1612."

=====Answer=====

This is fixed with a nice tool from Microsoft.  Please download and follow these instructions:

https://support.microsoft.com/en-us/help/17588/fix-problems-that-block-programs-from-being-installed-or-removed

After fixing the issue, then try to install Rhino again.
