====== Puis-je utiliser PanelingTools ? (Rhino pour Mac) ======
> **Résumé:** //Le module PanelingTools existe-t-il dans Rhino pour Mac ?//

Oui. Le module PanelingTools est maintenant intégré dans Rhino pour Mac. Il n'y a pas de menu PanelingTools et toutes les commandes sont disponibles dans la ligne de commandes uniquement.  Tapez **pt** pour voir toutes les commandes de PanelingTools dans la liste de saisie semi-automatique.
