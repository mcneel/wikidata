====== Rhino 6 30-user EDU Lab License  ======


Select license deployment type:
    * **[[rhino:install:wizard:start:windows:6:edu:stand_alone|30 stand-alone lab computers]]**
      * Use this option when 30 stand-alone computers will be used
      * Does not require Internet access
      * Rhino is installed manually on 30 computers
      * **Not recommended**

    * **[[rhino:install:wizard:start:windows:6:edu:zoo|30 computers in a single lab using school LAN]]**
        * Use this option when sharing the license in a school classroom lab
        * Uses the school LAN
        * Requires the **[[https://wiki.mcneel.com/zoo/home|Zoo]]** free floating license manager
        * Does not require Internet access
        * Rhino installation easily **[[https://wiki.mcneel.com/rhino/installingrhino/6|deployed]]**
        * **Recommended**

    * **[[rhino:install:wizard:start:windows:6:edu:team|30 Team members using Internet]]**
        * Use this option when you are sharing your license between multiple Team members
        * Allows off-campus license access
        * Requires occasional Internet access
        * Rhino is installed manually
        * Cloud Zoo Team created and managed by school Admin
