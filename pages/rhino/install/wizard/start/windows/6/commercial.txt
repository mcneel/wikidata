====== Rhino V6 Single Commercial License ======


Select license deployment type:
    * **[[rhino:install:wizard:start:windows:6:commercial:stand_alone|Single stand-alone computer]]**
      * Use this option when only one computer will be used
      * Does not require Internet access
      * Locks license to one computer
    * **[[rhino:install:wizard:start:windows:6:commercial:personal|Multiple computers, single user]]**
        * Use this option when you want to share you license between your personal computers
        * Requires occasional Internet access
    * **[[rhino:install:wizard:start:windows:6:commercial:team|Multiple computers, multiple users]]**
        * Use this option when you are sharing your license(s) between multiple users
        * Requires occasional Internet access
    * **[[rhino:install:wizard:start:windows:6:commercial:zoo|Multiple users on office LAN]]**
        * Use this option when you are sharing your license(s) between multiple users, limited to the office LAN
        * Does not require Internet access
