====== Muss ich Rhino deinstallieren, bevor ich eine neuere Version installiere? (Rhino 5 für Mac) ======
> **Zusammenfassung:** //Muss ich Rhino für Mac deinstallieren, bevor ich eine neuere Version installieren kann?//

Nein, das ist nicht nötig.  Sie können das Programmpaket - meist zu finden im Order "Programme" - überschreiben, wenn Sie eine neuere Version installieren.  Dadurch werden weder Ihre Voreinstellungen noch Ihre Lizenz überschrieben, da sich diese in anderen Ordnern befinden.  

Es können sogar zwei Versionen von Rhinoceros nebeneinander ausgeführt werden, indem Sie das zweite Programmpaket (Rhinoceros.app) umbenennen (z.B. RhinocerosKopie.app).
