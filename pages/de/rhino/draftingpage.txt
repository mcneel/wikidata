=====Schraffierungsmuster für Rhino 4=====

=====Schraffierungsmuster von Carlotto Vittorio=====

Laden Sie sich einfach die folgende Datei herunter und ersetzen Sie die Originaldatei (legen Sie von letzterer am Besten eine Sicherung an) im Ordner:

  * Windows XP: C:\Dokumente und Einstellungen\Alle Benutzer\Anwendungsdaten\McNeel\Rhinoceros\4.0\German\Support
  * Windows Vista und Windows 7: C:\ProgramData\McNeel\Rhinoceros\4.0\German\Support

=====Schraffierungsmuster für Rhino 5=====

1. Laden Sie [[http://wiki.mcneel.com/_media/legacy/en/hatchpatterns_1033.zip|dieses ZIP-Archiv]] herunter und entpacken Sie es. Benennen Sie hatchpatterns.txt in hatchpatterns.pat um. 

2. Starten Sie Rhino 5, gehen Sie auf Datei -> Eigenschaften -> Dokumenteigenschaften -> Anmerkung -> Schraffierung und importieren Sie die Datei hatchpatterns.pat.

=====Erstellung weiterer Linientypen und Druckbreiten=====

Bearbeiten Sie einfach die Textdateien im Ordner:

  * Windows XP: C:\Dokumente und Einstellungen\Alle Benutzer\Anwendungsdaten\McNeel\Rhinoceros\4.0\German\Support
  * Windows Vista: C:\ProgramData\McNeel\Rhinoceros\4.0\German\Support
