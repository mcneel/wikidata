====== Rhino a planté - Et maintenant? ======


Lorsque Rhino plante, une boule tournante apparaît pendant un moment et cette boîte de dialogue peut s'afficher :

{{:rhino:mac:rhinocrash.png?nolink|}}

**Ne cliquez pas** sur le bouton **Report** de cette boîte de dialogue. Cette action envoie un rapport à Apple, pas à nous et nous ne verrions jamais l'erreur. 

**Pour envoyer un rapport d'erreur à McNeel**

  - Cliquez sur le bouton **Reopen** de cette boîte de dialogue. Rhino redémarre. Rhino détecte qu'il a planté la dernière fois qu'il était ouvert et affiche la boîte de dialogue **Rhino Crash Report** :
  - Écrivez une description de ce que vous faisiez lorsque Rhino a planté afin que nous puissions reproduire votre problème et le réparer. Il est important d'inclure assez d'informations pour que nous reproduisions le plantage. Un message est alors créé et le rapport est envoyé par mail à McNeel & Associates. Le message contient toutes les informations que vous voyez dans la case Problem and System Information ainsi que votre propre description du problème. 
  - Cliquez sur le bouton **Send to McNeel**. Rhino se lance ensuite normalement.
