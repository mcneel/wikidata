====== 新しいバージョンをインストールする前に、Rhinoをアンインストールする必要はありますか（Rhino 5 for Mac） ======
> **Summary:** //Do I need to uninstall Rhino for Mac before installing a newer version?//

No, you do not.  You can overwrite the application bundle - most frequently located in the Applications folder - when you install a newer version.  This will not overwrite any of your preferences or license, which are stored in other folders.  

In fact, you can run two versions of Rhinoceros side-by-side if you so desire, just by renaming the application bundle (Rhinoceros.app) to something different (RhinocerosCopy.app - for example).
