====== Rhino 7 Studenten-Lab-Lizenz für 30 Nutzer  ======


===Bitte auswählen:===
    * **[[de:rhino:install:wizard:start:windows:6:commercial:zoo|30 Computer in einer einzigen Lab unter Verwendung einer Schule-LAN]] - Zoo (Empfohlen)**
        * Verwenden Sie diese Option beim Teilen der Lizenz in einer Klassenzimmer-Lab
        * Unter Verwendung der Schule-LAN
        * Benötigt den kostenlosen Floating-Lizenzmanager des **[[https://wiki.mcneel.com/de/zoo/home|Zoo]]**
        * Benötigt keinen Internetzugang
        * Ein Rhino-Konto wird nicht benötigt
        * Einfache **[[https://wiki.mcneel.com/rhino/installingrhino/6|Bereitstellung]]** der Rhino-Installation
        * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]

    * **[[https://wiki.mcneel.com/de/rhino/6/cloud_zoo/setup|30 Teammitglieder unter Verwendung von Internet]] - Cloud Zoo Team**
        * Wählen Sie diese Option wenn Sie Ihre Lizenz mit mehreren Teammitgliedern teilen
        * Lizenz ist auch außerhalb des Campus zugänglich
        * Gelegentlich wird ein Internetzugang benötigt
        * Rhino wurde manuell installiert
        * Von Schuladministrator erstelltes und verwaltetes Cloud Zoo Team
        * Jeder Student benötigt seinen eigenen Log-in bei [[https://accounts.rhino3d.com/help|Rhino Accounts]]
        * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]

    * **[[de:rhino:install:wizard:start:windows:6:commercial:stand_alone|Lizenzierung für 30 einzelne Computer]] (Nicht empfohlen)**
      * Wählen Sie diese Option wenn 30 einzelne Computer verwendet werden
      * Benötigt wird ein einmaliger Internetzugang wird für die Bestätigung auf jedem Computer oder eine [[https://wiki.mcneel.com/de/zoo/rhinovalidateoffline|Offline-Bestätigung]].
      * Internetzugang wird nicht benötigt, außer für die Aktualisierungen
      * Benötigt kein LAN
      * Rhino wird manuell auf 30 Computern installiert
      * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]

>Diese Wiki-Seite ist Teil eines Rhino-Installationsassistenten. [[de:rhino:install:wizard:start |Beginnen Sie am Ausgangspunkt]].
