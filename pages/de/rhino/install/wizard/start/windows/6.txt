====== Rhino 7, Rhino 6 oder Plug-in-Lizenztyp ======


===Bitte auswählen:===
    * **[[de:rhino:install:wizard:start:windows:6:commercial|Kommerzielle Einzelnutzer-Lizenz und 90tägige Testversion]]** //<- Klicken Sie für weitere Infos.//
      * Ein Benutzer pro Lizenz ist erlaubt
      * Kann im Einzelplatzmodus verwendet werden, oder über einen LAN- (Zoo-) oder Internet-basierten Manager zur zentralen Lizenzverwaltung (Cloud Zoo).
      * [[https://www.rhino3d.com/7/eula|Rhino Endbenutzer-Lizenzvertrag]]

    * **[[de:rhino:install:wizard:start:windows:6:educational|Studenten-Einzelnutzerlizenz]]** //<- Klicken Sie für weitere Infos.//
      * Ein Benutzer pro Lizenz ist erlaubt
      * Kann im Einzelplatzmodus auf dem Computer des Studierenden verwendet werden, oder mit dem Persönlichen Lizenzverwalter von Cloud Zoo
      * [[https://www.rhino3d.com/7/eula|Rhino Endbenutzer-Lizenzvertrag]]

    * **[[de:rhino:install:wizard:start:windows:6:lab|Studenten-Lab-Lizenz für 30 Nutzer]]** //<- Klicken Sie für weitere Infos.//
      * Bis zu 30 Nutzer sind gleichzeitig erlaubt
      * Kann über eine Schule-LAN mithilfe des [[https://wiki.mcneel.com/de/zoo/home|Zoo]] Zentralen Lizenzmanagers verwendet werden (empfohlen) oder im Einzelplatzmodus installiert werden.
       * Rhino-Installation **leicht gemacht**:
          * [[https://wiki.mcneel.com/de/rhino/installingrhino/7|Windows]]
          * [[https://wiki.mcneel.com/de/zoo/rhinomacdeployment|Mac]]
      * [[https://www.rhino3d.com/7/eula|Rhino Endbenutzer-Lizenzvertrag]]
