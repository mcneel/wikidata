====== Rhino 5: Error _mcneel._zoo5 ======
**Problem:**\\ Rhino 5 cannot run and shows an error message about "_mcneel._zoo5".\\ This problem happens when Rhino 5 runs for the first time after it's been insatalled.

{{http://wiki.mcneel.com/_media/rhino/rhino5/mcneel_zoo5.png}}

**Cause:**\\ An unknown installation problem.

**Solution:**\\
  - Delete the ***.lic** file under:\\ **For Windows Vista/7/8**\\ C:\ProgramData\McNeel\Rhinoceros\5.0\License Manager\Licenses\\\ **For Windows XP**\\ C:\Documents and Settings\All Users\Application Data\McNeel\Rhinoceros\5.0\License Manager\Licenses\ \\ 
  - Run Rhino 5.
  - Click **Enter a License**.
  - Enter your Rhino 5 license.
  - Click **OK**.
