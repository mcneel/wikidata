=====Downloads=====

**SectionTools Plugin for Rhino 7 for Windows:** Last updated December 3, 2020.
  * Use the new PackageManager command in Rhino 7 to search for and install SectionsTools.

**SectionTools Plugin for Rhino 6 for Windows:** Last updated July 29, 2019.
  * Make sure you have the latest [[https://www.rhino3d.com/download|Rhino 6]]. 
  * Download[[https://www.rhino3d.com/download/sectiontools/1/wip/rc| SectionTools for Rhino 6 Windows WIP]] for Rhino 6. Double-click and the plugin should install and load in Rhino. You need to load the toolbar using the toolbar manager.

**SectionTools Plugin for Rhino 5 only (does not work with Rhino 4):** Last updated December 20, 2013.
  * Make sure you have the latest service release [[http://www.rhino3d.com/download/rhino/5.0/evaluation|Rhino 5 or Rhino 5 Evaluation]]. 
  * Download[[https://www.food4rhino.com/app/sectiontools-rhino| SectionTools for Rhino 5]] for Rhino 5. Double-click and the plugin should install and load in Rhino. You need to load the toolbar using the toolbar manager.
