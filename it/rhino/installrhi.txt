====== Installazione dei file RHI per tutti gli utenti ======
Rhino 5.6 consente di selezionare se si desidera installare per un utente oppure per tutti gli utenti di un computer. L'implementazione dei file RHI si può inoltre si può automatizzare tramite script affinché sia invisibile all'utente.

=====Implementazione dei file RHI=====
  - Assicurarsi di aver implementato Rhino 5.6 o successive in rete.
  - Eseguire "C:\Programmi\Rhinoceros 5 (64-bit)\System\x64\rhiexec.exe" "C:\Full\Path\To\Installer.rhi" /admin /silent

 L'opzione /admin si installa per tutti gli utenti del computer.
L'opzione /silent si installa senza mostrare l'interfaccia utente.
