====== Installation von Rhino auf mehreren Computern ======

Die **[[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Lizenzvereinbarung]]** erlaubt Ihnen die Installation von Rhino auf allen Computern, die direkt unter Ihrer Kontrolle stehen, solange Sie nachweisen können, dass Rhino immer nur auf einem dieser Computer ausgeführt wird. Rhino wird auf Grundlage der "gleichzeitigen Verwendung" und nicht "pro Installation" lizenziert. 

Hier finden Sie den entsprechenden Auszug aus der Lizenzvereinbarung:\\

//"Robert McNeel & Associates gewährt Ihnen ein nicht ausschließliches Nutzungsrecht zur Benutzung der Software auf einem beliebigen Computer in Ihrem Besitz, so lange die Anzahl der Nutzer, die zeitgleich die Software benutzen, nicht die Anzahl der erworbenen Lizenzen überschreitet. Wenn die Software auf einem Server ausgeführt wird, darf die Anzahl der Nutzer, die zeitgleich die Software benutzen, die Anzahl der sich in Ihrem Besitz befindlichen Lizenzen nicht überschreiten."//

  * Rhino darf **nicht** an Freunde oder Familienmitglieder "verliehen" werden.
  * Mit dem kostenlosen Lizenzmanager **[[http://wiki.mcneel.com/de/zoo/home|Zoo]]** kann die gemeinschaftliche Verwendung von Lizenzen verwaltet werden, wenn mehr Anwender als Lizenzen vorhanden sind.
  * Einzelnutzer mit mehreren Computern benötigen den Zoo **[[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|nicht]]**.\\
