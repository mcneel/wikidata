===== Development Tools =====
  * [[http://www.dependencywalker.com/|Dependency Walker]] - Useful for troubleshooting system errors related to loading and executing modules.
  * [[http://weblogs.asp.net/kennykerr/archive/2007/07/10/manifest-view-1-0.aspx|Manifest View]] - Useful for viewing the embedded manifest of an EXE, DLL, or plug-in.
  * [[http://technet.microsoft.com/en-us/sysinternals/bb896653.aspx|Process Explorer]] - Useful for tracking down DLL-version problems or handle leaks, and provide insight into the way Windows and applications work.
  * [[http://en.wiki.mcneel.com/content/upload/files/RhinoGuidgen.zip|Rhino Guidgen]] - Visual Studio-like Guidgen utility, but with a few new formats.
  * [[http://visualstudiogallery.msdn.microsoft.com/7c8341f1-ebac-40c8-92c2-476db8d523ce|Spell Checker]] - Nice spelling checker for Visual Studio 2010.
  * [[http://www.jetbrains.com/resharper|ReSharper]] - .NET code quality analyzer - slick!
  * [[http://tortoisesvn.net/|TortoiseSVN]] - The coolest interface to Subversion version control.
  * [[http://vld.codeplex.com/|Visual Leak Detector]] - A free, robust, open-source memory leak detection system for Visual C++.
  * [[http://winmerge.org/|WinMerge]] - An invaluable differencing and merging tool.
  * [[http://www.gipsysoft.com/zoomplus/|Zoom+]] - A very handy screen magnification tool.
