====== Rhino 5 30사용자 교육 실습실용(Lab) 라이선스  ======


===하나를 선택하세요:===
    * **[[https://wiki.mcneel.com/ko/rhino/installingrhino/5.0|학교 LAN을 사용하는 30대의 컴퓨터 (권장합니다)]]**
        * 학교 교실/실습실에서 라이선스를 공유할 때 이 옵션을 사용하세요.
        * 학교 LAN을 사용합니다.
        * 무료 플로팅 라이선스 관리자인 Windows용 **[[https://wiki.mcneel.com/ko/zoo/home|Zoo]]** 가 필요합니다.
        * Rhino 설치는 쉽게 **[[https://wiki.mcneel.com/ko/rhino/installingrhino/5.0|배포]]**할 수 있습니다.
        * 인터넷 연결이 필요하지 않습니다.
        * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[ko:rhino:install:wizard:start:windows:5:edu:stand_alone|단일 실습실의 독립실행형 컴퓨터 30대 (권장하지 않습니다)]]**
      * 30대의 독립실행형 컴퓨터가 사용되는 경우, 이 옵션을 사용하세요.
      * 인터넷 연결이 필요하지 않습니다.
      * 30대의 컴퓨터에 Rhino를 수동으로 설치합니다.
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|최종 사용자 사용권 계약 - End User License Agreement]]

>이 위키 페이지는 Rhino 설치 마법사의 일부입니다. [[ko:rhino:install:wizard:start |처음부터 시작하려면 여기를 클릭하세요]].
