====== Asistente de instalación de Rhino 5 y Rhino 6 ======

Este asistente le ayudará a determinar la mejor manera y los requisitos específicos para instalar y activar la licencia de Rhino 5, Rhino 6, Rhino 7 y una amplia lista de plug-ins. Aparecerán opciones y requisitos en cada página. Siempre puede volver atrás o empezar de nuevo si cambia de opinión.
