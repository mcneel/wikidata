====== Comment supprimer un module de Rhino pour Mac ? ======
{{ :rhino:mac:ff.png?100|}}



**Pour supprimer un module sur Mac, vous devez accéder au dossier Bibliothèque d'utilisateur.**

  - Ouvrez le **Finder** sur votre Mac. 

  - Dans la partie supérieure de l'écran vous verrez des menus déroulants.
{{ :rhino:mac:menu.png?400|}}
  - L'un d'entre eux s'appelle **Aller**. Cliquez sur **Aller** pour déployer le menu déroulant.
  - Maintenez ensuite la touche **Options** du clavier enfoncée et remarquez que le mot Bibliothèque apparaît dans la liste du menu **Aller**.
  - Sélectionnez **Bibliothèque** -> **Application Support -> McNeel -> 7.0 (ou votre version) -> Plug-ins**. {{:rhino:mac:pluginsmac.png?600|}}///Users/<login name>/Library/Application Support/McNeel/Rhinoceros/7.0/Plug-ins//
  - **Supprimez** le fichier du module à partir de cet emplacement (//Twimmotion, Datasmith ou tout autre // module chargé dans votre programme. Vous pouvez le supprimer pour le désinstaller, le renommer ou le déplacer pour éviter qu'il ne soit chargé dans le programme.)
  - **Signalez** le problème au développeur du module en cause.
