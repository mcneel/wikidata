====== Select Linear and Non-linear Curves ======
> **Developer:** //[[developer:rhinoscript|RhinoScript]]//
> **Summary:** //Demonstrates how to select linear and non-linear curves using [[developer:rhinoscript|RhinoScript]]//

=====Selecting non-linear curves=====

The following [[developer:rhinoscript|RhinoScript]] subroutine will select all non-linear curves in the document:

<code vb>
 Sub SelNonLinearCrv()
   Dim arrCurves, strCurve
   arrCurves = Rhino.ObjectsByType(4)
   If IsArray(arrCurves) Then
     Rhino.EnableRedraw False
     For Each strCurve In arrCurves
       If Not Rhino.IsCurveLinear(strCurve) Then
         Rhino.SelectObject strCurve
       End If
     Next
     Rhino.EnableRedraw True
   End If
 End Sub
</code>

Here is the toolbar button macro version of the above:

<code vb>
_-NoEcho
_-RunScript (

End If
strCurve = Null
arrCurves = Null
)
</code>

=====Selecting linear curves=====
The following [[developer:rhinoscript|RhinoScript]] subroutine will select all linear curves in the document:

<code vb>
 Sub SelLinearCrv()
   Dim arrCurves, strCurve
   arrCurves = Rhino.ObjectsByType(4)
   If IsArray(arrCurves) Then
     Rhino.EnableRedraw False
     For Each strCurve In arrCurves
       If Rhino.IsCurveL
       inear(strCurve) Then
         Rhino.SelectObject strCurve
       End If
     Next
     Rhino.EnableRedraw True
   End If
 End Sub
</code>
