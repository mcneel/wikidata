====== Rhino V5 のインストール ======

>Rhino 5 for Windows のインストールはこちらの手順をご利用ください

  - Rhinoのインストーラーパッケージを**[[https://www.rhino3d.com/download/rhino/5/latest|ダウンロード]]**します
    - メールアドレスを入力し、"次へ"をクリックします
    - "今すぐ 日本語 をダウンロード" または別の言語を選択します
  - **インストーラーを実行します** - ローカル管理者の権限が必要です
    - "Rhino 5をインストール" をクリック
  - **ライセンス情報を入力します**
      - 名前: 必須
      - E-mail（メールアドレス）: 必須
      - 組織: オプション
    - **ライセンスの種類を選択します**
      - Rhinoを評価 - ライセンスキーは不要です
      - ライセンスキー - お持ちのRhinoライセンスキー24文字（例: RH50-XXXX...）を入力します
      - Zoo - [[https://wiki.mcneel.com/ja/zoo/home|LANを使ったZoo]]をインストール・ご利用の場合のみ
    - **使用許諾契約書に同意** - クリックしてお読みください
      - McNeelに登録情報を送信 - オプション
      - "次へ”をクリック
{{:ja:rhino:install:rhino5install02.png?300|}}

**Rhino 5 がインストールされ、デスクトップ ショートカットが2つ作成されます**
  * **Rhinoceros 5** - 32ビット版 Rhino V5 
    * Rhino 4 用にデザインされた32ビットのプラグインをサポートします
    * RAMの上限は4GBです

  * **Rhinoceros 5（64ビット版）**
    * 64ビット版WindowsシステムのRAMを全て利用します

===== Rhinoのインストールに失敗した場合 =====
  * Rhinoのインストールに失敗した場合、インストラクションを含めエラーについて説明するダイアログが表示されます。
  * [[https://wiki.mcneel.com/ja/rhino/installproblems|MSI エラーコード]]は表示されましたか?
  * エラーログを開くよう指示されていると思いますので、ファイルに保存して弊社McNeelのテクニカルサポート（米国: <tech@mcneel.com> ）または日本のサポート（<support@rhino3d.co.jp>）までご送信ください。
