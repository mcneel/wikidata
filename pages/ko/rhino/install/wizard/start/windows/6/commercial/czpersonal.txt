====== Cloud Zoo - 개인용 ======

>안내: 이 옵션을 사용하여 귀하가 소유한 하나의 라이선스를 귀하의 모든 컴퓨터에서 사용할 수 있습니다. 귀하의 라이선스는 저희 서버에 위치하게 되므로, 어떤 컴퓨터에서도 귀하가 해당 라이선스에 액세스할 수 있습니다. Rhino를 처음 사용할 때 인터넷 연결이 필요합니다. 그 이후에는 Rhino에서 일주일에 한 번 정도의 인터넷 연결을 통해 라이선스 사용 기간이 연장됩니다. 약 2주 동안 인터넷 연결 없이 Rhino를 사용할 수 있으나 계속해서 Rhino를 사용하려면 인터넷 연결이 필요합니다.
>귀하의 개인용 계정을 타인과 절대 **공유**하지 마세요.

===초기 Rhino 응용 프로그램 기반 설정===
  - Rhino 또는 플러그인을 설치합니다. 
  - Rhino를 시작합니다.
    - 기본 설정을 따릅니다.
  - 플러그인을 시작합니다. 
    - 기본 설정을 따릅니다.

===초기 웹 기반 설정===
  - **사용자의 Rhino 계정에 로그인**
    - https://accounts.rhino3d.com/?controller=home 을 방문하세요.
      - 아직 Rhino 계정이 없다면 Rhino 계정을 만듭니다.
      - 귀하의 자격 증명을 저장합니다. 나중에 필요하게 됩니다.
  - 귀하의 라이선스를 **개인용 라이선스**에 추가합니다.
    - "나의 계정" 페이지에서 라이선스를 클릭합니다.
    - 개인용 라이선스 아래에서 [귀하의 이름] (나) 을 클릭합니다.
    - "라이선스 [귀하의 이름] (나)" 페이지에서 라이선스 추가를 클릭합니다.
    - 귀하의 라이선스 키를 입력합니다.
    - 라이선스 추가를 클릭합니다.
  - **Rhino를 시작합니다**.
    - 로그인을 클릭합니다.
    - 귀하의 계정 자격 증명을 제공합니다.
    - Rhino가 시작합니다.

===다른 컴퓨터===
  - **Rhino를 설치합니다**.
  - **Rhino를 시작합니다**.
    - 로그인을 클릭합니다.
    - 귀하의 계정 자격 증명을 제공합니다.
    - Rhino가 시작합니다.

>이 위키 페이지는 Rhino 설치 마법사의 일부입니다. [[ko:rhino:install:wizard:start |처음부터 시작하려면 여기를 클릭하세요]].
