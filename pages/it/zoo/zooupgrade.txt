====== Aggiornare il server LAN Zoo ======
> **Prodotto:** //[[it:zoo:home|Zoo]]//
> **Sommario:** //Aggiornare il server LAN Zoo.//

Per utilizzare Rhino 7, occorre effettuare l'aggiornamento a **LAN Zoo** o spostare la licenza su **[[https://wiki.mcneel.com/rhino_accounts/home|Cloud Zoo]]** (spesso Ã¨ l'opzione migliore). 

===== Prerequisito  =====

[[https://wiki.mcneel.com/it/zoo/zoodotnet|LAN Zoo richiede Microsoft .NET Framework 4.8]].


===== Download e installazione =====
  * [[http://www.rhino3d.com/download/zoo/latest|Download di LAN Zoo qui.]]

  * Esegui l'installazione sullo stesso server dell'attuale LAN Zoo.


===== INFORMAZIONI =====

  * LAN Zoo aggiorna la versione precedente di LAN Zoo.

  * Le licenze verranno spostate automaticamente su LAN Zoo. 

  * Durante l'installazione di LAN Zoo, gli utenti possono eseguire Rhino come normalmente.

  * L'installazione richiede 15 secondi circa.
