====== LAN Zoo は Microsoft .NET Framework 4.8 が必要です ======
> **製品:** //[[ja:zoo:home|Zoo]]//
> **要約:** //LAN Zooと Microsoft .NET Framework 4.8 についての説明//


LAN Zoo は [[https://dotnet.microsoft.com/ja-jp/download/dotnet-framework/net48|Microsoft .NET Framework 4.8]] が必要です。

**[[http://windows.microsoft.com/en-us/windows/windows-update|Windows Update]]** を使うことで、Microsoft .NET Framework 4.8 をシステムに確実にインストールすることができます。
