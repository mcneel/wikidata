=====Circle Packing Script (version 2.0)=====

This script generates a circle packing with any number of selected circles.

While the script is running it will print progress information to the status bar.

This version works in Rhino 4, runs faster, and won't get caught in infinite loops.

**//Runs in Rhino 4 only.//**
