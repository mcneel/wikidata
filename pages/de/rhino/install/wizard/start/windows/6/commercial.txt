====== Rhino 6 Kommerzielle Einzelnutzer-Lizenz ======


===Bitte auswählen:===
    * **[[de:rhino:install:wizard:start:windows:6:commercial:czpersonal|Mehrere Computer, ein Benutzer]] - Cloud Zoo Persönlich**
        * Wählen Sie diese Option, wenn Sie Ihre Lizenz auf Ihren eigenen Computern teilen wollen
        * Gelegentlich wird ein Internetzugang benötigt
        * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]

    * **[[de:rhino:install:wizard:start:windows:6:commercial:czteam|Mehrere Computer, mehrere Benutzer]] - Cloud Zoo Team**
        * Wählen Sie diese Option wenn Sie Ihre Lizenz(en) mit mehreren Nutzern teilen
        * Gelegentlich wird ein Internetzugang benötigt
        * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]

    * **[[de:rhino:install:wizard:start:windows:6:commercial:zoo|Mehrere Benutzer auf einer Büro-LAN]] - Zoo**
        * Wählen Sie diese Option wenn Sie Ihre Lizenz(en) mit mehreren Nutzern bei Beschränkung auf die Büro-LAN teilen
        * Benötigt den kostenlosen Floating-Lizenzmanager des Windows **[[https://wiki.mcneel.com/de/zoo/home|Zoo]]**
        * Einfache **[[https://wiki.mcneel.com/rhino/installingrhino/6|Bereitstellung]]** der Rhino-Installation
        * Benötigt keinen Internetzugang
        * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]

    * **[[de:rhino:install:wizard:start:windows:6:commercial:stand_alone|Einzelcomputer-Lizenzierung]]**
      * Wählen Sie diese Option wenn nur ein Computer verwendet wird
      * Sperrt die Lizenz zur **ausschließlichen** Verwendung auf einem Computer
      * Benötigt wird ein einmaliger Internetzugang wird für die Bestätigung, oder eine [[https://wiki.mcneel.com/de/zoo/rhinovalidateoffline|Offline-Bestätigung]].
      * Nach der Bestätigung wird kein Internetzugang mehr benötigt, außer für die Aktualisierungen.
      * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]

>Diese Wiki-Seite ist Teil eines Rhino-Installationsassistenten. [[de:rhino:install:wizard:start |Beginnen Sie am Ausgangspunkt]].
