====== Signaler des problèmes avec Rhino pour Mac ======


Cliquez [[http://discourse.mcneel.com/category/rhino-for-mac|ici]] pour ouvrir le forum de Mac. Vous pouvez chercher si votre problème est mentionné et ajouter votre propre commentaire au sujet ou cliquer sur **Create Topic** en haut de la page des forums pour commencer un nouveau sujet. 

Pour inclure des fichiers dans votre message, tels qu'une capture d'écran ou un fichier .3dm, en haut de la fenêtre **Create a new topic**, cliquez sur le bouton Upload {{:rhino:mac:9-16-2013_2-11-02_pm.png?nolink|}}.

==== Nous envoyer un fichier volumineux ====

Si vous avez un petit fichier qui peut nous aider à reproduire un problème, vous pouvez le joindre à votre message sur le [[http://discourse.mcneel.com/category/rhino-for-mac|forum de discussion]].\\

Si votre fichier est très volumineux
  - Ouvrez la page [[http://www.rhino3d.com/upload|Chargement d'un fichier pour McNeel]] 
  - Entrez vos informations et indiquez **<mactech@mcneel.com>** comme adresse du destinataire chez McNeel. Merci de donner une description détaillée de votre problème. 

==== Comment écrire un rapport d'erreur détaillé ====
Nous souhaitons résoudre les problèmes que vous rencontrez mais nous devons tout d'abord les reproduire. Un bon rapport d'erreur nous aide beaucoup. Indiquez autant de détails que possible, même si cela vous paraît évident. Un bon rapport comprend :

  - Chaque étape que vous avez réalisée pour reproduire le problème. Lorsque vous lancez des commandes, indiquez comment vous l'avez lancé (en cliquant sur un bouton, en la tapant...). Indiquez précisément la commande que vous avez lancée. 
  - Ce qui s'est passé après chaque étape. 
  - Ce qui aurait dû se passer. 
  - Un modèle qui montre le problème. 
  - Une capture d'écran de l'erreur ou du problème. 
  - Une description de votre matériel et de la version de Rhino :
    - Ouvrez le panneau Preferences de Rhino.
    - Sélectionnez l'onglet **OpenGL**. Vous obtiendrez ainsi une description de votre ordinateur. 
    - Cliquez sur le bouton **Copy to clipboard**. Le contenu de l'onglet est alors copié dans votre presse-papiers.
    - Collez la description de votre ordinateur dans votre message.
  - Ce problème se produit-il avec tous les modèles ou juste un seul ?  Si le problème ne se produit qu'avec certains modèles, nous aurions besoin d'un copie de votre modèle pour le reproduire. 

Faites comme si vous étiez la //seule// personne ayant ce problème.
