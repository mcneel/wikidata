====== Résoudre les problèmes de connexion des bras USB de FARO ======

**Pour Rhino 3.0/4.0**

Si vous rencontrez des problèmes de connexion entre Rhino 3.0 et votre bras USB FARO, suivez ces instructions :

  - **Fermez Rhino**
  - **Désactivez** le bras FARO
  - **Débranchez** le bras FARO de votre ordinateur
  - Dans le menu **Démarrage**, cliquez sur **Panneau de configuration**
  - Double-cliquez sur **Ajout/Suppression de programmes**
  - Sélectionnez **FARO Arm USB Driver** et cliquez sur **Supprimer** Retirez complètement le pilote du bras USB Faro. Vérifiez que toutes les copies du pilote USB FARO sont supprimées.
  - **Redémarrez** votre ordinateur.
  - **Téléchargez** les pilotes USB FARO de la page de ressources de Rhino : http://www2.rhino3d.com/resources/display.asp?listing=402
  - **Installez** les pilotes du bras USB FARO que vous venez de télécharger.
  - **Redémarrez** votre ordinateur.
  - **Connectez** le câble USB de FARO à votre ordinateur.
  - **Allumez** le bras USB Faro.
  - L'**Assistant Ajout de nouveau matériel** s'ouvrira. Suivez les étapes à l'écran pour installer le pilote.
  - Lorsque l'assistant est terminé, **lancez Rhino**.
  - Dans le menu **Outils**, cliquez sur **Numériseur 3D** puis sur **Connecter**.
  - À l'invite **Sélectionner le numériseur**, cliquez sur l'option **Numériseur**.
  - À l'invite **Numériseur**, cliquez sur **FARO USB Digitizer**
  - Suivez les instructions pour connecter votre numériseur.
