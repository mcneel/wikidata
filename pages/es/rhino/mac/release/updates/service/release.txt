====== ¿Hay previstas verisiones Service Release o actualizaciones? ======
> **Resumen:** //Cuando salga Rhino 5 para Mac, ¿habrá versiones Service Release o actualizaciones?//

**Sí**.  De hecho, las actualizaciones de Rhino 5 para Mac incluirán correcciones de errores y nuevas funciones.  Al [[http://www.rhino3d.com/mac-feature-compare|comparar Rhino 5 para Mac y Rhino 5 para Windows]], verá que Rhino 5 para Mac no tiene las mismas funciones que Rhino 5 para Windows.  Las funciones como el **Soporte de plug-ins**, los **Diseños**, las **Sesiones de trabajo ** y las **Herramientas de animación ** no se incluyen en la versión inicial, pero se añadirán en las actualizaciones posteriores.  Quienes compren Rhino 5 para Mac recibirán las actualizaciones **sin coste adicional**.

===== Planificación del ciclo de actualización de versiones de Rhino 5 para Mac =====

**5.1**\\
Estamos trabajando activamente para incluir el soporte de plug-ins en nuestra primera actualización  (5.1).  El SDK del plug-in .NET C# es nuestra principal prioridad.  Ya hemos superado las dificultades técnicas, ahora solo los queda terminar temas de documentación y soporte.

**5.x**\\
Tenemos previsto agregar las Sesiones de trabajo, los Diseños y las Herramientas de animación durante el ciclo de publicación de versiones de Rhino 5 para Mac.  Actualmente no tenemos definido el orden de prioridad de estas funciones.

**Sin previsión**\\
Actualmente no hay ninguna fecha establecida para agregar Grasshopper a Rhino 5 para Mac.
