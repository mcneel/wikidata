====== Rhino 7, Rhino 6, 또는 플러그인 라이선스 방식 유형 ======


===하나를 선택하세요:===
    * **[[ko:rhino:install:wizard:start:windows:6:commercial|단일 사용자 상업용 및 90일 평가판 라이선스]]** //<- 상세 정보를 보시려면 클릭하세요.//
      * 한 라이선스당 한 명의 사용자가 허용됩니다.
      * 독립실행형으로 사용하거나, LAN (Zoo)이나 인터넷 기반 플로팅 라이선스 관리자 (Cloud Zoo) 를 통해 함께 사용할 수 있습니다. 
      * [[https://www.rhino3d.com/7/eula|Rhino 최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[ko:rhino:install:wizard:start:windows:6:educational|단일 사용자 교육용 라이선스]]** //<- 상세 정보를 보시려면 클릭하세요.//
      * 한 라이선스당 한 명의 사용자가 허용됩니다.
      * 학생의 컴퓨터에서 독립실행형으로 사용하거나, Cloud Zoo 개인용 라이선스 관리자를 통해 사용할 수 있습니다. 
      * [[https://www.rhino3d.com/7/eula|Rhino 최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[ko:rhino:install:wizard:start:windows:6:lab|30 사용자 교육 실습실용 라이선스(Educational Lab License)]]** //<- 상세 정보를 보시려면 클릭하세요.//
      * 30명의 동시 사용자가 허용됩니다.
      * [[https://wiki.mcneel.com/ko/zoo/home|Zoo]]플로팅 라이선스 관리자(권장됨)를 사용하여 학교 LAN상에서 사용하거나, 독립실행형으로 설치할 수 있습니다. 
       * Rhino 설치는 쉽게 **배포할 수 있습니다**:
          * [[https://wiki.mcneel.com/ko/rhino/installingrhino/7|Windows]]
          * [[https://wiki.mcneel.com/ko/zoo/rhinomacdeployment|Mac]]
      * [[https://www.rhino3d.com/7/eula|Rhino 최종 사용자 사용권 계약 - End User License Agreement]]
