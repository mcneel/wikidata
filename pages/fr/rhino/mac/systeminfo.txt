=====Recueil d'informations système sur votre ordinateur=====


  - Dans le menu **Apple**, sélectionnez **À propos de ce Mac**.  Une boîte de dialogue est affichée.
  - Dans la boîte de dialogue, cliquez sur **Plus d’infos…**.  Cette boîte de dialogue est remplacée par une nouvelle. 
  - Cliquez sur le bouton **Rapport système...**.  Une autre boîte de dialogue est affichée.
  - Dans le menu de l'application Informations système, sélectionnez **Fichier > Enregistrer...**.  Une boîte de dialogue est affichée.
  - Enregistrez le fichier.
  - Fermez la boîte de dialogue Informations système.
  - Envoyez le fichier enregistré à mactech@mcneel.com, avec une description de votre problème.
  - Dans votre message, n'oubliez pas de mentionner les messages de discourse.mcneel.com qui pourraient parler de votre problème.
