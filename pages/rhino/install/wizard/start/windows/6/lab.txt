====== Rhino 6 30-user EDU Lab License  ======


===Choose one:===
    * **[[rhino:install:wizard:start:windows:6:commercial:zoo|30 computers in a single lab using school LAN]] - Zoo (Strongly Recommended)**
        * **Requires no user management and minimal administration maintenance**
        * Use this option when limiting the license in a school classroom lab
        * Uses the school LAN
        * Requires the **[[https://wiki.mcneel.com/zoo/home|Zoo]]** free floating license manager
        * Does not require Internet access
        * No Rhino account needed
        * Rhino installation easily **deployed**
          * [[https://wiki.mcneel.com/rhino/installingrhino/6|Windows]]
          * [[https://wiki.mcneel.com/zoo/rhinomacdeployment|Mac]]
        * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[https://wiki.mcneel.com/rhino/6/cloud_zoo/setup|30 Team members using Internet]] - Cloud Zoo Team**
        * **Requires significant Team management and ongoing administration**
        * Use this option when you are sharing your license among multiple Team members
        * Allows off-campus license access
        * Requires occasional Internet access
        * Rhino is installed manually by each student
        * Cloud Zoo Team created and managed by school Admin
        * Requires each student has their own [[https://accounts.rhino3d.com/help|Rhino Account]] login
        * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:commercial:stand_alone|30 single computers licensing]] (Not Recommended)**
      * **Requires the most ongoing effort by IT to setup, replace, and re-image classroom computers**
      * Use this option when 30 individual, school-owned computers will be used
      * Requires one time Internet access for validation on each computer or an [[https://wiki.mcneel.com/zoo/rhinovalidateoffline|offline validation.]]
      * Does not require Internet access except for updates
      * Does not require LAN
      * Rhino is installed manually on 30 school-owned computers
      * [[https://www.rhino3d.com/6/eula|End User License Agreement]]
