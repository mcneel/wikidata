====== 3-user EDU Lab License  ======


Deployment type:
    * **Stand-alone** - Not recommended - The license can be used to manually install and validate the license on 30 individual computers.

    * **Zoo** - Recommended - [[https://wiki.mcneel.com/zoo/home|Floating license manager]] The license can be deployed on a Windows Server running the Zoo. Rhino itself can be [[https://wiki.mcneel.com/rhino/installingrhino/5.0|silently deployed]] onto the lab workstations pre-configured to use the Zoo server
