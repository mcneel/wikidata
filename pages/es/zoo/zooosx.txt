====== Zoo LAN para macOS ======
> **Producto:** //[[zoo:home|Zoo]]//
> **Resumen:** //Sobre el funcionamiento de Zoo LAN en macOS.//

===== Pregunta =====
Actualmente, si queremos usar Rhino para Mac con Zoo LAN, tenemos que usar la versión de Zoo LAN para Microsoft Windows. ¿Está previsto portar Zoo LAN a macOS?

===== Respuesta =====
Sí, actualmente estamos en el proceso de desarrollar una versión de Zoo LAN para macOS. Cuando tengamos una versión disponible para probar, publicaremos un mensaje en el foro de [[http://discourse.mcneel.com/|Discourse]].

Si no tiene un sistema de Windows donde ejecutar Zoo LAN, puede usar [[http://www.vmware.com/products/fusion-pro/|VMWare Fusion Pro]], que permite ejecutar Windows en un Mac.
