====== Ein oder mehrere Computer, die einem einzigen Nutzer gehören  ======


Bereitstellungstyp:
====Einzelplatzsystem====
    * Installieren Sie Ihre Einzelnutzerlizenz auf den Computern, die Ihnen gehören und die Sie persönlich kontrollieren.
    * Sie können Rhino nicht auf mehreren Computern gleichzeitig ausführen.
