===== Entfernen einer Lizenz aus dem Zoo oder aus einer Einzelcomputer-Installation =====

Um Ihrem Rhino-Konto eine bereits verwendete Lizenz hinzuzufügen, müssen Sie sie zuerst aus dem Zoo oder von dem Gerät, auf dem sie zuerst verwendet wurde, entfernen.


==== Lizenz aus dem Zoo entfernen ====
  - **Öffnen Sie die Zoo-Administratoranwendung**.
  - **Wählen Sie die Lizenz aus**.
  - Klicken Sie auf die Schaltfläche **Löschen**.

==== Lizenz aus einem Einzelcomputer entfernen ====
  - **Starten Sie Rhino**.
  - Klicken Sie im Menü **Werkzeuge** auf **Optionen**, dann klicken Sie auf **Lizenzen** und anschließend auf **Lizenzschlüssel ändern**.
  - Klicken Sie im unteren Bereich des Dialogfensters auf **Lizenzen entfernen**.

==== Hinzufügen der Lizenz zu Cloud Zoo ====
  - Gehen Sie auf die Seite [[https://www.rhino3d.com/licenses/?controller=add_licenses|Lizenzen Hinzufügen]].
  - Fügen Sie die Lizenz Ihrem Konto hinzu oder einem der Teams, denen Sie angehören.
