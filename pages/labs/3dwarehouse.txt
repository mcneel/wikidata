====== 3d Warehouse Reader ======

> **Summary:** //Browse and Load models directly from [[http://sketchup.google.com/3dwarehouse/|Google 3d Warehouse]]//

\\

[[http://en.wiki.mcneel.com/content/upload/files/SketchUpGetModel_20061211.exe|Download the latest build]] of the [[http://sketchup.google.com/3dwarehouse/|Google 3d Warehouse]] plugin which runs on Rhino4 Beta of December 7 or later.

 build# 12/11/06 

{{http://en.wiki.mcneel.com/content/upload/images/sketchup.jpg}}
\\

The download includes a plugin installer. After you download the installer file follow these steps to install:

  - Double click on the Installer named: [[http://en.wiki.mcneel.com/content/upload/files/SketchUpGetModel_20061211.exe|SketchUpGetMode.exe]]
  - Open Rhino 4.0
  - Go to Tools Pulldown > Options
  - Plug-ins
  - Click on the Install Button
  - Browse for the c:\Program Files\[[rhino:rhinoceros|RhinoCeros]] 4.0\Plug-ins
  - Select the SketchUpGetModel.rhp file

\\

Instructions to use:

  - Type in the command SketchUpGetModel
  - Select a model in [[http://sketchup.google.com/3dwarehouse/|Google 3d Warehouse]]
  - Click on the Download Button to get the model in Rhino
  - The model will be saved in a directory under My Documents\My 3D Warehouse
\\
