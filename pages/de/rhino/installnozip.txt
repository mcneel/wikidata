====== Das Installationsprogramm wird entpackt, aber es passiert nichts ======
> **Zusammenfassung:** //Was tun, wenn nach dem Download von Rhino nichts weiter geschieht.//
> ** Weitere Infos zu Installationsproblemen:** //[[installproblems|Problembehandlung bei der Installation von Rhino 5]]//

=====Problem=====
Das Installationsprogramm wird entpackt, aber danach passiert nichts mehr.

=====Ursache=====
Dieses Problem wird vom Microsoft PCA (Program Compatibility Assistant) verursacht, weil vielleicht ein früherer Installationsversuch von Rhino abgebrochen wurde.

=====Antwort=====
  - Starten Sie Windows neu
  - Öffnen Sie den Datei-Explorer und wechseln Sie ins Verzeichnis der heruntergeladenen Datei.
  - Halten Sie die Umschalttaste gedrückt
  - Klicken Sie mit der rechten Maustaste auf das Installationsprogramm und führen Sie es aus.
  - Es wird eine kleiner Dialog zur Auswahl der Sprache angezeigt.
  - Lassen Sie die Umschalttaste los
  - Wählen Sie Ihre Sprache und fahren Sie fort
**//Hinweis://** Es kann vorkommen, dass Sie eine Nachricht erhalten, wonach Sie versuchen würden, Rhino auf einem nicht unterstützten Betriebssystem zu installieren. Dies Lösung zu diesem Problem ist in **[[http://wiki.mcneel.com/de/rhino/rhino5/win7_not_supported|diesem Artikel]]** beschrieben.
