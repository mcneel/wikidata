====== Install Error 1619 (Rhino 5 for Windows) ======
> **Summary:** //How to fix the installation Error 1619.//
> **Related installation help:** //[[installproblems|Troubleshooting Rhino 5 Installation]]//

=====Problem=====
The Windows Installer returns error code 1619 when the installation package can not be opened.  Most likely, the reference to the .msi file in the command line is in error.

=====Answer=====
To fix this problem, restart the computer and then run the installer in steps.  The steps are included in [[rhino/error1603|Install Error 1603]].
