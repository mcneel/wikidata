====== Type de licence de Rhino 5======

===Choisir :===
    * **[[fr:rhino:install:wizard:start:windows:5:commercial|Licence individuelle commerciale et évaluation de 90 jours]]**
      * Un utilisateur est autorisé par licence
      * La licence peut être utilisée de façon autonome ou sur un réseau local (sauf pour l'évaluation) en utilisant le gestionnaire gratuit de licences flottantes [[https://wiki.mcneel.com/fr/zoo/home|Zoo]]
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Contrat de licence de l’utilisateur final]]

    * **[[fr:rhino:install:wizard:start:windows:5:educational|Licence éducation individuelle]]**
      * Un utilisateur est autorisé par licence
      * Installation uniquement de manière autonome
        * L'ordinateur doit appartenir à l'utilisateur
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Contrat de licence de l’utilisateur final]]

    * **[[fr:rhino:install:wizard:start:windows:5:lab|Licence Lab éducation pour 30 utilisateurs]]**
      * 30 utilisateurs simultanés sont autorisés par licence
      * La licence peut être utilisée sur le réseau local d'une école en utilisant le gestionnaire gratuit de licences flottantes [[https://wiki.mcneel.com/fr/zoo/home|Zoo]] pour Windows (recommandé) ou sur des installations autonomes.
      * L'installation de Rhino est facilement [[https://wiki.mcneel.com/fr/rhino/installingrhino/5.0|déployée]]
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Contrat de licence de l’utilisateur final]]
