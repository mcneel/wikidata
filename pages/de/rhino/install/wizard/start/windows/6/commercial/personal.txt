====== Installierung von Rhino 6 zur Verwendung auf mehreren Computern ======

>Hinweis: Verwenden Sie diese Option, um Ihre kommerzielle Einzelnutzerlizenz auf mehreren Computern gleichzeitig zu verwenden. Für diese Art der Lizenzimplementierung sind eine gelegentliche Internetverbindung sowie ein Log-in bei Rhino Accounts erforderlich. Teilen Sie Ihre Anmeldeinformationen für das Rhino-Konto mit niemandem.

  - **[[https://www.rhino3d.com/download/Rhino/6.0/release|Download]]** des Rhino-Installationspakets
    - Geben Sie Ihre E-Mail-Adresse ein und klicken Sie auf "Weiter"
    - Klicken Sie auf "Jetzt Deutsch herunterladen" oder wählen Sie eine andere Sprache
  - **Führen Sie das Installationsprogramm aus** - Lokale Administratorrechte werden benötigt
    - Akzeptieren Sie die AGB und die Datenschutzrichtlinien
      - Klicken Sie zum vorherigen Lesen auf die Links - empfohlen
    - Klicken Sie auf "Jetzt installieren"
      - Optional - Klicken Sie auf das Zahnrad-Symbol für weitere Optionen
    - Klicken Sie auf "Schließen", wenn die Installation abgeschlossen ist
  - **Bereitstellung Ihrer Lizenz**
    - Starten Sie Rhino
      - Rhinoceros 6 benötigt eine Lizenz:
        - {{:de:rhino:install:wizard:start:windows:6:commercial:v6_license.png?400|}}
        - Wählen Sie "**Einloggen**"
          - Daraufhin wird sich Ihr Webbrowser öffnen und Sie erhalten eine Aufforderung, sich in Ihr Rhino-Konto einzuloggen.
          - Geben Sie Ihre Anmeldeinformationen für das Log-in ein
            - Erstellen Sie ein neues Konto falls Sie keines haben
          - Fügen Sie Ihre Lizenz Ihren **persönlichen** Lizenzen hinzu
          - Schließen Sie das Browserfenster wenn das Log-in abgeschlossen ist
        - Rhino wird gestartet

  - **Rhino auf anderen Computern starten**
    - Starten Sie Rhino
      - Rhinoceros 6 benötigt eine Lizenz:
        - Wählen Sie "Einloggen"
        - Geben Sie Ihre Anmeldeinformationen für das Log-in bei Rhino Accounts ein
        - Rhino wird gestartet

=== Ist die Rhino-Installation fehlgeschlagen? ===
  * Wenn das Rhino-Installationsprogramm fehlschlägt, erhalten Sie in einem Dialogfenster Informationen und Anleitungen.
  * Haben Sie einen [[https://wiki.mcneel.com/de/rhino/6/installproblems|MSI-Fehlercode]] erhalten?
  * Möglicherweise werden Sie afgefordert, das Fehlerprotokoll zu öffnen, es in einer Datei zu speichern und es an den Technischen Support von McNeel zu senden: <tech@mcneel.com>
