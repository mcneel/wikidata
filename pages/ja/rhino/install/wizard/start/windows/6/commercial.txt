====== Rhino V6 単一ユーザー商用版ライセンス ======


===以下のうち1つを選んでください:===
    * **[[jp:rhino:install:wizard:start:windows:6:commercial:czpersonal|複数のコンピューター、お1人での使用（単一ユーザー）]] - Cloud Zoo（個人）**
        * ライセンスをユーザーお1人で所有の複数のコンピューターで共有されたい場合はこのオプションをご利用ください
        *時々インターネットへのアクセスが必要です
        *[[https://www.rhino3d.com/jp/6/eula|エンドユーザー使用許諾契約書]]

    * **[[jp:rhino:install:wizard:start:windows:6:commercial:czteam|複数のコンピューター、複数のユーザー]] - Cloud Zoo（チーム）**
        * 複数のユーザーがライセンスを共有している場合はこのオプションをご利用ください
        * 時々インターネットへのアクセスが必要です
        * [[https://www.rhino3d.com/jp/6/eula|エンドユーザー使用許諾契約書]]

    * **[[jp:rhino:install:wizard:start:windows:6:commercial:zoo|社内LANでの複数のユーザー]] - Zoo**
        * 社内LAN限定で複数のユーザーがライセンスを共有している場合はこのオプションをご利用ください
        * 無償のフローティンライセンスマネージャー、Windows **[[https://wiki.mcneel.com/ja/zoo/home|Zoo]]** が必要です
        * Rhinoのインストールは簡単に** [[https://wiki.mcneel.com/ja/rhino/installingrhino/6|展開]] **することができます
        * インターネットアクセスは不要です
        * [[https://www.rhino3d.com/jp/6/eula|エンドユーザー使用許諾契約書]]

    * **[[jp:rhino:install:wizard:start:windows:6:commercial:stand_alone|コンピューター1台でのライセンス管理]]**
      * コンピューターを1台しかお使いにならない場合はこちらのオプションをご利用ください
      * 1台のコンピューター**専用**のライセンスを確認します
      * バリデーションのため1度インターネットにアクセスするか、 [[https://wiki.mcneel.com/ja/zoo/rhinovalidateoffline|オフラインでバリデーションを行います]]
      * バリデーション後はアップデート以外、インターネットアクセスは不要です
      * [[https://www.rhino3d.com/jp/6/eula|エンドユーザー使用許諾契約書]]

>こちらのウィキページはRhinoインストールウィザードの一部です。[[jp:rhino:install:wizard:start |始めからやり直す]]
