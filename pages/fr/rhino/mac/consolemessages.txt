=====Recueillir les messages de la Console générés par Rhino=====

**Pour trouver les messages de la Console**

  - Dans **Applications > Utilitaires**, lancez le programme **Console**.
  - Dans la colonne de gauche, cliquez sur **Tous les messages**.
  - Dans la case **Filtre**, tapez **rhinoceros**.

Vous voyez maintenant les messages du journal qui concernent Rhino. Un certain nombre de messages auront probablement la même date et heure. 

{{:rhino:mac:consolemessages.png?direct&300|}}

**Pour sélectionner et envoyer ces messages** 
  - Cliquez sur le dernier puis faites défiler jusqu'au message le plus récent que vous voulez envoyer à McNeel.  
  - **Appuyez sur Maj et cliquez** sur ce message pour sélectionner tous les messages entre les deux.
  - Appuyez sur  **Commande+C** pour copier les messages.
  - Commencez un nouveau message électronique
  - Collez les messages de la Console dans le corps du message. Indiquez pourquoi vous nous envoyez ces messages.
  - Envoyez le tout à <mactech@mcneel.com>.

Fermez la **Console**.
