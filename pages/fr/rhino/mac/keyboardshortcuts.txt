====== Raccourcis des menus dans Rhino pour Mac ======

La version WIP 2012-12-17 de Rhino pour Mac ajoute des raccourcis de menu et de clavier à Rhino.  Cet article décrit comment les utiliser. 

Si vous avez utilisé les Préférences Système pour attribuer des raccourcis de menu dans Rhino (comme décrit [[http://mac.rhino3d.com/forums/t/2224.aspx|ici]]), nous vous recommandons de supprimer ces paramètres avant d'utiliser le nouveau système de raccourci clavier intégré. Les paramètres des Préférences Système interféreront avec le système intégré de façon indéfinie. 

=== Raccourcis standards du menu de Rhino ===

Rhino pour Windows présente un grand nombre de raccourcis clavier et, dans la plupart des cas, Rhino pour Mac a adopté les mêmes. Une partie de ces raccourcis n'est pas optimale pour Mac, mais nous pouvons utiliser l'assignation actuelle des touches comme point de départ d'une discussion.  Rhino pour Mac aura toujours un ensemble de raccourcis claviers définis par défaut qui ne pourra pas être modifié et il vaut mieux que cet ensemble de raccourcis soit utile et naturel pour les utilisateurs de Mac.  N'hésitez pas à suggérer des modifications au niveau des raccourcis clavier définis par défaut si vous pensez qu'ils seraient intéressants pour tous les autres utilisateurs, pour cela, utilisez [[http://mac.rhino3d.com/forums/t/3417.aspx|ce sujet du forum]].

Vous pouvez également définir pour vous-même un ensemble de raccourcis clavier, il n'est donc pas nécessaire de mentionner les raccourcis spécifiques à votre utilisation de Rhino pour Mac.  Vous pouvez toujours ajouter ces définitions spéciales à votre propre copie de Rhino.  Veuillez limiter vos suggestions ici aux modifications qui devraient être disponibles pour tout le monde.

J'ai mentionné que certains raccourcis clavier définis par défaut n'étaient pas optimaux. Certains raccourcis clavier définis par défaut utilisent des touches disponibles uniquement sur un [[http://store.apple.com/us/product/MB110LL/B/apple-keyboard-with-numeric-keypad|clavier disposant d'un pavé numérique]].  Par exemple, Vue > Annuler un changement de vue est associé à la touche Début, mais cette touche n'existe pas sur les claviers d'ordinateurs portables.  Plutôt que de supprimer ces raccourcis complètement, nous avons pensé qu'il serait mieux de les laisser afin qu'ils soient disponibles pour ceux qui peuvent en profiter.

D'autres raccourcis clavier définis par défaut utilisent les touches de fonction (F1-F12 ou F1-F19 sur la version avec clavier numérique).  Apple a ajouté ses propres actions sur le système aux touches de fonction, ce qui peut rendre peu approprié leur utilisation dans une application.  Vous devez maintenir la touche fn lorsque vous appuyez sur une des touches de fonction pour que le programme considère que vous avez utilisé une touche de fonction normale.

Vous pouvez également modifier le mode de fonctionnement de la touche fn + fonction dans Préférences Système > Clavier > Clavier. Vous y trouverez la case "Utiliser les touches F1, F2 et ainsi de suite, comme des touches de fonction standard".  Cette option n'est pas cochée par défaut, mais son activation rend l'utilisation des touches de fonction beaucoup plus facile dans Rhino.  N'oubliez pas que ce paramètre concerne tout le système et que vous changez le comportement des touches de fonction sur tout votre Mac.

=== Raccourcis de menu et Raccourcis clavier ===

Dans Rhino pour Windows, les utilisateurs peuvent ajouter des raccourcis clavier.  Ces raccourcis peuvent être associés à un élément de menu de l'application. Dans Windows, un utilisateur entre la chaîne de commandes à exécuter lorsqu'un raccourci clavier est tapé.

Les recommandations d'Apple indiquent que toutes les commandes des applications devraient être disponibles dans le menu de l'application. Afin d'appliquer cette recommandation, Apple prend en charge les raccourcis de menu, le même concept que les raccourcis clavier dans Windows, mais Apple exige que ces raccourcis soient toujours rattachés à un élément de menu.

Rhino dispose d'un large éventail  de commandes qui peuvent être personnalisées et il n'est pas très pratique de rattacher chaque commande de Rhino, ainsi que toutes ces variations, au menu de l'application.  Certaines commandes, moins souvent utilisées dans Rhino, doivent être tapées sur le clavier afin de les exécuter.  Ceci signifiait aussi, que sur Mac, toute commande qui n'était pas dans le menu de l'application ne pouvait pas être rattachée un raccourci de menu.

Nous trouvions cela trop restrictif pour Rhino qui possède un large éventail de commandes et nous avons modifié les techniques d'attribution des raccourcis menu afin de permettre l'assignation de raccourcis clavier à n'importe quelle commande, et non pas uniquement aux commandes se trouvant dans le menu de l'application.

=== Voir les raccourcis clavier standard ===

Dans Rhino pour Mac, les raccourcis claviers sont rattachés à des macros de commandes au lieu d'être rattachés directement à un élément de menu.  Vous pouvez voir les raccourcis clavier par défaut en sélectionnant Rhinoceros > Commandes > Personnaliser dans le menu de l'application, puis en sélectionnant l'onglet Raccourcis clavier. 

{{:rhino:mac:default_keyboard_shortcuts.png?700|}}

La liste des raccourcis clavier définis par défaut se trouve dans la partie supérieure de la fenêtre.  En cliquant sur un des raccourcis, la commande attribuée à celui-ci est sélectionnée dans la partie inférieure, et les informations sur la commande sont affichées dans le panneau de droite.  Ce panneau dispose d'un nouveau champ utilisé pour définir un raccourci clavier.

Si la macro de commandes est rattachée à un élément de menu de l'application, le raccourci clavier sera affiché dans le menu de l'application.

{{:rhino:mac:default_menu_shortcut_example.png?300|}}

Cependant, même si le raccourci n'est pas rattaché un élément de menu de l'application, il fonctionnera toujours dans Rhino pour Mac.

=== Définir vos propres raccourcis clavier ===

Vous ne devez pas modifier les raccourcis clavier ou les commandes définis par défaut.  Rhino pour Mac aura toujours un ensemble de commandes définies par défaut. Lorsque vous ferez appel à notre service d'assistance, nous pouvons vous demander de rétablir l'ensemble de commandes défini par défaut dans Rhino pour Mac afin d'éviter toute confusion sur l'action d'un élément de menu ou d'un raccourci clavier particulier.

Pour modifier l'ensemble de commandes défini par défaut, vous devez créer une copie de celui-ci.  Cliquez sur le bouton du menu déroulant en haut à gauche de l'Éditeur de commandes, où est indiqué "Default (read-only)".  Cliquez sur l'option "Duplicate" et indiquez le nom de votre nouvel ensemble de commandes. 

=== Suppression des raccourcis clavier ===

Pour supprimer des raccourcis clavier, sélectionnez l'onglet Keyboard Shortcuts en haut de la fenêtre, puis sélectionnez le raccourci que vous voulez supprimer.  Cliquez sur le bouton "-" au milieu à gauche de la fenêtre pour supprimer le raccourci.  Cette opération ne supprime pas la commande elle-même, elle supprime uniquement le raccourci assigné à la commande.

Vous remarquerez que tous les raccourcis de Rhino sont indiqués, y compris ?C pour Copier et ?Q pour Quitter Rhino. Vous pouvez créer une certaine confusion en supprimant et en modifiant les raccourcis clavier standard de Mac pour les assigner à des commandes de Rhino.  Ceci n'est pas recommandé mais tout de même possible.

=== Ajout de raccourcis clavier ===

Pour ajouter un raccourci clavier, vous devez d'abord trouver la commande qui recevra le raccourci en tapant une partie de son nom dans le champ de recherche en bas à gauche.  Sélectionnez le bouton de commande afin d'afficher les informations dans le panneau inférieur droit  Cliquez dans le champ "Keyboard shortcut" et tapez le raccourci que vous voulez attribuer. Par exemple, si vous voulez assigner le raccourci Commande-Maj-J à une commande, maintenez la touche Commande ainsi que la touche Maj tout en tapant la lettre J. Presque tous les raccourcis claviers sont désactivés lorsque vous tapez dans ce champ, vous pouvez donc utiliser presque n'importe quelle combinaison pour un raccourci.  La touche Effacer fonctionne toujours et vous pouvez donc l'utiliser pour supprimer un raccourci assigner à une commande.

Si vous voulez assigner un raccourci clavier à une macro de commandes qui ne se trouve pas dans l'ensemble de commandes, vous devez tout d'abord créer une nouvelle macro.  Cliquez sur le bouton "+" en bas à gauche pour créer une nouvelle commande Untitled. Remplissez les différents champs et tapez le raccourci que vous voulez utiliser.

=== Remarques sur vos raccourcis clavier ===

Aucun système ne recherche les erreurs sur les raccourcis clavier.  Rhino vous laissera ainsi assigner le même raccourci à plusieurs commandes. Vous pouvez également assigner des raccourcis qui sont réservés par OS X et qui ne fonctionneront donc pas dans Rhino.  En effet, le système d'exploitation lancera tout d'abord la commande qui correspond à ce raccourci.

Les raccourcis clavier ne sont pas limités à des combinaisons avec les touches Commande, Option, Maj ou Contrôle. Tout comme dans Adobe illustrateur, vous pouvez assigner une lettre simple à une commande.  Par exemple, vous pouvez assigner **C** à la commande Cercle.  Lorsque vous taperez la lettre C alors qu'aucune commande n'est active, la commande Cercle sera lancée.  Si vous assignez la C en tant que raccourci d'une commande de Rhino, vous ne pourrez pas taper directement d'autres commandes commençant par la C.

En revanche, vous pourrez toujours taper ces commandes en ouvrant la fenêtre Command Search avec ?K ou en sélectionnant Outils > Commandes > Search... dans le menu de l'application, puis en tapant le nom de la commande dans la boîte de dialogue. 

=== Raccourcis et alias ===

Les raccourcis clavier d'une seule lettre sont différents des alias des commandes.  Les premiers n'ont pas besoin de la touche Retour pour être activés.  Les deuxièmes peuvent être constitués de plusieurs lettres et nécessitent l'utilisation de la touche Retour pour être lancés. Le raccourci clavier "C" est complètement différent de l'alias "C" et ils exécuteront différentes commandes. 

=== Ensembles de commandes multiples ===

Vous pouvez créer plusieurs ensembles de commandes.  Si vous avez déjà un ensemble de commandes et que vous souhaitez tester les raccourcis d'une seule lettre, dupliquez un ensemble et modifiez la copie pour faire des essais. 

Utilisez Rhinoceros > Commands > "Your command set name" dans le menu de l'application pour passer d'un ensemble à un autre. Les raccourcis clavier et de menu sont automatiquement actualisés lorsque vous modifiez l'ensemble de commandes. Pas besoin de relancer Rhino pour commencer à utiliser le nouvel ensemble de commandes.
