====== Scripts de Python en Rhino para Mac ======
Rhino para Mac es compatible con un plug-in de prueba que añade scripts de Python a Rhino. Para tener esta opción, debe:

  - [[http://www.rhino3d.com/download/wenatchee/5.0/wip|Instalar la última versión de Rhino OSX]]
  - Ejecutar esta nueva versión de Rhino. \\ Se registrará la extensión .macrhi con la aplicación de Rhino. Solo es necesario hacerlo una vez. Ahora, al hacer doble clic en cualquier archivo .macrhi, se ejecutará Rhino.
  - [[http://download.rhino3d.com/IronPython/5.0/wip/download/|Descargar el instalador de plug-ins .macrhi IronPython]]. \\ Se trata de un instalador de plug-ins de Rhino para Mac para Python. 
  - Haga doble clic en el archivo para instalar el plug-in en Rhino. \\ Rhino para Mac se iniciará y se instalará el plug-in.
  - Cierre y reinicie Rhino. \\ Ahora debería aparecer el nuevo comando **EjecutarScriptDePython** (_RunPythonScript).

====Comando EjecutarScriptDePython====
Hay dos versiones de este comando:

**EjecutarScriptDePython** - Muestra el diálogo Abrir archivo que permite seleccionar un script de Python (archivo .py) El plug-in incluye un conjunto de scripts de ejemplo que se pueden ejecutar y el diálogo debe dirigir a esta carpeta la primera vez que se ejecuta.

**-EjecutarScriptDePython** - La versión "con guión" de este comando permite introducir un nombre de archivo de script. Si el script existe en el directorio Scripts de su equipo, se ejecutará. Puede utilizar el punto para ejecutar un script en un subdirectorio del directorio Scripts. Por ejemplo, si desea ejecutar el archivo CurveLength.py en el subdirectorio Samples, puede escribir samples.curvelength en la solicitud.

**Nota**: Todas las muestras incluidas en este plug-in deben funcionar tanto en [[http://mac.rhino3d.com|la WIP de Rhino para Mac]] como en [[http://www.rhino3d.com/|Rhino 5 para Windows]] .
