===== Zoo 또는 단일 컴퓨터 설치에서 라이선스 제거하기 =====

사용 중인 라이선스를 Rhino 계정에 추가하려면, 우선 사용 중인 Zoo 또는 장치에서 라이선스를 제거해야 합니다.


==== Zoo에서 라이선스 제거하기 ====
  - **Zoo Admin 어플리케이션을 시작합니다**.
  - **제거하려는 라이선스를 클릭합니다**.
  - **Delete** 단추를 클릭합니다.

==== 단일 컴퓨터에서 라이선스 제거하기: ====
  - **Rhino를 시작합니다**.
  - **도구** 메뉴에서 **옵션**을 클릭한 후, **라이선스**를 클릭하고, **라이선스 키 변경**을 클릭합니다.
  - 대화상자의 아래쪽에 있는 **라이선스 제거**를 클릭합니다.

==== Rhino 계정에 라이선스를 추가하려면 ====
  - [[https://www.rhino3d.com/licenses/?controller=add_licenses|라이선스 추가 페이지]]를 방문합니다.
  - 귀하의 계정 또는 귀하가 속한 팀에 라이선스를 추가합니다.

[[https://wiki.mcneel.com/ko/rhino_accounts/home|Cloud Zoo 홈페이지로 되돌아가려면 여기를 클릭]]하세요.
