====== Automating installation of Flamingo NXT ======


The Flamingo installer can be automated by System Administrators who want to deploy Rhino 5 throughout an organization.

These techniques will work for automated installation by running the MSI from a folder that contains a customized FlamingoSetup.ini.

The downloaded FlamingoNXT installer is a 7-Zip archive that can be extracted into it original folder structure. Withing the Installers folder will be an example FlamingoSetup.ini which can be modified for an automated install.

The specifics for modifying the ini file can be found on the [[rhino:installingrhino:5.0|Rhino Installer instructions]].
