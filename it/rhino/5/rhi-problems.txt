=====Risoluzione dei problemi di installazione dei file RHI=====

I file **RHI** sono dei pacchetti di installazione di Rhino.\\
Essi semplificano la creazione e l'installazione dei programmi di installazione dei plug-in per Rhino.

==== Problemi di installazione ====
-->Windows non riesce ad aprire questo file#
**Causa:** Le associazioni di file tra Windows ed i file RHI non sono correttamente impostate.
\\
**Soluzione:**
  - Nel Pannello di controllo, aprire Programmi e funzionalità
  - Cliccare con il tasto destro su Rhinoceros 5 e quindi su Ripara
  - Cliccare con il tasto destro su Rhinoceros 5 a 64 bit e quindi su Ripara
  - Fare doppio clic sul programma di installazione RHI per installarlo.
\\
Se non funziona, seguire le istruzioni per l'**Associazione manuale dei file RHI** qui sotto.
<--

-->Pacchetto non compatibile#
Ci sono vari motivi che causano questo errore. I più probabili sono:
  - **Download danneggiato:** un file RHI danneggiato che non può essere installato. Spesso ciò risulta da un download incompleto. Eliminare il file RHI, cancellare i file temporanei del Web browser ed effettuare di nuovo il download.
  - **RHI creato in modo scorretto:** Se siete sviluppatori, è possibile che non abbiate creato correttamente il file RHI. Seguire le istruzioni per la [[developer/rhinoinstallerengine/authoring|creazione dei file RHI]].
\\
<--

-->Errore di apertura del file di Rhinoceros 5.0#
**Causa:** In qualche modo, Rhino 5 è stato associato erroneamente ai file RHI.
\\
**Soluzione:** Seguire le istruzioni per l'**Associazione manuale dei file RHI** qui sotto.
<--

-->Associazione manuale dei file RHI#
-->Su Windows a 64 bit#
  - Cliccare con il tasto destro sul file RHI, quindi fare clic su **Apri con...**
  - Nella finestra di dialogo, spuntare la casella **Usa sempre il programma selezionato per aprire questo tipo di file**.
  - Nella finestra di dialogo Apri, cliccare su **Sfoglia...**
  - Andare a **C:\Programmi\Rhinoceros 5 (64-bit)\System\x64**, quindi selezionare **rhiexec.exe** (Se avete installato Rhino in un percorso diverso, seguite quel percorso)
  - Cliccare su **Apri**
  - Cliccare su **OK**

<--

-->Su Windows a 32 bit#
  - Cliccare con il tasto destro sul file RHI, quindi fare clic su **Apri con...**
  - Nella finestra di dialogo, spuntare la casella **Usa sempre il programma selezionato per aprire questo tipo di file**.
  - Nella finestra di dialogo Apri, cliccare su **Sfoglia...**
  - Andare a **C:\Programmi\Rhinoceros 5\System\x86**, quindi selezionare **rhiexec.exe** (Se avete installato Rhino in un percorso diverso, seguite quel percorso)
  - Cliccare su **Apri**
  - Cliccare su **OK**
