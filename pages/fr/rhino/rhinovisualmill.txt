Module //RhinoCAM// :

Le module //RhinoCAM// sert de pont de données entre Rhino 3.0 et Visual Mill 5.0 ce qui permet un transfert fluide des données de conception entre Rhino et Visual Mill. Ce produit permet d'avoir Rhino et Visual Mill en même temps et il facilite le transfert de données entre les deux logiciels, créant ainsi un véritable environnement CFAO pour l'utilisateur.

Afin d'utiliser le module, Rhino 3.0 devra être installé sur votre ordinateur. Une fois le module installé, l'élément //RhinoCAM// est inséré dans la barre de menu de Rhino 3.0. Ce menu vous permettra d'accéder aux fonctions qui transfèrent les données vers Visual Mill 5.0.

[[http://www.mecsoft.com/Mec/Downloads/plugins.shtml|Ce module peut être téléchargé ici ]]
