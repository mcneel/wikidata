====== La WIP est limitée dans le temps. Que se passe-t-il quand la version expire? (Rhino 5 pour Mac) ======
> **Résumé:** //Rhino pour Mac est limité dans le temps. Que se passe-t-il quand la version expire?//

Cette expiration permet de s'assurer que tous les utilisateurs testent la dernière version et qui ne communiquent pas des problèmes qui ont déjà été corrigés.  Avant que chaque version WIP de Rhino n'expire, une nouvelle WIP sortira et pourra être [[http://www.rhino3d.com/go/download/rhino-for-mac/wip/latest|téléchargée]]. RhinoWIP recherche les mises à jour à chaque démarrage et vous avertit lorsqu'une nouvelle version est disponible. Un message d'information sera également publié dans la [[http://discourse.mcneel.com/category/rhino-for-mac|section de Rhino pour Mac]] du [[http://discourse.mcneel.com/ |forum de discussion]] chaque fois qu'une nouvelle version WIP sera disponible.
