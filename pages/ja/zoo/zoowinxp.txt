====== LAN Zoo が Windows XP で動作しません ======
> **製品:** //[[ja:zoo:home|Zoo]]//
> **要約:** //Windows XP での LAN Zoo の動作に関する説明//

===== 質問 =====
Windows XP で LAN Zoo を実行しようとしているのですが、起動するとすぐ LAN Zoo がフリーズしてしまい、全く動きません。Windows XP は LAN Zoo でサポートされていないのしょうか?

===== 回答 =====
  - Zooは Microsoft .NET Framework 4.8 が必要です。
  - .NET Framework 4.8 およびそれ以降のバージョンは Windows XP でサポートされて__いません__。
