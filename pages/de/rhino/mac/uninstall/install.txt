====== Muss ich Rhino deinstallieren, bevor ich eine neuere Version installiere? (Rhino für Mac) ======
> **Zusammenfassung:** //Muss ich Rhino für Mac deinstallieren, bevor ich eine neuere Version installieren kann?//

Nein, das ist nicht nötig.  Sie können das Programmpaket - meist zu finden im Order "Programme" - überschreiben, wenn Sie eine neuere Version installieren.  Damit wird weder Ihre Lizenz noch irgendeine Ihrer Einstellungen überschrieben. Diese befinden sich jeweils in einem anderen Ordner.  

Tatsächlich können Sie, wenn Sie möchten, zwei Rhino-Versionen nebeneinander ausführen. Benennen Sie einfach das Programmpaket (Rhinoceros.app) zu etwas anderem um (z.B. RhinocerosCopy.app).
