====== Mac용 Rhino에서 RhinoScript 프로그램을 실행할 수 있나요? ======
> **요약:** //Mac용 Rhino에서 Rhinoscript를 사용할 수 있습니까?//

아니요. RhinoScript는 VisualBasic을 기본으로 하며, VisualBasic은 OS X에서 실행되지 않습니다. 

Mac과 Windows 버전 모두에서 사용하는 스크립팅 작성에는 Python 사용을 권장합니다. Windows와 Mac 모두에서 사용하는 Python 스크립팅 지원은 [[http://discourse.mcneel.com/category/scripting | Rhino 스크립팅 지원 포럼]]을 참조하세요.
