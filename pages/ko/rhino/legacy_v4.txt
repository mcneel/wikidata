=====  (64비트) Windows 7, 8.1, 10 에서 Rhino 4 실행하기 =====

Rhino 4는 Windows Xp에서 실행되도록 개발되었으나, 현대 버전의 Windows에서도 실행할 수 있습니다. Windows XP 버전 이후에 출시된 Windows에서 Rhino 4를 실행하는 데 필요한 몇 가지 필수 조건과 제한 사항을 안내합니다.

==== 제한 사항과 필수 조건 ====
  * **메모리** - 새로운 64 비트 컴퓨터 시스템은 아마도 16GB 이상의 RAM이 탑재되어 있을 것입니다. Rhino 4는 32 비트 응용 프로그램입니다. 새 컴퓨터의 탑재 RAM 용량과 상관없이, 최대 4GB RAM을 사용하는 제한이 있음을 뜻합니다. 
  * **레거시 응용 프로그램** - Rhino 4는 Microsoft가 **사용자 계정 컨트롤**을 Windows에 추가하기 전에 릴리스되었습니다. 이 환경에서 Rhino 4를 실행하려면 Rhino 4를 **Windows XP SP3 호환 모드**에서 실행해야 합니다. 그렇지 않으면 Windows는 원래 지정된 대로, Rhino를 종료할 때 Rhino에서 도구모음이나 다른 설정을 저장하지 못하도록 블록합니다. Windows 호환 모드는 바탕화면 바로가기 설정 중 일부입니다.
  * **저해상도** - 오래된 Windows XP 에서보다 더 높은 화면 해상도에서 새 컴퓨터가 실행될 것입니다. Rhino 4가 출시 될 때 지원되던 낮은 화면 해상도(예:1024 x 768)로 Windows를 제한하고 실행해야합니다. 그렇게 하지 않으면 **레이어 패널** 과 같은 일부 도구는 비어 있고 Rhino 4의 도구모음 아이콘은 너무 작아 읽을 수 없습니다.

==== Rhino 6 사용을 고려해 보세요 ====
Rhino 6의 [[https://www.rhino3d.com/6/features|새 기능과 성능]] 외에 RAM, 디스플레이 해상도와 같은 새로운 64비트 Windows 시스템의 성능을 완전히 활용하기 위해, Rhino 6로의 업그레이드를 고려해보시기 바랍니다. 지난 2007년 1월 Rhino 4가 릴리스되었을 때, Rhino 4에서 32비트 Windows Xp의 기능이 지원되었습니다. 그 이후 Windows와 컴퓨터 하드웨어는 많은 발전을 이루어왔습니다. 이러한 새 기능과 성능을 최대한 활용하려면 그에 맞게 개발된 Rhino 버전을 사용하셔야 합니다.\\
