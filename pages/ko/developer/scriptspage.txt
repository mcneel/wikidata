====== 매크로, 스크립트, 도구모음, 작업영역 ======

======Rhino 사용자들이 공개하는 Rhino 매크로, 스크립트, 도구모음======

=====소개=====

매크로와 스크립트는 작업의 자동화된 시퀀스를 가능하게 하는 Rhino 명령과 옵션의 문자열을 말합니다. 이러한 스크립트 (시퀀스) 는 "단추 누르기" 또는 어느 한 앨리어스의 입력을 통하여 반복 실행될 수 있습니다. 더욱 복잡한 스크립트 작업데 대해서는 [[developer:rhinoscript|RhinoScript]] 페이지를 참조하시기 바랍니다.

매크로와 스크립트를 공유하고자 하시는 사용자 여러분들께서는 이 wiki 페이지를 이용해 주시기 바랍니다. 첫 번째 단락에서는 스크립트가 소개된 외부 사이트가 소개되어 있습니다. 두 번째 단락은 스크립트 또는 매크로를 직접 포스트하실 수 있는 공간입니다. 세 번째 단락에서는 사용자 지정된 도구모음과 작업영역 (workspaces) (.tb 파일) 을 공유하실 수 있습니다. 마지막 부분에서는 매크로, 스크립트, 도구모음 설정, 가져오기, 설치에 대한 설명 링크가 소개되어 있습니다.

플러그인과 관련하여, //[[http://www2.rhino3d.com/resources/default.asp?language=|Rhino 리소스 페이지]] ** , **[[http://kr.wiki.mcneel.com/default.aspx/McNeel/RhinoHomeLabs.html|Rhino Labs]] ** 을 참조하시거나 하단에 소개된 외부 사이트를 참조하시기 바랍니다.

=====스크립트가 있는 외부 사이트 소개=====

주: 링크와 간략한 설명을 추가해 주시기 바랍니다. 웹사이트 없이 스크립트만을 공유하시려면 여기에 별도로 페이지를 만드실 수 있습니다.

**[[http://www.reconstructivism.net/index.htm?scripts.htm|David Rutten - Reconstructivism.net 사이트 (영어) ]]**
놀라운 도구들로 가득합니다!

**[[http://www.rhino3.de/|Jess Maertterer - Rhino 3DE 사이트 (영어) ]]**
다양한 스크립트, 플러그인과 튜토리얼

**[[http://www.ob.com/tutorials/#scripts|Off Broadway Rhino 스크립트 페이지 (영어) ]]**
30 개 이상의 Rhino 스크립트

**[[ClementGreinersScriptsPage|Clement Greiners 스크립트 페이지 (영어) ]]**
스크립트와 [[developer:rhinoscript|RhinoScript]] 함수

**[[GCWebsterScriptsPage|G. C. Webster 스크립트 페이지 (영어) ]]**
스크립트, 작업영역, 매크로.

** [[http://www.rayflectar.com/Rhino/RhinoScripts-Gallery.htm|Thomas Anagnostou 스크립트 (영어) ]]**
GearGen (involute gears), Harmony (distribute), IsoMatch, 등.

** [[PeterHarris|Peter Harris' tools (영어) ]]**
Monkey 를 사용하여 스크립트로 플러그인을 만들었습니다. 플러그인 하나에 많은 명령이 포함되어 있어 설정하기 쉽습니다.

** [[http://2e5.com/kite/parametric/|Parametric Kite Design with Rhino (영어) ]]**
2e5.com 에 소개된 글: Python 을 사용하여 매개변수적 카이트 디자인에 사용될 Rhino scripts 만들기.

=====스크립트와 매크로=====

주: 공유하고자 하시는 스크립트와 매크로를 올려주세요. 스크립트나 매크로가 몇 줄 이상 되신다면, .txt 또는 .rvb 파일 형태로 업로드하셔서 이 페이지에는 짧막한 설명과 함게 링크를 적어주시기 바랍니다. 설치 방법에 대해서는 페이지 아랫부분을 참조하시기 바랍니다.

**[[http://wiki.mcneel.com/_media/legacy/en/NamedViewMgr071.zip|Named View Manager v0.7.1 - Mattias D'Hooghe]]**

Named View Manager 는 표준 명명된 뷰 대화 박스를 대체하며, 보다 연장된 기능을 포함하고 있습니다.

이를 사용하여 명명된 뷰를 추가, 이름 바꾸기, 삭제할 수 있으며, 명명된 뷰를 신속하게 불러올 수 있습니다. 레이어 상태를 명명된 뷰에 첨부할 수 있으며, "CPlanes"을 명명된 뷰에 첨부할 수 있습니다. 생생한 슬라이드쇼 프레젠테이션도 가능합니다. 사용 설명에 대해서는, zip 파일에 포함된 readme 텍스트 파일을 참조하시기 바랍니다.

** [[http://wiki.mcneel.com/_media/legacy/en/Intersect-2-sets.rvb|Custom intersection script - Emilio Morello]]**
이 스크립트를 사용하여 두 개체 집합을 선택할 수 있으며, 각 집합 내부의 교차 없이도 커브 사이에 교차를 만듭니다.

**[[http://wiki.mcneel.com/_media/legacy/en/Drawer-script.zip|Drawer script for carpenters/interiorbuilders - Tobias Nagel]]**

옆 부분 사이에 앞과 뒤 부분이 들어가는 전통적인 형태의 옷장을 만들 수 있습니다. 모든 사이즈는 캐비넷과 재질, 옷장 레일 두께 내부의 사용 가능한 공간을 기준으로 계산됩니다. 설치 및 사용법은 zip 파일에 포함된 메뉴얼을 참조하세요.

**[[http://wiki.mcneel.com/_media/legacy/en/smarttarget.zip|Smart Target]] 과 [[http://wiki.mcneel.com/_media/legacy/en/levelcamera.zip|Level Camera]] scripts - Sam Page**
Smart Target 스크립트에서는 개체스냅 투영과 개체스냅 설정 해제 ( Project and Disable Osnaps ) 상태를 인식하여, 현재 켜진 상태인 경우에는 투영을 해제하고, 반대의 경우에는 개체스냅을 켭니다. 새로운 대상을 선택하여 개체스냅 투영과 개체스냅 설정 해제를 각각의 상태로 되돌릴 수 있습니다.

Level Camera 스크립트는 카메라의 롤을 꺼내 대상 z 값을 카메라 z 값과 동일하도록 설정합니다.

**[[http://wiki.mcneel.com/_media/legacy/en/FitVolume.zip|Fit Volume]]**
Scales selected solid (closed polysurface) object to given volume in litres. Currently, script works with unit settings to millimetres, metres or no units.

** [[MacroCollection|매크로 모음]]**
자동화 작업과 같이 작업 속도를 향상시키는 다양한 매크로를 소개합니다.

**[[http://wiki.mcneel.com/_media/legacy/en/Inverter_applet.zip|Inverter v1.0 by Daniel Piker]]**
This inverts objects with respect to any given sphere. Inversion is a mathematical way of turning things inside out. See txt file in zip for instructions

**[[http://wiki.mcneel.com/_media/legacy/en/Sweep1Simple.zip|Sweep1Simple v0.6 by Björn Syse]]**
이 플러그인 스크립트는 Sweep1Simple" 명령을 추가합니다. 이 명령은 레일 1개와 1개 이상의 교차 단면 커브를 사용하여 스윕 실행된 서페이스를 만듭니다. 교차 단면 커브는 레일상에 배치될 필요가 없으나, 결국엔 서페이스가 레일을 따르게 됩니다. 버그를 발견하시면 <bjorn@syse.se> 앞으로 메일을 보내 주세요.
