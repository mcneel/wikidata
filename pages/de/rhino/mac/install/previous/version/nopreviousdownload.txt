====== Kann ich eine Vorgängerversion von Rhino für Mac herunterladen? ======
> **Zusammenfassung:** //In der aktuellen Version von Rhino für Mac tritt ein Problem auf. Kann ich die vorherige Version herunterladen?//

Nein, Vorgängerversionen von Rhino für Mac sind derzeit nicht verfügbar.  

Wenn Sie einen Fehler in der aktuellen stabilen Release-Version finden, berichten Sie ihn bitte in unserem [[http://discourse.mcneel.com/c/rhino-for-mac|Discourse-Forum für Rhino für Mac]].
