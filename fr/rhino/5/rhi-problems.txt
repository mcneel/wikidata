=====Identification des problèmes d'installation de fichiers RHI=====

Les fichiers **RHI** sont des packs d'installation de Rhino.\\
Ils sont prévus pour simplifier la création et l'installation de programmes d'installation de modules pour Rhino.

==== Problèmes d'installation ====
-->Windows ne peut pas ouvrir ce fichier#
**Cause :** L'association de fichier n'est pas définie correctement pour les fichiers RHI dans Windows.
\\
**Solution :**
  - Dans le panneau de configuration, ouvrez Programmes et fonctionnalités
  - Cliquez avec le bouton de droite sur Rhinoceros 5, puis sélectionnez Réparer
  - Cliquez avec le bouton de droite sur Rhinoceros 5 64-bits, puis sélectionnez Réparer
  - Double-cliquez sur le fichier RHI pour l'installer.
\\
Si cette option ne fonctionne pas, veuillez suivre les instructions données ci-dessous pour **associer manuellement des fichiers RHI**.
<--

-->Le fichier est ouvert par erreur dans Rhinoceros 5#
**Cause :** D'une certaine façon, Rhino 5 a été associé par erreur avec les fichiers RHI. 
\\
**Solution :** Veuillez suivre les instructions données ci-dessous pour **associer manuellement des fichiers RHI**.
<--

-->Associer manuellement des fichiers RHI#
-->Sur Windows 64 bits#
  - Cliquer avec le bouton de droite sur le fichier RHI, puis cliquez sur **Ouvrir avec...**
  - Dans la boîte de dialogue Ouvrir avec, sélectionnez la case **Utiliser cette application pour tous les fichiers RHI**.
  - Dans la boîte de dialogue Ouvrir avec, cliquez sur **Parcourir...**
  - Ouvrez **C:\Program Files\Rhinoceros 5 (64-bit)\System\x64** puis sélectionnez **rhiexec.exe** (Si vous avez installé Rhino dans un autre dossier, ouvrez celui-ci)
  - Cliquez sur **Ouvrir**
  - Cliquez sur **Accepter**

<--

-->Sur Windows 32 bits#
  - Cliquer avec le bouton de droite sur le fichier RHI, puis cliquez sur **Ouvrir avec...**
  - Dans la boîte de dialogue Ouvrir avec, sélectionnez la case **Utiliser cette application pour tous les fichiers RHI**.
  - Dans la boîte de dialogue Ouvrir avec, cliquez sur **Parcourir...**
  - Ouvrez **C:\Program Files\Rhinoceros 5\System\x86** puis sélectionnez **rhiexec.exe** (Si vous avez installé Rhino dans un autre dossier, ouvrez celui-ci)
  - Cliquez sur **Ouvrir**
  - Cliquez sur **Accepter**
