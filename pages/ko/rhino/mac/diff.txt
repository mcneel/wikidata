====== 기존 Windows용 Rhino 사용자를 위한 안내 ======

Windows용 Rhino를 사용하셨나요? Mac 으로 바꿀 생각이십니까? 아니면 그냥 궁금하신 건가요? 그런 분들을 위한 간단한 가이드입니다...


----

===== 도구모음은 어디에 있나요? =====
기본적으로 Mac용 Rhino는 기존에 Windows용 Rhino를 사용하셨던 분이 아니라고 가정합니다. Windows에서와 같은 도구모음을 사용하시려면 **Rhinoceros** > **환경설정** > **테마**에서 **Windows용 Rhino**를 선택하세요. 이 설정이 실행되려면 새 모델링 창을 시작해야 합니다. Mac용 Rhino 테마에 대한 안내는 다음 항목을 참조하세요...

----

===== 도구모음과 도구 팔레트 ===== 
  * Mac용 Rhino에는 //도구 단추//가 표준 Macintosh 도구모음에서 Rhino 명령을 시작합니다.
  * Mac용 Rhino에는 도구 단추 모음이 있는 //도구 팔레트//가 있습니다. \\  Mac용 Rhino를 시작할 때 자주 사용하는 도구 단추가 있는 도구 팔레트가 문서 창의 왼쪽에 열립니다.
  * 도구 팔레트 단추에는 하나 이상의 명령과 메뉴가 포함될 수 있습니다.
  * 도구 팔레트 단추에 마우스를 가져가면 단추의 도구 설명이 표시됩니다.\\  도구 설명에 2줄의 텍스트가 있으면 해당 단추로 두 개의 명령(메인, 대체)이 실행됩니다.
    * 도구 팔레트 단추를 **클릭**하면 **메인** 명령이 실행됩니다.
    * **대체** 명령을 실행하려면 **Option** 키를 누르고 해당 도구 단추를 **클릭**합니다.
  * 메뉴가 있는 도구 팔레트 단추에는 왼쪽 하단 모서리에 작은 삼각형이 있습니다.
    * 해당 메뉴를 열려면, 도구 팔레트 단추를 클릭하고 잠시 그 상태를 유지합니다.
  * 도구 팔레트 메뉴는 떠 있는 도구 팔레트로 표시할 수 있습니다.
    * 떠 있는 도구 팔레트로 표시하려면 **Option** 키를 누른 상태에서 마우스 단추를 클릭하고 잠시 그 상태를 유지합니다.

----

===== 마우스 ===== 

**권장 사항**

  * 마우스 오른쪽 단추로 쉽게 탐색하려면 스크롤 휠이 부착된 2 단추식 마우스를 권장합니다. 하지만, 단추가 1개인 마우스로도 모든 작업이 가능합니다.
  * 단추가 1개인 마우스로 마우스 오른쪽 단추 기능을 실행하려면 **Control** 키를 누른 상태에서 마우스 단추를 클릭합니다.

**탐색**

  * **마우스 오른쪽 단추로 끌기**와 **Shift+마우스 오른쪽 단추 끌기** 작업은 Windows용 Rhino와 동일합니다. [[http://docs.mcneel.com/rhino/mac/help/ko-kr/user_interface/shortcuts.htm|Rhino 도움말 - 바로 가기]]를 참조하세요. \\ 뷰 투영 스타일과 상관없이, **Shift** 키를 누르면 항상 초점 이동(pan)이 실행되고, **Command** 키를 누르면 항상 확대/축소(zoom)되며, **Command+Shift** 키를 누르면 항상 뷰가 회전(rotate)합니다.
  * 스크롤 휠을 사용하여 뷰를 확대/축소합니다.
  * 1 단추식 마우스를 사용하는 경우, **Control**, **Command**, **Shift** 키를 같이 눌러 뷰를 회전할 수 있습니다.

----

===== 오른쪽 클릭 메뉴 ===== 

  * 뷰포트에서 오른쪽 클릭하면 상황에 맞는 메뉴가 표시됩니다.
  * 기본적으로 오른쪽 클릭은 마지막 명령을 반복하는 **Enter** 키의 역할을 **하지 않습니다**. \\ 이 액션을 변경하려면 **Rhinoceros** > **환경설정** > **마우스** > **마우스**에서 **상황에 맞는 메뉴 사용**을 끕니다.

----

===== 명령행 ===== 

Windows용 Rhino의 명령행에 익숙하시다면 Windows용 Rhino 명령행의 모든 기능은 Mac용 Rhino에 있습니다. 명령을 시작하려면 입력을 시작합니다. 왼쪽 사이드바에 있는 명령 필드에 텍스트가 표시됩니다. 명령을 시작하려면 Enter 키 또는 스페이스 바를 클릭하세요. Windows용 Rhino의 프롬프트 라인에 표시되는 모든 명령 옵션이 명령 옵션 대화상자에 표시됩니다. \\ \\

명령 옵션값을 입력하시려면 각 옵션 프롬프트에는 괄호로 둘러싸인 알파벳이 있습니다. 해당 옵션을 실행하려면 해당 글자를 입력하고 Enter 키를 누르세요. 

명령 옵션 대화상자에서 값을 입력하려면 커서를 움직일 필요가 없습니다. 입력하면 Rhino에서 사용자가 입력한 값이 자동으로 대화상자에 입력됩니다. Windows용 Rhino에서의 명령행 실행 방법은 Mac용 Rhino에서도 완전히 동일합니다.

----

===== 키보드 =====

  * Windows용 Rhino에서는 많은 작업에 **Control** 키를 사용합니다. Mac 버전에서는 대신 **Command** 키를 사용합니다.
    * 예를 들어, 개체를 선택 해제할 때 **Command** 클릭은 **Ctrl** 클릭을 대체합니다.
    * 또한, **Command** 키는 **Ctrl** 키를 대신하여 탐색에 사용됩니다:

| | | |**키** | **액션** | **+ Command 키** |
| | | |**왼쪽 화살표** | 왼쪽 회전 | 왼쪽 초점 이동 |
| | | |**오른쪽 화살표** | 오른쪽 회전 | 오른쪽 초점 이동 |
| | | |**위쪽 화살표** | 위로 회전 | 위로 초점 이동 |
| | | |**아래쪽 화살표** | 아래로 회전 | 아래로 초점 이동 |

----

===== 노트북 컴퓨터 관련 사항 ===== 


**블루투스 마우스**\\ 노트북 컴퓨터는 트랙패드가 탑재되어 있으므로, 블루투스 마우스 (예:Logitech V270)를 추가로 사용하시면 유용합니다.

**키보드에 없는 키**\\ 노트북 컴퓨터의 키보드에는 별도의 **Home**, **End**, **PageDn**, **PageUp** 키가 없고, **fn** 키와 다른 키를 조합하여 같은 기능을 실행할 수 있습니다. \\

| **키** | **+ fn key** |
| **위쪽 화살표** | Page Up |
| **아래쪽 화살표** | Page Down |
| **왼쪽 화살표** | Home |
| **오른쪽 화살표** | End |
| **Delete** | Delete |
