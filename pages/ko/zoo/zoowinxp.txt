====== Windows XP에서 LAN Zoo가 실행되지 않습니다 ======
> **제품:** //[[ko:zoo:home|Zoo]]//
> **요약:** //Windows XP에서 LAN Zoo를 실행하는 방법을 안내합니다.//

===== 질문 =====
Windows XP에서 LAN Zoo를 실행하려고 합니다. 컴퓨터에서 Zoo를 시작하면 갑자기 LAN Zoo가 멈추고 아무런 반응이 없습니다. Windows XP에서 LAN Zoo가 지원됩니까?

===== 답변 =====
  - Zoo를 사용하시려면 Microsoft .NET Framework 4.8 가 필요합니다.
  - .NET Framework 4.8 및 그 이후 버전이 Windows XP에서 지원되지 __않습니다__.
