====== Tipos de licencia de Rhino 6, Rhino 7 o plug-in ======


===Elija una:===
    * **[[rhino:install:wizard:start:windows:6:commercial|Licencia Comercial monopuesto y Evaluación de 90 días]]** //<- Haga clic para obtener más información.//
      * Permite un usuario por licencia
      * Se puede usar de forma independiente o compartir a través de una LAN (Zoo) o un administrador de licencias flotantes basado en Internet (Cloud Zoo)
      * [[https://www.rhino3d.com/7/eula|Contrato de licencia de usuario final de Rhino]]

    * **[[rhino:install:wizard:start:windows:6:educational|Licencia Educacional monopuesto.]]** //<- Haga clic para obtener más información.//
      * Permite un usuario por licencia
      * Se puede usar de forma independiente en el equipo del estudiante o con el administrador de licencias personales de Cloud Zoo
      * [[https://www.rhino3d.com/7/eula|Contrato de licencia de usuario final de Rhino]]

    * **[[rhino:install:wizard:start:windows:6:lab|Licencia Lab EDU para 30 usuarios]]** //<- Haga clic para obtener más información.//
      * Permite 30 usuarios al mismo tiempo
      * Se puede usar en una red de un centro docente utilizando el gestor de licencias flotantes [[https://wiki.mcneel.com/es/zoo/home|Zoo]] o se puede instalar de forma independiente.
       * La instalación de Rhino se **implementa** fácilmente:
          * [[https://wiki.mcneel.com/es/rhino/installingrhino/7|Windows]]
          * [[https://wiki.mcneel.com/es/zoo/rhinomacdeployment|Mac]]
      * [[https://www.rhino3d.com/7/eula|Contrato de licencia de usuario final de Rhino]]
