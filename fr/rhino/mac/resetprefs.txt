====== Rétablir les préférences de Rhino ======
Suivez cette procédure pour supprimer toutes les modifications réalisées sur les préférences de Rhino et rétablir les valeurs par défaut. 

  - **Fermez** Rhino.
  - Dans le **Finder**, maintenez la touche **Option** et sélectionnez **Bibliothèque** dans le menu **Aller** menu. \\ Le dossier Bibliothèque s'ouvre.
  - Dans le dossier **Bibliothèque**,  sélectionnez le dossier **Préférences**. 
  - Trouvez le fichier **com.mcneel.rhinoceros.plist** et déplacez-le dans la **Corbeille**.  \\ Vous pouvez également déplacer le fichier sur votre bureau si vous voulez restaurer ces préférences dans le future.
