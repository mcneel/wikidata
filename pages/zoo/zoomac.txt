====== Rhino for Mac and the Zoo ======
> **Product:** //[[zoo:home|Zoo]]//
> **Summary:** //Floating Rhino 5 for Mac licenses using the Zoo 6 for Windows.//

===== Requirements =====
  * The Zoo 6 for Windows. //**[[http://www.rhino3d.com/download/zoo/6/latest|Download...]]**//
  * A Windows computer on the network with your Macs. (You can run Windows on a Mac using [[http://www.vmware.com/products/fusion/|VMWare]].)


===== Notes =====

  * Zoo 6 for Windows supports [[http://www.rhino3d.com/mac|Rhino for Mac]], as well as all of our latest [[:zoo:products|Windows products]]. 
  * //Our older Windows products require an older version of the Zoo for Windows//.
  * [[http://wiki.mcneel.com/zoo/zooosx|Zoo for OS X]] is in development.
