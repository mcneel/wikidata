====== Error de instalación 1714 (Rhino 6 para Windows) o Error del sistema 1612 ======
> **Resumen:** //Cómo resolver el error del instalación 1714//
> **Ayuda con la instalación:** //[[es:rhino:6:installproblems|Solucionar problemas con la instalación de Rhino]]//

=====Problema=====
El archivo de registro muestra el error "Rhino Installer Engine -- Error 1714. No se puede eliminar la versión anterior de Rhino Installer Engine.  Póngase en contacto con el grupo de soporte técnico.  Error del sistema 1612."

=====Respuesta=====

Este error se soluciona con una herramienta de Microsoft. Descárguela y siga estas instrucciones:

https://support.microsoft.com/es-es/help/17588/fix-problems-that-block-programs-from-being-installed-or-removed

Después de resolver el problema, intente volver a instalar Rhino.
