====== MeshRepair V4 SR1 Plugin ======
> **Summary:** //Mesh Repair Wizard//


**[[http://download.mcneel.com/download.asp?id=MeshRepair_7_31_2007&language=|Download MeshRepair Plug-in of July 30]]**

**Please note that this plugin works with Rhino V4 SR1 or higher.  It will not load with SR0**

===== Notes =====
This plug-in is included in the [[http://download.rhino3d.com/rhino/5.0/wip/|Rhino 5.0 WIP]].


=====Commands:=====

The plugin has one command "MeshRepair".  It opens a dockable window.


=====Flow:=====

Call MeshRepair command to open the dockable MeshRepair wizard


**Page 1:**

Select a mesh then Click CheckMesh.  If the mesh is bad, RepairMesh button is enabled and full mesh details show on the window.

From this point on, all commands are applied to the mesh(s) you selected only.


{{http://en.wiki.mcneel.com/content/upload/images/mr_p0.PNG}}


**Page2:**

You can check any number of items you want to repair, then click Repair, naked edges count is updated.

Once you've done everything you could in this page, you can click next.


{{http://en.wiki.mcneel.com/content/upload/images/mr_p2.PNG}}


**Page 3:**

Filling small gaps calls Rhino command **MatchMeshEdges**.

If you hit **Draw Graph** button, you can get a feedback about how many of the naked edges would be resolved around what distance.

The graph gives feedback about number of naked edges in the range between min and max distances.

The wizard claculates 11 sample points in the range.


{{http://en.wiki.mcneel.com/content/upload/images/mr_p3.PNG}}


**Page 4:**

Additional commands to help repairing meshes for special cases.


{{http://en.wiki.mcneel.com/content/upload/images/mr_p4.PNG}}


Pressing **Finish** button in the wizard clears the list of meshes examined.
