====== Rhinoceros wurde nicht vom App Store heruntergeladen ======

> **HINWEIS:** // Im Folgenden wird davon ausgegangen, dass Sie macOS Sierra (10.12) oder neuer verwenden.  Für ähnliche Dialoge auf früheren Versionen von macOS siehe [[https://support.apple.com/kb/ph18657?locale=en_US|dieses Apple Support-Thema]].//

===== Problem =====

Wenn Sie Rhino für Mac zum ersten Mal ausführen, sehen Sie, abhängig von Ihren Systemeinstellungen, möglicherweise einen Dialog mit der folgenden Warnung: 

**"Rhinoceros.app" kann nicht geöffnet werden, weil es nicht vom App Store heruntergeladen wurde.**

{{:rhino:mac:rhino-mac-app-store-warning-01.png|}}

===== Lösungen =====

Es bestehen zwei Möglichkeiten, diese Einschränkung zu vermeiden:

  - Ändern Sie Ihre Einstellungen zu Sicherheit und Datenschutz, um Anwendungen zuzulassen, die nicht aus dem App Store stammen
  - Machen Sie eine "Ausnahme von der Regel" für Rhino für Mac

Gehen wir alle Optionen in dieser Reihenfolge durch...

**Option 1: Ändern Sie Ihre Einstellungen zu Sicherheit und Datenschutz**

Navigieren Sie zu //Systemeinstellungen// > //Sicherheit// > //Allgemein// (Reiter).  Suchen Sie den Abschnitt //Apps-Download erlauben von://...

{{:rhino:mac:rhino-mac-app-store-warning-02.png|}}

Wenn der Dialog "geschlossen" ist, klicken Sie unten auf das Vorhängeschloss um den Dialog zu "öffnen" und Änderungen vorzunehmen.

Dann wählen Sie den App-Store und die jeweilige Entwickler-Option.  Jetzt können Sie den Dialog schließen.  

Wenn Sie wieder versuchen Rhinoceros zu starten, werden Sie einen anderen Warndialog vorfinden, mit dem Sie Rhinoceros starten können.  Sobald Sie Rhinoceros geöffnet haben, werden Sie diese Warnung nicht mehr erhalten.

** Option 2: Eine Ausnahme machen**

Wenn Sie Ihre Einstellungen in //Sicherheit// nicht ändern möchten, ist das kein Problem.  Sie können für Rhino für Mac eine Ausnahme machen.  Hierfür müssen Sie Rhinoceros das nächste Mal auf spezielle Weise öffnen...

Suchen Sie Rhinoceros, entweder in Ihrem Ordner //Programme// oder in Ihrem //Dock//.  Anstatt jetzt mit der linken Maustaste auf das Rhinoceros-Symbol zu klicken, benutzen Sie die rechte Maustaste (für die Optionen).  Darauf erscheint ein Menü, in dem Sie die Option //Öffnen// wählen. 

{{:rhino:mac:rhino-mac-app-store-warning-03.png|}}

Nun erscheint ein anderes Warndialogfeld, das Ihnen die Möglichkeit gibt, Rhinoceros zu öffnen.  Sobald Sie Rhinoceros geöffnet haben, werden Sie diese Warnung nicht mehr erhalten.
