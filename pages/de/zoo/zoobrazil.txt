======Brazil 2.0 und Zoo======
> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //Verwendung von 2.0 mit dem Zoo.//

=====Überblick=====
[[http://brazil.rhino3d.com/|Brazil 2.0]] kann entweder mit Rhino 4.0 oder Rhino 5 verwendet werden. Netzwerklizenzen von [[http://brazil.rhino3d.com/|Brazil 2.0]] können daher entweder mit dem [[de:zoo4:home|Zoo 4.0]] für Rhino 4.0 oder dem [[de:zoo:home|Zoo 6]] für Rhino 5 verwendet werden.

=====Weitere Informationen=====
Nach der Erstinstallation ist [[http://brazil.rhino3d.com/|Brazil 2.0]] so konfiguriert, dass es den [[de:zoo4:home|Zoo 4.0]] zur Verwaltung der Netzwerklizenzen verwendet. 

Um [[http://brazil.rhino3d.com/|Brazil 2.0]] so zu rekonfigurieren, dass es einen [[de:zoo:home|Zoo 6]] für die Netzwerkverwaltung verwendet, benötigen Sie ein Hilfsprogramm.

Wechseln Sie dazu im Explorer des Rhino-Arbeitsplatzes in den folgenden Ordner:

====Windows 64-Bit====
<code>
C:\Programme (x86)\Gemeinsame Dateien\McNeel Shared\License Manager
</code>

====Windows 32-Bit====
<code>
C:\Programme\Gemeinsame Dateien\McNeel Shared\License Manager
</code>

Um [[http://brazil.rhino3d.com/|Brazil 2.0]] so zu konfigurieren, dass es einen [[de:zoo:home|Zoo 6]] verwendet, führen Sie das Hilfsprogramm **RegRhinoLM5.exe** aus.

Um [[http://brazil.rhino3d.com/|Brazil 2.0]] so zu konfigurieren, dass es einen [[de:zoo4:home|Zoo 4.0]] verwendet, führen Sie das Hilfsprogramm **RegRhinoLM4.exe** aus.

Diese Änderung muss auf jedem Arbeitsplatz vorgenommen werden, der über den Zoo verwaltet wird.
