=====Downloads=====

**SectionTools Plug-in for Rhino 6 for Windows:** Last updated October 11th, 2017
  * Make sure you have the latest [[https://www.rhino3d.com/download|Rhino 6]]. 
  * Download[[https://www.rhino3d.com/download/sectiontools/1/wip/rc/| SectionTools Installer]] for Rhino 6. Double-click and the plug-in should install and load in Rhino. You need to load the toolbar using the toolbar manager.

**SectionTools Plug-in for Rhino 5 only (does not work with Rhino 4):** Last updated December 20th, 2013
  * Make sure you have the latest service release [[http://www.rhino3d.com/download/rhino/5.0/evaluation|Rhino 5 or Rhino 5 Evaluation]]. 
  * Download[[http://www.rhino3d.com/download/SectionTools/1.0/wip/| SectionTools Installer]] for Rhino 5. Double-click and the plug-in should install and load in Rhino. You need to load the toolbar using the toolbar manager.
