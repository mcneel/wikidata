======Zoo - gestionnaire de licences======
[[zoo:home|{{:zoo:zoo_small.png|Zoo Home}}]]

=====Présentation=====
Le Zoo permet de partager des licences de Rhino, Flamingo, Penguin, Brazil et Bongo (Codes CD) avec les utilisateurs du même groupe de travail. Lorsque ces produits sont installés comme noeuds de groupes de travail au lieu d'être installés de façon autonome, le système fonctionne ainsi : 

  * Lorsqu'un noeud du groupe de travail est lancé, une licence est demandée au Zoo. 
  * Une licence libre est assignée au noeud. 
  * Lorsque un noeud est fermé, la licence est rajoutée à la liste des licences dans le Zoo.

===== Caractéristiques =====
  * **Pas de coût supplémentaire** - Aucune version spéciale de Rhino, Flamingo, Penguin ou Bongo n'est nécessaire. 
  * **Aucun matériel spécial n'est nécessaire** - Le Zoo fonctionne sur tous les systèmes d'un réseau de travail. 
  * **Aucun logiciel spécial n'est nécessaire** - Le Zoo fonctionne sur tous les systèmes Windows, dont Windows XP, Windows Vista, Windows 7 et Windows Server 2000, 2003 et 2008.
  * **Configuration simple** - Installez le Zoo sur un système du groupe et tapez les clefs de licence dans le zoo au lieu de les taper dans chaque système. 
  * **Retrait** - Les clés de licence peuvent être retirées afin que les portables puissent être déconnectés du réseau. Le code peut être réintroduit dans le zoo quand le portable est reconnecté au réseau. 
  * **Robuste** - Les noeuds de groupe continueront à fonctionner même si la connexion au réseau est interrompue ou si le serveur s'arrête.
  * **Compatible réseau routeur** - Le Zoo fonctionne aussi bien sur les réseaux à un seul segment que sur les réseaux où les systèmes sont séparés par des routeurs. 
  * **Environnement mixte** - Il est possible d'avoir un mélange de noeuds de groupe de travail et de noeuds autonomes sur le même domaine de Windows ou groupe de travail. 
  * **Un service** - Si vous redémarrez le système où est installé le Zoo, celui-ci démarrera automatiquement. 

** Nouveautés du Zoo 5.0**

Zoo 5.0 est une toute nouvelle version du Zoo. Entièrement réécrit, le Zoo 5,0 apporte quelques nouvelles fonctions par rapport aux versions précendentes, tels que :
  * **Prise en charge du protocole IP standard** - les clients communiquent avec Zoo 5.0 en utilisant le protocole HTTP (Port 80). Cette nouveauté permet aux administrateurs de système de rediriger plus facilement les requêtes du Zoo sur des réseaux complexes, voire même sur Internet.
  * **Compatible avec le pare-feu** - Pour que Zoo 5.0 fonctionne, le client doit consulter périodiquement le serveur mais il ne débute jamais de connexion vers le client. Les administrateurs n'ont plus besoin d'ouvrir des ports sur les ordinateurs clients.
  * **Limite de retrait de licences** - La version précédente de Zoo permet de retirer des licences pour les ordinateurs portables. Mais il n'existait aucune limite en termes de durée. Zoo 5.0 permet à l'administrateur d'indiquer la durée du retrait des licences.
  * **Prise en charge des modules tiers** - La version précédente de Zoo ne traitait que les licences des produits de McNeel ( Rhino,  Flamingo, Bongo, Brazil et Penguin par exemple). Zoo 5.0 permet aux développeurs de modules d'ajotuer une prise en charge de leurs produits par le Zoo.


=====Configuration requise=====
  * Un système sous Windows :
    * Windows XP, Windows Vista, Windows 7 ou Windows 8.
    * Windows Server 2003, Windows Server 2008 ou Windows Server 2008 R2.
  * [[http://www.microsoft.com/en-us/download/details.aspx?id=17718|Microsoft .NET Framework 4]]
  * Une connexion réseau configurée pour TCP/IP.
