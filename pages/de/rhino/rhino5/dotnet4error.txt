====== Installationsfehler des .NET Framework 4 (Fehlercode -2145124330) ======
> **Zusammenfassung:** //Die Installation des .NET Framework 4.0 wird mit einem Fehler abgebrochen (Fehlercode -2145124330)//
> **Weitere Infos:** //[[de/rhino/installproblems|Installationsfehler bei Rhino 5]]//

**Lösung:**\\ Reparieren Sie zunächst das .NET Framework 4 und installieren Sie anschließend Rhino 5.
  - Laden Sie das [[http://www.microsoft.com/en-us/download/details.aspx?id=30135|Microsoft .NET Framework Repair Tool]] herunter.
  - Führen Sie das Reparaturprogramm aus.
  - Klicken Sie immer auf "Weiter" und am Ende auf "Fertig stellen".
  - Installieren Sie Rhino 5 erneut.

**Hinweis:** Wenn das Repair Tool Ihr Problem nicht behebt, können Sie es auch mit dem [[http://wiki.mcneel.com/de/rhino/rhino5/dotnet4_not_recognized|.NET Framework Cleanup Tool]] versuchen.
