=====Background=====
PanelingTools plugin helps generate 2D and 3D cellular patterns and populate them over rectangular grids.

The plug-in supports intuitive design of paneling concepts as well as rationalize complex geometry into a format that is suitable for analysis and fabrication. The plug-in is closely integrated with Rhino 5 and is widely used for architectural and other building designers.

=====Download=====

===For Rhino 5 for Windows 32 and 64 bit===
  * Download and double-click **[[http://www.rhino3d.com/download/PanelingTools/1.0/wip| PanelingTools installer]].** Once installed, when open new session of Rhino, the installed version will be loaded (no need to drag&drop or use PluginManger to load). Toolbars are also installed, but you need to load first time using Tools/Toolbar Layout...
  * PanelingTools plugin also extends Grasshopper.  **[[http://www.grasshopper3d.com/group/panelingtools|Click for details...]]** 
  * {{:labs:singlestrokefont.zip|Single stroke font}} (optional). Add this font to your Windows font directory. For example,in Windows 7, drag the RhSS.ttf file into Control Panel>Appearance and Personalization>Fonts. The font will then be accessible from Rhino. This type of font is used mainly to tag parts for laser cutting. 

===For Rhino Mac WIP===
  * PanelingTools plugin is part of the **[[http://www.rhino3d.com/download/rhino-for-mac/5/evaluation/|Rhino for Mac WIP...]]** . You do not need to download or install anything. Please note that the Paneling Tools menu is not implemented since custom user interface changes are not yet working on the Mac, but all the commands are available. Type "pt" to see all the Paneling Tools commands in the AutoComplete list.  Let us know what you think!
