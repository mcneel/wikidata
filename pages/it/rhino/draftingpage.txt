====Modelli di riempimento per Rhino 5 e versioni successive ====

1. Scarica [[https://wiki.mcneel.com/_media/legacy/en/hatchpatterns_1033.zip|questo file zip]] ed estrai il contenuto. Rinomina il file hatchpatterns.txt in hatchpatterns.pat. 

2. In Rhino 5 o versione successiva, aprire //File>ProprietÃ >Annotazione>Riempimento// e importa semplicemente il file hatchpatterns.pat.


===== Modelli di riempimento per Rhino 4 =====
===Modelli di riempimento di Carlotto Vittorio===

Scarica semplicemente questo file di testo e sostituisci l'originale nel seguente percorso (ti consigliamo di salvare una copia dell'originale):

  * Win XP: //C:\Documents and Settings\All Users\Application Data\McNeel\Rhinoceros\4.0\English\Support//
  * Vista & Win7: //C:\ProgramData\McNeel\Rhinoceros\4.0\English\Support//
