====== Rhino 6 Cloud Zoo - Team======

>Hinweis: Verwenden Sie diese Option um Ihre Lizenzen mit den Mitgliedern eines Teams zu teilen. Ihre Lizenzen befinden sich auf unseren Servern, so dass ihr Team von jedem beliebigen Computer aus darauf zugreifen kann. Zu Beginn wird eine Internetverbindung benötigt. Danach können Sie Ihre Lizenz aufrechterhalten, indem Sie Rhino etwa ein Mal pro Woche mit Internetverbindung ausführen. Sie können Rhino für ca. 2 Wochen ohne Internetverbindung ausführen, ehe die Lizenz aktualisiert werden muss.


===Konfiguration durch Aufruf der Website===
  - **Loggen Sie sich in Ihr Rhino-Konto ein**
    - Besuchen Sie https://accounts.rhino3d.com/?controller=home
      - Erstellen Sie ein Rhino-Konto falls Sie noch keines haben
      - Speichern Sie Ihre Anmeldeinformationen. Sie werden sie später benötigen
  - **Ein Team gründen**
    - Auf Ihrer Seite "Mein Konto" klicken Sie auf **Teams**
      - Klicken Sie auf Neues Team
      - Fügen Sie einen Namen und eine Beschreibung hinzu
  - **Fügen Sie Ihre Lizenzen dem Team hinzu**
    - Auf Ihrer Seite "Mein Konto" klicken Sie auf **Lizenzen**
    - Unter Team-Lizenzen klicken Sie auf das von Ihnen gegründete Team
    - Auf der Team-Seite klicken Sie auf Lizenz hinzufügen
    - Wählen Sie das zutreffende Produkt
    - Geben Sie Ihren Lizenzschlüssel ein
    - Klicken Sie auf Lizenz hinzufügen
  - **Laden Sie Benutzer ein, Ihrem Team beizutreten**
    - Auf Ihrer Seite "Mein Konto" klicken Sie auf **Teams**
      - Klicken Sie auf Verwaltung Ihrer Teams
      - Klicken Sie auf Mitglieder einladen
      - Geben Sie die E-Mail-Adressen für die Benutzer ein, die Zugriff auf die Team-Lizenzen erhalten sollen
      - Klicken Sie auf Senden
      - Wenn Benutzer auf die Einladungen reagieren, werden Sie als Eigentümer des Teams eine Annahmebestätigung erhalten
  - **Starten Sie Rhino**
    - Klicken Sie auf Einloggen
    - Geben Sie Ihre Anmeldeinformationen für das Konto ein
    - Rhino wird gestartet

===Andere Computer===
  - **Installieren Sie Rhino**
  - **Starten Sie Rhino**
    - Klicken Sie auf Einloggen
    - Geben Sie Ihre Anmeldeinformationen für das Konto ein
    - Rhino wird gestartet
