====== Installation der Rhino 6 Einzelcomputer-Lizenzierung ======

>Hinweis: Verwenden Sie diese Option für ein Einzelplatzsystem oder wenn kein Internetzugang vorhanden ist. Diese Option wird die Lizenz für den ausschließlichen Gebrauch auf einem einzigen Computer sperren.

  - **[[https://www.rhino3d.com/download/Rhino/6.0/release|Download]]** des Rhino-Installationspakets
    - Geben Sie Ihre E-Mail-Adresse ein und klicken Sie auf "Weiter"
    - Klicken Sie auf "Jetzt Deutsch herunterladen" oder wählen Sie eine andere Sprache
  - **Führen Sie das Installationsprogramm aus** - Lokale Administratorrechte werden benötigt
    - Akzeptieren Sie die AGB und die Datenschutzrichtlinien
      - Klicken Sie zum vorherigen Lesen auf die Links - Empfohlen
    - Klicken Sie auf "Jetzt installieren"
      - Optional - Klicken Sie auf das Zahnrad-Symbol für weitere Optionen
    - Klicken Sie auf "Schließen", wenn die Installation abgeschlossen ist
  - **Bereitstellung Ihrer Lizenz**
    - Starten Sie Rhino
      - Rhinoceros 6 benötigt eine Lizenz:
        - Wählen Sie "**Geben Sie Ihren Lizenzschlüssel ein**"
      - Geben Sie Ihren mit RH60-XXXX... beginnenden Lizenzschlüssel ein und klicken Sie auf Fortfahren
        - Wählen Sie "Nur auf diesem Computer verwenden"
        - Antworten Sie beim Warnungshinweis mit Ja
        - Rhino wird gestartet
      - Registrieren Sie sich und bestätigen Sie 
        - Zur Registrierung und Bestätigung Ihrer Lizenz befolgen Sie einfach die Aufforderungen 

=== Ist die Rhino-Installation fehlgeschlagen? ===
  * Wenn das Rhino-Installationsprogramm fehlschlägt, erhalten Sie in einem Dialogfenster Informationen und Anleitungen.
  * Haben Sie einen [[https://wiki.mcneel.com/de/rhino/6/installproblems|MSI-Fehlercode]] erhalten?
  * Möglicherweise werden Sie aufgefordert, das Fehlerprotokoll zu öffnen, es in einer Datei zu speichern und es an den Technischen Support von McNeel zu senden: <tech@mcneel.com>
