====== Rhino 6 Cloud Zoo - Personnel ======

>Remarque : Utilisez cette option pour partager votre licence individuelle sur plusieurs ordinateurs. Votre licence sera conservée sur nos serveurs afin que vous puissiez y accéder depuis n'importe quel ordinateur. Une connexion initiale à Internet est nécessaire. Par la suite, une connexion à Internet une fois par semaine environ suffira pour maintenir la licence. Vous pouvez utiliser Rhino sans connexion à Internet pendant environ 2 semaines avant d'avoir besoin de rafraîchir la licence. 

===Configuration initiale de l'application Rhino===
  - Installez Rhino
  - Lancez Rhino
    - Suivez les options par défaut

===Configuration initiale sur Internet===
  - **Connectez-vous à votre compte Rhino** 
    - Visitez https://accounts.rhino3d.com/?controller=home
      - Créez un compte Rhino si vous n'en avez pas encore un. 
      - Enregistrez vos informations de connexion. Vous en aurez besoin par la suite
  - **Ajoutez votre clé licence dans vos licences personnelles**
    - Sur votre page "Mon compte", cliquez sur Licences
    - Sous Licences personnelles, cliquez sur [votre nom] (moi)
    - Dans la page "Licences [votre nom] (moi)", cliquez sur Ajouter une licence
    - Sélectionnez le produit correspondant
    - Indiquez votre clé de licence
    - Cliquez sur Ajouter une licence
  - **Lancez Rhino**
    - Cliquez sur Connexion
    - Entrez les informations sur votre compte
    - Rhino démarrera 

===Autres ordinateurs===
  - **Installez Rhino**
  - **Lancez Rhino**
    - Cliquez sur Connexion
    - Entrez les informations sur votre compte
    - Rhino démarrera
