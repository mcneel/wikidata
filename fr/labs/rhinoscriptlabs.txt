=====Présentation=====

[[developer:rhinoscript|RhinoScript]] est un langage de scriptage basé sur le langage VBScript de Microsoft. Avec RhinoScript, vous pouvez ajouter rapidement des fonctions à Rhino ou automatiser des tâches répétitives.

La version lab de [[developer:rhinoscript|RhinoScript]] contient des ajouts et des modifications par rapport à la version de [[developer:rhinoscript|RhinoScript]] incluse avec **Rhino 4.0 SR5**.

=====Nouveautés=====


====Nouvelles méthodes====

  * AddPlanarMesh - Crée un maillage plan à partir d’une courbe plane fermée.
  * BoxMorphObject - Déformation d'un objet par sa boîte de contour.
  * CurrentDetail - Change la vue détaillée actuelle en vue de mise en page.
  * CurrentDetail - Renvoie les vues détaillées d'une vue de mise en page.
  * DivideCurveEquidistant - Fractionne une courbe pour que la distance linéaire entre les points soit égale.
  * ExtrudeCurveTapered - Extrude une courbe vers un point.
  * IsDetail - Vérifie qu'une vue détaillée est sur une vue de mise en page.
  * IsLayout - Vérifie qu'une vue est sur une vue de mise en page.
  * IsLayoutObject - Vérifie qu'un objet réside dans une vue de mise en page ou dans l'espace du modèle.
  * Layer State Methods - Plusieurs nouvelles méthodes d'état des calques ont été ajoutées.
  * ObjectLayout - Renvoie ou change la mise en page ou l'espace de modélisation d'un objet.
  * ProjectCurveToMesh - Projette une ou plusieurs courbes sur un ou plusieurs maillage.
  * ProjectPointToMesh - Projette un ou plusieurs points sur un ou plusieurs maillage.
  * RemoveCurveKnot - Supprime un nœud dans une courbe.
  * RemoveSurfaceKnot - Supprime une ligne de nœuds dans une surface.

====Méthodes modifiées====

  * AddSweep1 - Une erreur avec les courbes de rail fermées a été corrigée.
  * AddSweep2 - Une erreur avec les courbes de rail fermées a été corrigée.
  * MeshVertexColors - Un argument supplémentaire, optionnel, a été ajouté.
  * RenameView - Correction d'un bogue grave.

=====Installation=====


  - Vérifiez que [[http://download.rhino3d.com/download_rel.asp?rel=452|**Rhino 4.0 SR5**.]] est installé.
  - Fermez toutes les copies de Rhino 4.0 ouvertes.
  - Dans le dossier **\Program Files\Rhinoceros 4.0\Plug-ins**, faites une copie de sauvegarde des fichiers suivants :
    - [[developer:rhinoscript|RhinoScript]].rhp
    - [[developer:rhinoscript|RhinoScript]].chm - Vous devrez peut-être le déverrouiller (clic avec le bouton de droite sur le fichier CHM et clic sur "Propriétés..." Dans l'onglet Général, désactivez la case "Lecture seule")
    - [[developer:rhinoscript|RhinoScript]].tlb
  - Décompressez le fichier téléchargé et copiez son contenu dans le dossier indiqué ci-dessus.
  - Lancez Rhino.
