======Hinzufügen von Rhino-Lizenzen zum LAN Zoo======

> **Produkt:** //[[de:zoo:home|Zoo]]//
> **Zusammenfassung:** //Erklärung, wie Sie dem LAN Zoo Rhino-Lizenzen hinzufügen und diese bestätigen können//

>Wenn Sie dem LAN Zoo eine Rhino-Lizenz hinzufügen möchten, werden Sie aufgefordert, diese online zu bestätigen. Rhino-Lizenzen müssen bestätigt werden, bevor sie benutzt werden können. 
>
>**Administratoren:** Wenn Sie hinter einer Firewall sind, öffnen Sie den https-Zugang (TCP port 443) zu api.mcneel.com

<color red>**Wichtig: Wenn Sie ein Upgrade auf eine höhere Version von Rhino durchführen, bewahren Sie den Lizenzschlüssel der vorherigen Version auf. Sie können jeden Lizenzschlüssel nur einmal für ein Upgrade benutzen.**</color>

Die einzige Information, die während der Lizenzbestätigung benötigt wird, ist der Lizenzschlüssel, der beim Kauf von Rhino dabei ist. 

Eine Registrierung ist nicht notwendig, auch keine persönlichen Informationen. Es werden keine Computer-spezifischen Informationen und auch keine Informationen aus Ihrem Computer an Robert McNeel & Associates gesendet. 

=====Hinzufügen einer Rhino-Lizenz=====
Zum Hinzufügen einer Lizenz zum LAN Zoo führen Sie den **Zoo Administrator** (ZooAdmin.exe) aus und wählen Sie **Edit → Add**.

{{zoo:rhinovalidate0.png}}

Um Lizenzen zum LAN Zoo hinzuzufügen:
  * Wählen Sie Ihr **Rhino**-Produkt aus der Drop-down-Liste.
  * Geben Sie den **registrierten Besitzer** des Produkts und seine **Organisation** ein.
  * Geben Sie Ihren **Lizenzschlüssel** ein.
  * Klicken Sie auf **OK**, um mit der Lizenzbestätigung zu beginnen.
