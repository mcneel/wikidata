====== Digitalización 3D ======
Las herramientas de los **digitalizadores 3D** pueden ser útiles para **[[http://en.wikipedia.org/wiki/Reverse_engineering|ingeniería inversa]].** Con un digitalizador, puede generar un modelo informático a partir de un objeto físico muestreando las coordenadas 3D de una en una para crear curvas de entrada para los comandos de superficie. Dado que cada comando de creación de superficies requiere curvas de entrada específicas, un modelador experimentado de Rhino debe decidir cuáles son las mejores herramientas de creación de superficies para duplicar la forma de un objeto. Con un digitalizador 3D, puede muestrear los puntos necesarios para crear las curvas de entrada requeridas. 

Los digitalizadores 3D también pueden utilizarse para la inspección. Tomando muestras de puntos en zonas críticas del modelo físico y comparando esos puntos con un modelo digital, se puede juzgar la precisión del modelo digital.
