====== NextEngine File Import ======
> **Summary:** //Plugin to import NextEngine SCN files//

=====Description=====

This plugin for Rhino4 will allow you to import NextEngine SCN files.

----

\\

\\

[[http://download.mcneel.com/download.asp?id=RhinoLabsNextEngine&language=|Click here to download plugin]]

=====Installation instructions (Only has to be done once)=====

  - Copy the contents of the zip file to your "[[rhino:rhinoceros|RhinoCeros]] 4.0\Plug-ins" directory.
  - Install [[http://www.microsoft.com/DOWNLOADS/details.aspx?displaylang=en&FamilyID=9b2da534-3e03-4391-8a4d-074b9f2bc1bf|Microsoft Visual C++ 2008 SP1 Redistributable]] (required to make regsvr32 work on some systems).
  - Run regsvr32 on pathtoplugins\NEModel.dll. (should look something like this  <color blue>regsvr32 "C:\Program Files\[[rhino:rhinoceros|RhinoCeros]] 4.0\Plug-ins\NEModel.dll"</color>)
  - Start Rhino.
  - Go to the Tools - Options - Plug-ins page.
  - Click on the Install button.
  - Browse to the folder where you copied the three files in step 1 above.
  - Select import_SCN.rhp and click the Open button.
  - Click the OK button to close the Rhino Options page.

===== Use Instructions=====

  - Go to the File pulldown and use the Open or Import command
  - In the Files of Type dropdown, select NextEngine .SCN
  - Select your SCN file.
  - Once Imported, you may need to run Zoom Extents to see the model.
