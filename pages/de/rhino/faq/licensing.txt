===== Häufig gestellte Fragen: Lizenzierung in Rhino 6, Rhino 7 und Rhino 8 =====

====Welche Lizenzoptionen stehen mir für Rhino 6 zur Verfügung?====
Siehe [[https://www.rhino3d.com/features/administration/licensing/|Rhino-Lizenzoptionen]] für nähere Angaben.

====Gemeinsame Nutzung von Lizenzen====

==Kann ich eine Lizenz von Rhino auf mehreren Computern verwenden?==
Ja, und zwar unter Verwendung eines unserer Manager für zentral verwaltete Lizenzen. Der LAN-Zoo funktioniert über die LAN Ihres Büros oder der Cloud Zoo verwendet gelegentlich eine Internetverbindung, um Einzelnutzerlizenzen zentral für mehrere Computer zu verwalten.

==Ist es möglich, eine Lizenz für einen einzigen Computer auf mehreren Computern zu verwenden?==
Für jeden weiteren Computer brauchen Sie eine Extralizenz. Bei Rhino 5 war mehr als eine Bestätigung pro Einzelnutzerlizenz zulässig. Bei Rhino 6, 7 und 8 ist dies nicht der Fall. Rhino 6, 7 und 8 fügen die Persönliche Lizenzverwaltung von Cloud Zoo hinzu, damit Sie eine einzelne Lizenz mit einer gelegentlichen Internetverbindung auf mehreren Computern verwalten können.

==Wie kann ich eine oder mehr Lizenzen mit mehreren Leuten teilen?==
Fügen Sie Ihre Lizenzen einem Team im Cloud Zoo (empfohlen) oder dem LAN Zoo hinzu.
[[https://www.rhino3d.com/features/administration/licensing/|Rhino-Lizenzoptionen]]

====Upgrade auf Rhino 8 ====

==Kann ich nach einem Upgrade von Rhino 7 auf Rhino 8 weiterhin Rhino 7 verwenden?==
Ja. Durch ein Upgrade weiten Sie Ihre Lizenz aus, um auf Funktionen von Rhino 8 zugreifen zu können, d.h. Sie fügen keine neue Lizenz hinzu. Somit können Sie Rhino 6 wie auch ältere Versionen von Rhino auf dem gleichen Computer verwenden. Allerdings können Sie nicht Rhino 7 auf einem Computer verwenden, während jemand anderes das Rhino-8-Upgrade auf einem anderen Gerät verwendet.

==Wenn ich meine Upgradelizenz für Rhino 8 im Cloud Zoo deponiere, kann ich dann immer noch Rhino 7 installieren?==
Ja, Rhino 7 kann installiert werden, selbst wenn es auf Rhino 8 aktualisiert wurde. Vergewissern Sie sich, beide auf dem Zoo oder Cloud Zoo zu haben. 

==Kann ich meine Lizenz für Rhino 5 im Cloud Zoo deponieren?==
Nein. Rhino 5 unterstützt den Cloud Zoo nicht.

==Was geschieht, wenn ich meine Upgradelizenzen für Rhino 7 und Rhino 8 im Zoo deponiere?==
Zur Unterstützung der Lizenzschlüssel für Rhino 8 benötigen Sie Zoo 7 SR 8 (oder neuer). Der Zoo fasst Ihre Upgrade-Codes für Rhino 7 und Rhino 8 in einem Lizenz-"Cluster" zusammen und wacht darüber, dass immer nur eine auf einmal verwendet wird. Wenn ein Anwender sowohl Rhino 7 als auch Rhino 8 vom gleichen Computer aus ausführt, verwendet er dabei nur eine Lizenz.

====Rhino 8 im Anzeigemodus verwenden ====

==Wie kann ich Rhino ausschließlich im Anzeigemodus verwenden?==

Um Rhino im Anzeigemodus zu verwenden benötigen Sie einen Lizenzschlüssel für die Testversion. Falls der, über den Sie bereits verfügen, verfallen ist, ist dies die ideale Gelegenheit.  Falls Sie über keinen Lizenzschlüssel für die Testversion verfügen, fordern Sie bei https://www.rhino3d.com/download/rhino/evaluation einen an.

Bei Bestätigung der Lizenz Ihrer Testversion verwenden Sie nicht die Option zum Teilen oder zur Zentralen Verwaltung Ihrer Lizenz für mehrere Computer, vergewissern Sie sich stattdessen, die Lizenz für den alleinigen Gebrauch auf Ihrem Computer zu beschränken.

Falls Sie bereits über eine Lizenz verfügen und diese befindet sich auf Cloud Zoo, müssen Sie sie zuerst aus Ihrem Rhino-Konto entfernen:

  * Gehen Sie auf https://accounts.rhino3d.com/ und loggen Sie sich mit Ihrer E-Mail-Adresse in Ihrem Konto ein.
  * Scrollen Sie auf der Seite nach unten und klicken Sie auf Lizenzen und auf Ihrem Namen unter Persönliche Lizenzen.
  * Sobald sie entfernt ist, starten Sie Rhino 8 neu. 
  * Bei der Aufforderung zur Eingabe einer neuen Lizenz wählen Sie die Option "Geben Sie Ihren Lizenzschlüssel ein" und geben Sie den Lizenzschlüssel Ihrer Testversion ein, dann klicken Sie auf Weiter. Vergewissern Sie sich, NICHT die Option "Zu Ihrem Rhino-Konto hinzufügen" wählen und klicken Sie auf die Option "Nur auf diesem Computer verwenden" unten links und dann auf Ja.  

Solange die Lizenz Ihrer Rhino-Testversion gültig ist, können Sie Ihre Arbeiten abspeichern; sobald sie abgelaufen ist, können Sie jedes Rhino-Modell öffnen, die Ansicht ändern, Bemaßungen verwenden usw., jedoch keine Dateien mehr speichern.
