===== Ajouter sur votre compte Rhino une clé de licence en cours d'utilisation dans le Zoo ou sur un appareil autonome =====

Afin d'ajouter sur votre compte Rhino une licence en cours d'utilisation, vous devez la supprimer du Zoo ou de l'appareil où elle est utilisée. 

==== Supprimer la licence de son emplacement actuel ====

== Si la licence est sur le Zoo : ==
  - Retirez la licence du Zoo.

== Si la licence est sur un appareil autonome : ==
  - Ouvrez **Outils** -> **Options** -> **Licences** -> **Modifier la clé de licence**.
  - Cliquez sur **Supprimer une licence** en bas de la boîte de dialogue.

==== Ajouter la licence à un compte Rhino ====
  - [[https://www.rhino3d.com/licenses/?controller=add_licenses|Ouvrez la page Ajouter une licence]]. 
  - Ajoutez la licence dans votre compte ou dans une des équipes à laquelle vous appartenez.
