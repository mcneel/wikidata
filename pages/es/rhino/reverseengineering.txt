====== Ingeniería inversa ======
Rhinoceros no es un software de ingeniería inversa automática.

En general, se utiliza la nube de puntos o la malla poligonal como referencia para remodelar el objeto con superficies NURBS.

Se pueden extraer curvas de sección de una nube de puntos (comando SecciónNubeDePuntos) o de una malla poligonal con Sección o Contorno.

También hay algunos plug-ins disponibles para ingeniería inversa en http://www2.rhino3d.com/resources/default.asp?category=13

Véase también http://www.flexicad.com/produkte/erweiterungen/rhinoreverse/uebersicht.htm.

----
El comando Parche es la única herramienta que se puede usar directamente para crear una superficie de un grupo de puntos. Pero también se puede usar el comando SecciónNubeDePuntos en Rhino para hacer la ingeniería inversa de las curvas en primer lugar y luego usar esas curvas para construir superficies NURBS.

Hay pocas posibilidades para hacer esto:

SMURF Plug-in for Rhino 3.0, http://www2.rhino3d.com/resources/display.asp?language=&listing=93  Smurf es un plug-in para Rhino que permite la conversión interactiva de mallas poligonales en parches de superficies NURBS.

El plug-in SMURF requiere una malla poligonal. Si tiene una nube de puntos, es necesario convertir la nube de puntos en una malla en primer lugar.

> **Convertidores:** ////
  * [[rhino:points2polys|Points2Polys]] es un convertidor GRATUITO de nubes de puntos a mallas poligonales de Paraform que está diseñado para usar en datos de puntos RAW no estructurados (unos 500.000 puntos). http://www.paraform.com/ppdl/

  Busque en el archivo del grupo de discusión de Rhino más información sobre esta herramienta.

  * Busque en el archivo del grupo de discusión y en la página de recursos de Rhino otros traductores de nubes de puntos más recientes.
