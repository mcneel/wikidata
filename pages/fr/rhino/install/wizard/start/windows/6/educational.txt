====== Licence Éducation individuelle de Rhino 6 ======


===Choisir :===
    * **[[fr:rhino:install:wizard:start:windows:6:commercial:czpersonal|Plusieurs ordinateurs, un seul utilisateur]] - Cloud Zoo Personnel**
      * Utilisez cette option lorsque vous souhaitez partager votre licence sur plusieurs ordinateurs personnels
      * Une connexion occasionnelle à Internet est nécessaire
      * Les licences éducation individuelles ne peuvent pas être ajoutées dans une équipe du Cloud Zoo
      * [[https://www.rhino3d.com/6/eula|Contrat de licence de l'utilisateur final]]

    * **[[fr:rhino:install:wizard:start:windows:6:commercial:stand_alone|Licence sur un ordinateur individuel]]**
      * Utilisez cette lorsqu'un seul ordinateur sera utilisé
      * La licence est verrouillée **exclusivement** sur un ordinateur
      * Aucun accès à Internet n'est nécessaire
      * [[https://www.rhino3d.com/6/eula|Contrat de licence de l'utilisateur final]]
