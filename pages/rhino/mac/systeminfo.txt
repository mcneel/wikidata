=====Collecting System Information for your computer=====


  - In the **Apple** menu, select **About This Mac**.  A new dialog appears..
  - In the dialog, click **More Info...**.  The dialog is replaced with a new dialog.
  - In the new dialog, click the **System Report...** button.  Another dialog appears.
  - In the System Information application menu, select **File > Save...**.  A Save dialog appears.
  - Save the file somewhere.
  - Quit System Information.
  - Email the saved file to mactech@mcneel.com, along with a description of your problem.
  - In your email, be sure to reference any discourse.mcneel.com posts that are involved with the problem.
