======"Rhinoceros 라이선스가 이미 사용 중입니다" 경고======
> **제품:** //[[:ko:zoo4:home|Zoo 4.0]]//
> **요약:** //"Rhinoceros 라이선스가 이미 사용 중입니다..." 경고 해결 방법.//

=====질문=====
Rhino 4.0을 시작하면 다음 오류가 표시됩니다:

**//이 Rhinoceros 라이선스는 <컴퓨터 이름>에서 이미 사용 중입니다.//**

=====답변=====
이 메시지가 표시되는 경우는 다음과 같습니다:
  - 동일한 Rhino 4.0 CD 키가 한 대보다 많은 수의 컴퓨터에서 Rhino를 설치하는 데 사용되었습니다.
  - Zoo 4.0 서버에 추가한 Rhino 4.0 CD 키 중 하나가 독립실행형 노드 설치에 이미 사용되었습니다.

모든 컴퓨터가 작업그룹 (또는 네트워크) 노드로 실행되어야 한다면, Rhino 를 실행할 때 Zoo 4.0에서 라이선스를 취득해야 하므로 다음 안내대로 실행하시면 문제를 해결하실 수 있습니다.

  - Rhino 4.0을 시작합니다.
  - 도움말 -> Rhinoceros 정보를 선택합니다.
  - 정보 대화상자에 라이선스가 독립실행형으로 표시되면 다음을 실행하세요:
    - Rhino를 닫고 관리자 권한으로 Rhino를 실행하여 Rhino를 다시 시작합니다. 
    - 도구 -> 라이선스 관리자 -> 작업그룹 노드를 지정합니다.
    - Rhino를 닫습니다.

Rhino를 다시 시작하면 Rhino가 작업그룹 노드로 실행됩니다.
