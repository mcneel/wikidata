====== Changer la méthode d'activation de la licence de Rhino 6 ======

Il existe maintenant trois options pour activer votre licence dans Rhino 6. Sélectionnez **une** des options suivantes :
  * **Ordinateur unique**, exclusivement sur un seul ordinateur
  * Partagée sur votre réseau en utilisant le **Zoo**
  * Partagée sur le **Cloud Zoo** (Nouveau dans Rhino 6)
[[https://www.rhino3d.com/6/new/licensing-and-administration|Plus d'informations sur ces options...]]

==== Pour changer les méthodes d'activation de la licence====
  - Retirez votre licence de sa position actuelle (voir ci-dessous)
  - Ajoutez votre licence où vous le souhaitez

==== Pour retirer une licence activée directement sur un ordinateur ====

  - **Lancez Rhino 6**
  - Dans le menu **Outils**, cliquez sur **Options**
  - Sélectionnez l'onglet **Licences**
  - Cliquez sur **Changer de clé de licence**
  - Cliquez sur **Retirer la licence**

Votre licence est maintenant disponible pour être utilisée ailleurs. 

> Remarque : Vous pouvez également annuler une validation de licence sur un ordinateur unique en désinstallant Rhino 6. 

==== Pour retirer une licence du Zoo ====
  Ouvrez **Zoo 6 Administrator** sur le serveur exécutant le Zoo (des autorisations d'administrateur son nécessaires)
  - Sélectionnez la licence à supprimer
  - Cliquez sur le bouton **X** dans la barre d'outils

Votre licence est maintenant disponible pour être utilisée ailleurs.

==== Pour retirer une licence activée sur le Cloud Zoo ==== 
  - Visitez la page [[https://www.rhino3d.com/licenses|Licences sur le site de Rhino]]
  - Cliquez sur **votre nom** dans la sections Licences personnelles
  - Cliquez sur le nom de la licence
  - Cliquez sur **Retirer**


Votre licence est maintenant disponible pour être utilisée ailleurs.

==== Pour retirer une licence d'une équipe du Cloud Zoo ====
  - Visitez la page [[https://www.rhino3d.com/licenses|Licences sur le site de Rhino]]
  - Cliquez sur le **nom de l'équipe** qui possède la licence
  - Cliquez sur le nom de la licence
  - Cliquez sur **Retirer** (Vous devez être le propriétaire de l'équipe ou un des administrateurs)


Votre licence est maintenant disponible pour être utilisée ailleurs.
