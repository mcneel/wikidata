====== Cómo eliminar un plugin de Rhino para Mac ======
{{ :rhino:mac:ff.png?100|}}
Para eliminar un plugin en el Mac, debe acceder a la carpeta de la biblioteca User Library.

  - Abra el **Finder** en su Mac. 

  - En la parte superior de la pantalla verá menús desplegables.
{{ :rhino:mac:menu.png?400|}}
  - Uno de menús indica **Ir**. Haga clic en el menú **Ir** para verlo desplegado.
  - A continuación, mantenga pulsada la tecla **Opciones** del teclado y observe que aparece la palabra Biblioteca en la lista del menú Ir.
  - Seleccione Biblioteca -> Soporte de aplicaciones -> McNeel -> 7.0 (o su versión) -> Plug-ins. {{:rhino:mac:pluginsmac.png?600|}}/Users/<login name>/Library/Application Support/McNeel/Rhinoceros/7.0/Plug-ins
  - Borre el archivo del plugin desde aquí (//Twimmotion, Datasmith o el plugin que sea// que tenga cargado. Puede borrarlo para desinstalarlo, cambiarle el nombre o moverlo para que no se cargue.)


Asegúrese de informar al desarrollador del plugin del fallo.
