
=====Convalidare Rhino offline=====

Ci sono due ragioni per farlo: se il computer non è collegato a internet. Oppure se il computer con può collegarsi al server di convalida per via del firewall o di un altro programma o configurazione che ne blocca l'accesso. Per verificare l'accesso al server di convalida, usa questo [[https://api.mcneel.com/|link.]] Dovresti visualizzare una pagina di un sito che non è per esseri umani. Se non puoi raggiungere la pagina, probabilmente il firewall locale ne ha bloccato l'accesso. Le licenze di Rhino vanno convalidate prima di poter essere usate. Per i computer offline, è possibile eseguire la convalida della licenza in modalità offline. - All'apertura di Rhino, viene notificato che il programma non può connettersi al server della convalida. {{ :zoo:validate01.png?direct&200 |}} - Fai clic sul pulsante **"Ottieni il codice di convalida manualmente"**. - Apparirà la seguente finestra di dialogo. {{ :zoo:validate02.png?direct&200 |}} - Inserisci **codice di licenza**, **nome dispositivo** e **ID dispositivo**. - Usa un sistema con accesso a Internet. - Sul browser web, consulta https://www.rhino3d.com/validate. - Inserisci nel modulo le informazioni relative a **indirizzo e-mail**, **codice di licenza** di Rhino, **nome dispositivo** e **ID dispositivo**. {{ :zoo:validate2.jpg?direct&200 |}} - Se esegui l'aggiornamento, dovrai inserire anche il codice di licenza della versione di Rhino precedente. - Il codice di convalida verrà visualizzato sullo schermo. Stampalo o salvalo su un file di testo. //Questo codice deve essere usato entro 24 ore dalla creazione//. - Torna al sistema offline e fai clic sul pulsante **Continua**. - Inserisci il codice di convalida nella procedura guidata per la convalida, quindi fai clic su Avanti. Rhino verrà correttamente convalidato offline. **Nota:** per la distribuzione in rete e l'utilizzo di LAN Zoo offline, i dati necessari vengono mostrati quando provi ad aggiungere una licenza a Zoo. Per istruzioni specifiche sulla convalida tramite Zoo, consulta https://wiki.mcneel.com/it/zoo/rhinovalidateoffline#validating_offline ====== File della guida in linea ====== La guida in linea di Rhino 6 è online. Per usare Rhino offline: - [[https://www.rhino3d.com/download/Rhino/6.0/Rhino6OfflineHelp|Scarica il link del programma di installazione della guida offline]] - Esegui il programma di installazione e assicurati di selezionare **Per tutti gli utenti** durante il processo di installazione. - Ulteriori informazioni su [[https://wiki.mcneel.com/it/rhino/installrhi|Installazione dei file RHI per tutti gli utenti]].
