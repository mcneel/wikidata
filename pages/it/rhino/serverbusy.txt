====== Errore server occupato ======
Questo errore può verificarsi quando si apre o si chiude qualsiasi versione di Rhino per Windows.

{{ :rhino:server_busy.png?400|}}

È molto difficile capire cosa non va in queste finestre di dialogo di Windows, molto poco utili e “dolorose”.
Il server nella finestra di dialogo **//Server Busy//** non si riferisce al nostro server delle licenze.

Si tratta di una finestra di dialogo di errore di Windows piuttosto generica.
Questo è legato a un **server COM** in esecuzione sul computer stesso.

Informazioni sui client e server Microsoft COM disponibili [[https://learn.microsoft.com/en-us/windows/win32/com/com-clients-and-servers|qui]].

==== Consigli per risolvere il problema: ====
  - Applicare tutti gli aggiornamenti di Microsoft.
  - Scaricare e installare l'ultima versione di Rhino da [[https://www.rhino3d.com/download/|qui]].
  - Riavviare il computer.
  - Aprire **TaskManager** di Windows e terminare l'attività per tutti i processi di Rhino ancora aperti.
  - Riavviare almeno ogni settimana come parte della manutenzione del computer.

È possibile scoprire quando altri segnalano questo messaggio sul nostro [[https://discourse.mcneel.com/|Forum di supporto]].
Esiste anche un report relativo al plug-in Enscape.
