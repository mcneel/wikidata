====== Come disinstallare Rhino per Mac ======
> **Riepilogo:** //Seguite i seguenti passi per rimuovere completamente Rhino per Mac dal computer.//

Seguite i seguenti passi per rimuovere completamente Rhino per Mac dal computer:

  - Chiudi tutte le applicazioni che eseguono Rhino.
  - Rimuovi l'applicazione dalla cartella "Applicazioni".
  - Rimuovi la directory **/Libreria/Application Support/McNeel**. Questa directory contiene i modelli dei tutorial ed i manuali scaricati; può non esistere nel computer.  Questa cartella contiene anche il file di licenza (per Rhino 5 e Rhino 6). 
  - Rimuovi la directory  **~/Libreria/Application Support/McNeel**  dalla home directory. Questa directory viene usata opzionalmente da Rhino ed è possibile che non esista nel vostro computer. Può contenere i materiali creati da voi, gli script di Python ed altre risorse.
  - Rimuovi il file **~/Libreria/Preferenze/com.mcneel.rhinoceros.plist** dalla home directory. Questo file contiene tutte preferenze peri Rhino 5 e Rhino 6.

Rhino 7 o versione successiva

  - Rimuovi la directory **/Users/Shared/McNeel**. Questa cartella contiene le informazioni di licenza usate da Rhino 7 e versioni successive.
  - Rimuovi **~/Library/Preferences/com.mcneel.rhinoceros.n.plist** (dove si trova la versione di Rhino) nella home directory. Questo file contiene tutte le vostre preferenze di Rhino.
