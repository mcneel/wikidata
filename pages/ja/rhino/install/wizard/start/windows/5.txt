====== Rhino V5 ライセンスの種類 ======

===以下のうち1つを選んでください:===
    * **[[jp:rhino:install:wizard:start:windows:5:commercial|単一ユーザー商用版、90日間評価版ライセンス]]**
      * 1ライセンスにつきユーザーお1人のご使用が認められます
      * スタンドアロンまたは無償のフローティングライセンスマネージャー、[[https://wiki.mcneel.com/ja/zoo/home|Zoo]] for Windows を利用してLAN経由（評価版を除く）で使うことができます
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|エンドユーザー使用許諾契約書]] （※日本語は15-16ページをご覧ください）

    * **[[jp:rhino:install:wizard:start:windows:5:educational|単一ユーザー教育版ライセンス]]**
      * 1ライセンスにつきユーザーお1人のご使用が認められます
      * スタンドアロンでの展開のみ
        *コンピューターはユーザー本人が所有していること
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|エンドユーザー使用許諾契約書]] （※日本語は15-16ページをご覧ください）

    * **[[jp:rhino:install:wizard:start:windows:5:lab|30ユーザー教育ラボラトリーライセンス]]**
      * 同時に30ユーザーまでお使いになれます
      * 無償のフローティングライセンスマネージャー、[[https://wiki.mcneel.com/ja/zoo/home|Zoo]] for Windows を利用して校内LAN経由（推奨）で、またはスタンドアロンで使うことができます
      * Rhinoのインストールは簡単に[[https://wiki.mcneel.com/ja/rhino/installingrhino/5.0|展開]]することができます
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|エンドユーザー使用許諾契約書]] （※日本語は15-16ページをご覧ください）
