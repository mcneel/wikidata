====== DOTNET Framework 4.0 설치관리자 오류 (종료 코드 -2145124330) ======
**문제:**\\ Rhino 5 를 설치하는 도중 DOTNET Framework 4.0 설치 관리자에서 오류 발생 (종료 코드 -2145124330).

**해결 방법:**\\ Rhino 5 를 설치하기 전에 .NET Framework 를 복구합니다. 
  - [[http://www.microsoft.com/en-us/download/details.aspx?id=30135|Microsoft .NET Framework Repair Tool]] 다운로드
  - Repair  Tool 을 실행합니다. 
  - 계속해서 "Next" 단추를 클릭하여 설치를 종료합니다. 
  - Rhino 5 를 설치합니다.
