====== Rhino 5 30-user EDU Lab License  ======


Select license deployment type:
    * **[[rhino:install:wizard:start:mac:5:edu:stand_alone|30 stand-alone computers in a single lab]]**
      * Use this option when 30 stand-alone computers will be used
      * Does not require Internet access
      * Rhino is installed manually on 30 computers
      * **Not recommended**

    * **[[rhino:install:wizard:start:mac:5:edu:zoo|30 computers using the school LAN]]**
        * Use this option when sharing the license in a school classroom lab
        * Uses the school LAN
        * Requires Windows **[[https://wiki.mcneel.com/zoo/home|Zoo]]** free floating license manager
        * Does not require Internet access
        * Rhino installation easily **[[https://wiki.mcneel.com/zoo/rhinomacdeployment|deployed]]**
        * **Recommended**
