====== Ausführen des Zoos unter Windows XP ======
> **Produkt:** //[[de:zoo:zoo5|Zoo 5]]//
> **Zusammenfassung:** //Anleitung zum Ausführen des Zoos unter Windows XP//

===== Problem =====
Ich versuche, den Zoo unter Windows XP auszuführen. Sobald wir den Zoo auf dem entsprechenden Computer ausführen, stürzt das Programm ab und nichts funktioniert mehr. Läuft der Zoo 5 nicht unter Windows XP?

===== Lösung =====
Zoo 5 läuft ab dem **Windows XP Service Pack 3**. 

Zoo 5 benötigt das **Microsoft .NET Framework 4**.

Der Zoo-Installer enthält dieses Framework nicht, da er dadurch rund 50 MB größer wäre.

**Downloads**
  * [[http://www.microsoft.com/de-de/download/details.aspx?id=17851|Microsoft .NET Framework 4 (Webinstaller)]]
  * [[http://www.microsoft.com/de-de/download/details.aspx?id=17718|Microsoft .NET Framework 4 (Standalone-Installer)]]
