====== Die WIP-Version hat ein Zeitlimit. Was passiert, wenn sie abläuft? (Rhino 5 für Mac) ======
> **Zusammenfassung:** //Die WIP-Version hat ein Zeitlimit. Was passiert, wenn sie abläuft?//

Das Ablaufen einer Version soll sicherstellen, dass alle Nutzer die jeweils letzte Version verwenden und keine Fehler berichten, die schon behoben sind.  Bevor eine RhinoWIP-Version abläuft, steht Ihnen bereits eine neue zum [[http://www.rhino3d.com/go/download/rhino-for-mac/wip/latest|Download]] zur Verfügung. RhinoWIP sucht bei jedem Start nach Aktualisierungen und meldet, wenn eine neue Version verfügbar ist. Zudem wird jedes Mal eine Nachricht im [[http://discourse.mcneel.com/category/rhino-for-mac|Rhino-Mac-Bereich]] des [[http://discourse.mcneel.com/ |Discourse-Forums]] gepostet, wenn eine neue Rhino-WIP-Version verfügbar ist.

Siehe auch: [[de:rhino:mac:wip|Rhino für Mac WIP (Work-In-Progress)]].
