======Rhino 5如何寻找Zoo服务器主机？======
> **产品：** //[[zh-tw:zoo:zoo5|Zoo 5]]//
> **摘要：** //Rhino 5如何寻找Zoo服务器管理主机概述。//

=====概述=====
不像Rhino 4.0可以自动在同个网域内寻找Zoo 4.0服务器主机,Rhino 5需要知道Zoo 5服务器主机名称。

当Rhino 5要从Zoo 5取得一个授权，会依照以下顺序确认查找Zoo 5服务器主机名称：
  - 查找本地计算机的Windows注册表主机名称或您服务器主机IP位置。
  - 查找目前使用者的Windows注册表主机名称或您服务器主机IP位置。
  - 查询您的网域名称系统(DNS)服务器为预设Zoo服务器主机名称。
 

=====更多讯息=====

====注册表查找-目前使用者====
当您查找登录编辑程序HKEY_CURRENT_USER的Zoo server名称或IP位置。Rhino 5 32及64位元系统将到以下位置寻找：

<code>
Hive:  HKEY_CURRENT_USER
Key:   Software\McNeel\Rhinoceros\5.0\License Manager
Name:  Server
Type:  REG_SZ
Value: <host name or IP address>
</code>

====注册表查找-Local Machine====

若您使用32位元Windows作业系统，查找登录编辑程序HKEY_CURRENT_USE的Zoo server名称或IP位置。Rhino 32位元程序会寻找下列位置：

<code>
Hive:  HKEY_LOCAL_MACHINE
Key:   Software\McNeel\Rhinoceros\5.0\License Manager
Name:  Server
Type:  REG_SZ
Value: <host name or IP address>
</code>

若您使用64位元Windows作业系统，查找登录编辑程序HKEY_CURRENT_USE的Zoo server名称或IP位置。Rhino 32及64位元程序__皆__会寻找下列位置：

<code>
Hive:  HKEY_LOCAL_MACHINE
Key:   SOFTWARE\Wow6432Node\McNeel\Rhinoceros\5.0\License Manager
Name:  Server
Type:  REG_SZ
Value: <host name or IP address>
</code>
====使用DNS====
若Rhino 5无法通过登录编辑程序确认您的Zoo 5服务器主机名称，则会查询您的DNS服务器名称为Zoo预设名称，以下则是主机名称：

<code>
__mcneel.__zoo5
</code>

举例，若贵公司主机名称为"mycorp.com"，那么您可以增加一个DNS别名为<nowiki>"__mcneel.__zoo5.mycorp.com"</nowiki>用来指向您的Zoo服务器名称。
