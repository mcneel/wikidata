====== Windows용 Rhino와 Mac용 Rhino의 차이 ======
=== Mac전용 사용자 인터페이스 추가 사항 ===

Mac 버전에서만 사용하실 수 있는 명령과 기능이 많습니다. 자세한 정보는 [[http://wiki.mcneel.com/rhino/mac/userinterfacefeatures|이 wiki 페이지]]를 참조하세요.

-->마우스#

**권장 사항**

  * 마우스 오른쪽 단추로 쉽게 탐색하려면 스크롤 휠이 부착된 2 단추식 마우스를 권장합니다. 하지만, 단추가 1개인 마우스로도 모든 작업이 가능합니다.
  * 단추가 1개인 마우스로 마우스 오른쪽 단추 기능을 실행하려면 **Control** 키를 누른 상태에서 마우스 단추를 클릭합니다.\\ \\

**탐색**

  * **마우스 오른쪽 단추로 끌기**와 **Shift+마우스 오른쪽 단추 끌기** 작업은 Windows와 동일합니다. [[http://docs.mcneel.com/rhino/5/help/en-us/user_interface/shortcuts.htm|Rhino 도움말 - 바로 가기]]를 참조하세요. \\ 뷰 투영 스타일과 상관없이, **Shift** 키를 누르면 항상 초점 이동(pan)이 실행되고, **Command** 키를 누르면 항상 확대/축소(zoom)되며, **Command+Shift** 키를 누르면 항상 뷰가 회전(rotate)합니다.
  * 스크롤 휠을 사용하여 뷰를 확대/축소합니다.
  * 1 단추식 마우스를 사용하는 경우, **Control**, **Command**, **Shift** 키를 같이 눌러 뷰를 회전할 수 있습니다.

<--

-->오른쪽 클릭 메뉴#

  * 뷰포트에서 오른쪽 클릭하면 상황에 맞는 메뉴가 표시됩니다.
  * 기본적으로 오른쪽 클릭은 마지막 명령을 반복하는 **Enter** 키의 역할을 **하지 않습니다**. \\ 이 액션을 변경하려면 **Rhinoceros** > **환경설정** > **마우스** > **마우스**에서 **상황에 맞는 메뉴 사용**을 끕니다.

<--

-->명령행#

Windows용 Rhino의 명령행에 익숙하시다면 Windows Rhino의 명령행의 모든 기능은 Mac용 Rhino에서도 같습니다. 명령행을 활성화하려면 입력을 시작하세요. **명령 검색** 창이 표시됩니다. 명령을 시작하려면 Enter 또는 Return을 누르거나 스페이스바를 누르세요.

Windows용 Rhino의 프롬프트에 표시되는 모든 명령 옵션이 대화상자로 표시됩니다. 이 대화상자를 화면에서 마우스로 끌어오면 Rhino가 해당 위치를 기억합니다.

명령 옵션 값을 입력하려면 Legacy 환경설정 패널에서 **Always provide an input field in command options dialog for typed values** 항목을 선택합니다. 각 옵션 프롬프트에서 한 글자씩 밑줄이 생깁니다. 해당 글자를 입력하고 Return 키를 누르면 해당 옵션이 적용됩니다. 

명령 옵션 대화상자에서 값을 입력하려면 커서를 움직일 필요가 없습니다. 입력하면 Rhino에서 사용자가 입력한 값이 자동으로 대화상자에 입력됩니다. Windows용 Rhino에서의 명령행 실행 방법은 Mac용 Rhino에서도 완전히 동일합니다.

-->도구모음과 도구 팔레트#

  * Mac용 Rhino에는 표준 Macintosh 도구모음에서 Rhino 명령을 적용하는 //도구 단추//가 있습니다.
  * Mac용 Rhino에는 또한 도구 단추 모음이 있는 //도구 팔레트//가 있습니다. \\  자주 사용되는 도구 단추의 도구 팔레트는 Mac용 Rhino가 시작할 때 문서 창의 왼쪽에 열립니다.
  * 도구 팔레트 단추에는 하나 이상의 명령과 메뉴가 포함될 수 있습니다.
  * 도구 팔레트 단추에 마우스를 가져가면 단추의 도구 설명이 표시됩니다.\\  도구 설명에 2줄의 텍스트가 있으면 해당 단추로 두 개의 명령(메인, 대체)이 실행됩니다.
    * 도구 팔레트 단추를 **클릭**하면 **메인** 명령이 실행됩니다.
    * **대체** 명령을 실행하려면 **Option** 키를 누르고 해당 도구 단추를 **클릭**합니다.
  * 메뉴가 있는 도구 팔레트 단추에는 왼쪽 하단 모서리에 작은 삼각형이 있습니다.
    * 해당 메뉴를 열려면, 도구 팔레트 단추를 클릭하고 잠시 그 상태를 유지합니다.
  * 도구 팔레트 메뉴는 떠 있는 도구 팔레트로 표시할 수 있습니다.
    * 떠 있는 도구 팔레트로 표시하려면 **Option** 키를 누른 상태에서 마우스 단추를 클릭하고 잠시 그 상태를 유지합니다.

-->키보드#

  * Windows용 Rhino는 많은 작업에 **Control** 키를 사용합니다. Mac 버전에서는 그 대신 **Command** 키를 사용합니다.
    * 예를 들어, 개체를 선택 해제할 때 **Command** 클릭은 **Ctrl** 클릭을 대체합니다. 
    * 또한, **Command** 키는 **Ctrl** 키를 대신하여 탐색에 사용됩니다:

| | | |**키** | **액션** | **+ Command 키** |
| | | |**왼쪽 화살표** | 왼쪽 회전 | 왼쪽 초점 이동 |
| | | |**오른쪽 화살표** | 오른쪽 회전 | 오른쪽 초점 이동 |
| | | |**위쪽 화살표** | 위로 회전 | 위로 초점 이동 |
| | | |**아래쪽 화살표** | 아래로 회전 | 아래로 초점 이동 |

-->노트북 컴퓨터 관련 사항#


**블루투스 마우스**\\ 노트북 컴퓨터는 1 단추식 마우스와 함께 출시되므로, 블루투스 마우스 (Logitech V270 등) 를 추가로 사용하시면 유용합니다.\\ \\

**Trackpad**\\ 두 손가락을 trackpad에서 위/아래로 슬라이딩하면 현재 뷰가 확대 또는 축소됩니다. \\ 이 액션을 변경하려면 **Rhinoceros** > **환경설정** > **Trackpad** 에서 **뷰를 초점 이동, 회전하려면 두 손가락 사용** 항목을 선택할 수 있습니다.\\

**키보드에 없는 키**\\ 노트북 컴퓨터의 키보드에는 별도의 **Home**, **End**, **PageDn**, **PageUp** 키가 없고, **fn** 키와 다른 키를 조합하여 같은 기능을 실행할 수 있습니다. \\

| **키** | **+ fn key** |
| **위쪽 화살표** | Page Up |
| **아래쪽 화살표** | Page Down |
| **왼쪽 화살표** | Home |
| **오른쪽 화살표** | End |
| **Delete** | Delete |
