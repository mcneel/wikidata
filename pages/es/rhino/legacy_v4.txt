===== Ejecutar Rhino V4 en Windows 7, 8.1 o 10 (64 bits) =====

Rhino V4 fue diseñado para ejecutarse en Windows XP, pero también puede ejecutarse en las versiones modernas de Windows. Estas son las limitaciones y los requisitos para ejecutar Rhino V4 en las versiones más recientes de Windows.

==== Limitaciones y requisitos ====
  * **Memoria** - Su nuevo equipo de 64 bits probablemente venga con una memoria de 16GB o más. Rhino 4 es un aplicación de 32 bits. Esto significa que está limitado a usar un máximo de 4 GB de RAM, independientemente de la cantidad de RAM que tenga su nuevo sistema. 
  * **Aplicación antigua** - Rhino V4 se lanzó antes de que Microsoft agregara el **Control de cuentas de usuario** en Windows. Para evitar esta función, deberá ejecutar Rhino V4 en **Modo de compatibilidad con Windows XP SP3** o Windows impedirá que Rhino guarde los cambios en las barras de herramientas y otras configuraciones cuando se cierre Rhino. Las opciones del modo Compatibilidad de Windows forman parte las propiedades del acceso directo del Escritorio.
  * **Baja resolución** - Su nuevo equipo probablemente pueda funcionar con resoluciones de pantalla mucho más altas que su antiguo sistema de Windows XP. Deberá limitar Windows y ejecutarlo con una resolución de pantalla más baja (como 1024 x 768), que era compatible cuando se lanzó Rhino V4. Si no lo hace, algunas herramientas como el **Panel de capas** estarán en blanco y los iconos de la barra de herramientas de Rhino V4 se verán muy pequeños.

==== Considere probar Rhino V6 ====
Además de las [[https://www.rhino3d.com/6/features|nuevas funciones y características]] de Rhino V6, para aprovechar al máximo las funciones de su nuevo sistema Windows de 64 bits, incluida la memoria RAM, resolución, etc., debería considerar actualizar a Rhino V6. En enero de 2007, cuando se lanzó Rhino V4, era compatible con las características de Windows XP de 32 bits. Windows y el hardware informático han avanzado mucho desde entonces. Para aprovechar estas nuevas características y funciones, necesitará una versión de Rhino diseñada para ello.\\

Pruebe la [[https://www.rhino3d.com/download/rhino-for-windows/6/evaluation|licencia de evaluación de 90 días de Rhino V6]] gratuita y compruebe si le gusta más en su nuevo equipo.
