====== Rhino 6 license type ======


===Choose one:===
    * **[[rhino:install:wizard:start:windows:6:commercial|Single Commercial and 90-day Evaluation License]]**
      * Allows one user per license
      * May be used stand-alone or over a LAN (except evaluation) using the Windows [[https://wiki.mcneel.com/zoo/home|Zoo]] free-floating license manager
      * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:educational|Single Educational License]]**
      * Allows one user per license
      * Stand-alone deployment only
        * User must own computer 
      * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:lab|30-user EDU Lab License]]**
      * Allows 30 concurrent users
      * May be used over a school LAN using the Windows [[https://wiki.mcneel.com/zoo/home|Zoo]] free floating license manager (recommended), or stand-alone
       * Rhino installation easily **deployed**:
          * [[https://wiki.mcneel.com/rhino/installingrhino/6|Windows]]
          * [[https://wiki.mcneel.com/zoo/rhinomacdeployment|Mac]]
      * [[https://www.rhino3d.com/6/eula|End User License Agreement]]
