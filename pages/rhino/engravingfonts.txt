====== Engraving Fonts for Rhino ======

Is there a single stroke font you recommend when working with Rhino and RhinoCam?

Yes. Dowload these [[http://wiki.mcneel.com/_media/legacy/en/MecSoft_Font-1.zip|engraving fonts]]

  * 1. Download and unzip the attached font file to a temporary folder on your computer.
  * 2. Copy **MecSoft_Font-1.ttf** to **C:\Windows\Fonts.** The font will now be installed on your computer.
  * 3. Open Rhino 4.0
  * 4. Select **TextObject** command
  * 5. Make sure to select curves and allow single stroke font.
  * 6. Select **MecSoft_Font-1** from the list of fonts.
