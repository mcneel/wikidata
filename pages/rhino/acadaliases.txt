====== AutoCAD Aliases for Rhino ======

==== Question ====
AutoCAD has a number of useful command aliases. For example, if you enter "L" at the AutoCAD command prompt, the Line command will run. Can these aliases be build into Rhino's standard set of aliases?

==== Answer ====
Rather than build AutoCAD aliases into Rhino, we have provided a standard set of AutoCAD aliases that you can import into Rhino.

=== Setting up Rhino for Windows ===
  - **{{:rhino:acadaliasesforrhino.zip|Download}}** and unzip the AutoCAD Aliases for Rhino text file.
  - Launch Rhino for Windows and click **Tools -> Options**.
  - Navigate to the **Aliases** page and click the **Import** button.
  - Select the AutoCAD Aliases for Rhino text file you just unzipped.

===Setting up Rhino for Mac===
  - **{{:rhino:acadaliasesforrhino.zip|Download}}** and unzip the AutoCAD Aliases for Rhino text file.
  - Launch Rhino for Mac and click **Rhinoceros -> Preferences**.
  - Click the **Aliases** option, and then click **Import aliases from file...** drop down menu item.
  - Select the AutoCAD Aliases for Rhino text file you just unzipped.
