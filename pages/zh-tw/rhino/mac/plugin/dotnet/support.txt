====== Rhino for Mac版本可支援.NET外掛嗎? ======
> **摘要：** //目前已開發出可在 Windows 和 Mac 系統上執行的 .NET 插件。//


您可以編寫可在 Windows 版 Rhino 和 Mac 版 Rhino 上執行的外掛程式。開發者必須使用 RhinoCommon（C# .NET）；使用 C/C++ 編寫的外掛程式在 Mac 版 Rhino 上不支援。

有關 Mac 版 Rhino 支援的外掛程式列表，請參閱 [[http://www.food4rhino.com/browse?sort_by=ds_changed&f[0]=im_field_unified_type%3A773&f[1]=im_field_platform_app%3A726|Food4Rhino]]。

開發者資訊請參閱我們[[http://developer.rhino3d.com/guides/rhinocommon/|開發者指南中的 RhinoCommon 部分]]。
