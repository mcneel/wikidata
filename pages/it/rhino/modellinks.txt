  * [[http://www.rhino3dmodels.com|rhino3dmodels.com]] - Oltre 750 modelli di Rhino scaricabili gratuitamente. rhino3dmodels.com funziona con un sistema di scambio. Per poter scaricare un file, occorre prima caricare. Per caricare, occorre registrarsi. Se il file caricato viene approvato, si può procedere con il download. Il numero di file scaricabili corrisponde al numero di file approvati e caricati: se caricate un file approvato, potete scaricare un file, se ne caricate due, potete scaricarne due, e così via.

=====Cataloghi CAD=====
  * [[http://www.mcmaster.com/|Catalogo McMaster-Carr]] McMaster-Carr mette a disposizione molti dei suoi modelli di viterie e bullonerie. Non sono perfetti ed alcuni di essi vanno un po' ricostruiti, comunque è molto meglio che iniziare da zero. Una volta trovato il pezzo desiderato, nella pagina corrispondente (con un piccolo schizzo del pezzo), troverete un pulsante per il download nella parte superiore; selezionate IGES per scaricare il file e lo potrete aprire direttamente in Rhino! --Sam

----
  * [[http://catalog.tycoelectronics.com|Catalogo Tyco Electronics]] Simile a McMaster-Carr:  Una volta trovato il pezzo desiderato, nella pagina corrispondente, alla voce "Corporate P/N", troverete un link a "CAD Geometry" e, se esiste il pezzo in 3D, selezionate IGES per scaricare un file IGES zippato. - GRATIS
