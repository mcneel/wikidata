======Rhino Polyhedra======
> **Summary:** //Rhinoceros 5 Labs plug-in to create polyhedra.//
> **Updated:** //December 30, 2013//

=====Description=====
The Rhino Polyhedra plug-in for Rhinoceros 5 will allow you to create a variety of [[http://en.wikipedia.org/wiki/Polyhedron|polyhedra.]]

{{:labs:rhinopolyhedralg.png|}}

Rhino Polyhedra supports both 32-bit and 64-bit Rhinoceros 5.

=====Installation=====
  - Download the plug-in. File is in Rhino Installer file (.RHI) form.
  - Double-click in the file to launch the Rhino Installer and install the plug-in.

=====Usage=====
1. Run the **Polyhedron** command, or

{{:labs:polyhedron.png?360}}

2. Run the **PolarZonohedron** command.

{{:labs:polarzonohedron.png?360}}
