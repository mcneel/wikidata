======Zoo - ライセンスマネージャー======
[[zoo:home|{{:zoo:zoo_small.png|Zoo Home}}]]

=====概要=====
Zooは同じネットワークでRhino、Flamingo、Penguin、Bongo、Brazilのライセンス（CD-Key）を共有できるようにするものです。これらの製品がスタンドアロンの代わりにワークグループノードとしてインストールされている場合、次のように動作します：

  * ワークグループノードを開始すると、（アプリケーションは）Zooからのライセンスを要求します。 
  * 使用されていないライセンスがそのノードに割り当てられます。
  * ノードがシャットダウンされると、キーはZooのライセンスプールに返却されます。

** Zooは無償です。**

**[[http://download.rhino3d.com/zoo/4.0/release/download/|Zoo 4.0のダウンロード]]** - Rhino 4、Flamingo、Penguin、Brazil、Bongo用（//Rhino 5はサポートされていません。//）

===== 機能 =====
  * **余分なコストが不要** - 特別なバージョンのRhino、Flamingo、Penguin、Bongo、Brazilは必要ありません。
  * **特別なハードウェアは必要ありません** - Zooは、同じネットワークワークグループの中のどのシステム上でも動きます。
  * **特別なソフトウェアは必要ありません** - ZooはWindows XP、Vista、7、Windows Server 2000、2003、2008を含む、どのWindowsシステムでも動作します。
  * **シンプルなセットアップ** - ワークグループのいずれかのシステムにZooをインストールし、個々のシステムの代わりにライセンスキーを入力します。
  * **チェックアウト** - ライセンスキーはチェックアウトが可能で、ノートパソコンのユーザーはネットワークから接続を切断することができます。ノートパソコンをネットワークに再接続する際は、再びキーをチェックインすることができます。
  * **安定性** - ネットワーク接続やサーバーがダウンしても、ワークグループ・ノードはそのまま機能し続けます。
  * **ルータブル** - Zooはシングルセグメントのネットワークでも、システムがルーターで分けられたネットワークでも動作します。
  * **混在環境** - 同じWindowsドメインやワークグループでの、ワークグループノードとスタンドアロンノードの混在をサポートしています。
  * **サービスとして動作** - Zooのあるシステムを再起動すると、Zooは自動的に起動します。

** Zoo 5.0の新機能**

Zoo 5.0はZooの全く新しいバージョンです。完全に書き換えられたZoo 5.0は、以前のバージョンにはなかった次を含む新しい機能を提供します。
  * **標準インターネット・プロトコルのサポート** - クライアントはHTTP（ポート80）を使ってZoo 5.0と通信しますので、システム管理者は複雑なネットワークでのZooの要求をルーティングしやすくなり、インターネットでも可能になります。
  * **ファイアウォールフレンドリー** - Zoo 5.0はクライアントに定期的にサーバーをチェックするよう要求しますが、クライアントに対して接続を開始することはありません。管理者がクライアントのコンピューターでファイアウォールのポートを開く必要はなくなりました。
  * **ライセンスの制限付きチェックアウト** - Zooの過去のバージョンではノートパソコンのユーザーや出張のために、ライセンスのチェックアウトができましたが、チェックアウトの期間に制限はありませんでした。Zoo 5.0では管理者がライセンスをチェックアウトする期間を指定できるようになりました。
  * **サードパーティプラグインのサポート** - Zooの過去のバージョンは（Rhino、Flamingo、Bongo、Brazil、Penguinなど）McNeel製品のライセンスのみを管理していましたが、Zoo 5.0はサードパーティのプラグイン開発者がZooへの製品サポートを追加できるようになりました。


=====動作環境=====
  * Windowsベースのシステム:
    * Windows XP、Windows Vista、Windows 7、またはWindows 8
    * Windows Server 2003、Windows Server 2008、またはWindows Server 2008 R2
  * [[http://www.microsoft.com/en-us/download/details.aspx?id=17718|Microsoft .NET Framework 4]]
  * TCP/IPが設定がされたネットワーク接続
