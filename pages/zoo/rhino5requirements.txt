======Zoo - System Requirements======
> **Product:** //[[zoo:home|Zoo 5]]//
> **Summary:** //Outlines the system requirements for running Zoo 5.//

=====System Requirements=====
  * A Windows-based system.
  * [[http://www.microsoft.com/en-us/download/details.aspx?id=17718|Microsoft .NET Framework 4]]
  * A network connection configured for TCP/IP.

Note, Zoo 5 communicates using **[[[:zoo:window7firewall|TCP Port 80]]**. So, the Windows-based system you choose to install Zoo 5 on should not already have other software running that uses **[[[:zoo:window7firewall|TCP Port 80]]**, including:

  * [[http://www.iis.net/|Internet Information Services (IIS) for WindowsÂ® Services]]
  * [[http://httpd.apache.org/|Apache HTTP Server]]
  * [[zoo:zooskype|Skype]]
