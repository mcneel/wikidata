=====What's new in Bongo 2.0?=====
  * [[bongo:IK|Inverse Kinematics]]
  * [[bongo:CurveEditor|Curve Editor]]
  * [[bongo:BetterTweening|Much better Tweening]]
  * System to be able to animate any property in any supported plug-in. For example
    * [[bongo:RDKSun|The RDK sun]]
    * [[bongo:RDKMaterial|RDK Material properties]]
    * [[bongo:RDKTexure|RDK Texture properties]]
    * [[bongo:RDKEnvironment|RDK Environment properties]]
  * [[bongo:Zoomabletimeline|Zoomable timeline]]
  * [[bongo:ImprovedKeyframeEditor|Much improved Keyframe Editor]]
    * [[bongo:EditmultKeyframes|Editing of multiple Keyframes]] 
  * [[bongo:Morphing|Simple morphing]]
  * [[bongo:NamedViewsAnimation|Named Views Animation]]
  * [[bongo:LayerAnimation|Animating Layers]]
  * [[bongo:RPC|Animate RPC's(only supported in Rhino 5 WIP)]]
  * [[bongo:AnimateVideoTexture|Animate Video Texure(RDK)]]
  * Support for new video file formats, like WMV
  * Support for 64-bit Windows (compatible with 64-bit Rhino 5 WIP)
