====== Rhino Won't Start with VCRUNTIME Error======
We have seen this with Rhino 7 & 8.
Related to Forum posts

  * [[https://discourse.mcneel.com/t/rhino-cycles-won-t-load-issue/208464/9?u=theoutside|rhino-cycles-won-t-load-issue]]
  * [[https://discourse.mcneel.com/t/starting-rhino-8-after-new-installation/174958|Rhino not starting after new install]]

**Problem: Rhino won't start with an error that points to Windows VCRUNTIME dll** 

//"C\Windows\SYSTEM32\VCRUNTIME140_1.dll is either not designed to run on Windows or it contains an error. Try installing the program again using the original installation media or contact your system administrator or the software vendor for support. Error status Oxc000012f"//

or 

//I’ve tried launching Rhino in Safe Mode and also attempted a repair installation, but it still doesn’t work
If I inhibit RhinoCycles from loading and start Rhino 8 normally, it will start with no issues. If I turn back on RhinoCycles it will crash at start.// 


===== Help Us Help Others =====

This problem is a growing issue for us, and we wish we could just make it go away. Until we know more, customers like you will need to ask us for help and potentially wait days until this issue is resolved.

  - To **Apps & features**, click the //Windows Start// button, type **Apps & Features**, and press Enter.
  - Click //Sort by// > //Install date//
  - Send a screenshot of the most recently installed apps to support@mcneel.com, with the subject "VC Runtime Fix Screenshot"




===== Possible Fix =====

This sounds like the redistributable issue that has been reported.
Try a repairing on **Microsoft C++ Redistributables**.
//Start with x86 C++ Redistributable first, then the x64 C++ Redistributable.//

  - **To Repair 64-bit Microsoft C++ Redistributables**
    - In Apps & Features, Right-click //Microsoft C++ Redistributable (x64)//, then click Change.
    - In //Microsoft C++ Redistributable Setup//, click //Repair//.
    - //Close// when setup is successful.
    - //Start Rhino// If it starts, you're done. If not, continue. 
  - **To Repair 32-bit Microsoft C++ Redistributables**
    - In Apps & Features, Right-click //Microsoft C++ Redistributable (x86)//, then click Change.
    - In //Microsoft C++ Redistributable Setup//, click //Repair//.
    - //Close// when setup is successful.
    - //Start Rhino//. If it starts, you're done. If not, continue. 
  - **Install Windows Updates**
    - To open **Windows Update**, click the //Windows Start// button, type **Windows Update**, and press Enter.
    - Make sure that all Windows updates have been successfully installed.
