======Validación de la licencia de Rhino 5======
Rhino 5 dispone de un nuevo sistema de validación de licencias que permite reducir la piratería de las licencias de Rhino. La validación de licencias asocia la licencia con su dirección de correo electrónico (o dominio de correo electrónico en el caso de una licencia Lab) y, en las licencias de actualización, con el código del licencia anterior. 

El contrato de licencia de Rhino permite tener Rhino instalado en varios ordenadores, siempre que la licencia esté en uso en un solo ordenador a la vez. El sistema de validación permite instalar la licencia en varios ordenadores, siempre que la licencia se valide con la misma información cada vez.

===== Actualización de versiones anteriores =====
Cuando actualice Rhino desde una versión anterior, la validación de licencias asociará la licencia de la versión anterior de Rhino con la licencia a la que se va a actualizar. Una licencia de una versión anterior solo puede utilizarse para actualizar una licencia de Rhino 5. Por ejemplo, una empresa tiene cuatro licencias de Rhino 4 y actualizaciones a Rhino 5. Cada instalación de actualización debe asociarse a una licencia de Rhino 4 diferente para que la validación funcione.

===== El proceso de validación =====
Después de instalar Rhino, dispondrá de un período de 30 días para completar el proceso de validación.

Rhino mostrará el Asistente de validación de licencias. Siga las instrucciones en pantalla para validar la licencia.

Si recibe un error durante la validación de licencias, póngase en contacto con el [[http://www.rhino3d.com/support|Soporte técnico de McNeel]] para obtener ayuda. Incluya la descripción del error cuando se ponga en contacto con el soporte técnico.

===== Validación sin conexión =====
Si su ordenador no está conectado a Internet, el Asistente de validación de licencias mostrará instrucciones para validar Rhino a través de un navegador web en otro ordenador. El proceso funciona de la siguiente manera:

  - Visite https://www.mcneel.com/validate/rhino/5
  - Introduzca la dirección de e-mail y el código de licencia
  - Si se solicita, introduzca el código de licencia de una versión anterior de Rhino
  - El código de validación aparecerá en la pantalla
  - Imprima o anote el código de validación
  - Vuelva al equipo que no tiene conexión a Internet
  - Inicie Rhino 5.
  - Introduzca el código de validación en el Asistente de validación y haga clic en Siguiente

Rhino se validará correctamente sin conexión.
