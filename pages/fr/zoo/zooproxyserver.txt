======Utiliser LAN Zoo avec un proxy======
> **Produit :** //[[fr:zoo:home|Zoo]]//
> **Résumé:** //Explique comment contourner un serveur proxy  pour accéder à un serveur LAN Zoo.//

De nombreuses structures utilisent de nos jours des proxy pour protéger leurs utilisateurs des intrusions et pour utiliser moins d'adresses IP publiques. Un proxy est un serveur placé entre vous et Internet. Il reçoit toutes les demandes d'un client sur Internet, remplit la demande puis renvoie les informations au client.

=====Contourner un serveur proxy=====
Il est parfois nécessaire de contourner le proxy et d'accéder au site Internet ou à la ressource directement. Comme par exemple pour le serveur Zoo, une ressources disponible localement, qui est disponible à travers un accès réseau direct.

Windows permet d'ajouter et de configurer des proxy dans les paramètres de l'explorateur Internet.

Vous pouvez choisir d'activer l'option **Ne pas utiliser de serveur proxy pour les adresses locales** afin de contourner toutes les adresses locales :

  - Cliquez sur **Démarrer** puis sur **Panneau de configuration**.
  - Cliquez sur **Réseau et Internet** puis sur **Options Internet**.
  - Dans la boîte de dialogue Options Internet, ouvrez l'onglet **Connexions**.
  - Sous **Paramètres de réseau local**, cliquez sur le bouton Paramètres réseau.
  - Cochez la case **Ne pas utiliser de serveur proxy pour les adresses locales**.
  - Cliquez sur **OK** pour terminer le processus de configuration du proxy.
