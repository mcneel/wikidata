====== Rhino para Mac Work-In-Progress (WIP) ======

Si ha comprado Rhino 7 para Mac, tiene acceso a la [[http://www.rhino3d.com/go/download/rhino-for-mac/wip/latest|WIP de Rhino para Mac]].  La versión WIP sirve para testear las nuevas funciones, reparar errores y realizar otras mejoras.

**[[http://www.rhino3d.com/go/download/rhino-for-mac/wip/latest|DESCARGUE LA WIP]]** (requiere el código de licencia de Rhino 7 para Mac) \\
__**La WIP de Rhino para Mac NO debe usarse para trabajos importantes, ya que puede ser inestable.**__

La [[http://www.rhino3d.com/go/download/rhino-for-mac/wip/latest|WIP de Rhino para Mac]] se puede ejecutar en el mismo sistema junto con la versión comercial (aunque no simultáneamente). Hay dos aplicaciones diferentes:

{{ :undefined:rhinowipindock.png?direct |}}

Rhinoceros (la versión comercial) y la WIP de Rhino para Mac tienen dos sistemas de actualización diferentes.  La WIP de Rhino para Mac se actualizará con más frecuencia que la versión comercial.

Rhinoceros y la WIP de Rhino para Mac comparten las preferencias y la licencia.

**ExplicitHistory** (Grasshopper) **requiere:**
  * 10.12 Sierra
  * 10.11 El Capitan
