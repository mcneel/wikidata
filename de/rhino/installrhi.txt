====== Installation von RHI-Dateien für alle Benutzer ======
Ab Rhino 5.6 können Sie auswählen, ob Sie das Programm nur für einen oder gleich für alle Benutzer des Computers installieren möchten. Zudem kann unter Verwendung eines Skripts eingestellt werden, dass die Bereitstellung von RHI-Dateien im Hintergrund abläuft.

=====Bereitstellung von RHI-Dateien=====
  - Es muss mindestens Rhino 5.6 in Ihrem Netzwerk bereitgestellt sein.
  - Führen Sie folgende Befehle aus: "C:\Programme\Rhinoceros 5 (64-bit)\System\x64\rhiexec.exe" "C:\Vollständiger\Pfad\Zu\Installer.rhi" /admin /silent

Die Option "/admin" sorgt für die Installation für alle Benutzer des Computers.
Die Option "/silent" sorgt für eine Installation ohne Anzeige der Benutzeroberfläche.
