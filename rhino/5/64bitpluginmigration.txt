===== 64-bit Rhino 5.0 Plug-ins =====
On March 30, 2011, McNeel will release an update to 64-bit Rhino 5.0 that will not be compatible with existing 64-bit C++ plug-ins.

==== Background Information: ====
We've updated the 64-bit builds of Rhino 5.0 to be compiled with Visual Studio 2010. This makes it possible for developers to write plug-ins with the newest programming tools available from Microsoft. 32-bit Rhino 5.0 is (an all 32-bit plug-ins) are still compiled with Visual Studio 2005. This allows 32-bit Rhino 5.0 to load Rhino 4.0 plug-ins.
