====== Zoo Requires Microsoft .NET Framework 4.5 ======
> **Product:** //[[zoo:home|Zoo]]//
> **Summary:** //Discusses Zoo and the Microsoft .NET Framework 4.5//


The Zoo requires **[[https://www.microsoft.com/en-us/download/details.aspx?id=30653|Microsoft .NET Framework 4.5]]**. 

You can ensure that Microsoft .NET Framework 4.5 is installed on your system by using **[[http://windows.microsoft.com/en-us/windows/windows-update|Windows Update]]**.

To detect the versions of the  Microsoft .NET Framework installed on your system, use the **

[[http://mcneel.s3.amazonaws.com/zoo/Utils/DetectFX.exe|Microsoft .NET Framework Detection Utility (Win32)]]**.

[[http://mcneel.s3.amazonaws.com/zoo/Utils/DetectFXx64.exe|Microsoft .NET Framework Detection Utility (x64)]]**.
