====RhinoCore.dll Could Not Be Loaded====

**Symptoms:** You start Rhino 8 and see the error "RhinoCore.dll Could Not Be Loaded"
{{:rhino:rhinocore-dll-could-not-be-loaded.png?400|}}

**Cause:** One or more dependencies of Rhino failed to install or need to be repaired.

====Solution====

  - Download and Run [[https://www.microsoft.com/en-us/download/details.aspx?id=30135|.NET Framework Repair tool]]
  - Download and install the [[https://aka.ms/vs/17/release/vc_redist.x64.exe|latest Visual Studio 2015/2019/2022 runtimes]] from Microsoft
