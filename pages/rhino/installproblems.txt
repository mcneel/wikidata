====== Troubleshooting Rhino 5 Installation ======
> This article is for Rhino 5. A similar article is available for [[http://wiki.mcneel.com/rhino/installproblems_legacy|Rhino 3 and 4]].

**Important Note -** Nearly all of the problems and fixes listed on this page can be avoided if Windows is fully up to date with **important updates** using **[[http://windowsupdate.microsoft.com/|Windows Update]]**, before installing Rhino. To find the status of Window update on your computer got to the [[http://windowsupdate.microsoft.com/|Windows Update locator page on Microsoft's website]].\\

Once Windows update is open on the computer, click on the Check for Updates button. After installing new updates, please try and install Rhino again.

In the case Windows update seems to be stalled, shows an error or does nto seem to finish installing updates, go to [[https://support.microsoft.com/en-us/kb/2700567|Microsoft Update Troubleshooting]].

**Reporting Install Problems -**
The Rhino V5 installer will display a log file when the installation process fails.  This log file is very helpful to our technical support in solving any problems you may have. 
  - When the error message appears, there is a link on the dialog. 
  - Click the link to open the log and "Save As.." as a txt file.
  - Attach the text file to an email to [[tech@mcneel.com]]
