=====Rhino 5의 레이아웃 소개=====

====레이아웃이란?====

레이아웃은 모델을 배치하고 주석을 달고, 출력하는 데 사용됩니다. 레이아웃의 용도:
{{ :rhino:anno_layout_tutorial1.png?300|}}

      * 모델의 뷰(디테일)을 다른 배율과 방향으로 배치합니다.
      * 제목 블록과 일반적인 노트를 레이아웃에 주석으로 추가합니다.
      * 디테일의 배율을 지정하고 레이아웃을 1=1로 출력합니다.
      * 디테일 또는 레이아웃에 텍스트, 치수를 만듭니다.
      * 텍스트, 치수, 해치에 주석 배율을 사용합니다.
      * 레이어 표시 여부와 디테일에 적용되는 색을 제어합니다.
      * 레이아웃 하나, 일부, 또는 전체를 플로터, 프린터, PDF로 출력합니다.

==== 배경 ====

Rhino 3의 레이아웃 전에는 제목 블록에 모델의 여러 뷰를 배치하는 작업은 Rhino가 아닌 다른 CAD 소프트웨어에서 해야 했습니다. Rhino 4 에서 레이아웃 공간이 추가되었으며, Rhino 5의 관련 기능은 다음과 같이 향상되었습니다:

**새 명령**
  * 표시 순서는 레이아웃에 그려지는 디테일 뷰의 순서를 제어합니다. 
  * DisableClippingPlane - 활성 뷰포트에서 선택된 클리핑 평면을 끕니다. 속성 -> 클리핑 평면 창에서도 사용할 수 있습니다.
  * ImportLayout - 다른 Rhino 파일에서 레이아웃 페이지를 가져옵니다.
  * 여러 페이지 레이아웃 인쇄 - 일부 또는 모든 레이아웃을 한 번에 인쇄할 수 있습니다.
  * ShowLayersInDetail - 디테일 뷰에서 레이어를 표시합니다.
  * HideLayersInDetail - 디테일 뷰에서 선택된 레이어를 숨깁니다.
  * ShowSelectedInDetail - ShowSelected 명령처럼 실행됩니다. 단, HideInDetail 명령으로 이전에 숨겨진 개체에만 실행되는 점이 다릅니다.
**향상된 기능**
  * HideInDetail - 활성 디테일과 비활성 디테일을 포함/제외시키는 옵션이 추가되었습니다.
  * 텍스트와 치수 스타일 크기 조정 - 텍스트와 치수의 크기를 레이아웃 뷰포트와 모델 뷰포트에서 서로 다르게 조정할 수 있습니다. 레이아웃에서 텍스트와 치수가 서로 다른 크기인 경우, 다른 뷰포트에서 텍스트와 치수를 같은 높이로 보이게 쉽게 설정할 수 있습니다.
  * 하이브리드 인쇄 - 가능한 경우에, 보다 높은 화질로 인쇄되도록 래스터와 벡터 인쇄가 모두 사용됩니다.

===== Rhino 5를 위한 레이아웃 튜토리얼 (PDF)=====

이 튜토리얼에서는 용지 레이아웃 뷰포트에서 모델을 설정하고 출력하는 과정을 소개합니다. 파일을 다운로드하여 튜토리얼을 스스로 따라하며 학습하실 수 있습니다.

튜토리얼 내용:
  - 레이아웃 기능을 사용하여 디테일과 제목 블록이 있는 플롯 용지를 만듭니다. 
  - 디테일의 배율을 설정하고 해당 디테일의 크기에 맞게 텍스트, 치수를 추가합니다. 디테일마다 레이어 표시 상태와 레이어 색을 설정합니다.
  - 플롯 용지에 제목 블록, 노트, 해당 모델과 직접 관련이 없는 개체에 대해 주석을 추가합니다. 
  - 인쇄 너비, 개체의 인쇄색를 지정하고 인쇄 미리보기 모드를 사용합니다. 
  - 레이아웃 1:1 또는 전체 크기의 PDF로 출력합니다.
