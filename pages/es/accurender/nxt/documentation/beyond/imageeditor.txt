====== Editor de imágenes de nXt ======

El Editor de imágenes de nXt es una potente herramienta independiente que permite editar archivos de imagen nativos (.nXtImage) producidos por cualquiera de las plataformas de nXt.  Estos archivos nativos conservan toda la información recopilada durante el renderizado. Ejecute el Editor desde la carpeta nXt del menú Inicio.

Con el editor de imágenes, se puede:

  * Ajustar la configuración del indicador de tono
  * Cambiar la intensidad de cualquier canal de iluminación
  * Añadir efectos especiales, basados en las imágenes, como neblina o desenfoque de profundidad
  * Guardar una imagen de mapa de tonos en un formato como .jpg o .png
  * Guardar la información de luminancia en formato HDR
  * Ver y guardar canales de máscara adicionales (alfa, la distancia, material) para usar en la composición avanzada
  * Guardar en formato de archivo de Piranesi© (*.epx), que puede utilizarse para crear renderizados no fotorrealistas
  * Utilizar la aritmética de imagen para tareas como la unión de una imagen generada por distintos nodos en el Render Farm.
  * Guardar la configuración de iluminación utilizada para generar este renderizado.  Esta configuración de iluminación puede utilizarse posteriormente para generar más renderizados.


===== Cargar una imagen =====

Después de guardar los resultados del renderizado con el formato .nXtImage, vaya al menú Archivo->Abrir... para cargar la imagen en el editor.  

Al mover el cursor sobre la imagen, la barra de estado muestra la siguiente información sobre cada píxel:

  * Coordenada del píxel, calculada desde la esquina inferior izquierda
  * Color color resultante y alfa en formato RGBA del píxel después del mapeado de tonos
  * El valor de luminancia almacenado en cada píxel como RGB y canal Y único.  Este es el resultado directo del cálculo de iluminación.  Si se utilizó información de iluminación exacta para producir el renderizado, estos valores tendrán unidades de candelas/m^2.
  * Distancia de cada píxel desde el visor en metros.  Los valores negativos indican un píxel del fondo.
  * Nombre del material usado para renderizar el píxel.


===== Trabajo con canales de máscara =====

Los archivos nXtImage contienen tres canales adicionales que pueden utilizarse como máscaras para la composición avanzada en la mayoría de editores de bitmaps.  Estos canales llevan información de alfa (transparencia), distancia y material de cada píxel, codificada en una imagen en escala de grises.  Cada canal se puede ver y guardar en un archivo .png.

Utilice el menú desplegable de Vista para ver cada canal de máscara.
Utilice la opción Archivo->Guardar máscara para guardar un canal de máscara como imagen PNG.

**Notas adicionales:**

  * El canal Alfa puede incluirse con una imagen de mapa de tonos seleccionando un formato de archivo con Alfa al guardar una imagen de mapa de tonos.
  * Los canales de distancia y materiales no están suavizados y pueden mostrar algunos artefactos de bordes duros.  Agregar una pequeña cantidad de desenfoque gaussiano a una máscara antes de usarla puede ayudar a suavizar estos bordes.
  * El canal de materiales solo codificará únicamente 255 materiales diferentes.  Si el dibujo contiene más materiales, algunos colores de máscara se repetirán.
