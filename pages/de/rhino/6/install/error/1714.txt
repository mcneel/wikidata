====== Installationsfehler 1714 (Rhino 6 für Windows) oder Systemfehler 1612 ======
> **Zusammenfassung:** //Behebung des Installationsfehlers 1714//
> **Weitere Infos zu Installationsproblemen:** //[[:de:rhino:6:installproblems|Problembehandlung: Installation von Rhino 6]]//

=====Problem=====
Die Protokolldatei zeigt an: "Rhino Installer Engine -- Fehler 1714. Die ältere Version der Rhino Installer Engine kann nicht entfernt werden.  Setzen Sie sich mit dem technischen Support-Team in Verbindung.  Systemfehler 1612."

=====Antwort=====

Dieses Problem kann mit Hilfe von Microsoft behoben werden.  Bitte laden Sie hier das Tool herunter und befolgen Sie die Anleitungen:

https://support.microsoft.com/de-de/help/17588/fix-problems-that-block-programs-from-being-installed-or-removed

Nachdem das Problem behoben ist, versuchen Sie eine erneute Installation von Rhino.
