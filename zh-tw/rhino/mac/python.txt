====== Mac Rhino支援Python腳本 ======
Mac Rhino 支援一個測試外掛，可讓您在Rhino中增加Python語言的腳本，在增加此支援之前，您必須：

  - [[http://www.rhino3d.com/download/wenatchee/5.0/wip|安裝最新版 OSX Rhino]]
  - 開啟此最新版Rhino。 \\ This will register the .macrhi extension with the Rhino application. You only need to do this once. Now double-clicking any .macrhi file will launch Rhino.
  - [[http://download.rhino3d.com/IronPython/5.0/wip/download/|下載 IronPython .macrhi 外掛程式安裝檔]]. \\ 此連結為Mac Rhino 給Python外掛程式安裝檔下載連結。 
  - 雙擊此檔案，安裝外掛程式。 \\ Mac Rhino將會開啟並安裝此外掛程式。
  - 重新開啟Rhino。 \\ 您的Rhino將會多一個名為 **RunPythonScript** 的新指令。

====RunPythonScript 指令====
此指令共有兩個版本：

**RunPythonScript** - 執行此指令將會顯示一個對話框讓您選取一個python腳本檔(.py file.)。此外掛包含一些範例檔讓您可選擇執行，當您第一次執行此指令，對話框預設指定這個範例檔資料匣。

**-RunPythonScript** - 此指令前方加上"減號"的版本，可讓直接你輸入一個腳本檔名，若腳本存在您電腦的Scripts目錄下，將可執行。您也可以執行放在子目錄的腳本。例如，若您想要執行Samples目錄下的CurveLength.py腳本檔，請在提示列輸入samples.curvelength即可執行腳本。

**備註**：所有的範例皆可在兩個平台[[http://mac.rhino3d.com|Mac Rhino WIP]] 及[[http://www.rhino3d.com/|Windows Rhino 5]]作業。
