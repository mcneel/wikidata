======Flamingo nXt 3.0 et le Zoo======
> **Produit :** //[[fr:zoo:home|Zoo]]//
> **Résumé :** //Discute de l'utilisation de Flamingo nXt avec le Zoo//

[[http://nxt.flamingo3d.com/|Flamingo nXt 3.0]] peut être utilisé avec Rhino 4.0 ou Rhino 5. Pour cette raison, les licences réseau Flamingo nXt peuvent être gérées soit par [[zoo4:home|Zoo 4.0]], le gestionnaire de licences réseau de Rhino 4.0, soit par [[zoo:home|Zoo 6]], le gestionnaire de licences réseau de Rhino 5.

=====Plus d'informations=====
Lors de l'installation initiale, Flamingo nXt est configuré pour utiliser un serveur [[zoo4:home|Zoo 4.0]] pour les licences réseau. 

Pour reconfigurer [[http://nxt.flamingo3d.com/|Flamingo nXt]] afin d'utiliser un serveur [[zoo:home|Zoo 6]] pour les licences réseau, vous devez exécuter un programme utilitaire.

Dans l'explorateur de l'ordinateur où se trouve Rhino, naviguez jusqu'au dossier suivant :

====Sur Windows 64 bits====
<code>
C:\Program Files (x86)\Common Files\McNeel Shared\License Manager
</code>

====Sur Windows 32 bits====
<code>
C:\Program Files\Common Files\McNeel Shared\License Manager
</code>

Pour configurer [[http://nxt.flamingo3d.com/|Flamingo nXt]] afin d'utiliser un serveur [[zoo:home|Zoo 6]], exécutez l'utilitaire **RegRhinoLM5.exe.

Pour configurer [[http://nxt.flamingo3d.com/|Flamingo nXt]] afin d'utiliser un serveur [[zoo4:home|Zoo 4.0]], exécutez l'utilitaire **RegRhinoLM4.exe.

Ce changement doit être effectué sur chaque poste de travail Rhino desservi par le Zoo.
