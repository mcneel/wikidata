====== ¿Puedo ejecutar un programa de RhinoScript en la versión para Mac? ======
> **Resumen:** //¿Rhinoscript existe para Rhino para Mac?//

No. RhinoScript está basado VisualBasic y VisualBasic no es compatible con OS X. 

Se recomienda usar Python para scripting tanto en la versión para Mac como Windows. Visite el [[http://discourse.mcneel.com/category/scripting | foro de Rhino de soporte sobre scripting]] para obtener ayuda sobre el lenguaje de scripting Python para Windows y Mac.
