====== Disabling Plugins on Rhino for Windows ======



{{ :rhino:8:plugins_that_do_not_ship.png?300|}}
**In Rhino for Windows you can disable the plugin by:**//Italic Text//
  - Start the Rhino application with the //Safemode//shortcut 
  - Type the **Options** command
  - Go to the Plugins page
  - Select the filter //Plugins that do not thip with Rhino//
  - uncheck the "enable" on the plugins your suspect or all.
  - Close and restart Rhino normally to test.
  - Report your crashing to the offending plugin developer.
