===== Foire aux questions : Licences dans Rhino 6, Rhino 7 et Rhino 8 =====

====Quelles sont les options de licence pour Rhino ?====
Consultez [[https://www.rhino3d.com/features/administration/licensing/|Rhino Licensing options]] pour de plus amples informations.

====Partager des licences====

==Puis-je utiliser une licence de Rhino sur plusieurs ordinateurs ?==
Oui, en utilisant l’un de nos gestionnaires de licences flottantes gratuit. Le LAN Zoo fonctionne sur votre réseau local et le Cloud Zoo utilise une connexion Internet occasionnelle pour partager des licences entre plusieurs ordinateurs.

==Et si je veux installer une licence autonome sur plusieurs ordinateurs ?==
Vous devrez avoir un numéro de licence pour chaque ordinateur. Rhino 5 permettait de valider une clé de licence plusieurs fois. Ce n’est pas le cas pour Rhino 6, 7 et 8. Rhino 6, 7 et 8 permettent de gérer des licences personnelles sur le Cloud Zoo afin de pouvoir utiliser une licence simple sur plusieurs ordinateurs alternativement avec une connexion à Internet occasionnelle.

==Comment puis-je partager une ou plusieurs licences avec plusieurs personnes ?==
Ajoutez vos licences à une équipe dans le Cloud Zoo (recommandé) ou ajoutez vos licences dans le LAN Zoo.
[[https://www.rhino3d.com/features/administration/licensing/|Options de licence de Rhino]]

====Mise à jour vers Rhino 8 ====

==Si je réalise une mise à jour de Rhino 7 vers Rhino 8, puis-je toujours utiliser Rhino 7 ?==
Oui. Lorsque vous réalisez une mise à jour, vous élargissez votre licence afin d’accéder aux fonctions de Rhino 8 ; vous n'ajoutez pas une nouvelle licence. Cela signifie que vous pouvez utiliser Rhino 6 et les anciennes versions de Rhino sur le même ordinateur. Vous ne pouvez pas utiliser Rhino 7 sur un ordinateur alors qu'une autre personne utilise la mise à jour de Rhino 8 sur un autre.

==Si j'ajoute ma licence de mise à jour de Rhino 8 dans le Cloud Zoo, puis-je toujours installer Rhino 7 ?==
Oui, Rhino 7 peut être installé, même s'il a été mis à jour vers Rhino 8. Les deux licences doivent cependant être validées de la même façon (ordinateur unique, Zoo ou Cloud Zoo). 

==Puis-je ajouter ma licence de Rhino 5 dans le Cloud Zoo ?==
Non. Rhino 5 n'est pas compatible avec le Cloud Zoo.

==Si j'ajoute mes licences de Rhino 7 et de mise à jour vers Rhino 8 dans le Zoo, que se passe-t-il ?==
Vous devrez utiliser Zoo 7 SR 8 (ou une version supérieure) pour ajouter des clés de licence de Rhino 8. Le Zoo regroupera vos clés de Rhino 7 et de mise à jour vers Rhino 8 dans une "licence groupée", et vérifiera qu'une seule est utilisée à un moment donné. Un utilisateur qui lance Rhino 7 et Rhino 8 sur le même ordinateur n'utilisera qu'une seule licence.

====Utiliser Rhino 8 en version visionneuse ====

==Comment puis-je utiliser Rhino en mode visionneuse uniquement ?==

Pour utiliser Rhino en mode visionneuse, vous avez besoin d’une clé de licence d’évaluation. Le cas de figure idéal est lorsque vous en avez déjà une et qu’elle a expiré.  Si vous n’avez pas de clé de licence d’évaluation, demandez-en une sur https://www.rhino3d.com/download/rhino/evaluation.

Au moment de valider votre licence d’évaluation, ne sélectionnez pas l’option d’utilisation partagée ou flottante de la licence entre ordinateurs ; assurez-vous de choisir l’option Verrouiller la licence sur l’ordinateur.

Si vous possédez déjà une licence et qu’elle est sur le CloudZoo, vous devrez d’abord la supprimer de votre compte Rhino :

  * Rendez-vous sur https://accounts.rhino3d.com/ et connectez-vous à votre compte avec votre adresse e-mail.
  * Descendez jusqu’en bas de la page et cliquez sur Licences, puis sur votre nom sous Licences personnelles.
  * Une fois supprimée, redémarrez Rhino 8. 
  * Lorsque Rhino vous demande de saisir une nouvelle licence, sélectionnez l’option « Indiquez votre clé de licence » et tapez votre clé de licence d’évaluation, puis cliquez sur Continuer. Attention : NE CLIQUEZ PAS SUR « Ajouter dans votre compte Rhino » et cliquez sur l’option « Utiliser uniquement sur cet ordinateur » en bas à gauche, puis cliquez sur Oui.  

Tant que votre licence d'évaluation de Rhino est valide, vous pourrez enregistrer votre travail. Une fois qu'elle aura expiré, vous pourrez ouvrir votre modèle de Rhino, modifier la vue, utiliser des cotes, etc., mais vous ne pourrez plus enregistrer les fichiers.
