====== Benutzerdefineirte Schraffierungsmuster für Rhino für Rhino======

Um ein Schraffierungsmuster in Rhino zu erstellen, gibt es mehrere Möglichkeiten. Die erste und einfachste, wenn Sie AutoCAD haben, ist die Importierung von ACAD.PAT in Rhino. Zweitens können Sie sie von Internet-Sites wie [[http://www.dotsoft.com/acadhatch.htm|DotSoft]] herunterladen.

Sie können auch mithilfe eines der untenstehenden Tutorials eine PAT-Datei von Grund auf in Notepad erstellen.
Da Rhino das gleiche Format wie AutoCAD verwendet, eignet sich jegliches Tutorial zur Erstellung eines AutoCAD-Musters.

Konsultieren Sie bitte:
  * [[http://www.dotsoft.com/acadhatch.htm|DotSoft]]
  * [[http://www.afralisp.net/autocad/tutorials/creating-hatch-patterns.php|AfraLISP]]
  * [[http://www.cadhatch.com/free-downloads/4588167679|Kostenlose CADhatch-Downloads]]
  * [[http://www.ellenfinkelstein.com/acadblog/creating-custom-hatch-patterns/|AutoCAD Bible, Ellen Finkelstein]]

Laden Sie eine [[http://s3.amazonaws.com/mcneel/rhino/4.0/misc/hatch patterns for rhino.pat|Beispieldatei mit Rhino-Schraffierungsmustern]] zur Verwendung herunter.
Importieren Sie sie in Ihre Rhino-Datei in Rhino von:
**Optionen -> Dokumenteigenschaften -> Schraffierung -> Laden**.
