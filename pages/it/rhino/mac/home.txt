[[http://www.rhino3d.com/download/wenatchee/5.0/wip|Scaricate]] la versione work-in-progress (WIP) corrente. (È gratuita, per ora.)
==============

====Rhino per Mac è quasi pronto per la produzione.====

  * Funzionalità a cui stiamo ancora lavorando per la prima versione:
    * **Importazione e/o esportazione** per alcuni formati file 
    * Pulizia dell'**interfaccia utente**
    * **Localizzazione** in tutte le lingue di Rhino
    * **Documentazione** e tutorial 

  * Queste funzionalità **NON** saranno nella prima versione. **//<- Importante//**
    * **Grasshopper**
    * **Layout**
    * **Sessioni di lavoro**
    * I/O file di **SketchUP**
    * Importazione file di **Solidworks**
    * Supporto **GHS**
    * I/O file **KML, WMF, ZPR e SCN**
    * **Plug-in di terze parti**

**Ogni una o due settimane**, verrà resa disponibile una nuova versione WIP. Ciascuna versione sarà valida per circa **60 giorni**.

====Requisiti di sistema====
  * Processori Intel Core 2 Duo a 64 bit o successivi
  * OS X 10.8.5 o qualsiasi versione di OS X 10.9.  Abbiamo riscontrato che dei bug nelle versioni precedenti di 10.8 potevano causare dei problemi in Rhino. Si richiede quindi l'ultima versione di 10.8.
  * Si raccomandano 4GB di RAM o superiore
  * Un mouse a più pulsanti (preferibilmente con rotellina)
  * Lo [[http://www.3dconnexion.com/products/spacenavigator.html|SpaceNavigator]] è supportato.

--> Hardware non supportato#
    * OS X 10.6 o versioni precedenti
    * Lo **Spaceball** e lo **SpaceExplorer**
