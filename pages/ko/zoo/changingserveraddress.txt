======Zoo 서버의 IP 주소 변경======
> **제품:** //[[ko:zoo:zoo5|Zoo 5]]//
> **요약:** //Zoo 서버의 IP 주소 변경과 그 결과에 대해 안내합니다.//

=====질문=====
라이선스 서버 중 일부의 IP 주소 변경을 고려 중입니다. 이 중 한 서버에서 Rhinoceros 5용 Zoo 5 라이선스 서비스를 호스팅 중입니다. 서버의 IP 주소를 변경하면 라이선스를 사용할 수 없게되는 건 아닌지 확인 바랍니다. 호스트 이름과 MAC 주소에는 변동이 없습니다.

=====답변=====
Zoo 5 설치 또는 라이선스는 서버의 IP 또는 MAC 주소와 묶여있지 않습니다. Zoo 와 관련하여, Zoo 서비스를 호스팅하는 서버의 IP 주소를 변경하셔도 이로 인해 발생하는 문제는 없습니다. 

Rhino 설치에서 서버의 (DNS) 도메인 이름을 사용하여 Zoo 서버를 찾는 경우에는 마이그레이션도 원활하게 진행됩니다. Rhino 설치에서 서버의 IP 주소를 사용하여 Zoo를 찾는다면 각각의 작업 스테이션에서 리디렉션이 필요합니다.
