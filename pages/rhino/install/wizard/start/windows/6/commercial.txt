====== Commercial License ======


===Choose one:===
    * **[[rhino:install:wizard:start:windows:6:commercial:czpersonal|Multiple computers, single user]] - Cloud Zoo Personal**
        * Use this option when you want to share your license among your personal computers
        * Supports unexpired evaluation license keys
        * Requires occasional Internet access [[rhino:install:wizard:start:windows:6:commercial:czpersonal:firewall|(over HTTPS TCP port 443)]]
        * [[https://www.rhino3d.com/7/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:commercial:czteam|Multiple computers, multiple users]] - Cloud Zoo Team**
        * Use this option when you are sharing your licenses among multiple users
        * Requires occasional Internet access [[rhino:install:wizard:start:windows:6:commercial:czpersonal:firewall|(over HTTPS TCP port 443)]]
        * [[https://www.rhino3d.com/7/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:commercial:zoo|Multiple users on office LAN]] - Zoo**
        * Use this option when you are sharing your licenses among multiple users, limited to the office LAN
        * Does not support evaluation license keys
        * Requires Windows **[[https://wiki.mcneel.com/zoo/home|Zoo]]** free floating license manager
        * Rhino installation easily **deployed**:
          * [[https://wiki.mcneel.com/rhino/installingrhino/7|Windows]]
          * [[https://wiki.mcneel.com/zoo/rhinomacdeployment|Mac]]
        * Does not require Internet access
        * [[https://www.rhino3d.com/7/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:commercial:stand_alone|Single computer licensing]]**
      * Use this option when only one computer will be used.
      * Use this option with an **expired evaluation** license for use as a **file viewer**.
      * Locks license **exclusively** to one computer.
        * Note: Contact Sales for help if your computer is lost or damaged.
      * Requires one-time Internet access for validation or an [[https://wiki.mcneel.com/zoo/rhinovalidateoffline|offline validation.]]
      * After validation, it does not require Internet access except for updates.
      * [[https://www.rhino3d.com/7/eula|End User License Agreement]]
