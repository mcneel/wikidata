====== Zoo pour OS X ======
> **Produit :** //[[fr:zoo:home|Zoo]]//
> **Résumé :** //Discussion sur le Zoo et son utilisation sur OS X.//

===== Question=====
Actuellement, si on veut utiliser Rhino pour Mac avec le Zoo, on doit utiliser la version du Zoo qui fonctionne sur Microsoft Windows. Est-il prévu de développer une version du Zoo pour Apple OS X ?

===== Réponse=====
Oui, nous travaillons actuellement sur le développement d'une version du Zoo fonctionnant sur OS X. Lorsqu'elle sera disponible pour les premiers tests, nous posterons un messages sur notre forum de discussion [[http://discourse.mcneel.com/|Discourse]]. 

Si vous ne possédez pas de système Windows pour utiliser le Zoo, vous pouvez utiliser [[http://www.vmware.com/products/fusion-pro/|VMWare Fusion Pro]], qui permet de lancer Windows sur un Mac.
