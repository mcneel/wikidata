====== Rhino 5 Einzelnutzer-Studentenlizenz  ======


    * **[[rhino:install:wizard:start:windows:5:edu:stand_alone|Ein Einzelplatzsystem]]**
      * Sie können Ihre Einzelnutzerlizenz auf den Computern verwenden, die Ihnen gehören und die Sie persönlich kontrollieren
      * Sie können Rhino nicht auf mehreren Computern gleichzeitig ausführen
      * Benötigt keinen Internetzugang
