==== Errore di installazione 9 - Rhino 6 ====

Errore di blocco non valido (-2147024887)

> **Sommario:** //Come risolvere l'errore di installazione 9//

=====Problema=====
Un file relativo all'installazione di Rhino è danneggiato.

=====Risposta=====
**Passo 1: Esegui nuovamente il download**
  - Scarica il programma di installazione più recente di Rhino 6 qui: [[http://www.rhino3d.com/download/rhino/6/latest|Download completo di Rhino 6]].
  - Esegui il file scaricato per avviare il processo di installazione.
  - Se riscontri problemi di installazione, salva una copia del file di registro sul desktop.
\\
**Passo 2: Disabilita la scansione virus**
  - Alcuni scanner di virus e malware scansionano i file eliminati utilizzati da Rhino durante l'installazione. In generale, si consiglia di abilitare sempre la scansione virus.
  - Disabilita temporaneamente la scansione virus.
  - Scarica il programma di installazione più recente di Rhino 6 qui: [[http://www.rhino3d.com/download/rhino/6/latest|Download completo di Rhino 6]].
  - Esegui il file scaricato per avviare il processo di installazione.
  - Se riscontri problemi di installazione, salva una copia del file di registro sul desktop.
  - Abilita la scansione virus.
\\
**Se il problema persiste**

È frustrante e ne siamo spiacenti. Invia per e-mail il file di registro di installazione a tech.eu@mcneel.com.
