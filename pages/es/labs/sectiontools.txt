=====Descargas=====

**SectionTools para Rhino 6 para Windows:** Última actualización 14 agosto, 2018.
  * Asegúrese de tener la última versión de [[https://www.rhino3d.com/download|Rhino 6]]. 
  * Descargue[[https://www.rhino3d.com/download/sectiontools/1/wip/rc| SectionTools para Rhino 6 para Windows WIP]] para Rhino 6. Haga doble clic y el plug-in se instalará y cargará en Rhino. Tiene que cargar la barra de herramientas mediante el administrador de barras de herramientas.

**SectionTools para Rhino 5 (no funciona con Rhino 4):** Última actualización 20 diciembre, 2013.
  * Asegúrese de tener la última service release de [[http://www.rhino3d.com/download/rhino/5.0/evaluation|Rhino 5 o Evaluación de Rhino 5]]. 
  * Descargue[[https://www.food4rhino.com/app/sectiontools-rhino| SectionTools para Rhino 5]] para Rhino 5. Haga doble clic y el plug-in se instalará y cargará en Rhino. Tiene que cargar la barra de herramientas mediante el administrador de barras de herramientas.
