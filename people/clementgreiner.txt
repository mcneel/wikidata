===== Plugins for Rhino V4=====

**[[rhino:bakerender|BakeRender]]** Plugin to bake textures with BakeAIR directly from Rhino. //(Version 2.03, works with XP 32/64 bit with Rhino4 SR4/20080807 or higher, does NOT work with Rhino 5)//

**[[labs:txtemplate|TxTemplate]]** Plugin to create image based UV-maps from meshes or render meshes with various controls over colors, line widths, transparency and layered background texture. //(Version 1.0, works with XP and Windows 7 (32/64 bit) with Rhino4 SR7/20080807 or higher)//

**[[http://en.wiki.mcneel.com/content/upload/files/UnwrapRenderMesh.zip|UnwrapRenderMesh]]** Plugin to unwrap the underlying RenderMesh of a Polysurface. This plugin can either be used to view the packed texture coordinates Rhino creates by default or to create basic texturing templates with other Programs like Illustrator or Photoshop. The command **UnwrapRenderMesh** will be available from the commandline once the plugin is installed. //(Version 2.2, works with XP 32/64 bit with Rhino4 SR4/20080807 or higher)//

**[[labs:uvcwrap|UVcWrap]]** New pseudo UV-wrapping Plugin to unwrap a mesh's texture coordinates and transform them as real Rhino mesh objects. After the transformation has been done, the unwrapped meshes can be re-wrapped to the original mesh. //(Version 0.5b, tested with XP 64 bit with Rhino4 SR5b/20090226 or higher)//

**[[http://en.wiki.mcneel.com/content/upload/files/GripRandomizer.zip|GripRandomizer]]** Simple plugin to randomly move the selected grips of a curve, surface or mesh. If no grips (control points) are selected, all grips are randomized. Distance and direction of the random movement is scriptable via the commandline. //(Version 1.01, works with XP 32/64 bit with Rhino4 SR4/20080807 or higher)//

===== Scripts for Rhino V3======
**[[http://en.wiki.mcneel.com/content/upload/files/CurvatureGraph.zip|CurvatureGraph.rvb]]**
Script for turning "CurvatureGraph" display curves into real usable Rhino curves

**[[http://en.wiki.mcneel.com/content/upload/files/CopyObjectsToPoints_RH3.zip|CopyObjectsToPoints_RH3.zip]]**
Script for Rhino 3 to copy objects to points or points in a pointcloud

**[[http://en.wiki.mcneel.com/content/upload/files/CopyObjectsToSrfNormal_RH3.zip|CopyObjectsToSrfNormal_RH3.zip]]**
Script for Rhino 3 to copy objects to points on a surface and orient the objects to the surface normal at these points

**[[http://en.wiki.mcneel.com/content/upload/files/DashCrvs.zip|DashCrvs.zip]]**
Script to dash curves (split curves into a number of segments, delete every second segment)

**[[http://en.wiki.mcneel.com/content/upload/files/FilletPolyCrvs.zip|FilletPolyCrvs.zip]]**
Script for Rhino 3 to fillet all segments of a polycurve with a defined radius

**[[http://en.wiki.mcneel.com/content/upload/files/ObjectNotes.zip|ObjectNotes.zip]]**
Scripts and toolbar for adding notes to objects (uses Object Data)

**[[http://en.wiki.mcneel.com/content/upload/files/RevolveAroundCrv.zip|RevolveAroundCrv.zip]]**
Script for Rhino 3 to revolve one curve around another curve with optional parameters

**[[http://en.wiki.mcneel.com/content/upload/files/SpherePipe_RH3.zip|SpherePipe_RH3.zip]]**
Script for Rhino 3 to create pipes from curves with spherical end caps (with optional join or merge of caps)

**[[http://en.wiki.mcneel.com/content/upload/files/TorusCreator.zip|TorusCreator.zip]]**
Script for Rhino 3 to create controled torii (eg. lumpy, spiral, knot, PQ, wave, regular)
