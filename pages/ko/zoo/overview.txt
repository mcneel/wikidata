======LAN Zoo - 기능 안내======
> **제품:** //[[ko:zoo:home|Zoo]]//
> **요약:** //LAN Zoo의 기능을 소개합니다.//

=====개요=====
LAN Zoo를 사용하시면 동일한 네트워크 작업그룹에 속한 사용자들이 Rhino, Flamingo, Penguin, Brazil, Bongo 라이선스 (CD 키) 를 공용할 수 있습니다. 이 제품들을 독립실행형 (stand-alone) 제품이 아닌, 작업그룹 노드로 설치하면 다음과 같이 실행됩니다: 

  * 작업그룹 노드가 시작되면 LAN Zoo에게 라이선스를 요청합니다.
  * 사용되지 않은 라이선스가 해당 노드로 배정됩니다. 
  * 노드가 종료되면 라이선스가 LAN Zoo의 라이선스 풀(pool)로 반납됩니다. 


=====기능=====
  * ** LAN Zoo는 무료입니다!**
  * **추가 비용이 없습니다.** Rhino, Flamingo, Penguin, Brazil, Bongo의 특별한 버전이 필요하지 않습니다. 
  * **특별한 하드웨어가 필요하지 않습니다.** LAN Zoo는 Windows 기반 시스템에서 실행됩니다. 
  * **별도의 소프트웨어가 필요하지 않습니다.** LAN Zoo는 Microsoft .NET Framework 4.8를 지원하는 일반적인 Windows 시스템 (Windows Vista 이상)에서 실행됩니다.
  * **설정이 간단합니다.** Windows 기반 시스템에 LAN Zoo를 설치하고 각각의 컴퓨터가 아닌 LAN Zoo에 라이선스 키를 입력합니다. 
  * **체크아웃.** 상업용(Commercial)과 교육 실습실(Educational Lab) 라이선스 키는 체크아웃이 가능합니다. 노트북 사용자는 네트워크와의 연결을 끊고 사용할 수 있습니다. 교육 실습실용 라이선스는 30일 체크아웃으로 제한됩니다. 노트북 컴퓨터가 다시 네트워크에 연결되면 라이선스 키를 다시 체크인할 수 있습니다. 안내: 단일 사용자 교육용(Single user Educational) 라이선스는 LAN Zoo에서 지원되지 않습니다.
  * **안정성.** 네트워크 연결이 끊어지거나 서버가 다운되더라도 네트워크 노드는 계속해서 실행됩니다.
  * **라우팅 가능.** Zoo는 단일 세그먼트 네트워크 및 라우터로 시스템이 구분된 네트워크에서도 모두 실행됩니다. 
  * **서비스로 실행** - LAN Zoo가 있는 시스템을 다시 부팅하면 Zoo가 자동으로 시작합니다. 

=====추가 정보=====
  * **표준 인터넷 프로토콜 지원.** 클라이언트는 **HTTP (포트 80)** 프로토콜을 사용하여 LAN Zoo와 연결됩니다. 시스템 관리자는 이러한 점을 사용하여 복잡한 네트워크나 인터넷을 통해 쉽게 Zoo 요청을 라우팅할 수 있습니다.
  * **방화벽 친화성.** LAN Zoo 에서는 클라이언트가 서버를 주기적으로 체크하도록 되어 있으나, 클라이언트 쪽으로 다시 연결을 시도하지 않습니다. 관리자는 클라이언트 컴퓨터의 방화벽 포트를 열어둘 필요가 없습니다.
  * **제한된 라이선스 체크아웃.** 이전 LAN Zoo 버전에서는 업무용 출장이나 노트북 컴퓨터 사용자를 위한 라이선스 체크아웃이 허용되었습니다. 단, 체크아웃이 허용되는 기간에는 제한이 없었습니다. 이제, LAN Zoo에서 관리자가 라이선스 체크아웃 기간을 지정할 수 있습니다.
  * **서드 파티 플러그인 지원.** LAN Zoo에서는 서드 파티 플러그인 개발자가 자신이 개발한 제품을 LAN Zoo에 추가할 수 있습니다.
