====== Rhino 6 or plug-in license type ======


===Choose one:===
    * **[[rhino:install:wizard:start:windows:6:commercial|Single Commercial and 90-day Evaluation License]]** //<- Pick for more details.//
      * Allows one user per license
      * May be used stand-alone or over a LAN using a free-floating license manager. Details [[rhino:install:wizard:start:windows:6:commercial|here.]]
      * Rhino evaluation must be installed stand-alone, floating license install not available for evaluation.
      * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:educational|Single Educational License]]** //<- Pick for more details.//
      * Allows one user per license
      * May be used stand-alone or with CloudZoo license manager
        * User must own computer(s) 
      * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:lab|30-user EDU Lab License]]** //<- Pick for more details.//
      * Allows 30 concurrent users
      * May be used over a school LAN using the Windows [[https://wiki.mcneel.com/zoo/home|Zoo]] free floating license manager (recommended) or installed stand-alone.
       * Rhino installation easily **deployed**:
          * [[https://wiki.mcneel.com/rhino/installingrhino/6|Windows]]
          * [[https://wiki.mcneel.com/zoo/rhinomacdeployment|Mac]]
      * [[https://www.rhino3d.com/6/eula|Rhino End User License Agreement]]
