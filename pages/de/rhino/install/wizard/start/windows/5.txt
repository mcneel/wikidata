====== Rhino 5 Lizenztyp ======

===Bitte auswählen:===
    * **[[de:rhino:install:wizard:start:windows:5:commercial|Kommerzielle Einzelnutzer-Lizenz und 90tägige Testversion]]**
      * Ein Benutzer pro Lizenz ist erlaubt
      * Kann als Einzelplatz verwendet werden oder über eine LAN (außer der Testversion) unter Verwendung des kostenlosen Floating-Lizenzmanagers im [[https://wiki.mcneel.com/de/zoo/home|Zoo]] für Windows
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Endbenutzer-Lizenzvertrag]]

    * **[[de:rhino:install:wizard:start:windows:5:educational|Studenten-Einzelnutzerlizenz]]**
      * Ein Benutzer pro Lizenz ist erlaubt
      * Nur Einzelplatzsystem-Bereitstellung
        * Computer muss Eigentum des Nutzers sein
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Endbenutzer-Lizenzvertrag]]

    * **[[rhino:install:wizard:start:windows:5:lab|Schule-Lab-Lizenz für 30 Nutzer]]**
      * Bis zu 30 Nutzer sind gleichzeitig erlaubt
      * Kann über eine Schule-LAN verwendet werden mithilfe des kostenlosen Floating-Lizenzmanagers im [[https://wiki.mcneel.com/de/zoo/home|Zoo]] für Windows (empfohlen) oder als Einzelplatz
      * Einfache [[https://wiki.mcneel.com/de/rhino/installingrhino/5.0|Bereitstellung]] der Rhino-Installation
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Endbenutzer-Lizenzvertrag]]
