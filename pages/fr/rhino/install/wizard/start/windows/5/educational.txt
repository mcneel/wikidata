====== Licence Éducation individuelle de Rhino 5 ======


    * **[[fr:rhino:install:wizard:start:windows:5:edu:stand_alone|Ordinateur autonome individuel]]**
      * Vous pouvez utiliser votre licence individuelle sur tous les ordinateurs vous appartenant et que vous contrôlez
