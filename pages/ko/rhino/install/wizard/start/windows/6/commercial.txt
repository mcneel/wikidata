====== Rhino 6 단일 사용자 상업용 라이선스 ======


===하나를 선택하세요:===
    * **[[ko:rhino:install:wizard:start:windows:6:commercial:czpersonal|여러 컴퓨터, 단일 사용자]] - Cloud Zoo 개인용**
        * 귀하가 본인의 여러 개인용 컴퓨터에서 본인의 라이선스를 사용할 때 이 옵션을 사용합니다.
        * 가끔 인터넷 연결이 필요합니다.
        * [[https://www.rhino3d.com/6/eula|최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[ko:rhino:install:wizard:start:windows:6:commercial:czteam|여러 대의 컴퓨터, 여러 명의 사용자]] -  Cloud Zoo 팀**
        * 귀하가 여러 명의 사용자와 귀하의 라이선스를 공유할 때 이 옵션을 사용합니다.
        * 가끔 인터넷 연결이 필요합니다.
        * [[https://www.rhino3d.com/6/eula|최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[ko:rhino:install:wizard:start:windows:6:commercial:zoo|회사 LAN에서의 여러 사용자]] - Zoo**
        * 회사 LAN로 제한하여 귀하가 귀하의 라이선스를 여러 사용자 간에 공유할 때  이 옵션을 사용합니다.
        * 무료 플로팅 라이선스 관리자인 Windows용 **[[https://wiki.mcneel.com/ko/zoo/home|Zoo]]** 가 필요합니다.
        * Rhino 설치는 쉽게 **[[https://wiki.mcneel.com/ko/rhino/installingrhino/6|배포]]**할 수 있습니다.
        * 인터넷 연결이 필요하지 않습니다.
        * [[https://www.rhino3d.com/6/eula|최종 사용자 사용권 계약 - End User License Agreement]]

    * **[[ko:rhino:install:wizard:start:windows:6:commercial:stand_alone|단일 컴퓨터 라이선스 방식]]**
      * 1대의 컴퓨터에서만 Rhino 6를 사용할 경우 이 옵션을 사용합니다.
      * 라이선스는 1대의 컴퓨터에서 **독점 사용**하도록 제한됩니다.
      * 유효성 검사를 받기 위한 인터넷 연결이 1회 필요하거나, 또는 [[https://wiki.mcneel.com/ko/zoo/rhinovalidateoffline|오프라인 유효성 검사]]가 필요합니다.
      * 유효성 검사를 완료한 이후에는, 업데이트할 때를 제외하고 인터넷 연결이 필요하지 않습니다.
      * [[https://www.rhino3d.com/6/eula|최종 사용자 사용권 계약 - End User License Agreement]]

>이 위키 페이지는 Rhino 설치 마법사의 일부입니다. [[ko:rhino:install:wizard:start |처음부터 시작하려면 여기를 클릭하세요]].
