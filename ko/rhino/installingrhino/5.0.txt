======Active Directory 또는 SMS를 사용한 Rhino 5 MSI 배포======
=====개요=====
이 방법은 Active Directory 또는 SMS를 사용하여 회사 전체에 Rhino 5를 배포하고하 하는 시스템 관리자 여러분께 가장 유용한 안내입니다.

또한, 사용자 지정된 Setup.ini가 포함된 폴더에서 MSI를 실행하여 자동화된 설치에도 사용하실 수 있습니다. 여기에는 Active Directory, SMS, 또는 다른 배포 도구가 필요하지 않습니다.

=====필수 조건=====
Rhino 5를 설치하시기 전에 대상 컴퓨터에 **Microsoft .NET 4.0 Full Framework**가 설치되어 있어야 합니다. 설치 관리자는 DVD의 Installers\Prerequisites 폴더에 있습니다.

=====배포 준비=====
   - **Setup.ini** 파일을 편집하고 설치하려는 MSI 파일의 옆에 둡니다. **안내:** # 기호 다음의 모든 문자는 설치 관리자에서 무시합니다. 줄의 중간에 # 기호가 있어도 무시되기는 마찬가지입니다.

=====설정 배포=====
  - **Active Directory Users & Computers** 를 엽니다.
  - Rhino 5 사용자 또는 Rhino 5 를 받을 컴퓨터를 포함한 그룹 또는 조직 구성 단위를 오른쪽 클릭하여 속성을 클릭합니다.
  - 그룹 정책 탭에서 새 그룹 정책의 이름을 **Install Rhino 5.0**로 지정하여 추가합니다.
  - **Install Rhino 5.0** 그룹 정책 개체를 선택하고 **편집...**을 클릭합니다.
  - 컴퓨터 기준 또는 사용자 기준 (해당 조직에 어느 방식이 가장 적합한지 판단하려면 그룹 정책을 읽어 확인하세요) 으로 설치할지 정합니다.
  - (이전 단계의 선택에 따라) 사용자 구성 또는 컴퓨터 구성 아래 **소프트웨어 설정**을 펼쳐 **소프트웨어 설치**에서 오른쪽 클릭하고 **새로 만들기 -> 패키지**를 클릭합니다.
  - **S:\Rhino 5.0\rh50.msi** 를 설치하기 위해 선택합니다.
  - 소프트웨어 배포 대화상자에서 **할당됨**을 선택하고 확인을 클릭합니다.
  - 오른쪽 창에서 Rhino 5.0 를 오른쪽 클릭하고 **속성**을 클릭합니다.
  - **로그온 시 이 응용 프로그램 설치** 확인란을 선택합니다. (또는 해당 조직에 필요에 따라 배포 옵션을 맞춤 지정합니다).
  - **HelpMedia.msi**와 **LanguagePack_en-us.msi** (기타 원하는 언어 팩) 를 배포하기 위해 6에서 10까지의 과정을 반복합니다.

======스크립트를 사용한 Rhino 5 설치 (배치 파일)======
CD 키 "AAAABBBBCCCCDDDDEEEE"를 가지고 32비트용 Rhino 5 (영어)를 독립 실행형 라이선스로 설치하려면 다음 스크립트를 실행합니다:

<code>
msiexec.exe /i"D:\Installers\setup_x86_en-us.msi" INSTALLDIR="C:\Program Files (x86)\Rhinoceros 5" RMA_CDKEY=AAAABBBBCCCCDDDDEEEE INSTALL_EN=1
 
</code>
 
CD 키 "AAAABBBBCCCCDDDDEEEE"를 가지고 64비트용 Rhino 5 (영어)를 독립 실행형 라이선스로 설치하려면 다음 스크립트를 실행합니다:
 
<code>
msiexec.exe /i"D:\Installers\setup_x64_en-us.msi" INSTALLDIR="C:\Program Files\Rhinoceros 5 (64-bit)" RMA_CDKEY=AAAABBBBCCCCDDDDEEEE INSTALL_EN=1 INSTALL_DE=1 INSTALL_FR=1
 
</code>
     
**자동 설치:** 사용자 인터페이스 없이 설치하려면 msiexec 에 몇 가지 옵션이 있습니다. Microsoft 의 [[http://technet.microsoft.com/en-us/library/cc759262(v=ws.10).aspx|Msiexec Command Line Options]] 안내에서 /q 스위치에 대한 자세한 정보를 참조하세요.

**안내:** 32 비트용 Rhino와 64 비트용 Rhino는 제각각 다른 디렉터리에 설치되어야 합니다.
이 매개변수와 관련 정보에 대한 설명은  **[[rhino:automatedinstalls|Rhino4Setup.ini의 편집]]**을 참조하세요.

======스크립트와 INI 파일을 사용한 Rhino 5 설치======
- 설치 DVD 에 있는 Installers 폴더에서 **Setup.ini**파일을 열어 수정합니다. 설치 프로세스에 설정하려는 매개변수에서 코멘트 (#)를 제거합니다.

**안내:** 한 줄에서 # 기호 뒤에 있는 모든 문자는 설치 관리자에서 무시합니다. 줄의 중간에 # 기호가 있어도 무시되기는 마찬가지입니다. 설정과 관련된 자세한 정보는 **[[ko:rhino:installingrhio:5.0:setupini|Setup.ini 편집]]**을 참조하세요.

- Setup.ini를 수정하여 32비트용 Rhino와 64비트용 Rhino가 서로 다른 경로에 설치되도록 지정해야 합니다. 같은 폴더에 32 비트용 Rhino와 64 비트용 Rhino를 모두 설치하면 문제가 발생합니다!
- MSI 파일을 시작하면 같은 폴더에 있는 INI 파일을 자동으로 찾아 그 안에 있는 매개변수를 읽고 자동으로 설치를 실행합니다.

**샘플 배치 파일:**\\
**안내:** 이 파일은 사용자가 Rhino 5 DVD의 Installers 폴더를 C:\Temp\Installers 폴더로 복사하고 **setup.ini** 파일을 편집하여 다음과 같이 배치 파일을 만들었다고 가정하여, 32 비트용과 64 비트용 Rhino 5 버전 (영어 인터페이스와 영어 도움말 지원)을 모두 설치합니다:
<code>

msiexec /i C:\Temp\Installers\rhino5setup_en-us_x64.msi /qb
msiexec /i C:\Temp\Installers\rhino5setup_en-us_x86.msi /qb
msiexec /i C:\Temp\Installers\LanguagePack_en-us.msi /qb
msiexec /i C:\Temp\Installers\HelpMedia.msi /qb

</code>

======언어팩이 없는 불완전한 Rhino 5 설치======
템플릿, 도구모음, 플러그인이 안 보입니까? 언어팩이 설치되지 않은 경우에 발생하는 현상입니다. MSI 스크립트로 설치했을 때 설치 스크립트에 오류가 있거나, CD로 설치했을 때 불완전하거나 또는 설치에 "실패"한 경우일 수 있습니다. 

위에 안내해 드린 MSI 설치 스크립트로 Rhino를 설치했으나 언어팩 줄이 누락됐다면 직접 MSI를 실행하여 컴퓨터에 언어팩을 설치해야 합니다. 언어팩은 Rhino 5 CD의 Installers 폴더에도 있습니다.

언어 지원이 설치되었는지 확인하시려면 다음 위치에서 한국어 파일, 플러그인, UI 폴더를 찾아 보세요:
C:\Users\<login>\AppData\Roaming\McNeel\Rhinoceros\5.0. 

이 폴더가 없거나, Rhino를 이미 열었으면 도구모음, 템플릿, 도움말이 없을 것입니다. 이것을 수정하려면 다음을 실행합니다:
     - 원하는 언어팩의 MSI 를 실행합니다.
     - Rhino를 열고 명령행에 **ToolbarReset** 명령을 실행합니다. 
     - Rhino를 닫고 다시 엽니다. 이렇게 하면 레지스트리에 있는 구성표가 삭제되고, 없어졌던 구성요소가 Rhino를 다시 열 때 다시 원래대로 설정됩니다.

만약 로드되지 않으면:
  - 제어판을 엽니다.
  - 프로그램 및 기능으로 갑니다.
  - Rhino 언어팩 엔트리로 스크롤합니다. 언어팩은 **Rhinoceros 5 Language Pack Installer (ko-KR)** 처럼 표기됩니다.
  - 오른쪽 클릭하여 **변경**을 선택합니다.
  - Rhino를 다시 엽니다.

======언어팩 MSI 찾기======
  - Rhino 제품 CD를 찾습니다.
  - Installer 폴더를 찾습니다.
  - LanguagePack_XX-XX.msi 를 실행합니다.
 
언어팩을 설치한 후:
  - Rhino를 엽니다.
  - ToolbarReset을 입력합니다.
  - Rhino를 닫고 다시 엽니다.
템플릿이 로드됩니다.
만약 로드되지 않으면:
  - 제어판을 엽니다.
  - 프로그램 및 기능으로 갑니다.
  - Rhino 언어팩 엔트리로 스크롤합니다. 언어팩은 **Rhinoceros 5 Language Pack Installer (ko-KR)** 처럼 표기됩니다.
  - 오른쪽 클릭하여 **변경**을 선택합니다.
  - Rhino를 다시 엽니다.

======Rhino 5 설치 - Zoo 서버 지정======
설치 과정에서 Zoo 서버를 지정하려면 다음 줄을 setup.ini 파일에 추가하세요:

<code>
ZOO_SERVER=host.domain.com # 완전히 검증된 도메인 이름 또는 IP 주소
</code>

또는 

ZOO_SERVER 속성을 명령행에 전달합니다:
<code>
ZOO_SERVER=host.domain.com
</code>

또는

[[ko:zoo:findzooserver#DNS 사용하기|DNS 입력 설정]]
