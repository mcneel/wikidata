====== Rhino V5 Single Commercial License ======


Select license type:
    * **[[rhino:install:wizard:start:windows:5:commercial:install|One or more computers owned by single user]]**
      *Use this option when you want to share you license between your personal computers
      * Does not require Internet access
    * **[[rhino:install:wizard:start:windows:5:commercial:install|Multiple users on office LAN]]**
      * Use this option when you are sharing your license(s) between multiple users over the office LAN
      * Requires Windows **[[https://wiki.mcneel.com/zoo/diagnostics|Zoo]]** server
      * Does not require Internet access
