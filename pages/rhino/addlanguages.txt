====== Adding language support to Rhino 4 ======

To add multiple language support to an existing Rhino 4 installation, follow these steps:

\\

**1.** Have the Rhino installation CD in the CD drive.

{{:legacy:en:ControlPanel.png}}

**2.** Open the  Windows **Add or Remove Programs** control panel.

**3.** Find **[[rhino:rhinoceros|Rhinoceros 4.0]]** in the list and click **Change**.

{{:legacy:en:Modify_Installation.png}}

**4.** In the next dialog, click the **Modify** radio button.

{{:legacy:en:Add_langiages.png}}

**5.** Select the languages to install from the list in the next dialog box and click **Next**. Follow the prompts through the rest of the installation process.
