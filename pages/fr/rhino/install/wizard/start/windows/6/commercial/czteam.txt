====== Rhino 6 Cloud Zoo - Équipe======

>Remarque : Utilisez cette option pour partager vos licences avec les membres d'une équipe. Vos licences seront conservées sur nos serveurs afin que votre équipe puisse accéder à celles-ci depuis n'importe quel ordinateur. Une connexion initiale à Internet est nécessaire. Par la suite, une connexion à Internet une fois par semaine environ suffira pour maintenir la licence. Vous pouvez utiliser Rhino sans connexion à Internet pendant environ 2 semaines avant d'avoir besoin de rafraîchir la licence.


===Configuration initiale sur Internet===
  - **Connectez-vous à votre compte Rhino**
    - Visitez https://accounts.rhino3d.com/?controller=home
      - Créez un compte Rhino si vous n'en avez pas encore un.
      - Enregistrez vos informations de connexion. Vous en aurez besoin par la suite
  - **Créez une équipe**
    - Sur votre page "Mon compte", cliquez sur **Équipes**
      - Cliquez sur Nouvelle équipe
      - Ajoutez un nom et une description
  - **Ajoutez vos licences dans votre équipe**
    - Sur votre page "Mon compte", cliquez sur **Licences**
    - Sous Licences d'équipe, cliquez sur l'équipe que vous avez créée
    - Dans la page de l'équipe, cliquez sur Ajouter une licence
    - Sélectionnez le produit correspondant
    - Indiquez votre clé de licence
    - Cliquez sur Ajouter une licence
  - **Invitez des utilisateurs à rejoindre votre équipe**
    - Sur votre page "Mon compte", cliquez sur **Équipes**
      - Cliquez sur le nom de l'équipe que vous venez de créer
      - Cliquez sur Action puis sur Inviter des membres
      - Indiquez les adresses mail des utilisateurs à qui vous voulez donner accès aux licences de l'équipe
      - Cliquez sur Envoyer
      - Lorsque les utilisateurs répondront aux invitations, en tant que Propriétaire de l'équipe, vous recevrez une confirmation de leur acceptation. 
  - **Lancez Rhino**
    - Cliquez sur Connexion
    - Entrez les informations sur votre compte
    - Rhino démarrera

===Autres ordinateurs===
  - **Installez Rhino**
  - **Lancez Rhino**
    - Cliquez sur Connexion
    - Entrez les informations sur votre compte
    - Rhino démarrera
