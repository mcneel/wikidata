====Installationsfehler 1603 - Rhino 6 ====

Schwerwiegender Fehler während der Installation. (-2147023293)

> **Zusammenfassung:** //Behebung des Installationsfehlers 1603//

=====Problem=====
Dieser Fehler kann von zahlreichen Problemen verursacht werden. Bitte führen Sie folgende Schritte aus.

=====Antwort=====
**Schritt 1: Führen Sie die Windows-Updates durch**
  - Starten Sie das [[http://windowsupdate.microsoft.com/|Windows Update]].
  - Wenn wichtige Updates verfügbar sind, installieren Sie diese durch Klick auf **Jetzt aktualisieren**.  Dieser Prozess kann einige Zeit in Anspruch nehmen.
  - Wenn beim Ausführen des Windows-Updates ein Problem auftritt, lesen Sie bitte die Hinweise zur [[http://support.microsoft.com/kb/906602|Problembehandlung bei der Installation von Updates]].
  - Wenn Sie das Update-Problem behoben haben, führen Sie das [[http://windowsupdate.microsoft.com/|Windows Update]] bis zum Ende aus.
\\
**Schritt 2: Installieren Sie das .NET 4.6 Framework**
  - Laden Sie den [[https://www.microsoft.com/de-de/download/details.aspx?id=49981|.NET 4.6 Framework Installer]] herunter und führen Sie ihn aus.
  - Wenn der Micorsoft .NET-Installer einen Fehler anzeigt: 
    - Schreiben Sie den Fehler auf 
    - Konsultieren Sie **Funktioniert es immer noch nicht?**, weiter unten.

\\
**Schritt 3: Installieren Sie Rhino**
  - Öffnen Sie den **Windows Explorer**.
  - Laden Sie sich das aktuelle Rhino-6-Installationsprogramm hier herunter: [[http://www.rhino3d.com/download/rhino/6/latest|Vollversion von Rhino 6]]
  - Führen Sie die heruntergeladene Datei aus, um mit der Installation zu beginnen.
  - Wenn die Installation fehlschlägt, speichern Sie eine Kopie der Protokolldatei auf Ihrem Desktop.
\\
**Funktioniert es immer noch nicht?**

Das ist ärgerlich und tut uns wirklich leid! Bitte senden Sie die Protokolldatei aus Schritt 3 an tech.eu@mcneel.com
