====== Rhino 5のインストールの場所をカスタマイズする ======

=====問題:=====
Rhino 5をシステムボリューム（c:）ではなく別の場所にインストールしたいのですが。

=====解決策:=====
  - Install Rhino 5をRhino DVDまたはhttp://www.rhino3d.com/download/5 \\ \\ からダウンロードできる最新のRhino 5のインストーラ
  - Select **Change installation folder** in the installation process.\\ \\ {{http://wiki.mcneel.com/_media/rhino/rhino5/change_installation_folder.png?&600}}
