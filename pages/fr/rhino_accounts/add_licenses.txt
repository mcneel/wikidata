===== Ajouter une licence à votre compte Rhino =====

  - Consultez votre courrier électronique pour trouver un message de McNeel ou d'un revendeur contenant les informations sur la licence.
  - Si le message contient un lien intitulé Installer votre clé de licence:
    * Cliquez sur le lien.
    * Lorsque le lien s'ouvre, cliquez sur **Compte Rhino**.
  - Si le message ne possède pas de lien intitulé Installer votre clé de licence :
    * Trouvez la clé de licence du produit dans le message.
    * Copiez la clé de licence.
    * [[https://www.rhino3d.com/licenses/?controller=add_licenses|Ouvrez la page Ajouter une licence]].
  - Ajoutez la clé de licence dans votre compte personnel ou dans une des équipes à laquelle vous appartenez.
