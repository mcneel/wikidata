====== Pourquoi Rhino affiche-t-il un message indiquant que la version a expiré ? (Rhino pour Mac) ======
> **Résumé:** //Rhino pour Mac peut informer que la version a expiré pour plusieurs raisons.//

=====Problème=====
L'expiration de Rhino pour Mac peut provenir d'un grand nombre de problèmes. 

=====Réponse=====
Plusieurs copies de Rhino sont installées et celle qui démarre est la version expirée. Vous devez supprimer **toutes** les copies de Rhino. 

Double cliquez sur un fichier 3DM. Si le fichier s'ouvre dans Rhino, une copie de Rhino est toujours installée quelque part. 
  - Cliquez avec le bouton de droite sur l'icône de Rhino dans le Dock, sélectionnez **Options > Afficher dans le Finder** afin de trouver cette copie.
  - Déplacez cette copie dans la Corbeille. 
  - Lorsqu'il n'existe plus de copies de Rhino, double cliquer sur un fichier 3DM l'ouvrira dans TextEdit. Lorsque ceci se produit, toutes les copies de Rhinoceros ont été supprimées.
  - Réinstallez maintenant la version que vous venez de télécharger.
