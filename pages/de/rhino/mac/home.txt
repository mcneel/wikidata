==============
====Die Entwicklung von Rhino für Mac ist bald abgeschlossen.====

  * An diesen Funktionen arbeiten wir noch für die erste Version:
    * Verbesserungen an der Benutzeroberfläche für **Rendering und Materialien**
    * **Bessere Unterstützung für Drucken**
    * Kleinere Verbesserungen an der **Benutzeroberfläche**
    * **Import und/oder Export** einiger Dateiformate
    * **Lokalisierung** Ordentliche Übersetzung in die unterstützten Sprachen
    * **Dokumentation** und Tutorials

  * Folgendes wird in der ersten Version noch **NICHT** funktionieren. **//<- Wichtig//**
    * **Grasshopper**
    * **Layouts**
    * **Arbeitsgruppen**
    * **Animationswerkzeuge**
    * **Paneling Tools**
    * **Solidworks**-Dateiimport
    * **GHS**-Support
    * Import/Export von **KML-, WMF-, ZPR- und SCN**-Dateien
    * **Plug-ins von Drittentwicklern**

Alle **ein oder zwei Wochen** steht ein neuer Build zur Verfügung. Dieser ist dann für rund **60 Tage** gültig.

====Systemanforderungen====
  * Prozessor: Intel Core 2 Duo 64-Bit oder neuer
  * OS X 10.8.5 oder eine beliebige Version von OS X 10.9.  Bugs in älteren Versionen von 10.8 könnten Probleme in Rhino verursachen, weswegen die neueste Version 10.8 benötigt wird.
  * 4 GB RAM (empfohlen wird etwas mehr)
  * Eine Mehr-Tasten-Maus (Mausrad wird empfohlen)
  * Der [[http://www.3dconnexion.com/products/spacenavigator.html|SpaceNavigator]] wird unterstützt. (Sie benötigen dazu den Treiber von 3DConnexion.)

==== Nicht unterstützt ====
    * OS X 10.7 und ältere Versionen
    * Spaceball und SpaceExplorer.
