====== Problembehandlung Installation von Rhino 5 ======
> Dieser Artikel bezieht sich auf Rhino 5. Ähnliche Artikel sind ebenfalls vorhanden für [[http://wiki.mcneel.com/rhino/installproblems_legacy|Rhino 3 und 4]].

**Wichtiger Hinweis:** Die meisten auf dieser Seite aufgelisteten Probleme können vermieden werden, wenn Sie vor der Installation von Rhino sicherstellen, dass Ihr Windows auf dem neuesten Stand ist. Führen Sie dazu gegebenenfalls ein **[[http://windowsupdate.microsoft.com/|Windows Update]]** durch Den aktuellen Update-Status der Windows-Version auf Ihrem Computer können Sie herausfinden, indem Sie den Anweisungen auf [[http://windowsupdate.microsoft.com/|dieser Microsoft-Seite]] folgen.\\

Wenn das Windows Update geöffnet ist, klicken Sie auf die Schaltfläche //Nach Updates suchen//. Nachdem Sie alle wichtigen Updates installiert haben, versuchen Sie eine erneute Installation von Rhino.

Wenn das Windows Update hängen bleibt, einen Fehler ausgibt oder ein sonstiges Problem verursacht, gehen Sie auf Microsofts [[https://support.microsoft.com/de-de/kb/2700567|Seite zur Behandlung von Problemen beim Windows Update]].

**Protokollierung von Installationsproblemen**

Falls die Installation von Rhino 5 fehlschlägt, legt das Installationsprogramm automatisch eine Protokolldatei an.  Diese Protokolldatei hilft unserem technischen Support dabei, Ihnen bei den aufgetretenen Problemen weiterzuhelfen. 
  - Im Dialog der Fehlermeldung finden Sie einen Link. 
  - Folgen Sie dem Link zum Öffnen der Protokolldatei und speichern Sie diese als Textdatei ab (**Speichern unter...**).
  - Senden Sie die Textdatei als Anhang einer E-Mail an [[tech.eu@mcneel.com]].
