====== Rhino 6 Einzelnutzer-Studentenlizenz ======


===Bitte auswählen:===
    * **[[rhino:install:wizard:start:windows:6:commercial:czpersonal|Mehrere Computer, ein Benutzer]] - Cloud Zoo Persönlich**
      * Wählen Sie diese Option, wenn Sie Ihre Lizenz auf Ihren eigenen Computern teilen wollen
      * Gelegentlich wird ein Internetzugang benötigt
      * Einzelne Studentenlizenzen unterstützen keine Cloud Zoo Teams
      * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]

    * **[[rhino:install:wizard:start:windows:6:commercial:stand_alone|Einzelcomputer-Lizenzierung]]**
      * Wählen Sie diese Option wenn nur ein Computer verwendet wird
      * Sperrt die Lizenz zur **ausschließlichen** Verwendung auf einem Computer
      * Benötigt keinen Internetzugang
      * [[https://www.rhino3d.com/6/eula|Endbenutzer-Lizenzvertrag]]
