====== Rhino Thumbnail Viewer ======
> **Summary:** //Windows Shell extension to view Rhino thumbnail preview images within Explorer.//
> **Important:** //This tool is included with [[http://www.rhino3d.com/|Rhinoceros 5]].//

=====Description=====

The Rhino Thumbnail Viewer is an extension to the Windows Shell that allows you to view thumbnail images of your Rhino 4.0, or prior, files automatically within Windows Explorer. Note, you do not have to have Rhino installed on your system to use this extension.

The Rhino Thumbnail Viewer support Rhino 4.0 or prior version files.

The Rhino Thumbnail Viewer supports Windows 2000 and Windows XP (both 32-bit and 64-bit versions).

{{http://en.wiki.mcneel.com/content/upload/images/RhinoThumbView1.png}}

The Rhino Thumbnail Viewer also supports Windows Vista and Windows 7 (both 32-bit and 64-bit versions).

{{http://en.wiki.mcneel.com/content/upload/images/RhinoThumbView2.png}}

=====Troubleshooting=====
After installation, if you do not see thumbnail preview images and you are using Windows XP, make sure to disable "Always show icons, never thumbnails" in the Folder Options of Windows Explorer.
