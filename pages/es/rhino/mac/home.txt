====Rhino para Mac sigue en desarrollo pero ya está casi listo para producción.====

  * Estas son las funciones con las que todavía estamos trabajando para la primera versión:
    * **Importación/Exportación** de algunos formatos de archivo 
    * **Mejoras en la interfaz de usuario**
    * **Documentación** y tutoriales. 

  * Estas funciones **NO** estarán en la primera versión. **//<- Importante//**
    * **Grasshopper**
    * **Diseños**
    * **E/S de archivos de Sketchup**
    * **Importación de archivos de SolidWorks**
    * **Compatibilidad con GHS**
    * **E/S de archivos KML, WMF, ZPR y SCN**
    * **Plug-ins externos**

Habrá una nueva versión WIP disponible **cada semana o cada dos semanas**. Cada versión podrá utilizarse durante unos **60 días**.

====Requisitos del sistema====
  * procesadores Intel Core Duo 2 de 64 bits
  * OS X 10.7.5, OS X 10.8.5 o cualquier versión de OS X 10.9.  Los errores de las versiones anteriores de 10.7 o 10.8 podrían causar problemas en Rhino, de modo que se requiere la última versión de 10.7 o 10.8.
  * 4 GB de RAM (se recomienda más)
  * Un ratón de varios botones (se recomiendan una rueda de desplazamiento)
  * El [[http://www.3dconnexion.com/products/spacenavigator.html|SpaceNavigator]] es compatible.

--> Hardware no compatible#
    * OS X 10.6 o versiones anteriores
    * **Spaceball** y **SpaceExplorer**
