====== macOS용 LAN Zoo ======
> **제품:** //[[ko:zoo:home|Zoo]]//
> **요약:** // macOS에서 LAN Zoo를 실행하는 방법을 안내합니다.//

===== 질문 =====
Mac용 Rhino에서 LAN Zoo를 사용하고 싶지만, LAN Zoo는 Microsoft Windows에서만 실행되기 때문에 Windows 시스템에서 호스트해야 합니다. LAN Zoo를 Apple macOS로 포팅할 계획이 있습니까?

===== 답변 =====
아니요. 현재, LAN Zoo를 macOS로 포팅하려는 계획은 없습니다. 계획이 변경된다면 [[http://discourse.mcneel.com/|Discourse]] 토론 게시판에 공지로 알려 드리겠습니다. 

LAN Zoo를 실행할 Windows 시스템이 없으시다면, [[http://www.vmware.com/products/fusion-pro/|VMWare Fusion Pro]]를 사용하여 macOS 시스템에서 Windows 가상 시스템을 실행할 수 있습니다.
