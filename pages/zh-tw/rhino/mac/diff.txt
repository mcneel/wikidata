====== Rhino Mac版本與Windows版本的不同 ======
=== Mac版本使用者介面新增特色 ===

有一些指令及功能是只有Mac版本才有的，請見[[http://wiki.mcneel.com/rhino/mac/userinterfacefeatures|介面特色wiki頁面]]了解詳情。 

-->滑鼠#

**建議**

  * 為了方便使用，建議使用具有左右鍵及滾輪的滑鼠。但所有操作也可以使用單鍵滑鼠完成。
  * 使用單鍵滑鼠需要右鍵功能時，按住**Control**鍵並點選滑鼠。\\ \\

**視圖巡覽快速鍵**

  * **按住右鍵拖曳** 及 **Shift+按住右鍵拖曳** 作業方式與Windows版本相同，請參考[[http://docs.mcneel.com/rhino/5/help/en-us/user_interface/shortcuts.htm|Rhino 說明 - 視圖巡覽快速鍵]]. \\ 所有視圖投影方式，按住**Shift**鍵加右鍵為平移，按住 **Command**鍵加右鍵為縮放視圖，按住**Command+Shift** 鍵加右鍵為旋轉視圖。
  * 可使用滾輪在視圖上縮放。
  * 若您使用單鍵滑鼠，請使用 **Control**+**Command**+**Shift**鍵來旋轉視圖。

<--

-->右鍵文字功能表#

  * 在視圖區塊上按右鍵會顯示一個文字功能表。
  * 預設並**不會**重複上一個指令。 \\ 若要改變此設定，在**Rhinoceros** > **Preferences** > **Mouse** > **Mouse**，關閉 **Enable context menus**。

<--

-->指令提示列#

若您使用Windows版本Rhino的指令提示列，所有功能也一樣在Mac Rhino，要輸入指令您只要直接開始在鍵盤輸入，將會出現一個**Command Search**視窗，輸入指令後，您可以使用Return鍵及空白鍵啟動指令。

接著會出現指令選項對話框，所有選項皆與Windows版本指令提示列一樣，您可以將對話框拖曳到螢幕上任何位置，並且Rhino會記住這個位置。

如果您想要鍵入命令選項值，請在Legacy preferences面板( Legacy preferences panel)勾選**Always provide an input field in command options dialog for typed values** 。
之後每個新的選項提示的其中一個字母將會被畫上強調底線。您可以鍵入字母並按下確認鍵(Return)來叫用此選項。 

在指令選項列中輸入數值時，您不需要移動滑鼠，直接輸入即可。所有使用習慣皆與Rhino Windows版本一樣。

<--

-->工具列及工具面板#

  * Mac Rhino 在標準Mac工具列有包含Rhino指令的 //工具按鈕//。
  * Mac Rhino 也提供包括Windows版本的一系列工具按紐的//工具列// \\  當Mac Rhino開啟時，左手邊會有一系列常用的工具按鈕。
  * 工具按紐可以包含一或兩個指令。
  * 滑鼠在工具按紐上稍停一下會出現指令提示。\\  若指令提示有兩行，代表包括兩個指令。
    * 左鍵**點擊**工具按紐會執行第一行的指令。.
    * 按住**Option**鍵並**點擊**左鍵則會執行第二行的指令。
  * 有工具列連結的按鈕右下角會有一個小三角。
    * 長按有工具列的按鈕，可以叫出工具列。
  * 工具列也可以顯示為浮動式工具列。
    * 若要顯示浮動式工具列，請按住**Option**鍵並使用滑鼠在圖示上按久一點。

<--

-->鍵盤#

  * Windows版的Rhino使用**Control**鍵做許多操作，在Mac版本，使用**Command**鍵來取代。
    * 例如：**Command** 鍵取代 **Ctrl**鍵來取消選取物件。
    * 同樣的，**Command** 鍵取代**Ctrl**鍵作為視圖巡覽快速鍵：

| | | |**鍵盤** | **動作** | **+ Command 鍵** |
| | | |**方向左鍵** | 左轉視圖 | 往左平移 |
| | | |**方向右鍵** | 右轉視圖 | 往右平移 |
| | | |**方向上鍵** | 上轉視圖 | 往上方平移 |
| | | |**方向下鍵** | 下轉視圖 | 往下方平移 |

<--

-->筆記型電腦相關問題#


**藍芽滑鼠**\\ 建議使用具有雙鍵、滾輪功能並有藍芽功能滑鼠\\ \\

**觸控式軌跡板(Trackpad)**\\ 在觸控式軌跡板上使用兩根手指向上或向下縮放視圖。\\ 若您想改變這個設定，請到**Rhinoceros** > **Preferences** > **Trackpad**,**使用兩根手指來平移或縮放視圖(Use two fingers to pan and rotate views)**.\\

**缺少的鍵盤鍵**\\ 筆電鍵盤沒有獨立的**Home**、**End**、 **PageDn** 或 **PageUp** 鍵，但可結合**fn** 鍵其他按鍵來複製相同指令。 \\

| **按鍵** | **+ fn 鍵** |
| **向上方向鍵** | 上一頁 |
| **向下方向鍵** | 下一頁 |
| **向左方向鍵** | Home |
| **向右方向鍵** | End |
| **Delete** | Delete |
