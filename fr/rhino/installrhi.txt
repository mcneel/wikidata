====== Installation de fichiers RHI pour tous les utilisateurs ======
Rhino 5.6 vous permet de choisir d'installer pour un utilisateur ou pour tous les utilisateurs de l'ordinateur. De plus, le déploiement des fichiers RHI peut être scripté pour être exécuté en arrière-plan. 

=====Déploiement des fichiers RHI=====
  - Vérifiez que Rhino 5.6 ou une version plus récente est déployé sur votre réseau.
  - Lancez "C:\Program Files\Rhinoceros 5 (64-bit)\System\x64\rhiexec.exe" "C:\Full\Path\To\Installer.rhi" /admin /silent

L'option /admin réalise l'installation pour tous les utilisateurs de l'ordinateur.
L'option /silent réalise l'installation sans montrer d'interface utilisateur.
