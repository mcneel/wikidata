======Options de licence de Rhino 6======

> **Produit:** //[[rhino:home|Rhinoceros]]//
> **Résumé:** //Découvrez les différentes options de licence de Rhino. Trouvez le type de licence qui correspond le mieux à votre situation.//

Nous avons modifié et clarifié le Contrat de licence de l'utilisateur final pour Rhino 6 ainsi que les modes d'utilisation de la licence du logiciel. 

Il existe maintenant trois options pour utiliser votre licence :
  - Licence autonome pour l'utiliser **exclusivement** sur un seul ordinateur.
  - Licences flottantes à utiliser sur le **réseau privé d'une entreprise** en utilisant le Zoo. [[zoo:home|Plus d'informations ici....]]
  - Partager votre licence sur **plusieurs ordinateurs** en utilisant le Cloud Zoo pour Rhino 6. [[rhino:6:cloud_zoo:setup| Cliquez ici pour voir comment configurer le Cloud Zoo...]]
    - Ajoutez votre licence et elle sera accessible sur tous les ordinateurs auquel vous avez accès, en utilisant Internet au lieu d'un réseau local, à travers votre compte Rhino.
    - Pour plusieurs utilisateurs, utilisez le Cloud Zoo pour créer des équipes, ajoutez plusieurs licences et invitez des utilisateurs à rejoindre l'équipe pour partager les licences sur Internet. 

Nous recommandons d'utiliser le Cloud Zoo pour une plus grande flexibilité. Une comparaison détaillée des options de licences est donnée sur la page [[https://www.rhino3d.com/6/new/licensing-and-administration|Licences et administration de Rhino 6]]

Vous pouvez passer d'un mode de licence à un autre mais ce n'est pas une opération automatique. Pour plus d'informations, consultez l'article : [[rhino:6:change-license|Changer de licence dans Rhino 6 pour utiliser le Cloud Zoo]].

Vous trouverez ici les réponses aux [[rhino:faq:licensing|Questions souvent posées]] sur les licences.
