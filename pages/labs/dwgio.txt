====== DWG/DXF File Import/Export for V4 SR1 ======

> **Summary:** //Rhino 4.0 Plug-ins to import and export [[rhino:autocad|AutoCAD]] [[accurender:dwg|dwg]] & dxf files//

=====Description=====

These plug-ins for Rhino 4.0 update the dwg/dxf import and export. First rename or move the originals then then place one or both of the following:

For Import - The //import_ACAD.rhp// file from this [[http://en.wiki.mcneel.com/content/upload/files/import_ACAD_11_26_07.zip|zip file]]  or

For Export - The //export_ACAD.rhp// file from this [[http://en.wiki.mcneel.com/content/upload/files/export_ACAD_11_26_07.zip|zip file]]

in your in your Rhino 4.0 installation plug-ins directory.

These will work with Rhino4 SR2.

=====Export=====
  * Fixed some alignment problems with [[rhino:autocad|AutoCAD]] paper space exported from Rhino layouts.
  * Convert characters that [[rhino:autocad|AutoCAD]] can't digest in the names of paper space pages to '_'.

===== Import=====
  * Changed to allow a few paper space pages that were being skipped because of uninitialized settings to be imported.
  * Fixed the positioning of center and right justified MText to match the [[rhino:autocad|AutoCAD]] drawing better.
