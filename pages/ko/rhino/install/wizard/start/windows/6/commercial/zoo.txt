====== LAN Zoo 라이선스 관리자 ======

>안내: LAN(로컬 영역 네트워크)로 제한하여 여러 사용자 간에 라이선스를 공유할 때 이 옵션을 사용합니다. 인터넷 연결이 필요하지 않습니다. 무료 Zoo 플로팅 라이선스 관리자의 설치와 설정이 필요합니다. Rhino는 워크스테이션에 자동 설치 배포될 수 있습니다.

Rhino 설치 관리자 패키지를 **[[https://www.rhino3d.com/download/Rhino/7/release|다운로드]]**합니다. 
    - 귀하의 이메일 주소를 입력하고 "다음"을 클릭합니다.
    - "한국어 지금 다운로드"를 클릭하거나 원하시는 다른 언어를 선택하세요.

--------

**설치 유형:** 하나를 선택합니다.


**설치 배포** - IT 관리자가 실행합니다. 
  * 설정 및 자동 설치 **[[https://wiki.mcneel.com/ko/rhino/installingrhino/7|배포]]** 안내에 따라 실행합니다. 
  * Rhino를 시작합니다 




**메뉴얼**
  * **설치 관리자를 실행**합니다. 로컬 관리자 권한이 필요합니다. 
        - 사용 약관과 개인 정보 취급 방침에 동의합니다.
          - 링크를 클릭하여 읽습니다. - 권장 사항입니다. 
        - "지금 설치"를 클릭합니다.
          - 선택 사항 - 고급 옵션을 보려면 톱니 바퀴 모양의 아이콘을 클릭합니다.
        - 설치가 완료되면 "닫기"를 클릭합니다. 
        - Rhino를 시작합니다. - **Rhinoceros 라이선스가 필요합니다** 대화상자가 표시됩니다. 
        - **[[https://wiki.mcneel.com/ko/zoo/home|Zoo를 사용합니다.]]**
        - **이 호스트 이름 또는 IP 주소인 Zoo 서버 사용** 항목을 선택합니다.
        - Zoo 서버의 FQDN 또는 IP 주소를 입력합니다.
        - **계속**을 클릭합니다.
        - Rhino가 시작합니다.

--------

=== Rhino 설치에 실패하셨습니까? ===
  * Rhino 설치 관리자가 실패하면, 오류에 대한 설명과 그 해결 방법이 안내된 대화상자가 표시됩니다.
  * [[https://wiki.mcneel.com/ko/rhino/7/installproblems|MSI 오류 코드]]를 받으셨나요?
  * 필요하다면, 오류 로그를 열어 파일에 저장하고 McNeel 기술 지원 <tech@mcneel.com> 에게 이메일로 보내주시기 바랍니다.
