====== Licence Lab Éducation de Rhino 5 pour 30 utilisateurs ======


===Choisir :===
    * **[[https://wiki.mcneel.com/fr/zoo/rhinomacdeployment|30 ordinateurs utilisant le réseau de l'école - Zoo]] (Recommandé)**
        * Utilisez cette option lorsque vous partagez la licence dans une salle de l'école
        * Rhino utilise le réseau de l'école
        * Le gestionnaire gratuite de licences flottantes **[[https://wiki.mcneel.com/fr/zoo/home|Zoo]]** sur Windows est nécessaire
        * L'installation de Rhino est facilement **[[https://wiki.mcneel.com/fr/zoo/rhinomacdeployment|déployée]]**
        * Aucun accès à Internet n'est nécessaire
        * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Contrat de licence de l’utilisateur final]]

    * **[[https://wiki.mcneel.com/fr/rhino/5/macinstall|30 ordinateurs individuels dans une seule salle]] (Non recommandé)**
      * Utilisez cette option lorsque 30 ordinateurs individuels seront utilisés
      * Aucun accès à Internet n'est nécessaire
      * Rhino est installé manuellement sur 30 ordinateurs
      * [[http://files.na.mcneel.com/rhino/5.0/docs/rh50_eula.pdf|Contrat de licence de l’utilisateur final]]

>Cette page wiki fait partie d'un assistant d'installation de Rhino. [[fr:rhino:install:wizard:start |Commencer du début]].
