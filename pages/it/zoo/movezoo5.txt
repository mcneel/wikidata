======Spostare lo Zoo 5 su un altro server======
> **Prodotto:** //[[zoo:zoo5|Zoo 5]]//
> **Sommario:** //Spiega come spostare il software del server dello Zoo 5 da un server ad un altro.//

=====Ulteriori informazioni=====
Di fatto, spostare il software del gestore delle licenze Zoo da un server ad un altro non è diverso dall'installare il software dello Zoo dall'inizio.

Nota: Non è possibile "spostare" o "copiare" i codici dei prodotti McNeel da un server dello Zoo ad un altro. I codici dei vari prodotti vanno aggiunti al nuovo server dello Zoo così come si è fatto per il server dello Zoo originale.

=====Passi=====
  * Scaricare l'ultima versione dello Zoo da **[[http://www.rhino3d.com/download/zoo/5.0/commercial|qui]]**.
  * Installare il software del server dello Zoo nel nuovo server.
  * Avviare l'amministratore dello Zoo.
  * Aggiungere i codici dei prodotti.
  * Se necessario, avviare il servizio Zoo.
  * Reindirizzare i client di Rhino verso il nuovo server dello Zoo modificando il [[zoo:findzooserver|Registro di sistema di Windows]] oppure il [[zoo:findzooserver|nome del server dello Zoo predefinito]] del server DNS.

Quando tutto funziona come previsto, disinstallare il software dello Zoo dal server precedente.
