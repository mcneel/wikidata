====== Rhino 6 Cloud Zoo - チーム======

>メモ: チームのメンバーとライセンスを共有される場合はこのオプションをご利用ください。お持ちのライセンスを弊社のサーバーに置いておく形になりますので、同じチームでしたらどのコンピューターからでもアクセスすることができます。最初にインターネットへ接続する必要があり、その後は週に1回程度インターネットにつながった状態でRhinoを実行するとライセンスはリフレッシュされます。インターネットがなくても、リフレッシュが必要になるまで2週間程度Rhinoを動作させることができます。


===最初にウェブベースでセットアップする===
  - **Rhinoアカウントにログイン**
    - https://accounts.rhino3d.com/?controller=home にアクセス
      - まだお持ちでない場合はRhinoアカウントを作成してください
      - 後で必要になりますので、資格情報を保存しておいてください
  - **チームを作成**
    - "マイアカウント" のページで **チーム** をクリック
      - 新規チーム をクリック
      - 名前 と 説明 を追加
  - **チームにお持ちのライセンスを追加**
    - "マイアカウント" のページで **ライセンス** をクリック
    - チームのライセンス の下にある、作成したチームをクリック
    - チームのページで、ライセンスを追加 をクリック
    - 製品を正しく選択
    - ライセンスキーを入力
    - ライセンスを追加 をクリック
  - **チームに参加するユーザーを招待**
    - "マイアカウント" のページで **チーム** をクリック
      - チームを管理 をクリック
      - メンバーを招待 をクリック
      - チームのライセンスにアクセスできるようにしたいユーザーのメールアドレスを入力
      - 送信 をクリック
      - ユーザーが招待に返信すると、チームの所有者としてあなたは受諾の確認を受け取ります
  - **Rhinoを起動**
    - ログイン をクリック
    - アカウントの資格情報を入力
    - Rhinoが起動します

===別のコンピューター===
  - **Rhinoをインストール**
  - **Rhinoを起動**
    - ログイン をクリック
    - アカウントの資格情報を入力
    - Rhinoが起動します
