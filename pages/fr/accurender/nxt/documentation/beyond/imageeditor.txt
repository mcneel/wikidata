====== Éditeur d'image nXt ======

L'éditeur d'images nXt est un outil indépendant qui permet de modifier des fichiers images natifs (.nXtImage) produits par une des plates-formes nXt.  Ces fichiers natifs conservent toutes les informations recueillies pendant le rendu. Lancez l'éditeur à partir du menu Démarrer, dossier nXt.

Avec l'éditeur vous pouvez :

  * Ajuster les paramètres du contrôle de tonalité
  * Changer l'intensité de tous les canaux d'éclairage
  * Ajouter des effets spéciaux calculés à partir de l'image, tels que du brouillard ou un flou de profondeur. 
  * Enregistrer une image dont les tonalités ont été réparties dans un format populaire comme .jpg ou .png
  * Enregistrer les informations sur la luminance dans un format HDR
  * Voir et enregistrer des canaux masqués supplémentaires (alpha, distance, matériau) pour les utiliser en composition avancée
  * Enregistrer au format Piranesi(c) (*.epx) qui peut être utilisé pour créer un rendu non photo réalistes
  * Utiliser l'arithmétique d'image pour recoller par exemple une image qui a été générée par plusieurs noeuds de la ferme de rendu.
  * Enregistrer les paramètres d'éclairage utilisés pour générer ce rendu    Ces paramètres peuvent ensuite être utilisés pour générer d'autres rendus.


===== Charger une image =====

Après avoir enregistré les résultats de votre rendu dans un fichier .nXtImage, utilisez le menu Fichier > Ouvrir pour charger l'image dans l'éditeur. 

Lorsque vous déplacez le curseur sur l'image, la barre d'état affiche les informations suivantes sur chaque pixel :

  * Les coordonnées du pixel, mesurées à partir du sommet inférieur gauche
  * La couleur et la valeur alpha, au format RVBA, du pixel après la répartition des tonalités
  * La valeur de luminance réelle enregistrée dans chaque pixel sous RVB et un seul canal Y.  Il s'agit du résultat direct du calcul de l'éclairage.  Si des informations précises sur l'éclairage ont été utilisées pour produire le rendu, ces valeurs auront des unités de candelas/m2.
  * La distance entre chaque pixel et le point de vue en mètres.  Les valeurs négatives indiquent un pixel de l'arrière-plan.
  * Le nom du matériau utilisé pour rendre le pixel.


===== Travailler avec des canaux de masque=====

Les fichiers nXtImage contiennent trois canaux supplémentaires qui peuvent être utilisés pour des tâches de composition avancée dans la plupart des éditeurs d'image.  Ces canaux portent des informations sur la transparence (alpha), la distance et le matériau pour chaque pixel. Les informations sont encodées dans une image en échelle de gris.  Chaque canal peut être observé et enregistré dans un fichier .png.

Utilisez le menu déroulant Afficher pour voir chaque canal de masque. 
Utilisez l'option Fichier > Enregistrer le masque pour enregistrer un canal de masque dans une image .png. 

**Remarques supplémentaires:**

  * Le canal alpha peut être inclus avec une image dont les tonalités ont été réparties en sélectionnant un format de fichier avec canal alpha lors de l'enregistrement d'une image dont les tonalités ont été réparties.
  * Les canaux de distance et de matériaux ne sont pas anticrénelés et peuvent laisser apparaître des artefacts aux bords marqués.  L'ajout d'une petite quantité de flou gaussien à un masque avant de l'utiliser peut aider à adoucir ces bords.
  * Le canal de matériaux n'encodera que 255 matériaux différents.  Si votre dessin contient plus de matériaux, certaines couleurs de masque seront répétées.
