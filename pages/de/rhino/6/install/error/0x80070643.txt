==== Installationsfehler 0x80070643 - Rhino 6 ====

e000: Fehler 0x80070643: Installation des MSI-Pakets fehlgeschlagen.

> **Zusammenfassung:** //Behebung des Installationsfehlers 0x80070643//

=====Problem=====
Die Installation durch Rhino schlägt fehl.  Die Datei Installation.log enthält die Zeile: 

e000: Fehler 0x80070643: Installation des MSI-Pakets fehlgeschlagen.

=====Antwort=====
Es handelt sich um ein Problem mit dem Microsoft .NET Framework.  Laden Sie das Microsoft .NET Framework Repair Tool herunter, um das .NET Framework wie in diesem Artikel erklärt zu reparieren: https://support.microsoft.com/de-de/help/976982/net-framework-update-installation-error-0x80070643-or-0x643

Danach installieren Sie Rhino neu.
