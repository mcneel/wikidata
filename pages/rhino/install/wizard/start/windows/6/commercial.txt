====== Rhino V6 Single Commercial License ======


===Choose one:===
    * **[[rhino:install:wizard:start:windows:6:commercial:czpersonal|Multiple computers, single user]] - Cloud Zoo Personal**
        * Use this option when you want to share you license among your personal computers
        * Requires occasional Internet access
        * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:commercial:czteam|Multiple computers, multiple users]] - Cloud Zoo Team**
        * Use this option when you are sharing your license(s) among multiple users
        * Requires occasional Internet access
        * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:commercial:zoo|Multiple users on office LAN]] - Zoo**
        * Use this option when you are sharing your license(s) among multiple users, limited to the office LAN
        * Requires Windows **[[https://wiki.mcneel.com/zoo/home|Zoo]]** free floating license manager
        * Rhino installation easily **[[https://wiki.mcneel.com/rhino/installingrhino/6|deployed]]**
        * Does not require Internet access
        * [[https://www.rhino3d.com/6/eula|End User License Agreement]]

    * **[[rhino:install:wizard:start:windows:6:commercial:stand_alone|Single computer licensing]]**
      * Use this option when only one computer will be used
      * Locks license **exclusively** to one computer
      * Requires one time Internet access for validation or an [[https://wiki.mcneel.com/zoo/rhinovalidateoffline|offline validation.]]
      * After validation, it does not require Internet access except for updates.
      * [[https://www.rhino3d.com/6/eula|End User License Agreement]]
