====== Zoo para OS X ======
> **Producto:** //[[zoo:home|Zoo]]//
> **Resumen:** //Sobre el funcionamiento del Zoo en OS X.//

===== Pregunta =====
Actualmente, si queremos usar Rhino para Mac con el Zoo, tenemos que usar la versión del Zoo para Microsoft Windows. ¿Está previsto portar el Zoo a Apple OS X?

===== Respuesta =====
Sí, actualmente estamos en el proceso de desarrollar una versión del Zoo para OS X. Cuando tengamos una versión disponible para probar, publicaremos un mensaje en el foro de [[http://discourse.mcneel.com/|Discourse]].

Si no tiene un sistema de Windows donde ejecutar el Zoo, puede usar [[http://www.vmware.com/products/fusion-pro/|VMWare Fusion Pro]], que permite ejecutar Windows en un Mac.
