====== Upgrading a LAN Zoo Server ======
> **Product:** //[[zoo:home|Zoo]]//
> **Summary:** //Upgrading a Zoo server.//

To support Rhino 7, you need to either upgrade to **LAN Zoo 7** or move to the **[[https://wiki.mcneel.com/rhino_accounts/home|Cloud Zoo]]** (often a better option). 

===== Prerequisite  =====

[[https://wiki.mcneel.com/zoo/zoodotnet|The Zoo Requires Microsoft .NET Framework 4.8]]

===== Download and Install =====
  * The Zoo requires [[https://dotnet.microsoft.com/download/dotnet-framework/net48|Microsoft .NET Framework 4.8]]. You can ensure that Microsoft .NET Framework 4.8 is installed on your system by using **[[http://windows.microsoft.com/en-us/windows/windows-update|Windows Update]]**.

  * [[http://www.rhino3d.com/download/zoo/7/latest|Download the LAN Zoo 7 here.]]

  * Install on the same server as your current LAN Zoo.

===== FYI =====

  * Zoo 7 replaces your older Zoo.

  * Your licenses will be automatically be moved to Zoo 7. 

  * Users running Rhino at the time of installation will not be affected.

  * The installation takes about 15 seconds.
