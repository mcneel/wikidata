====== Rhino 5 설치 ======

>Windows용 Rhino 5 설치 안내입니다. 

  - Rhino 설치 관리자 패키지를 **[[https://www.rhino3d.com/download/rhino/5/latest|다운로드]]**합니다.
    - 귀하의 이메일 주소를 입력하고 "다음"을 클릭합니다.
    - "한국어 지금 다운로드"를 클릭하거나 원하시는 다른 언어를 선택하세요.
  - **설치 관리자 실행**합니다. 로컬 관리자 권한이 필요합니다. 
    - "Rhino 5 설치"를 클릭합니다.
  - **라이선스 정보를 입력**합니다.
      - 이름: 필수
      - 이메일: 필수
      - 기관: 선택 사항
    - **라이선스 유형 선택**
      - Rhino 평가 - 라이선스 키가 필요하지 않습니다.
      - 라이선스 키 - 24 자리 Rhino 라이선스 키 (예: RH50-XXXX...) 를 입력합니다.
      - Zoo - [[https://wiki.mcneel.com/ko/zoo/home|LAN Zoo]] 설치의 경우에만 해당합니다.
    - **사용 약관에 동의합니다** - 클릭하여 사용 약관을 읽습니다. 
      - McNeel로 등록 정보를 보냅니다를 클릭합니다. - 선택 사항
      - "다음” 을 클릭합니다.
{{:rhino:install:wizard:start:windows:5:commercial:v5_installer.png?400|}}

**Rhino 5 가 설치되고 2개의 바탕화면 바로 가기가 생깁니다.**
  * **Rhinoceros 5** - 32 비트 Rhino 5 
    * Rhino 4용으로 개발된 32 비트 플러그인이 지원됩니다.
    * 4 GM RAM으로 제한됩니다.

  * **Rhinoceros 5 (64-bit)**
    * 64 비트 Windows 시스템의 모든 RAM을 활용합니다.

===== Rhino 설치에 실패하셨습니까? =====
  * Rhino 설치 관리자가 실패할 경우, 오류에 대한 설명과 해결 방법이 안내되어 있는 대화상자가 표시됩니다. 
  * [[https://wiki.mcneel.com/ko/rhino/installproblems|MSI 오류 코드]]가 대화상자에 표시되었나요?
  * 필요하다면, 오류 로그를 열어 파일에 저장하고 McNeel 기술 지원 <tech@mcneel.com> 에게 이메일로 보내주시기 바랍니다.
