====== Zoo Requires Microsoft .NET Framework 4.5 ======

> **Product:** //[[zoo:home|Zoo]]//
> **Summary:** //Discusses Zoo and the Microsoft .NET Framework 4.5//


The Zoo requires **[[https://www.microsoft.com/en-us/download/details.aspx?id=30653|Microsoft .NET Framework 4.5]]**. You can ensure that Microsoft .NET Framework 4.5 is installed on your system using [[http://windows.microsoft.com/en-us/windows/windows-update|Windows Update]].
