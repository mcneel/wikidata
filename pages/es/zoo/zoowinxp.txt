====== Zoo LAN no funciona en Windows XP ======
> **Producto:** //[[es:zoo:home|Zoo]]//
> **Resumen:** //Describe cómo se ejecuta el Zoo LAN en Windows XP//

===== Pregunta =====
Estamos intentando que el Zoo LAN funcione en Windows XP. Cuando iniciamos el Zoo LAN en esa máquina, el Zoo LAN se bloquea y no ocurre nada. ¿Windows XP no es compatible con Zoo LAN?

===== Respuesta =====
  - Zoo LAN requiere Microsoft .NET Framework 4.8.
  - .NET Framework 4.8 y las versiones posteriores __no__ son compatibles con Windows XP.
