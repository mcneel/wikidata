======E57 File Import======
> **Summary:** //Rhinoceros 5 Labs plug-in to import .E57 files.//
> **Updated:** //September 12, 2013//

[[http://wiki.mcneel.com/labs/e57import|{{:labs:e57128.jpg|E57 File Import}}]]

=====Description=====
This Labs plug-in will allow you to open or import .E57 files. The E57 file format is a compact, vendor-neutral format for storing point clouds, images, and metadata produced by 3D imaging systems, such as laser scanners. The file format is specified by the ASTM, an international standards organization, and it is documented in the ASTM E2807 standard. 

=====More Information=====
[[http://www.astm.org/Standards/E2807.htm|ASTM International]]\\
[[http://www.libe57.org/data.html|E57 Data Examples]]
=====Installation=====
  - Download the plug-in. File is in Rhino Installer file (.RHI) form.
  - Double-click in the file to launch the Rhino Installer.

=====Usage=====
  - From Rhino's **File** menu, select either the **Open** or **Import** command.
  - In the **Files of Type** drop down box, select the **E57 Files** file type.
  - Select your .E57 file.
  - Once opened or imported, you may need to **Zoom** to see the geometry.
