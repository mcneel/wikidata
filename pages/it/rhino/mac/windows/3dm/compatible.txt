====== I file 3DM di Rhino per Mac sono compatibili con Rhino per Windows? ======
> **Riepilogo:** //Esiste compatibilità tra i file 3DM della versione di Rhino per Mac e la versione di Rhino per Windows?//

Sì, i file di Rhino 5 sono identici. È inoltre possibile esportare i file di Rhino V4 per la compatibilità con Rhino V4 per Windows.
