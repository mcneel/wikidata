====== La WIP tiene un tiempo límite. ¿Qué pasa cuando caduca? (Rhino 5 para Mac) ======
> **Resumen:** //La WIP tiene un límite de tiempo. ¿Qué pasa cuando caduca?//

La fecha de caducidad sirve para asegurarnos de que todo el mundo está probando la última versión y que no se informe de errores que ya se han solucionado.  Antes de que caduque cada RhinoWIP, habrá una nueva WIP disponible para [[http://www.rhino3d.com/go/download/rhino-for-mac/wip/latest|descargar]]. RhinoWIP comprueba las actualizaciones de  cada vez que se inicia y le notifica si hay una nueva versión disponible. También se enviará un mensaje en la [[http://discourse.mcneel.com/category/rhino-for-mac|sección de Rhino para Mac]] del [[http://discourse.mcneel.com/ |foro de Rhino]] cada vez que haya una nueva RhinoWIP disponible.

Véase también: [[rhino:mac:wip|Rhino para Mac Work-In-Progress]].
