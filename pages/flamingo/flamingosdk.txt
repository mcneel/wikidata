====== Flamingo SDK Wiki ======

=====Overview=====
Flamingo SDK is a scripting tool based on Microsoft's VBScript language. With RhinoScript, you can quickly add functionality to Rhino, or automate repetitive tasks.

** .NET SDK Help **
[[http://s3.amazonaws.com/files.na.mcneel.com/flamingo/nxt/docs/en/FlamingoSDK.chm|Download the FlamingoSDK.chm help file]]
  * Note: if the Help file fails to open correctly:
    - Right click on the CHM file and choose properties
    - Click on the "Unblock" button at the bottom of the properties dialog to allow access to the help file.
