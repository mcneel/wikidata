====== Écriture de scripts en Python dans Rhino pour Mac ======
Rhino pour Mac prend en charge un module de test qui ajoute des scripts de Python dans Rhino. Afin d'ajouter cette prise en charge, vous devez :

  - [[http://www.rhino3d.com/download/wenatchee/5.0/wip|Installer la dernière version de Rhino pour OSX]]
  - Lancer cette nouvelle version de Rhino. \\ L'extension .macrhi sera associée à l'application Rhino. Cette opération n'est nécessaire qu'une seule fois. Rhino démarrera maintenant si vous double cliquez sur un fichier .macrhi.
  - [[http://download.rhino3d.com/IronPython/5.0/wip/download/|Télécharger le programme d'installation de module .macrhi IronPython]]. \\ Il s'agit d'un programme d'installation de module de Rhino pour Python. 
  - Double cliquer sur le fichier pour installler le module dans Rhino. \\ Rhino pour Mac démarrera et installera le module.
  - Fermer et redémarrer Rhino. \\ Vous devriez maintenant voir une nouvelle commande **RunPythonScript**.

====Commande _RunPythonScript====
Cette commande est disponible en deux versions :

**_RunPythonScript** - Affiche une boîte de dialogue Ouvrir un fichier qui vous permet de sélectionner un script de python (fichier .py.) Le module comprend un ensemble de scripts d'exemple que vous pouvez lancer et la boîte de dialogue devrait ouvrir ce dossier la première fois que vous la lancez. 

**-_RunPythonScript** - La version précédée d'un tiret permet de taper le nom du fichier de script. Si le script existe dans le répertoire de votre ordinateur, il sera lancé. Vous pouvez utiliser le point pour lancer un script dans un sous-répertoire du dossier Scripts. Par exemple, si vous voulez lancer le fichier CurveLength.py qui se trouve dans le sous-répertoire Exemples, vous pouvez taper exemples.curvelength dans l'invite. 

**Remarque** : Tous les exemples inclus dans ce module devraient fonctionner sur [[http://mac.rhino3d.com|Mac Rhino WIP]] et sur [[http://www.rhino3d.com/|Windows Rhino 5]].
