==== Errore di installazione 1601 - Rhino 6  ====

Errore del servizio di installazione (-2147023295)

> **Sommario:** //Come risolvere l'errore di installazione 1601//

=====Problema=====
Gli errori 1601 possono essere causati da una gran varietà di problemi. Segui tutti i passi in basso.

=====Risposta=====
**Passo 1: Esegui gli aggiornamenti di Windows**
  - Individua ed esegui il [[http://windowsupdate.microsoft.com/|servizio Windows Update]].
  - Se esistono aggiornamenti importanti da installare, fai clic sul pulsante di aggionamento per installarli.  Questo processo può richiedere molto tempo.
  - Se riscontri un errore quando cerchi di eseguire il programma di installazione di Windows Update, consulta l'articolo sulla [[http://support.microsoft.com/kb/906602|risoluzione dei problemi comuni di installazione di Windows Update]].
  - Se riesci a risolvere il problema di Windows Update, completa l'esecuzione del [[http://windowsupdate.microsoft.com/|servizio Windows Update]].
\\
**Passo 2: Installa .NET 4.6 Framework**
  - Scarica ed esegui il [[https://www.microsoft.com/download/details.aspx?id=49981| Programma di installazione .NET 4.6 Framework]].
  - Se il programma di installazione di Microsoft .NET mostra un errore: 
    - Prendi nota dell'errore. 
    - Vai alla sezione **Se il problema persiste**, in basso.

\\
**Passo 3: Installa Rhino**
  - Apri **Windows Explorer**.
  - Scarica il programma di installazione più recente di Rhino 6 qui: [[http://www.rhino3d.com/download/rhino/6/latest|Download completo di Rhino 6]].
  - Esegui file scaricato per avviare il processo di installazione.
  - Se riscontri problemi di installazione, salva una copia del file di registro sul desktop.
\\
**Se il problema persiste**

È frustrante e ne siamo spiacenti. Invia per e-mail il file di registro dal passo 3 a tech.eu@mcneel.com.
