====== Rhino V6 Single EDU License ======


Choose one:
    * **[[rhino:install:wizard:start:windows:6:commercial:stand_alone|Single stand-alone computer]]**
      * Use this option when only one computer will be used
      * Does not require Internet access
      * Locks license exclusively to one computer
    * **[[https://wiki.mcneel.com/rhino/6/cloud_zoo/setup|Multiple computers, single user - Cloud Zoo Personal]]**
      * Use this option when you want to share you license between your personal computers
      * Requires occasional Internet access
      * Single EDU licenses do not support Cloud Zoo Teams
