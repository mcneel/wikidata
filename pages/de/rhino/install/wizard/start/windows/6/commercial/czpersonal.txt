====== Rhino 6 Cloud Zoo - Persönlich ======

>Hinweis: Verwenden Sie diese Option um Ihre Einzellizenz auf allen Ihren Computern zu teilen. Ihre Lizenz befindet sich auf unseren Servern, so dass Sie von jedem beliebigen Computer aus darauf zugreifen können. Zu Beginn wird eine Internetverbindung benötigt. Danach können Sie Ihre Lizenz aufrechterhalten, indem Sie Rhino etwa ein Mal pro Woche mit Internetverbindung ausführen. Sie können Rhino für ca. 2 Wochen ohne Internetverbindung ausführen, ehe die Lizenz aktualisiert werden muss.

===Konfiguration durch erstmalige Rhino-Anwendung===
  - Installieren Sie Rhino
  - Starten Sie Rhino
    - Befolgen Sie die Vorgaben

===Konfiguration durch Aufruf der Website===
  - **Loggen Sie sich in Ihr Rhino-Konto ein**
    - Besuchen Sie https://accounts.rhino3d.com/?controller=home
      - Erstellen Sie ein Rhino-Konto falls Sie noch keines haben
      - Speichern Sie Ihre Anmeldeinformationen. Sie werden sie später benötigen
  - **Fügen Sie Ihre Lizenz Ihren persönlichen Lizenzen hinzu**
    - Auf Ihrer Seite "Mein Konto" klicken Sie auf Lizenzen
    - Unter Persönliche Lizenzen klicken Sie auf [Ihr Name] (ich)
    - Auf der Seite "Lizenzen [Ihr Name] (ich)" klicken Sie auf Lizenz hinzufügen
    - Wählen Sie das zutreffende Produkt
    - Geben Sie Ihren Lizenzschlüssel ein
    - Klicken Sie auf Lizenz hinzufügen
  - **Starten Sie Rhino**
    - Klicken Sie auf Einloggen
    - Geben Sie Ihre Anmeldeinformationen für das Konto ein
    - Rhino wird gestartet

===Andere Computer===
  - **Installieren Sie Rhino**
  - **Starten Sie Rhino**
    - Klicken Sie auf Einloggen
    - Geben Sie Ihre Anmeldeinformationen für das Konto ein
    - Rhino wird gestartet
