====== Venite da Rhino per Windows? ======

Venite da Rhino per Windows?  State pensando di passare a OS X?  O siete semplicemente curiosi?  Questa breve guida vi aiuterà ad orientarvi.


----

===== Dove si trovano le barre degli strumenti? =====
Di default, Rhino per Mac suppone che non veniate da Rhino per Windows.  Per visualizzare le barre degli strumenti in stile Windows, dal menu **Rhinoceros** > **Preferenze** > **Temi** selezionate **Rhino per Windows**.  Aprite quindi una nuova finestra di modellazione affinché la modifica abbia effetto.  Per maggiori informazioni sul tema Rhino per Mac, si veda qui sotto.

----

===== Barre strumenti e tavolozze degli strumenti ===== 
  * Rhino per Mac presenta dei //pulsanti strumento// che richiamano i comandi di Rhino nella barra strumenti standard di Macintosh.
  * In Rhino per Mac ci sono anche le //tavolozze degli strumenti//, finestre che contengono al loro interno un insieme di pulsanti strumento. \\  Quando si avvia Rhino per Mac, sulla sinistra della finestra del documento, appare una tavolozza degli strumenti con i pulsanti strumento usati più di frequente.
  * I pulsanti delle tavolozze degli strumenti possono contenere uno o due comandi ed anche un menu.
  * Passate sui pulsanti della tavolozza degli strumenti con il cursore per visualizzarne la relativa descrizione.\\  Se la descrizione ha due righe di testo, il pulsante contiene un comando primario ed un comando secondario.
    * **Fate clic** sul pulsante della tavolozza degli strumenti per richiamare il comando **primario**.
    * Per richiamare il comando **secondario**, mantenete premuto il tasto **Opzione** e **fate clic** sul pulsante strumento.
  * I pulsanti delle tavolozze degli strumenti contenenti un menu presentano un triangoletto in corrispondenza del vertice sinistro inferiore.
    * Per aprire il menu, fate clic e mantenete premuto brevemente il pulsante della tavolozza degli strumenti.
  * I menu delle tavolozze degli strumenti possono anche essere visualizzati come tavolozze degli strumenti mobili.
    * Per visualizzare una tavolozza degli strumenti mobile, mantenendo premuto il tasto **Opzione**, fate clic e mantenete premuto il tasto del mouse.

===== Mouse ===== 

**Consigli**

  * Per una più facile navigazione con il tasto destro del mouse, si raccomanda un mouse a due pulsanti con rotellina di scorrimento; tutte le operazioni sono comunque possibili anche usando un mouse a pulsante singolo.
  * Per riprodurre con un mouse a pulsante singolo le azioni eseguite con il tasto destro del mouse, mantenere premuto il tasto **Ctrl** e fare clic sul pulsante del mouse.

**Navigazione**

  * Le operazioni **Tasto destro del mouse e trascinamento** e **Maiusc + tasto destro del mouse e trascinamento** sono come in Rhino per Windows. Vedi [[http://docs.mcneel.com/rhino/mac/help/it-it/user_interface/shortcuts.htm|Guida in linea di Rhino - Scorciatoie]]. \\ A prescindere dallo stile di proiezione della vista, mantenere premuto il tasto **Maiusc** per eseguire sempre una panoramica, mantenere premuto il tasto **Comando** per eseguire sempre uno zoom e mantenere premuti i tasti **Comando+Maiusc** per eseguire sempre una rotazione della vista.
  * Usare la rotellina di scorrimento per eseguire degli zoom in/out in una vista.
  * Se avete un mouse a pulsante singolo, occorrerà mantenere premuti i tasti **Ctrl** + **Comando** + **Maiusc** per eseguire sempre una rotazione della vista.

----

===== Menu di scelta rapida (pulsante destro) ===== 

  * Cliccando con il pulsante destro su una vista si visualizza un menu contestuale.
  * Di default, il tasto destro **non** funge da **Invio** per ripetere l'ultimo comando. \\ Per cambiarlo, in **Rhinoceros** > **Preferenze** > **Mouse** > **Mouse**, disattivare l'opzione **Abilita menu contestuali**.

----

===== Linea di comando ===== 

Se siete abituati alla linea di comando di Rhino per Windows, troverete questa funzionalità anche in Rhino per Mac.  Per lanciare un comando, basta che iniziate a digitare. Il testo apparirà nel campo di inserimento dei comandi che si trova nella barra laterale sinistra.  Per eseguire il comando, potete premere Invio oppure la barra spaziatrice.  Apparirà una finestra di dialogo contenente tutte le opzioni di comando che appaiono nel prompt dei comandi di Rhino per Windows. \\ \\

Se desiderate digitare anche i valori delle opzioni di comando, vedrete che ciascuna opzione appare con una lettera sottolineata. Per richiamare un'opzione, potete digitarne la rispettiva lettera e premere Invio. 

Quando digitate i valori nella finestra di dialogo delle opzioni di comando, non occorre che spostiate il cursore.  Basta che li digitiate e Rhino li inserirà automaticamente nella finestra.  Tutte le vostre abitudini d'uso della linea di comando di Rhino per Windows funzionano in modo identico in Rhino per Mac.

----

===== Tastiera =====

  * Rhino per Windows usa il tasto **Ctrl** per molte operazioni. Nella versione per Mac, si usi il tasto **Comando** al suo posto.
    * Per esempio, **Comando** + clic sostituisce **Ctrl** + clic per deselezionare gli oggetti.
    * Il tasto **Comando** sostituisce il tasto **Ctrl** anche per le scorciatoie di navigazione:

| | | |**Tasto** | **Azione** | **+ tasto Comando** |
| | | |**Freccia sinistra** | Ruota a sinistra | Panoramica a sinistra |
| | | |**Freccia destra** | Ruota a destra | Panoramica a destra |
| | | |**Freccia su** | Ruota verso l'alto | Panoramica su |
| | | |**Freccia giù** | Ruota verso il basso | Panoramica giù |

----

===== Portatili ===== 


**Mouse Bluetooth**\\ Visto che i portatili si vendono con un trackpad, può risultare utile un mouse Bluetooth, come per esempio il mouse Logitech V270.

**Tasti mancanti**\\ La tastiera di un portatile non ha tasti separati per **Home**, **Fine**, **PaginaGiù** o **PaginaSu**; la funzione di questi tasti viene svolta dal tasto **fn** in combinazione con altri tasti. \\

| **Tasto** | **+ tasto fn** |
| **Freccia su** | Pagina su |
| **Freccia giù** | Pagina giù |
| **Freccia sinistra** | Home |
| **Freccia destra** | Fine |
| **Cancella** | Cancella |
