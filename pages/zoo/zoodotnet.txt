====== Zoo Requires Microsoft .NET Framework 4.8 ======
> **Product:** //[[zoo:home|Zoo]]//
> **Summary:** //Discusses Zoo and the Microsoft .NET Framework 4.8//


The Zoo requires [[https://dotnet.microsoft.com/download/dotnet-framework/net48|Microsoft .NET Framework 4.8]]. 

You can ensure that Microsoft .NET Framework 4.8 is installed on your system by using **[[http://windows.microsoft.com/en-us/windows/windows-update|Windows Update]]**.

To detect the versions of the  Microsoft .NET Framework installed on your system, use:

**[[http://mcneel.s3.amazonaws.com/zoo/Utils/DetectFX.exe|.NET Framework Detection Utility (Win32)]]**

**[[http://mcneel.s3.amazonaws.com/zoo/Utils/DetectFXx64.exe|.NET Framework Detection Utility (x64)]]**
