===== 일반적인 비디오 요구 사항 =====
Rhino 에는 고정된 하드웨어 요구 사항이 있습니다만 (일부 하드웨어 구성의 경우 성능이 더 좋거나 더 나쁠 수 있습니다), 현재 시판 중인 거의 대부분의 컴퓨터 시스템에서도 충분히 Rhino가 실행될 수 있을 만큼 Rhino는 유연하다고 믿고 있습니다. 낮은 사양의 컴퓨터나 **Intel(모든 모델) 등의 저가의 비디오 카드**에서 Rhino의 일부 최신 디스플레이 기능이 올바르게 실행되지 않거나, 성능이 저하될 수 있으나 일반적으로 작업을 완료하기에 크게 문제가 되지 않습니다. 안타깝게도 13인치 Apple MacBook 컴퓨터(13인치 "Retina" 포함)에는 문제 발생의 소지가 있는 Intel HD 그래픽 칩이 탑재되어 있으므로 사용하지 않으시는 것이 좋습니다.\\

Rhino는 Windows 범위에서 OpenGL 그래픽 시스템을 많이 사용합니다. 시중의 대부분의 그래픽 카드는 DirectX 를 사용하며, ¡°게임¡± 사용을 위해 개발되었습니다. 이들 중 대다수는 OpenGL 지원이 완벽하지 않습니다. Rhino에 가장 좋은 비디오 카드는 ¡°워크스테이션 그래픽¡±을 위해 개발되고 OpenGL 지원에 중점을 둔 제품들입니다. 그러나 좋은 비디오 카드에 나쁜 드라이버를 사용하는 것이 문제입니다. **비디오 카드의 드라이버를 항상 최신 드라이버로 업데이트하시기 바랍니다!**\\

<color red> **경고:**</color> 모든 드라이버가 설치되기 전에 Windows 레지스트리 복원 지점을 만드는 것은 아닙니다. 새 드라이버로 인해 Windows 설정이 손상되는 경우를 생각하여, 중요 Windows 시스템 구성요소를 업데이트하기 전에 **시스템 복원 지점**을 설정하는 것은 **일종의 보험**이라고 할 수 있습니다. 

  * Rhino 4에는 OpenGL 1.1 버전이 필요합니다.
  * Rhino 5에는 OpenGL 2.0 버전과 Shader 1.2 버전이 필요합니다.

===== 성능 벤치마크 =====
드라이버 개발자들이 더 좋은 결과를 얻기 위해 코드를 정밀하게 튜닝하는 데에만 몇 달이 걸리기도 합니다. 주어진 벤치마크가 뭔가 확실한 것을 증명하는 것은 아니지만 특정한 벤치마크의 성능이 특정한 GPU에서 더 좋아지거나 나빠지는 것은 증명됩니다.

벤치마크는 GPU의 실제 사용이나 성능을 보여주지는 않습니다. 
즉, 비디오 카드/GPU의 어느 한 지점에서 게임이 더 잘 실행된다고 해도 모든 게임이 해당 카드에서 더 잘 실행되는 것을 의미하지 않습니다. 

더 잘 실행되었다는 이유만으로 벤치마크가 언급되었을 가능성이 더 높습니다. 
비디오 카드가 실행되지 않거나 성능이 확연히 저하되는 벤치마크는 어떻습니까?

벤치마크는 마켓팅 방법이며, 경쟁 제품보다 훨씬 더 좋은 위치를 선점하기 위해 사용되므로 실제 사용 또는 성능을 구분하기 좋은 기준은 아닙니다. 

**이와 관련된 정보를 찾으시려면 사용자 포럼에서 실제 사용자의 의견과 소감을 참조하시기 바랍니다.** [[http://v5.rhino3d.com/forum/categories/support-1/listForCategory|Rhino 사용자 포럼]]에 문의하시는 것도 한 가지 방법입니다.
 
===== Intel 그래픽 카드와 Rhino 5 =====
**이 카드들은 권장 제품이 아니며, Rhino에서 사용하는 것을 피하셔야 합니다.**

이 제품을 구입하셨고 컴퓨터에 탑재된 유일한 비디오 카드라면(예: 하이브리드가 아닌 경우) 가속 하드웨어 모드를 사용하시고 테스트해보시길 권장합니다. Rhino 5가 설치될 때 Rhino 5에서 자동으로 이 설정이 해제되어 있을 수도 있습니다. 

{{:rhino:rhino_opengl_options.png?400|}}

//Rhino에서 도구 > 옵션 > OpenGL //

테스트하신 후에, 설정(소프트웨어 에뮬레이션 모드)이 해제되었을 때보다 Rhino 성능이 더 좋아졌다면 설정이 된 상태를 유지하세요. 
음영, 렌더링 표시, 분석 모드와 같은 많은 Rhino 5 기능이 OpenGL을 사용합니다. 
새 기능의 성능이 좀 떨어질 수 있으나, 아예 실행되지 않는 것보다는 바람직하다고 할 수 있습니다.

===== nVidia Quadro 카드 =====

이 제품은 고급 엔지니어링 레벨 비디오 카드입니다. 꾸준히 양호한 드라이버를 제공하고 성능 또한 우수합니다. 이제까지 오랫동안 권장해온 카드가 있다면 이 제품이라고 할 수 있습니다.\\
http://wiki.mcneel.com/rhino/videocards 에서 추가 **중요** 정보를 참조하세요.
===== Nvidia GeForce =====

Nvidia Geforce 제품군 (GT 200, 300, GTX 400, GTX 500)은 Rhino에 적합합니다.

GT 200과 300은 이전 칩셋을 기반으로 하며 속도는 빠릅니다. 새로운 NVidia FERMI 기반 GeForce 카드 (예: GeForce FX 400, 500 시리즈)는 사용하실 수 있으나, 더 고가의 카드에서 기대할 수 있는 성능의 향상에는 못미칩니다. 실제로 일부 경우, 200 또는 300 시리즈 카드보다 속도가 느리기도 합니다.


===== ATI Video =====
최신 AMD/ATI 4/5/6k 시리즈 Radeon HD 카드는 5 버전에 잘 맞습니다. 오래된 초기 버전은 DirectX 전용이므로 가급적 피하시기 바랍니다. 카드와 함께 번들된 드라이버가 오래된 버전인 경우를 종종 있습니다. ATI/AMD 웹사이트에서 최신 드라이버로 업데이트하시기 바랍니다. FirePro v7900 과 v5900 카드는 정말 좋습니다.
http://wiki.mcneel.com/rhino/videocards

===== 새 드라이버 =====
nVidia Quadro 3800 제품을 사용하신다면 최신 드라이버로 반드시 업데이트하시기 바랍니다.

업데이트된 nVidia 드라이버는 [[http://www.nvidia.com/Download/index.aspx?lang=en-us
|여기를 클릭하세요.]]

업데이트된 ATI 드라이버는 [[http://support.amd.com/us/Pages/AMDSupportHub.aspx
|여기를 클릭하세요.]] 

Windows 디스플레이 속성에서 곧바로 자동 업데이트를 실행하지 마십시오. 제조업제 웹사이트에 직접 가셔서 업데이트된 드라이버를 다운로드하시기 바랍니다. 다운로드를 완료하시면 대부분의 경우 EXE 파일 실행 과정으로 진행됩니다. 

바탕화면에 다운로드하시면 쉽게 찾아 설치하실 수 있습니다. 설치를 마친 후에는 실행 파일을 삭제하십시오.

<color red> **경고:**</color> 모든 드라이버가 설치되기 전에 Windows 레지스트리 복원 지점을 만드는 것은 아닙니다. 새 드라이버로 인해 Windows 설정이 손상되는 경우를 생각하여, 중요 Windows 시스템 구성요소를 업데이트하기 전에 **시스템 복원 지점**을 설정하는 것은 **일종의 보험**이라고 할 수 있습니다.
===== Crossfire =====
Crossfire에서 두 개를 실행하면 분명히 장점이 있을테지만 Rhino에서 크게 차이가 있을지는 확실하지 않습니다. Crossfire는 고해상도 그래픽과 게임에서 높은 프레임 속도가 요구될 때처럼 비디오 카드에서 많은 양의 데이터가 처리될 때 진가를 발휘합니다. 그러나 Rhino는 방식이 다릅니다. 대부분 (95%) Rhino의 디스플레이는 대기/유휴 상태라고 할 수 있습니다.

===== 비디오 메모리 ===== 
가급적 많은 비디오 메모리를 갖춘 카드를 권장합니다. 5 버전에서 작업하실 때 도움이 될 것입니다. 1GB 비디오 카드와 2GB 비디오 카드 중 선택하셔야 한다면 2GB 카드를 선택하십시오.
