====== Dois-je désinstaller Rhino avant d'installer une version plus récente ? (Rhino 5 pour Mac) ======
> **Résumé:** //Dois-je désinstaller Rhino avant d'installer une version plus récente ?//

Non, ce n'est pas la peine. Vous pouvez remplacer l'application, qui se trouve normalement dans le dossier Applications, lorsque vous installez une nouvelle version. Cette action ne remplacera pas vos préférences ni votre licence, qui sont enregistrées dans d'autres dossiers. 

D'ailleurs, vous pouvez lancer deux versions de Rhino ensemble si vous le souhaitez, en donnant simplement un autre nom au fichier de l'application (Rhinoceros.app > RhinocerosCopie.app par exemple).
