=====RHI 설치관리자 관련 문제 해결=====

**RHI** 파일은 **RH**ino **I**nstaller 패키지입니다.\\
Rhino용 플러그인의 생성 및 설치를 간소화하기 위해 고안되었습니다.

==== 설치 문제 ====
-->Windows 에서 이 파일을 열 수 없습니다#
**원인:** Windows에서 RHI 파일의 파일 연결이 올바르게 설정되어 있지 않습니다.
\\
**해결책:**
  - 제어판에서 프로그램 및 기능을 엽니다.
  - Rhinoceros 5 를 오른쪽 클릭하고 복구를 클릭합니다.
  - Rhinoceros 5 (64-bit)를 오른쪽 클릭하고 복구를 클릭합니다.
  - RHI 설치관리자를 두 번 클릭하여 설치합니다.
\\
이 방법으로 해결되지 않으면, 다음의 **직접 RHI 파일 연결하기** 항목을 실행해 보세요.
<--

-->호환되지 않는 패키지#
이 오류가 발생하는 데는 몇 가지 이유가 있을 수 있습니다. 가장 가능성이 높은 첫 번째 이유:
  - **손상된 다운로드:** 손상된 RHI 파일은 설치하실 수 없습니다. 파일이 불완전하게 다운로드된 경우가 대부분입니다. 해당 RHI 파일을 삭제하시고, 웹브라우저의 임시 파일을 지우신 후 다시 다운로드하시기 바랍니다. 
  - **잘못 작성된 RHI:** 귀하가 개발자이신 경우, RHI 파일이 올바르게 작성되지 않았을 수도 있습니다. [[developer/rhinoinstallerengine/authoring|RHI 파일 작성 안내]]를 참조하시기 바랍니다.
\\
<--

-->Rhinoceros 5 파일 열기 오류#
**원인:** 어떤 이유에서건, Rhino 5가 RHI 파일로 잘못 연결되어 있습니다.
\\
**해결책:** 다음의 **직접 RHI 파일 연결하기** 항목을 실행해 보세요.
<--

-->직접 RHI 파일 연결하기#
-->64 비트 Windows#
  - Rhino Installer Package (RHI) 파일을 오른쪽 클릭하고 **연결 프로그램**을 클릭합니다.
  - 연결 프로그램 대화상자에서 **이 종류의 파일을 열 때 항상 선택된 프로그램 사용** 확인란을 선택합니다.
  - 연결 프로그램 대화상자에서 **찾아보기...**를 클릭합니다.
  - **C:\Program Files\Rhinoceros 5 (64-bit)\System\x64** 을 찾아  **rhiexec.exe** 파일을 선택합니다. (다른 위치에 Rhino를 설치했다면 해당 위치를 찾으시기 바랍니다.)
  - **열기**를 클릭합니다.
  - **확인**을 클릭합니다.

<--

-->32 비트 Windows#
  - Rhino Installer Package (RHI) 파일을 오른쪽 클릭하고 **연결 프로그램**을 클릭합니다.
  - 연결 프로그램 대화상자에서 **이 종류의 파일을 열 때 항상 선택된 프로그램 사용** 확인란을 선택합니다.
  - 연결 프로그램 대화상자에서 **찾아보기...**를 클릭합니다.
  - **C:\Program Files\Rhinoceros 5\System\x86** 위치를 찾아 **rhiexec.exe** 파일을 선택합니다. (다른 위치에 Rhino를 설치했다면 해당 위치를 찾으시기 바랍니다.)
  - **열기**를 클릭합니다.
  - **확인**을 클릭합니다.
